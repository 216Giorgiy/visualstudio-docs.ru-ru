# <a name="assigning-licenses-in-the-visual-studio-subscriptions-administrator-portal"></a>Назначение лицензий на портале администратора подписок Visual Studio

## <a name="assigning-a-single-user"></a>Назначение одному пользователю
При наличии доступных лицензий для подписок Visual Studio вы можете назначить их новым пользователям, чтобы предоставить им доступ к преимуществам подписки. 
1.  Чтобы назначить одного подписчика Visual Studio, в верхней части таблицы нажмите кнопку **Добавить**.

    ![Добавление подписчика](_img\assign-license-add\assign-license-add.png)

2.  Введите сведения в поля формы для нового подписчика. Если ваша организация использует Azure Active Directory, это поле действует как функция поиска людей в текущем каталоге, позволяя выбрать подходящего пользователя в результатах поиска. Когда вы выберете нужного человека, его имя, адрес электронной почты для входа и адрес для уведомлений будут заполнены автоматически, как показано ниже. 

Если ваша организация использует разные адреса электронной почты для приема сообщений и входа в систему, здесь вы сможете ввести нужный адрес. Выберите гиперссылку "Адрес электронной почты для сообщений отличается от электронного адреса для входа?". 

Если требуется предоставить подписчику доступ к скачиванию программного обеспечения при входе на [портал управления подписками Visual Studio](https:/my.visualstudio.com?wt.mc_id=o~msft~docs), не снимайте флажок "Скачивания". Если снять этот флажок, пользователь не будет иметь доступ к скачиваемым файлам, однако сохранит доступ ко всем другим преимуществам подписки. По завершении нажмите кнопку **Добавить**.

   ![Ввод сведений о подписчике](_img\assign-license-add\add-subscriber-1.png)

   ![Ввод сведений о подписчике](_img\assign-license-add\add-subscriber-2.png)

3.  После добавления подписчика ему автоматически отправляется сообщение электронной почты о назначении, содержащее дальнейшие инструкции. Сообщение о назначении можно переотправить в любое время, выбрав подписчика и нажав кнопку **Отправить заново** в верхнем меню.

    ![Подписчик добавлен](_img\assign-license-add\add-subscriber-complete.png)

## <a name="bulk-assignments"></a>Массовые назначения
1.  Чтобы добавить сразу несколько подписчиков, перейдите на вкладку "Подписчики". На ленте в верхней части нажмите кнопку **Массовое добавление**. 

    ![Массовое добавление](_img\assign-license-add\bulk-assign-add.png)

2. Массовое назначение использует шаблон Microsoft Excel для отправки подписчиков. В диалоговом окне "Upload Multiple Subscribers" (Отправка нескольких подписчиков) нажмите кнопку **Скачать**, чтобы скачать этот шаблон. Всегда скачивайте последнюю версию шаблона. При использовании более старой версии массовая отправка может завершиться ошибкой.

    ![Отправка нескольких подписчиков](_img\assign-license-add\bulk-assign-upload.png)

3.  В таблице Excel заполните поля сведениями о людях, которым вы хотите назначить подписки. Поле "Ссылка" является необязательным. Если вы неправильно заполнили какую-либо часть шаблона, отображается сообщение об ошибке с описанием проблемы. По завершении сохраните файл на жестком диске.
**Чтобы обеспечить правильную отправку, соблюдайте следующие рекомендации:**
    - Проследите за тем, чтобы ни одно из полей формы не содержало запятые.
    - Удалите пробелы до и после полей формы, например имен пользователей.
    - Убедитесь, что имена и фамилии пользователей, состоящие из двух частей, не содержат лишние пробелы (например, имя "Maggie May" не следует вводить как "Maggie  May", так как система не удалит лишний пробел).

   ![Шаблон массового добавления](_img\assign-license-add\bulk-template.png)

4.  Вернитесь на портал администрирования подписок Visual Studio и нажмите кнопку **Обзор** в диалоговом окне "Отправка нескольких подписчиков". Перейдите к сохраненному файлу Excel и нажмите кнопку **ОК**. На экране отображается индикатор хода отправки. 

    ![Отправка при массовом добавлении](_img\assign-license-add\bulk-assign-upload-2.png)

Если шаблон содержит ошибки, отправка завершается сбоем и отображаются соответствующие ошибки, чтобы можно было исправить шаблон и повторить попытку.

   ![Сбой отправки](_img\assign-license-add\bulk-assign-upload-fail.png)

Если отправка выполнена успешно, отображается список подписчиков и сообщение с подтверждением.

   ![Отправка завершена](_img\assign-license-add\bulk-assign-upload-complete.png)
