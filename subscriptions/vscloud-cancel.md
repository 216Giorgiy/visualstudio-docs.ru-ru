# <a name="buying-visual-studio-cloud-subscriptions---cancelling-subscriptions"></a>Приобретение облачных подписок на Visual Studio — отмена подписок

Чтобы отменить облачные подписки на Visual Studio, войдите на [портал администрирования подписок](https://manage.visualstudio.com/) Visual Studio с учетной записью, имеющей доступ к подписке Azure, которая используется для первоначальной покупки.
1.  Убедитесь, что вы выбрали нужную подписку Azure (указана в верхнем левом углу).
2.  На панели навигации слева выберите **Сократить** в подписке, которую вы хотите отменить.
3.  Появится окно настройки числа подписок, связанных с вашей учетной записью.
4.  Укажите нужное число подписок (или выберите 0) и щелкните **Продолжить**.
5.  Просмотрите свой заказ и нажмите кнопку **Подтвердить**.
## <a name="faq"></a>Вопросы и ответы
### <a name="q--can-i-still-use-a-subscription-after-i-cancel"></a>Вопрос. Можно ли использовать подписку после отмены?
Ответ. Да, вы сохраняете доступ ко всем преимуществам на оставшийся период подписки.
