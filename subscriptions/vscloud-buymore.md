# <a name="buying-visual-studio-cloud-subscriptions---buying-additional-subscriptions"></a>Приобретение облачных подписок на Visual Studio — приобретение дополнительных подписок

Чтобы приобрести дополнительные облачные подписки на Visual Studio, войдите на [портал администрирования подписок](https://manage.visualstudio.com/) Visual Studio с учетной записью, имеющей доступ к подписке Azure, используемой для первоначальной покупки.

1.  Убедитесь, что вы выбрали нужную подписку Azure (указана в верхнем левом углу).
2.  На панели навигации слева выберите **Купить еще** в подписке, которую вы хотите приобрести.
3.  Укажите нужное число подписок и щелкните **Продолжить**.
4.  Просмотрите заказ, а затем нажмите кнопку **Подтвердить**.
5.  Когда вы вернетесь на портал администрирования подписок, вы увидите, что число доступных подписок увеличилось.

Дополнительные сведения см. в [документации](/visualstudio/subscriptions/) по назначению подписок и управлению ими.

## <a name="faq"></a>Вопросы и ответы
### <a name="q--how-long-do-i-have-to-wait-before-i-can-assign-subscriptions"></a>Вопрос. Как долго нужно ждать перед назначением подписок?
Ответ. Ваши подписки сразу же становятся доступными для назначения на портале администрирования подписок.
