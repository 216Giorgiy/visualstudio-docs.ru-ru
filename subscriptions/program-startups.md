# <a name="visual-studio-subscriptions-offered-to-startups"></a>Подписки Visual Studio для начинающих компаний
Начинающие компании, вступившие в программу Microsoft for Startups (ранее BizSpark), могут получить 5 [ежемесячных облачных подписок Visual Studio Enterprise](https://www.visualstudio.com/vs/pricing/). Это те же подписки, которые предлагаются на коммерческой основе, без специальных ограничений. Каждая подписка должна быть назначена конечному пользователю (который является лицензированным пользователем), прежде чем пользователь сможет запустить Visual Studio и VSTS распознает его как подписчика Visual Studio Enterprise.

[Узнайте больше о программе Microsoft for Startups](https://startups.microsoft.com/program-details/)

### <a name="sign-in-required-with-visual-studio"></a>Требуется войти в Visual Studio
Чтобы воспользоваться всеми облачными подписками Visual Studio, лицензированные пользователи должны войти в Visual Studio, указав идентификатор или имя пользователя, использованные для активации подписки Visual Studio. Для разблокировки продукта не предоставляются ключи. Для запуска Visual Studio необходимо выполнить вход. Подписчик Visual Studio может работать автономно в течение 30 дней, затем Visual Studio прекратит работу. При работе в сети Visual Studio время от времени автоматически обновляет состояние лицензии пользователя, чтобы проверить срок действия подписки.
