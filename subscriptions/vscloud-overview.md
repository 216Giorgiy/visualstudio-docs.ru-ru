# <a name="buying-visual-studio-cloud-subscriptions---overview"></a>Приобретение облачных подписок на Visual Studio — обзор

Благодаря нашим облачным подпискам вы можете арендовать Visual Studio, VSTS и необходимые преимущества для подписчиков без долгосрочного контракта. Выставление счетов осуществляется через подписку Microsoft Azure, при этом поддерживается оплата кредитной картой и покупки по контракту Enterprise Agreement*, а в скором времени также будут доступны покупки у партнеров [поставщиков облачных решений](https://www.microsoft.com/solution-providers/search). Если у вас еще нет подписки Azure, не беспокойтесь — ее настройка бесплатна и может быть выполнена при первой покупке.

Сравнение [преимущества облачных подписок и цены на них](https://www.visualstudio.com/vs/pricing/).

## <a name="before-you-start"></a>Перед началом работы
Перед приобретением выполните следующие действия для успешного выполнения транзакций. • Для расчета вам потребуется [подписка Azure](https://azure.microsoft.com/pricing/purchase-options/).  [Зарегистрироваться](https://portal.azure.com/) можно перед совершением первой покупки или в процессе ее совершения в Visual Studio Marketplace.  

• Чтобы оформить подписку Azure и совершать покупки в Visual Studio Marketplace, требуется [личная учетная запись Майкрософт](https://www.microsoft.com/account) либо [рабочая или учебная учетная запись](https://azure.microsoft.com/en-us/documentation/articles/sign-up-organization/), если вы совершаете покупки для организации и она управляет удостоверениями с помощью Azure Active Directory (Azure AD).
