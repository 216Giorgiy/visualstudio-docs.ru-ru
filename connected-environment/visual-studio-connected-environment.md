# <a name="vs-connected-environment-for-azure-container-service-aks-preview"></a>Подключенная среда VS для службы контейнеров Azure (AKS) (предварительная версия)
Подключенная среда VS позволяет осуществлять быструю разработку Kubernetes. Вы создаете среду на базе Kubernetes, которая полностью управляется в Azure и оптимизирована для разработки, а затем последовательно разрабатываете контейнеры в облаке с помощью знакомых вам средств, таких как VS Code, Visual Studio или командная строка.

Такой подход имеет несколько преимуществ:

* Работа в среде разработки, не имеющей инфраструктуры и отражающей рабочую среду, с полным доступом к облачным ресурсам.
* Отладка контейнеров Node.js и .NET Core непосредственно в Kubernetes с Visual Studio или VS Code. Для остальных языков доступен интерфейс командной строки.
* Совместное использование экземпляра Kubernetes в команде разработчиков для снижения расходов и минимизации настройки локальных компьютеров для новых членов команды.
* Разработка кода в изолированном пространстве и сквозное тестирование с другими компонентами без репликации или имитации зависимостей.
