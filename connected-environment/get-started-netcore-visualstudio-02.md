# <a name="get-started-on-connected-environment-with-net-core-and-visual-studio"></a>Начало работы в подключенной среде с .NET Core и Visual Studio

Предыдущий шаг: [установка средств](get-started-netcore-visualstudio-01.md)

## <a name="create-an-aspnet-web-app"></a>Создание веб-приложения ASP.NET
В Visual Studio 2017 создайте новый проект. Сейчас это должно быть **веб-приложение ASP.NET Core**. Присвойте проекту имя **webfrontend**.

![](images/NewProjectDialog1.png)

Выберите шаблон **Веб-приложение (модель-представление-контроллер)** и убедитесь, что выбрали **.NET Core** и **ASP.NET Core 2.0** в двух раскрывающихся списках в верхней части диалогового окна. Нажмите кнопку **ОК**, чтобы создать проект.

![](images/NewProjectDialog2.png)
