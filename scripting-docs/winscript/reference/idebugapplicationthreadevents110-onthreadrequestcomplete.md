# IDebugApplicationThreadEvents110::OnThreadRequestComplete
Вызов в поток с помощью передачи потока PDM завершен.  
  
> [!IMPORTANT]
>  [IDebugApplicationThreadEvents110 — интерфейс](../../winscript/reference/idebugapplicationthreadevents110-interface.md) реализуется PDM v11.0 и большим.  Обнаружен в activdbg100.h.  
  
## Синтаксис  
  
```cpp  
HRESULT OnThreadRequestComplete( void );  
```  
  
#### Параметры  
 Этот метод не имеет параметров.
