# <a name="idebugapplicationthreadevents110onthreadrequestcomplete"></a>IDebugApplicationThreadEvents110::OnThreadRequestComplete
Вызов в поток, используя PDM поток завершения переключения.  
  
> [!IMPORTANT]
>  [Idebugapplicationthreadevents110 — интерфейс](../../winscript/reference/idebugapplicationthreadevents110-interface.md) — реализованный PDM v11.0 и более поздней версии. Обнаружено в activdbg100.h.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT OnThreadRequestComplete( void );  
```  
  
#### <a name="parameters"></a>Параметры  
 Этот метод не имеет параметров.
