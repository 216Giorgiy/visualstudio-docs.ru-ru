# Интерфейс IDebugExpressionCallBack
Предоставляет события, связанные `IDebugExpression` состояний для отслеживания хода выполнения.  
  
## Методы  
 В дополнение к методам, наследуемым от интерфейса `IUnknown`, интерфейс `IDebugExpressionCallBack` предоставляет следующие методы.  
  
|Метод|Описание|  
|-----------|--------------|  
|[IDebugExpressionCallBack::onComplete](../../winscript/reference/idebugexpressioncallback-oncomplete.md)|Указывает, что вычисление выражений завершена.|
