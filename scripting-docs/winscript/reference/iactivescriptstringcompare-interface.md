# Интерфейс IActiveScriptStringCompare
Интерфейс `IActiveScriptCompare` используется для определения метод сравнения строк для обработчика скриптов.  
  
## Методы в порядке таблицы Vtable  
  
|Метод|Описание|  
|-----------|--------------|  
|[IActiveScriptStringCompare::StrComp](../../winscript/reference/iactivescriptstringcompare-strcomp.md)|Определяет метод сравнения строк для обработчика скриптов.|
