# Интерфейс IVariantChangeType
Позволяет вызывающему объекту преобразования ДРУГОГО значения из одного типа в другой.  
  
## Методы  
 В дополнение к методам, наследуемым от интерфейса `IUnknown`, интерфейс `IVariantChangeType` предоставляет следующие методы.  
  
|Метод|Описание|  
|-----------|--------------|  
|[IVariantChangeType::ChangeType](../../winscript/reference/ivariantchangetype-changetype.md)|Имеет другое значение и создается новый вариант с указанным типом.|
