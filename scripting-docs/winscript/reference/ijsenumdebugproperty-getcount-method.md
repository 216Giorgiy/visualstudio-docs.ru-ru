# <a name="ijsenumdebugpropertygetcount-method"></a>Метод IJsEnumDebugProperty::GetCount
Возвращает число свойств объекта.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
HRESULT GetCount(  
   ULONG *pCount  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `pCount`  
 [out] Количество свойств.  
  
## <a name="return-value"></a>Возвращаемое значение  
  
## <a name="requirements"></a>Требования  
 **Заголовок:** jscript9diag.h
