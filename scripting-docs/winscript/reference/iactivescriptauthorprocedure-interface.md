# Интерфейс IActiveScriptAuthorProcedure
В дополнение к методам, наследуемым от интерфейса `IUnknown`, интерфейс `IScriptAuthorProcedure` предоставляет следующий метод.  
  
## Методы в порядке таблицы Vtable  
  
|Метод|Описание|  
|-----------|--------------|  
|[IActiveScriptAuthorProcedure::ParseProcedureText](../../winscript/reference/iactivescriptauthorprocedure-parseproceduretext.md)|Анализирует процедура кода добавляет текст процедуры кода к скрипту создание обработчика и создает объект `IScriptEntry`, который соответствует процедуре кода.|
