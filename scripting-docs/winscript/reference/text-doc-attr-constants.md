# Константы TEXT_DOC_ATTR
Опишите атрибуты документа.  
  
## Синтаксис  
  
```  
typedef DWORD TEXT_DOC_ATTR;  
```  
  
## Константы  
  
|Константа|Значение|Описание|  
|---------------|--------------|--------------|  
|TEXT\_DOC\_ATTR\_READONLY|0x00000001|Документ доступен только для чтения.|  
|TEXT\_DOC\_ATTR\_TYPE\_PRIMARY|0x00000002|Документ первичный файл этой схемы документа.|  
|TEXT\_DOC\_ATTR\_TYPE\_WORKER|0x00000004|Документ работы.|  
|TEXT\_DOC\_ATTR\_TYPE\_SCRIPT|0x00000008|Документ файл сценария.|
