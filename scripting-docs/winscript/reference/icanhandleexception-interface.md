# Интерфейс ICanHandleException
Разрешает вызывающий объект обработчика скрипта, чтобы определить, какие исключения вызывающий объект.  
  
## Методы  
 В дополнение к методам, наследуемым от интерфейса `IUnknown`, интерфейс `ICanHandleException` предоставляет следующие методы.  
  
|Метод|Описание|  
|-----------|--------------|  
|[ICanHandleException::CanHandleException](../../winscript/reference/icanhandleexception-canhandleexception.md)|Определяет, если вызывающий объект обработчика скрипта может обработать определенное исключение.|
