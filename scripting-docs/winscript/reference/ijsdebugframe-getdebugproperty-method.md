# Метод IJsDebugFrame::GetDebugProperty
Возвращает браузер свойств для данного фрейма стеков.  
  
## Синтаксис  
  
```  
HRESULT GetDebugProperty(  
   IJsDebugProperty **ppDebugProperty  
);  
```  
  
#### Параметры  
 `ppDebugProperty`  
 \[out\] Объект, представляющий обозреватель свойств.
