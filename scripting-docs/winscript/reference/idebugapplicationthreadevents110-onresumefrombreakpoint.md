# IDebugApplicationThreadEvents110::OnResumeFromBreakPoint
Поток возобновляет от точки останова и будет активен еще раз.  
  
> [!IMPORTANT]
>  [IDebugApplicationThreadEvents110 — интерфейс](../../winscript/reference/idebugapplicationthreadevents110-interface.md) реализуется PDM v11.0 и большим.  Обнаружен в activdbg100.h.  
  
## Синтаксис  
  
```cpp  
HRESULT OnResumeFromBreakPoint( void );  
  
```  
  
#### Параметры  
 Этот метод не имеет параметров.
