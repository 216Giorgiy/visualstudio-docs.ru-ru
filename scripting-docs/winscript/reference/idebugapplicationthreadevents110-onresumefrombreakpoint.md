# <a name="idebugapplicationthreadevents110onresumefrombreakpoint"></a>IDebugApplicationThreadEvents110::OnResumeFromBreakPoint
Поток возобновление от точки останова и будет активен еще раз.  
  
> [!IMPORTANT]
>  [Idebugapplicationthreadevents110 — интерфейс](../../winscript/reference/idebugapplicationthreadevents110-interface.md) — реализованный PDM v11.0 и более поздней версии. Обнаружено в activdbg100.h.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT OnResumeFromBreakPoint( void );  
```  
  
#### <a name="parameters"></a>Параметры  
 Этот метод не имеет параметров.
