# Метод IJsDebugBreakPoint::Enable
Включает точку останова.  
  
## Синтаксис  
  
```  
HRESULT Enable(void);  
```  
  
## Возвращаемое значение  
  
## Заметки  
 Возвращает E\_UNEXPECTED, если вызван на удаленной точке останова.

 **Заголовок:** jscript9diag.h
