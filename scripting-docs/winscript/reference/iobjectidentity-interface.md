# Интерфейс IObjectIdentity
В дополнение к методам, наследуемым от интерфейса `IUnknown`, интерфейс `IObjectIdentity` предоставляет следующие методы.  
  
## Методы  
  
|Метод|Описание|  
|-----------|--------------|  
|[IObjectIdentity::IsEqualObject](../../winscript/reference/iobjectidentity-isequalobject.md)|Определяет, является ли объект равен текущему объекту.|
