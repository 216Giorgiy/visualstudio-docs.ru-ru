# Интерфейс IApplicationDebuggerUI
Отладчика, реализованный средой разработки \(ide\) \(в дополнение к `IApplicationDebugger`\) для предоставления внешний компонент несколько элементов управления с пользовательским интерфейсом \(пользовательским интерфейсом\) отладчика.  
  
 В дополнение к методам, наследуемым от интерфейса `IUnknown`, интерфейс `IApplicationDebuggerUI` предоставляет следующие методы.  
  
## Методы в порядке таблицы Vtable  
  
|Метод|Описание|  
|-----------|--------------|  
|[IApplicationDebuggerUI::BringDocumentToTop](../../winscript/reference/iapplicationdebuggerui-bringdocumenttotop.md)|Перемещение окно отладки документ, содержащий заданный в верхнюю часть пользовательского интерфейса отладчика.|  
|[IApplicationDebuggerUI::BringDocumentContextToTop](../../winscript/reference/iapplicationdebuggerui-bringdocumentcontexttotop.md)|Перемещение окно, содержащий заданный контекст рисования в верхнюю часть пользовательского интерфейса отладчика и прокручивает окно к контексту.|
