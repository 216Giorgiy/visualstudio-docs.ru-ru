# Метод IJsDebugProperty::GetPropertyInfo
Получает информацию об этом объекте.  
  
## Синтаксис  
  
```  
HRESULT GetPropertyInfo(  
   UINT nRadix,  
   JsDebugPropertyInfo *pPropertyInfo  
);  
```  
  
#### Параметры  
 `nRadix`  
 \[in\] Система счисления для использования.  
  
 `pPropertyInfo`  
 \[out\] Информация об объекте.
