# Интерфейс IDebugDocumentInfo
Предоставляет сведения о документе, который может быть или не быть создан.  
  
 В дополнение к методам, наследуемым от интерфейса `IUnknown`, интерфейс `IDebugDocumentInfo` предоставляет следующие методы.  
  
## Методы в порядке таблицы Vtable  
  
|Метод|Описание|  
|-----------|--------------|  
|[IDebugDocumentInfo::GetName](../../winscript/reference/idebugdocumentinfo-getname.md)|Возвращает указанное имя документа.|  
|[IDebugDocumentInfo::GetDocumentClassId](../../winscript/reference/idebugdocumentinfo-getdocumentclassid.md)|Возвращает идентификатор CLSID, указывающий тип документа.|
