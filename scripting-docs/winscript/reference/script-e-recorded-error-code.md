# <a name="scripterecorded-error-code"></a>Код ошибки SCRIPT_E_RECORDED
Ошибка прошло между обработчик скриптов и узлом. Узел должен передать код ошибки вызывающему объекту.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
