# Код ошибки SCRIPT_E_RECORDED
Ошибка передается между обработчиком скрипта и основным приложением.  Необходимости узла передать код ошибки вызывающему объекту.  
  
## Синтаксис  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
