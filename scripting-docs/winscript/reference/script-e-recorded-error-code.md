# <a name="scripterecorded-error-code"></a>Код ошибки SCRIPT_E_RECORDED
Ошибка прошло между обработчиком сценариев и узлом. Узел должен передать код ошибки в вызывающем объекте.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
