# Интерфейс IActiveScriptTraceInfo
Предоставляет методы для запуска и остановки трассировки.  
  
## Методы  
  
|Метод|Описание|  
|-----------|--------------|  
|[Метод IActiveScriptTraceInfo::StartScriptTracing](../../winscript/reference/iactivescripttraceinfo-startscripttracing-method.md)|Запускает трассировку скрипта.|  
|[Метод IActiveScriptTraceInfo::StopScriptTracing](../../winscript/reference/iactivescripttraceinfo-stopscripttracing-method.md)|Трассировка стопов.|
