# Интерфейс IDebugDocument
Базовый интерфейс для всех документов отладки.  
  
## Заметки  
 Интерфейс `IDebugDocument` предоставляет только методы, унаследованные от [Интерфейс IDebugDocumentInfo](../../winscript/reference/idebugdocumentinfo-interface.md).  Он не предоставляет никаких дополнительных методов.
