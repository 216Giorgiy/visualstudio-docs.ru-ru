# Код ошибки SCRIPT_E_REPORTED
Обработчик скриптов сообщал необработанное исключение в основное приложение через [IActiveScriptSite::OnScriptError](../../winscript/reference/iactivescriptsite-onscripterror.md).  Основное приложение может не учитывать эту ошибку.  
  
## Синтаксис  
  
```cpp  
SCRIPT_E_REPORTED   0x80020101L  
  
```
