# Метод IJsDebugBreakPoint::Delete
Удаляет точку останова.  
  
## Синтаксис  
  
```  
HRESULT Delete(void);  
```  
  
## Возвращаемое значение  
  
## Заметки  
 Возвращает S\_FALSE, если вызван на удаленном точки останова.

 **Заголовок:** jscript9diag.h
