# Интерфейс IDebugExtendedProperty
Расширяет интерфейс `IDebugProperty`.  
  
## Методы в порядке таблицы Vtable  
 Помимо методов, наследуемых из `IDebugProperty` этот интерфейс предоставляет следующие методы.  
  
|Метод|Описание|  
|-----------|--------------|  
|[IDebugExtendedProperty::GetExtendedPropertyInfo](../../winscript/reference/idebugextendedproperty-getextendedpropertyinfo.md)|Возвращает `ExtendedDebugPropertyInfo`, которое описывает это `IDebugExtendedProperty``.`|  
|[IDebugExtendedProperty::EnumExtendedMembers](../../winscript/reference/idebugextendedproperty-enumextendedmembers.md)|Перечисляет членов расширенного свойства.|  
  
## Требования  
 Заголовок: dbgprop.h
