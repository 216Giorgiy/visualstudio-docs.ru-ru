# Метод IActiveScriptTraceInfo::StopScriptTracing
Останавливает трассировку скрипта.  
  
## Синтаксис  
  
```  
HRESULT StopScriptTracing(void);   
}  
  
```  
  
#### Параметры  
 Этот метод не имеет параметров.  
  
## Возвращаемое значение  
 Возможные возвращаемые значения этого метода следующим образом:  
  
1.  S\_ОК. Успех.  
  
2.  E\_NOTIMPL: Не реализуется.
