# IDebugApplicationThreadEvents110::OnSuspendForBreakPoint
Определяет, является ли поток приостановил полностью точки останова и обычное выполнение не возобновляется.  
  
> [!IMPORTANT]
>  [IDebugApplicationThreadEvents110 — интерфейс](../../winscript/reference/idebugapplicationthreadevents110-interface.md) реализуется PDM v11.0 и большим.  Обнаружен в activdbg100.h.  
  
## Синтаксис  
  
```cpp  
HRESULT OnSuspendForBreakPoint( void );  
```  
  
#### Параметры  
 Этот метод не имеет параметров.
