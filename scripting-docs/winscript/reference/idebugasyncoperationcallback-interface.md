# Интерфейс IDebugAsyncOperationCallBack
Предоставляет события состояния, связанные с `IDebugAsyncOperation` отслеживания хода выполнения интерфейса.  
  
## Методы  
 В дополнение к методам, наследуемым от интерфейса `IUnknown`, интерфейс `IDebugAsyncOperationCallBack` предоставляет следующие методы.  
  
|Метод|Описание|  
|-----------|--------------|  
|[IDebugAsyncOperationCallBack::onComplete](../../winscript/reference/idebugasyncoperationcallback-oncomplete.md)|Сообщает, что результат асинхронной операции. доступен из отладочных|
