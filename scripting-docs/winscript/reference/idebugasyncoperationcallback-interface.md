# <a name="idebugasyncoperationcallback-interface"></a>Интерфейс IDebugAsyncOperationCallBack
Предоставляет состояние события, связанные с ходом `IDebugAsyncOperation` интерфейс оценки.  
  
## <a name="methods"></a>Методы  
 Помимо методов, наследуемых от `IUnknown`, `IDebugAsyncOperationCallBack` интерфейс предоставляет следующие методы.  
  
|Метод|Описание|  
|------------|-----------------|  
|[IDebugAsyncOperationCallBack::onComplete](../../winscript/reference/idebugasyncoperationcallback-oncomplete.md)|Сигналы, результат становится доступен из отладки асинхронной операции.|
