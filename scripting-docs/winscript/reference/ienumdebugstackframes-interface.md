# Интерфейс IEnumDebugStackFrames
Перечисляет кадры стека, соответствующий поток.  
  
## Методы  
 В дополнение к методам, наследуемым от интерфейса `IUnknown`, интерфейс `IEnumDebugStackFrames` предоставляет следующие методы.  
  
|Метод|Описание|  
|-----------|--------------|  
|[IEnumDebugStackFrames::Next](../../winscript/reference/ienumdebugstackframes-next.md)|Извлекает заданное количество сегментов в последовательности перечисления.|  
|[IEnumDebugStackFrames::Skip](../../winscript/reference/ienumdebugstackframes-skip.md)|Пропустить указанное количество сегментов в последовательности перечисления.|  
|[IEnumDebugStackFrames::Reset](../../winscript/reference/ienumdebugstackframes-reset.md)|Сбрасывает последовательность перечислений в начало.|  
|[IEnumDebugStackFrames::Clone](../../winscript/reference/ienumdebugstackframes-clone.md)|Создает перечислитель с тем же состоянием, что и текущий перечислитель.|
