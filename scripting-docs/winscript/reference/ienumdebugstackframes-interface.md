# <a name="ienumdebugstackframes-interface"></a>Интерфейс IEnumDebugStackFrames
Перечисляет кадры стека, соответствующий поток.  
  
## <a name="methods"></a>Методы  
 Помимо методов, наследуемых от `IUnknown`, `IEnumDebugStackFrames` интерфейс предоставляет следующие методы.  
  
|Метод|Описание|  
|------------|-----------------|  
|[IEnumDebugStackFrames::Next](../../winscript/reference/ienumdebugstackframes-next.md)|Возвращает указанное количество сегментов в последовательности перечисления.|  
|[IEnumDebugStackFrames::Skip](../../winscript/reference/ienumdebugstackframes-skip.md)|Пропускает указанное число сегментов в последовательности перечисления.|  
|[IEnumDebugStackFrames::Reset](../../winscript/reference/ienumdebugstackframes-reset.md)|Сбрасывает последовательность перечисления в начало.|  
|[IEnumDebugStackFrames::Clone](../../winscript/reference/ienumdebugstackframes-clone.md)|Создает перечислитель, который содержит том же состоянии, как у текущего перечислителя.|
