# <a name="profilerexternalobjectaddress-type"></a>Тип PROFILER_EXTERNAL_OBJECT_ADDRESS
Адрес объекта, например C++ выделенного объекта, вне кучи JavaScript внешнего объекта. Используется в [структура PROFILER_HEAP_OBJECT](../../winscript/reference/profiler-heap-object-structure.md) и [структура PROFILER_HEAP_OBJECT_RELATIONSHIP](../../winscript/reference/profiler-heap-object-relationship-structure.md).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
typedef void* PROFILER_EXTERNAL_OBJECT_ADDRESS;  
```
