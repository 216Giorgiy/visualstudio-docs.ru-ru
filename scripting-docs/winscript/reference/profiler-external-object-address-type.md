# Тип PROFILER_EXTERNAL_OBJECT_ADDRESS
Внешний адрес объекта, например объект C\+\+\-allocated, out кучи javascript.  Используемый в [Структура PROFILER\_HEAP\_OBJECT](../../winscript/reference/profiler-heap-object-structure.md) и [Структура PROFILER\_HEAP\_OBJECT\_RELATIONSHIP](../../winscript/reference/profiler-heap-object-relationship-structure.md).  
  
## Синтаксис  
  
```  
typedef void* PROFILER_EXTERNAL_OBJECT_ADDRESS;  
```
