# Метод IJsDebugFrame::GetName
Возвращает понятное имя кадра стека.  
  
## Синтаксис  
  
```  
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### Параметры  
 `pName`  
 \[out\] Имя кадра стека.
