# Интерфейс IBindEventHandler
Позволяет вызывающему объекту указать объект, который обрабатывает указанный обработчик событий.  
  
## Методы  
 В дополнение к методам, наследуемым от интерфейса `IUnknown`, интерфейс `IBindEventHandler` предоставляет следующие методы.  
  
|Метод|Описание|  
|-----------|--------------|  
|[IBindEventHandler::BindHandler](../../winscript/reference/ibindeventhandler-bindhandler.md)|Привязывает событие объекта.|
