# Код ошибки SCRIPT_E_PROPAGATE
Ошибка скрипта передачи вызывающему объекту, который может находиться в другом потоке.  Основное приложение должно передавать код ошибки вызывающему объекту.  
  
## Синтаксис  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
