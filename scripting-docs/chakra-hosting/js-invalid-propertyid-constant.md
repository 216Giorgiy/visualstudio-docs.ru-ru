# <a name="jsinvalidpropertyid-constant"></a>Константа JS_INVALID_PROPERTYID
Идентификатор недопустимого свойства.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
const JsPropertyId JS_INVALID_PROPERTYID = 0;  
```  
  
## <a name="requirements"></a>Требования  
 **Заголовок:** jsrt.h
