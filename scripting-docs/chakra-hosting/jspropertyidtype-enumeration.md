# Перечисление JsPropertyIdType
Перечисление типов свойства JavaScript.  
  
## Синтаксис  
  
```vb  
enum JsPropertyIdType;  
  
```  
  
## Участники  
  
### Значения  
  
|Имя|Описание|  
|---------|--------------|  
|JsPropertyIdTypeString|Перечисление типов свойства строки JavaScript.|  
|JsPropertyIdTypeSymbol|Перечисление типов свойства символа JavaScript.|  
  
## Требования  
 jsrt.h
