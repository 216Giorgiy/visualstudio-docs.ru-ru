# <a name="jssourcecontext-typedef"></a>Определение типа (Typedef) JsSourceContext
Файл cookie, который определяет скрипт для целей отладки.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
typedef DWORD_PTR JsSourceContext;  
```  
  
## <a name="requirements"></a>Требования  
 **Заголовок:** jsrt.h
