# Размещение среды выполнения JavaScript
API среды выполнения JavaScript \(JsRT\) предоставляют настольным приложениям, приложениям Магазина Windows и серверным приложениями, выполняемым в ОС Windows, способ добавления возможностей создания сценариев в приложение с использованием стандартизированного ядра JavaScript Chakra, которое также используется Microsoft Edge и Internet Explorer. Эти API доступны в Windows 10 и любой версии ОС Windows, если на компьютере установлен Internet Explorer версии 11.0.
