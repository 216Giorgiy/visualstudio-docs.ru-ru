# Перечисление JsRuntimeVersion
Версия среды выполнения.  
  
## Синтаксис  
  
```  
enum JsRuntimeVersion;  
```  
  
## Участники  
  
### Значения  
  
|Имя|Описание|  
|---------|--------------|  
|`JsRuntimeVersion10`|Создайте среду выполнения версии IE10.|  
|`JsRuntimeVersion11`|Создайте среду выполнения версии IE11.|  
|`JsRuntimeVersionEdge`|Создайте среду выполнения с самой высокой версией, присутствующей на компьютере во время выполнения.|
