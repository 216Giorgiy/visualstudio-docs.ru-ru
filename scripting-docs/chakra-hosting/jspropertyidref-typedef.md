# <a name="jspropertyidref-typedef"></a>Определение типа (Typedef) JsPropertyIdRef
Идентификатор свойства.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
typedef JsRef JsPropertyIdRef;  
```  
  
## <a name="remarks"></a>Примечания  
 Идентификаторы свойства используются для ссылки на свойства объектов JavaScript вместо использования строк.  
  
## <a name="requirements"></a>Требования  
 **Заголовок:** jsrt.h
