# Определение типа (Typedef) JsValueRef
Ссылка на значение JavaScript.  
  
## Синтаксис  
  
```  
typedef JsRef JsValueRef;  
```  
  
## Заметки  
 Значение JavaScript — это значение одного из следующих типов: Undefined, Null, Boolean, String, Number или Object.  
  
## Требования  
 **Заголовок:** jsrt.h
