# Ожидался символ &quot;=&quot; (JavaScript)
Предпринята попытка создать переменную для использования с операторами условной компиляции, но отсутствует знак равенства между переменной и значением, которое требуется ей присвоить.  
  
### Исправление ошибки  
  
-   Добавьте знак равенства.  Примеры.  
  
    ```javascript  
    @set @myvar1 = 1  
    ```
