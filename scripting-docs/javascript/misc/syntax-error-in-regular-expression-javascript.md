# Синтаксическая ошибка в регулярном выражении (JavaScript)
Структура поисковой строки нарушает одно или несколько грамматических правил регулярного выражения [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].  
  
### Исправление ошибки  
  
-   Убедитесь, что структура поисковой строки регулярного выражения соответствует синтаксису регулярного выражения [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].
