# Ожидался символ &quot;:&quot;
Предпринята попытка создать выражение с помощью троичного условного оператора, но не указано двоеточие между вторым и третьим операндами.  Троичный \(с тремя операндами\) условный оператор требует наличия двоеточия между вторым \(true\) и третьим \(false\) операндами.  
  
### Исправление ошибки  
  
-   Вставьте двоеточие между вторым и третьим операндами.
