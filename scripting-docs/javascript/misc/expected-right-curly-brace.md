# Ожидался символ &quot;}&quot;
Отсутствует закрывающая фигурная скобка, обозначающая конец текста функции, цикла, блока кода или инициализатора объекта.  Примером этой ошибки может быть цикл **for**, в котором стоит только открывающая фигурная скобка, обозначающая начало тела цикла.  
  
### Исправление ошибки  
  
-   Добавьте закрывающую фигурную скобку, завершающую функцию, цикл, блок кода или инициализатор объекта.
