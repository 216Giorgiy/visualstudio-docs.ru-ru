# Метка не найдена
Установлена ссылка на несуществующую метку.  Метки можно использовать для пометки блоки кода. В пределах области они должны быть уникальными.  
  
### Исправление ошибки  
  
-   Проверьте, что имя метки написано верно.  
  
-   Убедитесь, что все ссылки на метки установлены на метки, определенные в текущей области \(в том числе для опережающих определений\).
