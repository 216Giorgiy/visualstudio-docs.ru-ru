# <a name="expected-3939"></a>Ожидается &#39; {&#39;
Отсутствует открывающая фигурная скобка, обозначающая начало тела функции. Код, содержащийся в теле функции, даже если он представляет собой одну строку, должны содержаться в фигурных скобках влево и вправо. Обратите внимание, что использование фигурных скобок в функциях строже, чем в циклах.  
  
### <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте открывающая фигурная скобка, обозначающая начало тела функции.
