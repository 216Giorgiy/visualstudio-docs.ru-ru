# Ожидался символ &quot;{&quot;
Вы не включили открывающую фигурную скобку, обозначающую начало текста функции.  Код, составляющий текст функции, даже если это всего одна строка, должен находиться между открывающей и закрывающей фигурными скобками.  Обратите внимание, что использование фигурных скобок в тексте функций более строго, чем в циклах.  
  
### Исправление ошибки  
  
-   Добавьте открывающую фигурную скобку, обозначающую начало текста функции.
