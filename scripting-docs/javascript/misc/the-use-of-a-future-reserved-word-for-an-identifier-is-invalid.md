# Недопустимо использовать в качестве идентификатора зарезервированное на будущее слово
1049: Недопустимо использовать в качестве идентификатора зарезервированное на будущее слово.  
  
 Использование зарезервированного на будущее слова в качестве имени переменной, функции или другого определяемого пользователем идентификатора не допускается.  
  
 Дополнительные сведения о зарезервированных на будущее словах см. в разделе [Планируемые зарезервированные слова JavaScript](../../javascript/reference/javascript-future-reserved-words.md).
