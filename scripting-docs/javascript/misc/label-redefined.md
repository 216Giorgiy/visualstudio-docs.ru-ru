# Метка переопределена
Создана новая метка, но ей дано имя существующей метки.  Метки можно использовать для пометки блоки кода. В пределах области они должны быть уникальными.  
  
### Исправление ошибки  
  
-   Убедитесь, что все метки, используемые в программах, уникальны в рамках соответствующих областей.
