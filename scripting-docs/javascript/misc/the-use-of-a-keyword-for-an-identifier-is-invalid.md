# Недопустимо использовать ключевое слово в качестве идентификатора
1048: Недопустимо использовать ключевое слово в качестве идентификатора  
  
 Использование ключевого слова в качестве имени переменной, функции или другого определяемого пользователем идентификатора не допускается.  
  
 Дополнительные сведения о ключевых словах см. в разделах [Зарезервированные слова языка JavaScript](../../javascript/reference/javascript-reserved-words.md) и [Планируемые зарезервированные слова JavaScript](../../javascript/reference/javascript-future-reserved-words.md).
