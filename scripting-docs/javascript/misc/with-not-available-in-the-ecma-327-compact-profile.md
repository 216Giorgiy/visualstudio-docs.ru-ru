# Ключевое слово with недоступно в ECMA 327 Compact Profile
Инструкция 'with' недоступна в ECMA 327 Compact Profile \(ErrWithNotInCP  
  
 Не разрешено использовать ключевое слово with в Compact Profile.  
  
 Дополнительные сведения о ключевом слове with см. в разделе [Оператор with](../../javascript/reference/with-statement-javascript.md).
