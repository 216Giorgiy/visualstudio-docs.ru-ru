# Ожидалось ключевое слово &quot;@end&quot;
Предпринята попытка создать условно компилируемый блок кода, но в конце не указан оператор **@end**.  Для операторов [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] можно задавать условную компиляцию, заключая их в блок **@if\/@end**.  
  
### Исправление ошибки  
  
-   Добавьте соответствующий оператор **@end**.
