# <a name="expected-39end39"></a>Ожидается &#39; @end&#39;
Была предпринята попытка создания блока условно скомпилированного кода, но не содержит  **@end**  инструкции в конце. [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]операторы условной компиляции, заключив их в  **@if/ @end**  блока.  
  
### <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте соответствующий  **@end**  инструкции.
