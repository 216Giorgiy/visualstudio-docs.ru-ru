# Ожидался идентификатор (JavaScript)
В контексте, в котором необходим идентификатор, использован объект, отличный от идентификатора.  Идентификатор может быть:  
  
-   переменной,  
  
-   свойством,  
  
-   массивом,  
  
-   или именем функции.  
  
### Исправление ошибки  
  
-   Измените выражение, чтобы слева от знака равенства находился идентификатор.
