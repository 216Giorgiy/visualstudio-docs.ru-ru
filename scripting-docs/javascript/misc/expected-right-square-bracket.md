# Ожидался символ &quot;]&quot;
Установлена ссылка на элемент массива, но отсутствует правая квадратная скобка.  Любое выражение, которое ссылается на элемент массива, должно включать открывающую и закрывающую скобки.  
  
### Исправление ошибки  
  
-   Добавьте правую скобку в выражение, которое ссылается на элемент массива.
