# Ожидалось ключевое слово &quot;while&quot;
Условие `while` не включено в цикл `do...while`.  Оператор **do** должен иметь соответствующую проверку `while` в конце блока кода.  
  
### Исправление ошибки  
  
-   Включите оператор проверки `while`после закрывающей фигурной скобки.
