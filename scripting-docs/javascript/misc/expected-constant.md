# Ожидалась константа
Предпринята попытка использовать переменную \(не условной компиляции\) в операторе проверки условной компиляции.  Операторы проверки условной компиляции должны иметь значение константы.  
  
### Исправление ошибки  
  
-   Замените переменную литералом.  
  
-   Замените переменную переменной условной компиляции.
