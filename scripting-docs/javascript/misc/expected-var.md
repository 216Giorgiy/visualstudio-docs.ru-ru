# Ожидалось ключевое слово &quot;var&quot;
Ожидалось ключевое слово "var" \(ERRnoVar 1034\)\)  
  
 Не удалось добавить ключевое слово var, когда это было необходимо.  
  
 Дополнительные сведения о ключевом слове var см. в разделе [Оператор var](../../javascript/reference/var-statement-javascript.md).
