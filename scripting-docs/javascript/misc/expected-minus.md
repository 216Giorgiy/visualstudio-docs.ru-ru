# Ожидался символ &quot;/&quot;
Предпринята попытка создать литерал регулярного выражения, но отсутствует один из символов косой черты \(\/\).  Точно так же как строковые литералы записываются как символы внутри пары кавычек, литералы регулярных выражений записываются как символы внутри пары символов косой черты \(\/\).  
  
### Исправление ошибки  
  
-   Вставьте конечный символ прямой косой черты, обозначающий конец регулярного выражения.
