# Ожидался символ &quot;;&quot;
Предпринята попытка указать несколько операторов в одной строке, не разделяя их точкой с запятой.  Точки с запятой используются для завершения операторов.  Можно разместить в одной строке несколько операторов, но каждый из них должен отделяться от следующего при помощи точки с запятой.  
  
### Исправление ошибки  
  
-   Обозначьте конец каждого из операторов точкой с запятой.  
  
-   Убедитесь, что в вызовах функций скобки используются правильно.  
  
-   Убедитесь, что не забыли поставить точку с запятой в заголовке цикла **for**.
