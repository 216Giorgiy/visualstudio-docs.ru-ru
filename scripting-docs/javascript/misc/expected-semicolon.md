# <a name="expected-3939"></a>Ожидается &#39; &#39;
Предпринята попытка поместить более одной инструкции в строку, но и не разделяйте их точкой с запятой. Для завершения операторов используются точки с запятой. Несмотря на то, что можно разместить несколько операторов в одной строке, каждый из них должны быть заключены точками с запятой.  
  
### <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Отметьте конец каждая инструкция точкой с запятой.  
  
-   Убедитесь, что вызовы функций правильно используются скобки.  
  
-   Убедитесь, что не забыли точки с запятой в заголовке **для** цикла.
