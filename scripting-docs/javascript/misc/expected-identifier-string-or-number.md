# <a name="expected-identifier-string-or-number"></a>Ожидался идентификатор, строка или число
Используется неверный синтаксис для объявления литерала объекта. Свойства литерала объекта должен иметь идентификатор, строка или число. Объектный литерал (также называемый «инициализатором объектов») состоит из пары свойство: значение, заключенного в скобки список разделенных запятыми. Например:  
  
```JavaScript  
var point = {x:1.2, y:-3.4};  
```  
  
### <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Убедитесь, что используется правильный синтаксис литерала.
