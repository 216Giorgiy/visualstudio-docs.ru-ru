# Ожидался идентификатор, строка или число
Для объявления литерала объекта использован неверный синтаксис.  Свойства литерала объекта должны быть идентификаторами, строками или числами.  Литерал объекта \(также называемый "инициализатором объектов"\) состоит из списка заключенных в фигурные скобки пар "свойство:значение", разделенных запятой.  Примеры.  
  
```javascript  
var point = {x:1.2, y:-3.4};  
```  
  
### Исправление ошибки  
  
-   Используйте правильный синтаксис литерала.
