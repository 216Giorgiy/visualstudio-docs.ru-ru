# Недопустимое присваивание (JavaScript)
Совершена попытка присвоить значение идентификатору, доступному только на чтение.  Невозможно присвоить значение идентификатору, доступному только на чтение.  Например, объектам, определенным узлом, или внешним объектам COM.  
  
### Исправление ошибки  
  
-   Не присваивайте значения идентификаторам, доступным только на чтение.
