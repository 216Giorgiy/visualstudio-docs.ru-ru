# Предполагается наличие объекта Array или arguments
Не предоставлен массив, ожидавшийся в качестве аргумента.  Эта ошибка относится только к методу **Function.prototype.apply**.  Второй аргумент данной функции, если он указан, должен представлять собой либо объект `Array`, либо объект **Arguments**.  
  
### Исправление ошибки  
  
-   Укажите объект `Array` или объект **Arguments** в качестве второго аргумента.
