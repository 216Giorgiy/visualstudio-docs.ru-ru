# Длина массива должна быть конечным положительным целым числом
Вызывается конструктор **Array** с аргументом, который не является целым числом \(целые числа состоят из нуля и набора положительных целых чисел\).  
  
### Исправление ошибки  
  
-   При создании новых объектов `Array` можно использовать только положительные целые числа.  Если требуется создать массив с одним элементом, который не является целым числом, это необходимо делать в два этапа.  Сначала создайте массив с одним элементом, затем поместите значение в первый элемент \(array\[0\]\).  Ниже приведен пример кода, вызывающего эту ошибку.  
  
    ```javascript  
    var piArray = new Array(3.14159);  
    ```  
  
     В следующем примере показан правильный способ определения массива с одним числовым элементом.  
  
    ```javascript  
    var piArray = new Array(1);  
    piArray [0] = 3.14159;  
    ```  
  
     Верхнего ограничения на размер массива не существует, кроме максимально возможного значения целого числа \(приблизительно 4 миллиарда\).
