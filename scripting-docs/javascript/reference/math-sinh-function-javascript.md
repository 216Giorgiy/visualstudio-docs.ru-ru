# <a name="mathsinh-function-javascript"></a>Функция Math.sinh (JavaScript)
Возвращает ареа-синус числа.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.sinh(x)   
```  
  
## <a name="remarks"></a>Примечания  
 Возвращаемое значение равно `(exp(x) - exp(-x))/2`. Если значение x равно `NaN`, результат — `NaN`.
