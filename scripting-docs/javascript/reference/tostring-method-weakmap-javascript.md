# Метод toString (WeakMap) (JavaScript)
Возвращает строковое представление `WeakMap` объекта.  
  
## Синтаксис  
  
```javascript  
weakmapObj.toString()  
```  
  
#### Параметры  
 `weakmapObj`  
 Обязательное.  Объект `WeakMap`.  
  
## Значение свойства или возвращаемое значение  
 Строковое представление объекта `WeakMap`.
