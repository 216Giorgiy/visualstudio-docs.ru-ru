# Функция Math.log10 (JavaScript)
Возвращает логарифм с основанием 10 числа.  
  
## Синтаксис  
  
```  
Math.log10(x)   
```  
  
## Возвращаемое значение  
 Если значение `x` равно `NaN`, эта функция возвращает `NaN`.  Если значение `x` меньше нуля \(0\), эта функция возвращает `NaN`.
