# Метод resolvedOptions (Intl.Collator)
Возвращает объект, содержащий свойства и значения объекта средства упорядочения.  
  
## Синтаксис  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### Параметры  
 `collatorObj`  
 Объект `Collator`, из которого извлекается информация.  
  
## Заметки  
 Свойства возвращаемого объекта соответствуют вычисленным свойствам объекта `Collator`.
