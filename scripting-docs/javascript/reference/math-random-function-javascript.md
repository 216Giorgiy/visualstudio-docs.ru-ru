# <a name="mathrandom-function-javascript"></a>Функция Math.random (JavaScript)
Возвращает псевдослучайное число от 0 до 1.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.random( )  
  
```  
  
## <a name="remarks"></a>Примечания  
 Созданное псевдослучайное число — от 0 (включительно) до 1 (исключительно), то есть, возвращенное число может быть равным нулю, но он всегда будет меньше единицы. Генератор случайных чисел автоматически запускается при [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] первой загрузки.
