# Функция Math.random (JavaScript)
Возвращает псевдослучайное число между 0 и 1.  
  
## Синтаксис  
  
```  
  
Math.random( )  
  
```  
  
## Заметки  
 Псевдослучайно число создается в диапазоне от 0 \(включительно\) до 1 \(не включая 1\), то есть возвращенное число может быть нулем, но всегда будет меньше единицы.  Генератор случайных чисел автоматически инициируется при первой загрузке [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].
