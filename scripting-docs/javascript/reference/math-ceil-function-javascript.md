# Функция Math.ceil (JavaScript)
Возвращает наименьшее целое число, которое больше или равно числовому аргументу метода.  
  
## Синтаксис  
  
```  
  
Math.ceil(number)   
```  
  
## Заметки  
 Обязательный аргумент `number` представляет собой числовое выражение.  
  
 Возвращаемое значение — наименьшее целое число, которое больше или равно числовому аргументу метода.  
  
 **Относится к**: [Объект Math](../../javascript/reference/math-object-javascript.md)
