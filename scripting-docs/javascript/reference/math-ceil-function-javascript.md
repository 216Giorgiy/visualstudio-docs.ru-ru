# <a name="mathceil-function-javascript"></a>Функция Math.ceil (JavaScript)
Возвращает наименьшее целое число, больше или равно числовому аргументу.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.ceil(number)   
```  
  
## <a name="remarks"></a>Примечания  
 Обязательный аргумент `number` является числовым выражением.  
  
 Возвращает значение, равное наименьшее целое число больше или равно числовому аргументу целочисленное значение.  
  
 **Применяется к**: [объект Math](../../javascript/reference/math-object-javascript.md)
