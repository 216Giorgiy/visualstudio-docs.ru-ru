# <a name="tostring-method-string-1"></a>Метод toString (String) 1
Возвращает строковое представление строки.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
string.toString()  
```  
  
## <a name="parameters"></a>Параметры  
 `string`  
 Обязательный. Массив, для представления в виде строки.  
  
## <a name="return-value"></a>Возвращаемое значение  
 Строковое представление строки.  
  
## <a name="example"></a>Пример  
 Следующий пример иллюстрирует использование **toString** метод со строкой.  
  
```JavaScript  
var string = "this is a test";  
var strStr = string.toString();  
document.write(strStr);  
  
// Output: this is a test  
  
```
