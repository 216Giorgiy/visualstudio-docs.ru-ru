# <a name="bytelength-property-arraybuffer"></a>Свойство byteLength (ArrayBuffer)
Только для чтения. Длина буфера ArrayBuffer (в байтах).  
  
## <a name="syntax"></a>Синтаксис  
  
```JavaScript  
var arrayLength = arrayBuffer.byteLength;  
```  
  
## <a name="remarks"></a>Примечания  
  
## <a name="example"></a>Пример  
 Следующий пример показывает способ получения байтовая длина буфера arraybuffer.  
  
```JavaScript  
var req = new XMLHttpRequest();  
    req.open('GET', "http://www.example.com");  
    req.responseType = "arraybuffer";  
    req.send();  
  
    req.onreadystatechange = function () {  
        if (req.readyState === 4) {  
            var buffer = req.response;  
  
        alert(buffer.byteLength);  
        }  
    }  
  
```
