# <a name="mathatanh-function-javascript"></a>Функция Math.atanh (JavaScript)
Возвращает ареа-тангенс числа.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.atanh(x)   
```  
  
## <a name="remarks"></a>Примечания  
 Если значение `x` равно `NaN`, результат — `NaN`.  
  
 **Применяется к**: [объект Math](../../javascript/reference/math-object-javascript.md)
