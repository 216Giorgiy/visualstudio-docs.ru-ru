# Функция Math.trunc (JavaScript)
Возвращает целую часть числа, отбросив все цифры дробной части.  
  
## Синтаксис  
  
```vb  
Math.trunc(number)  
```  
  
## Заметки  
 Обязательный аргумент `number` является числовым выражением, для которого требуется усеченное целое число.  
  
 Если параметр `number` имеет значение `NaN`, возвращается значение `NaN`.
