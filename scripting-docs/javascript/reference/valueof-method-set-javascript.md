# Метод valueOf (Set) (JavaScript)
Возвращает примитивное значение указанного объекта.  
  
## Синтаксис  
  
```javascript  
setObj.valueOf()  
```  
  
#### Параметры  
 `setObj`  
 Обязательное.  Объект `Set`.  
  
## Значение свойства или возвращаемое значение  
 Возвращает экземпляр набора.
