# Функция Math.cbrt (JavaScript)
Возвращает кубический корень числа.  
  
## Синтаксис  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## Заметки  
 Если значение `number` равно `NaN`, результат — `NaN`.
