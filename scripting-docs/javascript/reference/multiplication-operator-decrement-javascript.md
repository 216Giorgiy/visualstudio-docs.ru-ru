# <a name="multiplication-operator--javascript"></a>Оператор умножения (*) (JavaScript)
Умножает значения двух выражений.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
result = number1*number2  
```  
  
## <a name="parameters"></a>Параметры  
 *результат*  
 Любая переменная.  
  
 `number1`  
 Любое выражение.  
  
 `number2`  
 Любое выражение.
