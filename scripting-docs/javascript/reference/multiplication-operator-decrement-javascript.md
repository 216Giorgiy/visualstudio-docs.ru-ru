# Оператор умножения (*) (JavaScript)
Умножает значения двух выражений.  
  
## Синтаксис  
  
```  
  
result = number1*number2  
```  
  
## Параметры  
 *result*  
 Любая переменная.  
  
 `number1`  
 Произвольное выражение.  
  
 `number2`  
 Произвольное выражение.
