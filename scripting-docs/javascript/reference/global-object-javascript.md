# Объект Global (JavaScript)
Встроенный объект, назначение которого заключается в сборе глобальных функций и констант в один объект.  
  
## Заметки  
 Объект `Global` никогда не используется напрямую, и его невозможно создать при помощи оператора `new`.  Он создается при инициализации обработчика скриптов, и его функции и константы становятся доступными немедленно.
