# <a name="mathlog2-function-javascript"></a>Функция Math.log2 (JavaScript)
Возвращает логарифм с основанием 2 числа.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
Math.log2(x)   
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Если значение `x` равно `NaN`, эта функция возвращает `NaN`. Если значение `x` меньше нуля (0), эта функция возвращает `NaN`.
