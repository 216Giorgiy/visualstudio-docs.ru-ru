# Метод toString (Set) (JavaScript)
Возвращает строковое представление набора.  
  
## Синтаксис  
  
```javascript  
setObj.toString()  
```  
  
#### Параметры  
 `setObj`  
 Обязательное.  Объект `Set`.  
  
## Значение свойства или возвращаемое значение  
 Строковое представление набора.
