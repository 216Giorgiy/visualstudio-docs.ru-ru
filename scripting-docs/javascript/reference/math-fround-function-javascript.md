# <a name="mathfround-function-javascript"></a>Функция Math.fround (JavaScript)
Возвращает ближайший формат числа, выраженный числом с плавающей запятой одиночной точности.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.fround(  
number  
)   
```  
  
## <a name="remarks"></a>Примечания  
 Если значение `number` равно `NaN`, результат — `NaN`.
