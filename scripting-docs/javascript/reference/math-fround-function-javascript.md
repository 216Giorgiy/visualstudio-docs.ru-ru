# Функция Math.fround (JavaScript)
Возвращает ближайший формат числа, выраженный числом с плавающей запятой одиночной точности.  
  
## Синтаксис  
  
```  
  
Math.fround(  
number  
)   
```  
  
## Заметки  
 Если значение `number` равно `NaN`, результат — `NaN`.
