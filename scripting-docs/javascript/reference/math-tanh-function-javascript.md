# Функция Math.tanh (JavaScript)
Возвращает гиперболический тангенс числа.  
  
## Синтаксис  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## Заметки  
 Возвращаемое значение равно `(exp(x) - exp(-x))/(exp(x) + exp(-x))`.  Если значение `x` равно `NaN`, результат — `NaN`.
