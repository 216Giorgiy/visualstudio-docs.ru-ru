# <a name="mathsin-function-javascript"></a>Функция Math.sin (JavaScript)
Возвращает синус числа.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.sin(radians)   
```  
  
## <a name="remarks"></a>Примечания  
 `radians` Аргумент является числовым выражением, содержащий значение угла в радианах.  
  
 Возвращаемое значение — это синус числового аргумента.
