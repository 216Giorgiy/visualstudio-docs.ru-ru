# Функция Math.sin (JavaScript)
Возвращает синус числа.  
  
## Синтаксис  
  
```  
  
Math.sin(radians)   
```  
  
## Заметки  
 Аргумент `radians` представляет собой числовое выражение, которое содержит угол в радианах.  
  
 Возвращаемое значение — синус числового аргумента.
