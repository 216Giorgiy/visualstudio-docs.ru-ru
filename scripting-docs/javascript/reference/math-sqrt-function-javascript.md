# <a name="mathsqrt-function-javascript"></a>Функция Math.sqrt (JavaScript)
Возвращает квадратный корень числа.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## <a name="remarks"></a>Примечания  
 Обязательный аргумент `number` является числовым выражением.  
  
 Если `number` имеет отрицательное значение, возвращаемое значение — `NaN`.
