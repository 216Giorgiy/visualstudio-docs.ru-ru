# Функция Math.sqrt (JavaScript)
Возвращает квадратный корень числа.  
  
## Синтаксис  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## Заметки  
 Обязательный аргумент `number` представляет собой числовое выражение.  
  
 Если значение `number` является отрицательным, то возвращаемое значение равно `NaN`.
