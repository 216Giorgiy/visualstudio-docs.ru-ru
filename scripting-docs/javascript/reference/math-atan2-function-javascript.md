# Функция Math.atan2 (JavaScript)
Возвращает угол \(в радианах\) от оси X до точки с координатами \(*y*,*x*\).  
  
## Синтаксис  
  
```  
  
Math.atan2(y, x)  
```  
  
## Параметры  
 *x*  
 Обязательный.  Числовое значение, представляющее координату x в декартовой системе координат.  
  
 *y*  
 Обязательный.  Числовое значение, представляющее координату y в декартовой системе координат.  
  
## Заметки  
 Возвращаемое значение лежит в диапазоне от *\-pi* до *pi*.  Оно представляет собой угол для указанной точки \(*y*,*x*\), выраженный в радианах.  
  
 **Относится к**: [Объект Math](../../javascript/reference/math-object-javascript.md)
