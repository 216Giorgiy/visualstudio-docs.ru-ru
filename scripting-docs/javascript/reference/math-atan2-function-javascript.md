# <a name="mathatan2-function-javascript"></a>Функция Math.atan2 (JavaScript)
Возвращает угол (в радианах) от оси X до точки (*y*,*x*).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.atan2(y, x)  
```  
  
## <a name="parameters"></a>Параметры  
 *x*  
 Обязательный. Числовое выражение, представляющее декартовых координат по оси x.  
  
 *y*  
 Обязательный. Числовое выражение, представляющее декартовы координату по оси y.  
  
## <a name="remarks"></a>Примечания  
 Возвращаемое значение находится между *-pi* и *pi*. Он представляет угол предоставленной (*y*,*x*) точки в радианах.  
  
 **Применяется к**: [объект Math](../../javascript/reference/math-object-javascript.md)
