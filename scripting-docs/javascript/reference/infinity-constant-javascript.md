# <a name="infinity-constant-javascript"></a>Константа Infinity (JavaScript)
Число, больше, чем наибольшее число с плавающей запятой. Возвращает начальное значение `Number.POSITIVE_INFINITY`. Минус бесконечность (-Infinity) меньше, чем наименьшее число с плавающей запятой.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
Infinity   
```  
  
## <a name="remarks"></a>Примечания  
 `Infinity` Константой входит в состав `Global` объекта и становится доступным при инициализации обработчика скриптов.
