# Метод clear (Set) (JavaScript)
Удаляет из набора все элементы.  
  
## Синтаксис  
  
```javascript  
setObj.clear()  
```  
  
#### Параметры  
 `setObj`  
 Обязательное.  Набор, который требуется очистить.
