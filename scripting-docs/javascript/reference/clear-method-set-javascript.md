# <a name="clear-method-set-javascript"></a>Метод clear (Set) (JavaScript)
Удаляет из набора все элементы.  
  
## <a name="syntax"></a>Синтаксис  
  
```JavaScript  
setObj.clear()  
```  
  
#### <a name="parameters"></a>Параметры  
 `setObj`  
 Обязательный. Набор для очистки.
