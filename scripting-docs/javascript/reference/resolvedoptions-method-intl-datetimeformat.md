# <a name="resolvedoptions-method-intldatetimeformat"></a>Метод resolvedOptions (Intl.DateTimeFormat)
Возвращает объект, содержащий свойства и значения объекта средства форматирования даты и времени.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Параметры  
 `dateTimeFormatObj`  
 `DateTimeFormat` Объект для получения информации от.  
  
## <a name="remarks"></a>Примечания  
 Свойства возвращаемого объекта соответствуют вычисляемого свойства `DateTimeFormat` объекта.
