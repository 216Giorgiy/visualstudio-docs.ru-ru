# Метод resolvedOptions (Intl.DateTimeFormat)
Возвращает объект, содержащий свойства и значения объекта средства форматирования даты\/времени.  
  
## Синтаксис  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### Параметры  
 `dateTimeFormatObj`  
 Объект `DateTimeFormat`, из которого извлекается информация.  
  
## Заметки  
 Свойства возвращаемого объекта соответствуют вычисленным свойствам объекта `DateTimeFormat`.
