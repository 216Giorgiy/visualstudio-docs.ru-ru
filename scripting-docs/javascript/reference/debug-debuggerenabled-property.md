# Свойство Debug.debuggerEnabled
Определяет, включена ли отладка для контекста скрипта.  Отладка может быть включена или отключена, если присоединен отладчик.  
  
## Синтаксис  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
