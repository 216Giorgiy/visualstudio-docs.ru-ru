# Оператор присваивания (=) (JavaScript)
Присваивает значение переменной.  
  
## Синтаксис  
  
```  
  
result = expression  
```  
  
## Параметры  
 `result`  
 Любая переменная.  
  
 `expression`  
 Произвольное числовое выражение.  
  
## Заметки  
 Оператор \= ведет себя другие операторы, поэтому содержащие его выражения имеют значение.  Это означает, что операторы присваивания можно объединять в цепочку следующим образом: `j = k = l = 0`.  В этом случае `j`, `k` и `l` равны нулю.
