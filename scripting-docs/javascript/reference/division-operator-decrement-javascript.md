# Оператор деления (/) (JavaScript)
Выполняет деление двух выражений.  
  
## Синтаксис  
  
```  
  
result = number1 / number2   
```  
  
## Параметры  
 `result`  
 Любая числовая переменная.  
  
 `number1`  
 Произвольное числовое выражение.  
  
 `number2`  
 Произвольное числовое выражение.
