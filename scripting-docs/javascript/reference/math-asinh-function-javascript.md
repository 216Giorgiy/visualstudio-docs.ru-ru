# Функция Math.asinh (JavaScript)
Возвращает ареа\-синус числа.  
  
## Синтаксис  
  
```  
  
Math.asinh(x)   
```  
  
## Заметки  
 Если значение `x` равно `NaN`, результат — `NaN`.  
  
 **Применимо к** [Объект Math](../../javascript/reference/math-object-javascript.md)
