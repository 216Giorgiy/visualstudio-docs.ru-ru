# <a name="mathasinh-function-javascript"></a>Функция Math.asinh (JavaScript)
Возвращает ареа-синус числа.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.asinh(x)   
```  
  
## <a name="remarks"></a>Примечания  
 Если значение `x` равно `NaN`, результат — `NaN`.  
  
 **Применяется к**: [объект Math](../../javascript/reference/math-object-javascript.md)
