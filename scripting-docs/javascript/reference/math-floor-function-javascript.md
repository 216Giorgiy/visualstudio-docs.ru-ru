# <a name="mathfloor-function-javascript"></a>Math.floor (JavaScript)
Возвращает наибольшее целое число, меньше или равно числовому аргументу.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.floor(number)   
```  
  
## <a name="remarks"></a>Примечания  
 Необходимая *номер* аргумент является числовым выражением.  
  
 Возвращаемое значение является целочисленное значение, равное наибольшее целое число, меньше или равно числовому аргументу.  
  
 **Применяется к**: [объект Math](../../javascript/reference/math-object-javascript.md)
