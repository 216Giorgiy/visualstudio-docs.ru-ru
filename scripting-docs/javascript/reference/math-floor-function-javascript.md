# Math.floor (JavaScript)
Возвращает наибольшее целое число, которое меньше или равно числовому аргументу метода.  
  
## Синтаксис  
  
```  
  
Math.floor(number)   
```  
  
## Заметки  
 Обязательный аргумент *number* представляет собой числовое выражение.  
  
 Возвращаемое значение — наибольшее целое число, которое меньше или равно числовому аргументу метода.  
  
 **Относится к**: [Объект Math](../../javascript/reference/math-object-javascript.md)
