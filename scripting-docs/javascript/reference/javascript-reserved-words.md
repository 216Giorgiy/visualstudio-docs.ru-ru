# <a name="javascript-reserved-words"></a>Зарезервированные слова языка JavaScript
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]имеет ряд зарезервированные слова, которые нельзя использовать в качестве идентификаторов. Зарезервированные слова имеют особое значение для [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] языка, как они являются частью синтаксиса языка. Использование зарезервированных слов приведет к ошибке компиляции при загрузке скрипта.  
  
 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]также есть список Планируемые зарезервированные слова. Эти слова не в данный момент частью [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] языка, несмотря на то, что они зарезервированы для будущего использования. Дополнительные сведения о будущем зарезервированных ключевых слов на языке JavaScript см. в разделе [Планируемые зарезервированные слова JavaScript](../../javascript/reference/javascript-future-reserved-words.md).  
  
 При выборе идентификаторов также важно избежать все слова, которые уже являются именами встроенных [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] объектов и функций, таких как `String` или `parseInt`.
