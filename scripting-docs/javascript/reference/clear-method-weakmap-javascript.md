# <a name="clear-method-weakmap-javascript"></a>Метод clear (WeakMap) (JavaScript)
Удаляет все элементы из объекта `WeakMap`.  
  
## <a name="syntax"></a>Синтаксис  
  
```JavaScript  
weakmapObj.clear()  
```  
  
#### <a name="parameters"></a>Параметры  
 `weakmapObj`  
 Обязательный. `WeakMap` Объекта для очистки.
