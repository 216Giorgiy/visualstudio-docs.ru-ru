# Метод clear (WeakMap) (JavaScript)
Удаляет все элементы из объекта `WeakMap`.  
  
## Синтаксис  
  
```javascript  
weakmapObj.clear()  
```  
  
#### Параметры  
 `weakmapObj`  
 Обязательное.  Объект `WeakMap`, который требуется очистить.
