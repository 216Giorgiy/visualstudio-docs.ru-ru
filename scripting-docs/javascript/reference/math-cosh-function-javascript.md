# Функция Math.cosh (JavaScript)
Возвращает гиперболический косинус числа.  
  
## Синтаксис  
  
```  
  
Math.cosh(x)   
```  
  
## Заметки  
 Возвращаемое значение равно `(exp(x) + exp(-x))/2`.  Если значение `x` равно `NaN`, результат — `NaN`.  
  
 **Применимо к** [Объект Math](../../javascript/reference/math-object-javascript.md)
