# <a name="mathcosh-function-javascript"></a>Функция Math.cosh (JavaScript)
Возвращает гиперболический косинус числа.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.cosh(x)   
```  
  
## <a name="remarks"></a>Примечания  
 Возвращаемое значение равно `(exp(x) + exp(-x))/2`. Если значение `x` равно `NaN`, результат — `NaN`.  
  
 **Применяется к**: [объект Math](../../javascript/reference/math-object-javascript.md)
