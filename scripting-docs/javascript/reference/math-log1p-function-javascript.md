# Функция Math.log1p (JavaScript)
Возвращает натуральный логарифм 1 плюс числа.  
  
## Синтаксис  
  
```  
Math.log1p(x)   
```  
  
## Возвращаемое значение  
 Если значение `x` равно `NaN`, эта функция возвращает `NaN`.  Если значение `x` меньше нуля \(0\), эта функция возвращает `NaN`.
