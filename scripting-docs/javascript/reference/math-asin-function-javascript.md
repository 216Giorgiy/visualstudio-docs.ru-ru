# <a name="mathasin-function-javascript"></a>Функция Math.asin (JavaScript)
Возвращает арксинус числа.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.asin(number)   
```  
  
## <a name="remarks"></a>Примечания  
 Необходимая `number` аргумент является числовым выражением, для которого требуется арксинус.  
  
 Возвращаемое значение — арксинус `number` аргумент в радианах.  
  
 **Применяется к**: [объект Math](../../javascript/reference/math-object-javascript.md)
