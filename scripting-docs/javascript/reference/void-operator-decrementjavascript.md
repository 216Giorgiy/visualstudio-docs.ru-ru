# <a name="void-operator-javascript"></a>Оператор void (JavaScript)
Запрещает выражению возвращать значение.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
void expression   
```  
  
## <a name="remarks"></a>Примечания  
 `expression` Может быть любым допустимым [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] выражение.  
  
 `void` Оператор вычисляет выражение и возвращает `undefined`. Это полезно в ситуациях, где следует вычислить выражение, но нежелательно результаты видимым для остальной части скрипта.
