# Оператор void (JavaScript)
Запрещает выражению возвращать значение.  
  
## Синтаксис  
  
```  
void expression   
```  
  
## Заметки  
 Аргумент `expression` — это любое допустимое выражение [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].  
  
 Оператор `void` вычисляет свое выражение и возвращает значение `undefined`.  Это удобно в ситуациях, когда нужно вычислить выражение, но результат должен быть скрыт от остальной части скрипта.
