# Свойство size (Map) (JavaScript)
Возвращает число элементов в сопоставлении.  
  
## Синтаксис  
  
```javascript  
sizeVar = mapObj.size  
```  
  
#### Параметры  
 `sizeVar`  
 Обязательное.  Любое число.  
  
 `mapObj`  
 Обязательное.  Любой объект `Map`.
