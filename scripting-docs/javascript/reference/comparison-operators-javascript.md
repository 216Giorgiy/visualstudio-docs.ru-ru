# Операторы сравнения (JavaScript)
Возвращает логическое значение, указывающее результат сравнения.  
  
## Синтаксис  
  
```  
  
expression1 comparisonoperator expression2  
```  
  
## Параметры  
 `expression1`  
 Произвольное выражение.  
  
 `comparisonoperator`  
 Любой оператор сравнения.  
  
 `expression2`  
 Произвольное выражение.  
  
## Заметки  
 При сравнении строк в [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] используется значение символа Юникода строкового выражения.  
  
 Ниже описано поведение различных групп операторов в зависимости от типов и значений выражений `expression1` и `expression2`.  
  
 Операторы отношения: `<`, `>`, `<=`, `>=`  
  
-   Попытка преобразовать `expression1` и `expression2` в числа.  
  
-   Если оба выражения являются строками, выполняется сравнение строк.  
  
-   Если какое\-либо из выражений имеет значение `NaN`, возвращается значение `false`.  
  
-   Отрицательный нуль равен положительному нулю.  
  
-   Отрицательная бесконечность меньше любых значений, включая саму себя.  
  
-   Положительная бесконечность больше любых значений, включая саму себя.  
  
 Операторы равенства: `==`, `!=`  
  
-   Если типы двух выражений различаются, выполняется попытка преобразовать их в строки, числа или логические значения.  
  
-   Значения типа `NaN` не равны никаким значениям, включая самих себя.  
  
-   Отрицательный нуль равен положительному нулю.  
  
-   Значение `null` равно и значению `null`, и значению `undefined`.  
  
-   Значения считаются равными, если они представляют собой идентичные строки, численно эквивалентные числа, один и тот же объект, идентичные логические значения или \(если они различных типов\) могут быть приведены к одному из перечисленных здесь случаев.  
  
-   Все прочие сравнения считаются неравными.  
  
 Операторы тождества: `===`, `!==`  
  
 Эти операторы работают так же, как и операторы равенства, за исключением того, что преобразование типов не выполняется.  Если типы обоих выражений не совпадают, то эти выражения всегда возвращают значение `false`.
