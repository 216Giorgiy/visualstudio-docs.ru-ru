# <a name="mathexp-function-javascript"></a>Функция Math.exp (JavaScript)
Возвращает число e (основания натуральных логарифмов), возведенное в степень.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.exp(number)   
```  
  
## <a name="remarks"></a>Примечания  
 Необходимая `number` аргумент является числовым выражением, представляющим степень *e*.  
  
 Возвращаемое значение является числом. Константа *e* является Эйлера число, равное примерно 2,71828 и `number` — переданный аргумент.  
  
 **Применяется к**: [объект Math](../../javascript/reference/math-object-javascript.md)
