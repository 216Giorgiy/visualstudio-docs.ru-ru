# Функция Math.exp (JavaScript)
Возвращает число e \(основание натуральных логарифмов\), возведенное в степень.  
  
## Синтаксис  
  
```  
  
Math.exp(number)   
```  
  
## Заметки  
 Обязательный аргумент `number` — это числовое выражение, представляющее степень числа *e*.  
  
 Возвращаемое значение является числом.  Константа *e* представляет собой число Эйлера, приблизительно равное 2,71828, а `number` — это указанный аргумент.  
  
 **Относится к**: [Объект Math](../../javascript/reference/math-object-javascript.md)
