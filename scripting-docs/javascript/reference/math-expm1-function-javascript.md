# <a name="mathexpm1-function-javascript"></a>Функция Math.expm1 (JavaScript)
Возвращает результат вычитания 1 из e (основания натуральных логарифмов), возведенного в степень.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.expm1(number)   
```  
  
## <a name="remarks"></a>Примечания  
 Обязательный аргумент `number` является числовым выражением, представляющим степень, в которую возводится e.  
  
 Возвращаемое значение является числом. Константа e — это число Эйлера, равное примерно 2,71828, а `number` — переданный аргумент.  
  
 **Применяется к**: [объект Math](../../javascript/reference/math-object-javascript.md)
