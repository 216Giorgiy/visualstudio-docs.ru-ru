# Свойство size (Set) (JavaScript)
Возвращает число элементов в наборе.  
  
## Синтаксис  
  
```javascript  
sizeVar = setObj.size  
```  
  
#### Параметры  
 `sizeVar`  
 Обязательное.  Любое число.  
  
 `setObj`  
 Обязательное.  Любой объект `Set`.
