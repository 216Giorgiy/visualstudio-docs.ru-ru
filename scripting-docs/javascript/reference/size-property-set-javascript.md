# <a name="size-property-set-javascript"></a>Свойство size (Set) (JavaScript)
Возвращает количество элементов в наборе.  
  
## <a name="syntax"></a>Синтаксис  
  
```JavaScript  
sizeVar = setObj.size  
```  
  
#### <a name="parameters"></a>Параметры  
 `sizeVar`  
 Обязательный. Любое число.  
  
 `setObj`  
 Обязательный. Любой объект `Set`.
