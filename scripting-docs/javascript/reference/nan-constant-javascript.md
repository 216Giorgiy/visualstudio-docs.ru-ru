# Константа NaN (JavaScript)
Специальная константа, которая определяет, что выражение не является числом.  
  
## Синтаксис  
  
```  
NaN   
```  
  
## Заметки  
 Константа `NaN` \(не число\) является членом объекта `Global` и становится доступной при инициализации обработчика сценариев.  
  
## Требования  
 Свойство `NaN` было введено в [!INCLUDE[jsv3text](../../javascript/reference/includes/jsv3text-md.md)] и доступно только для чтения в [!INCLUDE[jsv9textspecific](../../javascript/reference/includes/jsv9textspecific-md.md)].
