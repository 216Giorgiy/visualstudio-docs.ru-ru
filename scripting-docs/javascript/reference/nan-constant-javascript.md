# <a name="nan-constant-javascript"></a>Константа NaN (JavaScript)
Особая константа, которая указывает thatNot выражение не является числом.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
NaN   
```  
  
## <a name="remarks"></a>Примечания  
 `NaN` Константа (не число) является членом `Global` объекта и становится доступным при инициализации обработчика скриптов.
