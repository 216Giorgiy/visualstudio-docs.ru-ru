# <a name="mathtan-function-javascript"></a>Функция Math.tan (JavaScript)
Возвращает тангенс числа.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## <a name="remarks"></a>Примечания  
 Необходимая `radians` аргумент является числовым выражением, содержащий значение угла в радианах.  
  
 Возвращает значение числового аргумента тангенс `radians`.
