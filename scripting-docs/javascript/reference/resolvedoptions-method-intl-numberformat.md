# Метод resolvedOptions (Intl.NumberFormat)
Возвращает объект, содержащий свойства и значения объекта средства форматирования `NumberFormat`.  
  
## Синтаксис  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### Параметры  
 `numberFormatObj`  
 Объект `NumberFormat`, из которого извлекается информация.  
  
## Заметки  
 Свойства возвращаемого объекта соответствуют вычисленным свойствам объекта `NumberFormat`.
