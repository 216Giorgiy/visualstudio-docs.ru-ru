# Метод valueOf (Map) (JavaScript)
Возвращает примитивное значение указанного объекта.  
  
## Синтаксис  
  
```javascript  
mapObj.valueOf()  
```  
  
#### Параметры  
 `mapObj`  
 Обязательное.  Объект `Map`.  
  
## Значение свойства или возвращаемое значение  
 Возвращает экземпляр сопоставления.
