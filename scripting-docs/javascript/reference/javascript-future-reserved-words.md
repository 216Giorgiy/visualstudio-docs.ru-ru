# <a name="javascript-future-reserved-words"></a>Планируемые зарезервированные слова JavaScript
В JavaScript будущих зарезервированные ключевые слова должны не использоваться как идентификаторы, несмотря на то, что они не имеют особого значения в текущей версии.  
  
 Список зарезервированных слов на языке JavaScript см. в разделе [зарезервированные слова JavaScript](../../javascript/reference/javascript-reserved-words.md).  
  
## <a name="future-reserved-words"></a>Планируемые зарезервированные слова  
 Ниже приведены Планируемые зарезервированные слова.  
  
1.  класс  
  
2.  enum  
  
3.  экспорт  
  
4.  расширяет  
  
5.  импорт  
  
6.  Super  
  
## <a name="future-reserved-words-in-strict-mode"></a>Планируемые зарезервированные слова в строгом режиме  
 Ниже приведены Планируемые зарезервированные слова только в строгом режиме. Дополнительные сведения о строгом режиме см. в разделе [директива use strict](../../javascript/reference/use-strict-directive.md).  
  
1.  реализует  
  
2.  интерфейс  
  
3.  пакет  
  
4.  private  
  
5.  protected  
  
6.  public  
  
7.  static  
  
8.  yield
