# Функция Math.atan (JavaScript)
Возвращает арктангенс числа.  
  
## Синтаксис  
  
```  
  
Math.atan(number)   
```  
  
## Заметки  
 Обязательный аргумент `number` представляет собой числовое выражение, для которого требуется определить арктангенс.  
  
 Возвращаемое значение — арктангенс аргумента `number` в радианах.  
  
 **Относится к**: [Объект Math](../../javascript/reference/math-object-javascript.md)
