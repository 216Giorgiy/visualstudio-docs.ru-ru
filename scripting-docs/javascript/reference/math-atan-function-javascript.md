# <a name="mathatan-function-javascript"></a>Функция Math.atan (JavaScript)
Возвращает арктангенс числа.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
Math.atan(number)   
```  
  
## <a name="remarks"></a>Примечания  
 Необходимая `number` аргумент является числовым выражением, для которого требуется арктангенс числа.  
  
 Возвращает значение арктангенса `number` аргумент в радианах.  
  
 **Применяется к**: [объект Math](../../javascript/reference/math-object-javascript.md)
