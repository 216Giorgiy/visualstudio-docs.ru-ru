# <a name="managing-app-resources"></a>Управление ресурсами приложений

Файлы ресурсов приложения, такие как изображения, звуковые и текстовые файлы. являются неотъемлемой частью приложения и не компилируются вместе с ним. Каждая платформа, поддерживаемая Visual Studio для Mac, обрабатывает эти типы файлов по-разному. Дополнительные сведения о работе с ресурсами приложения в Visual Studio для Mac см. в приведенных ниже руководствах.

## <a name="xamarinforms"></a>Xamarin.Forms

Код Xamarin.Forms выполняется на нескольких платформах, каждая из которых имеет свою собственную файловую систему. Это означает, что чтение и запись файлов можно осуществить с помощью собственных API файлов на каждой платформе либо добавлением файлов в качестве внедренных ресурсов.

* [Работа с образами](https://developer.xamarin.com/guides/xamarin-forms/user-interface/images/)
* [Работа с файлами]( https://developer.xamarin.com/guides/xamarin-forms/application-fundamentals/files/)
