# <a name="remove-parameters-refactoring-c"></a>Рефакторинг для удаления параметров (C#)
`Remove Parameters`— Это операция оптимизации, который предоставляет простой способ удаления параметров из методов, индексаторов или делегатов. Удалите параметры изменения объявление; во всех местах, где вызывается член параметр удаляется, чтобы отразить новое объявление.  
  
 Выполнить операцию удаления параметров размещая указатель на метод, индексатор или делегата. Когда курсор находится в положении, для вызова Remove `Parameters` операцию, нажмите кнопку **рефакторинг** меню, нажмите сочетание клавиш или выберите команду в контекстном меню.  
  
> [!NOTE]
>  Невозможно удалить первый параметр метода расширения.  
  
### <a name="to-remove-parameters"></a>Для удаления параметров  
  
1.  Создайте консольное приложение с именем `RemoveParameters`, а затем замените `Program` следующим кодом.  
  
    ```csharp  
    class A  
    {  
        // Invoke on 'A'.  
        public A(string s, int i) { }  
    }  
  
    class B  
    {  
        void C()  
        {  
            // Invoke on 'A'.  
            A a = new A("a", 2);  
        }  
    }  
    ```  
  
2.  Поместите курсор на метод `A`, в объявлении метода или вызов метода.  
  
3.  Из **рефакторинг** последовательно выберите пункты **удалить параметры** для отображения **удалить параметры** диалоговое окно.  
  
     Можно также ввести сочетание клавиш CTRL + R, V, чтобы отобразить **удалить параметры** диалоговое окно.  
  
     Щелкните правой кнопкой мыши курсор, пункты **рефакторинг**, а затем нажмите кнопку **удалить параметры** для отображения **удалить параметры** диалоговое окно.  
  
4.  С помощью **параметры** поле, поместите курсор на `int i`, а затем нажмите кнопку **удалить**.  
  
5.  Нажмите кнопку **ОК**.  
  
6.  В **Предварительный просмотр изменений — удаление параметров** диалоговое окно, нажмите кнопку **применить**.  
  
## <a name="remarks"></a>Примечания  
 Параметры можно удалить из объявления метода или вызов метода. Поместите курсор в объявления метода или делегата имени и удаления параметров вызова.  
  
> [!CAUTION]
>  Удалите включает параметры, можно удалить параметр, который упоминается в тексте элемента, но не удаляет ссылки на этот параметр в теле метода. Это может привести к ошибкам построения в код. Тем не менее, можно использовать **Просмотр изменений** диалоговое окно просмотра кода перед выполнением операции рефакторинга.  
  
 Если удаляемый параметр изменяется во время вызова метода, удаление этого параметра приведет к удалению изменение. Например если вызов метода изменяется с  
  
```csharp  
MyMethod(param1++, param2);  
```  
  
 в  
  
```csharp  
MyMethod(param2);  
```  
  
 операцией рефакторинга `param1` не увеличивается.
