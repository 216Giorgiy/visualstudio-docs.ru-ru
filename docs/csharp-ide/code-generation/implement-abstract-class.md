# <a name="implement-an-abstract-class-in-c"></a>Реализация абстрактного класса на языке C# #
**Что:** позволяет сразу создавать код, необходимый для реализации абстрактного класса. 

**Когда:** требуется выполнить наследование от абстрактного класса.  

**Почему:** можно вручную реализовать все абстрактные члены по одному, однако эта функция автоматически создаст все сигнатуры метода. 

**Как:**

1. Поместите курсор в строке которого красной волнистой линией, указывающее, унаследованным от абстрактного класса, но не реализован все необходимые элементы.

   ![Выделенный код](media/abstract_highlight.png)

1. Затем выполните одно из следующих действий.
   * **Клавиатура**
     * Нажмите клавишу **Ctrl +.** для запуска **Быстрые действия и рефакторинг** и выбрать пункт **реализовать абстрактный класс** из контекстного меню окна предварительного просмотра.
   * **Мышь**
     * Щелкните правой кнопкой мыши и выберите **Быстрые действия и рефакторинг** и выбрать пункт **реализовать абстрактный класс** из контекстного меню окна предварительного просмотра.
     * Наведите указатель мыши на красной волнистой линией и нажмите кнопку ![Лампочки](media/bulb.png) значок, который отображается.
     * Нажмите кнопку ![Лампочки](media/bulb.png) значок, который отображается в левом поле, если текст курсор уже находится в строке с красной волнистой линией.

   ![Реализуйте класс preview](media/abstract_preview.png)

   >[!TIP]
   >Используйте [ **предварительного просмотра изменений** ](../../ide/preview-changes.md) ссылку в нижней части окна предварительного просмотра для просмотра всех изменений, сделанных перед началом выделения.
   >
   >Кроме того, можно использовать **документа**, **проекта**, и **решения** ссылки в нижней части окна предварительного просмотра для создания правильного метода подписи в нескольких классы, которые наследуют от абстрактного класса.

1. Абстрактный метод подписи будет автоматически создан, Готово к реализации.

   ![Реализуйте класс результата](media/abstract_result.png)
