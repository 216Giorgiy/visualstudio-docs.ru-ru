# <a name="generate-a-field-property-or-local-in-c"></a>Создать поле, свойство или локальной в C# #
**Что:** позволяет немедленно создать код для ранее необъявленный поля, свойства или локальной. 

**Когда:** вводит новое поле, свойство или локальный при вводе текста и хотите правильно объявите ее, автоматически.  

**Почему:** можно объявить поле, свойство или локальной справки, однако эта функция будет создать объявление, а затем введите автоматически. 

**Как:**

1. Поместите курсор в строке которого красной волнистой линией, указывающее, используется поле локальный или свойство, которое еще не существует.

   ![Выделенный код](media/field_highlight.png)

1. Затем выполните одно из следующих действий.
   * **Клавиатура**
     * Нажмите клавишу **Ctrl +.** для запуска **Быстрые действия и рефакторинг** и выбрать пункт **создания локального/свойства/поля** из контекстного меню окна предварительного просмотра.
   * **Мышь**
     * Щелкните правой кнопкой мыши и выберите **Быстрые действия и рефакторинг** и выбрать пункт **создания локального/свойства/поля** из контекстного меню окна предварительного просмотра.
     * Наведите указатель мыши на красной волнистой линией и нажмите кнопку ![Лампочки](media/bulb.png) значок, который отображается.
     * Нажмите кнопку ![Лампочки](media/bulb.png) значок, который отображается в левом поле, если текст курсор уже находится в строке с красной волнистой линией.

   ![Создание поля или свойства и локального предварительного просмотра](media/field_preview.png)

   >[!TIP]
   >Используйте [ **предварительного просмотра изменений** ](../../ide/preview-changes.md) ссылку в нижней части окна предварительного просмотра для просмотра всех изменений, сделанных перед началом выделения.

1. Поля, свойства или локальный создается автоматически с типом выводиться из использования.

   ![Создание локального/свойства/поля результата](media/field_result.png)
