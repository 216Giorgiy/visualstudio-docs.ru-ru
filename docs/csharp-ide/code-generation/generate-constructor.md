# <a name="generate-a-constructor-in-c"></a>Создается конструктор в C# #
**Что:** позволяет немедленно сформировать код для нового конструктора для класса. 

**Когда:** вводит новый конструктор и необходимо правильно объявите автоматически, или изменить существующий конструктор. 

**Почему:** можно объявить конструктор перед их использованием, однако этот компонент, с соответствующими параметрами, автоматически создаст. Кроме того изменение существующих конструктор требует обновления всех класса callsite среды, если только эта функция предназначена для автоматического обновления.

**Как:** существует несколько способов, чтобы создать конструктор:
- [Создать конструктор и выбора членов](#pick)
- [Создать конструктор из выбранных полей](#selection)
- [Создание в результате новый использования конструктора](#usage)
- [Добавьте параметр существующий конструктор](#addparameter)
- [Создание и инициализация поля из параметра конструктора](#create)

## <a id = "pick"></a>Создать конструктор и выбора членов
1. Поместите курсор в любую пустую строку в классе:

   ![Курсор в пустую строку](media/constructor1_highlight.png)

1. Затем выполните одно из следующих действий.
   * **Клавиатура**
     * Нажмите клавишу **Ctrl +.** для запуска **Быстрые действия и рефакторинг** и выбрать пункт **создать конструктор...**  из контекстного меню окна предварительного просмотра.
   * **Мышь**
     * Щелкните правой кнопкой мыши и выберите **Быстрые действия и рефакторинг** и выбрать пункт **создать конструктор...**  из контекстного меню окна предварительного просмотра.
     * Нажмите кнопку ![Лампочки](media/bulb.png) значок, который отображается в левом поле, если текст курсор уже находится в пустой строке в классе.

   ![Создание предварительного просмотра конструктора](media/constructor1_preview.png)

1. Появится диалоговое окно позволяет выбрать, какие члены, которые вы хотите использовать в качестве параметров конструктора, а также порядок их (с вверх и Стрелка вниз). Нажмите клавишу **ОК** для фиксации изменений.
  
   ![Диалоговое окно выбора членов](media/constructor1_dialog.png)

   >[!TIP] 
   >Вы можете проверить **добавьте null проверки** флажок для автоматического создания проверок значений null для параметров конструктора.

1. Конструктор будет создана с учетом выбранных параметров в указанном порядке.
   ![Создать результат конструктора](media/constructor1_result.png)

## <a id="selection"></a>Создать конструктор из выбранных полей
1. Выделить элементы, необходимо иметь в созданный конструктор: ![выделение элементов](media/constructor2_highlight.png)

1. Затем выполните одно из следующих действий.
   * **Клавиатура**
     * Нажмите клавишу **Ctrl +.** для запуска **Быстрые действия и рефакторинг** и выбрать пункт **сформировать конструктор «TypeName(...)»**  из контекстного меню окна предварительного просмотра.
   * **Мышь**
     * Щелкните правой кнопкой мыши и выберите **Быстрые действия и рефакторинг** и выбрать пункт **сформировать конструктор «TypeName(...)»**  из контекстного меню окна предварительного просмотра.
     * Нажмите кнопку ![Лампочки](media/bulb.png) значок, который отображается в левом поле, если текст курсор уже находится в строке с выбором.

     ![Создание предварительного просмотра конструктора](media/constructor2_preview.png)

1. Конструктор будет создана с учетом выбранных параметров.
     ![Создать результат конструктора](media/constructor2_result.png)

## <a id="usage"></a>Создание в результате новый использования конструктора
1. Поместите курсор в строке которого красной волнистой линией, указывающее, используется конструктор, который еще не существует.

   ![Выделенный код](media/constructor_highlight.png)

1. Затем выполните одно из следующих действий.
   * **Клавиатура**
     * Нажмите клавишу **Ctrl +.** для запуска **Быстрые действия и рефакторинг** и выбрать пункт  **создать конструктор в "*TypeName*" ** из контекстного меню окна предварительного просмотра.
   * **Мышь**
     * Щелкните правой кнопкой мыши и выберите **Быстрые действия и рефакторинг** и выбрать пункт  **создать конструктор в "*TypeName*" ** из контекстного меню окна предварительного просмотра.
     * Наведите указатель мыши на красной волнистой линией и нажмите кнопку ![Лампочки](media/bulb.png) значок, который отображается.
     * Нажмите кнопку ![Лампочки](media/bulb.png) значок, который отображается в левом поле, если текст курсор уже находится в строке с красной волнистой линией.

   ![Создание предварительного просмотра конструктора](media/constructor_preview.png)

   >[!TIP]
   >Используйте [ **предварительного просмотра изменений** ](../../ide/preview-changes.md) ссылку в нижней части окна предварительного просмотра для просмотра всех изменений, сделанных перед началом выделения.

1. Конструктор будут автоматически созданы со всеми параметрами выводиться из использования.

   ![Создать результат конструктора](media/constructor_result.png)

## <a id="create"></a>Создание и инициализация поля из параметра конструктора
1. Из существующего конструктора добавьте параметр:

   ![Создать конструктор выделения](media/constructor5_highlight.png)

1. Поместите курсор в пределах вновь добавленный параметр.

1. Затем выполните одно из следующих действий.
   * **Клавиатура**
     * Нажмите клавишу **Ctrl +.** для запуска **Быстрые действия и рефакторинг** и выбрать пункт **создать и инициализировать свойство «YourProperty»** или **создать и инициализировать поле «YourField»** из Всплывающее окно предварительного просмотра.
   * **Мышь**
     * Щелкните правой кнопкой мыши и выберите **Быстрые действия и рефакторинг** и выбрать пункт **создать и инициализировать свойство «YourProperty»** или **создать и инициализировать поле «YourField»**из контекстного меню окна предварительного просмотра.
     * Нажмите кнопку ![Лампочки](media/bulb.png) значок, который отображается в левом поле, если текст курсор уже находится в строке с добавленный параметр.

   ![Создание предварительного просмотра конструктора](media/constructor5_preview.png)

1. Свойство или поле будет создана и автоматически присваивается имя для сопоставления типов.

   ![Создать результат конструктора](media/constructor5_result.png)
