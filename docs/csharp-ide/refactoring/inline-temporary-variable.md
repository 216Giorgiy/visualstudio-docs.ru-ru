# <a name="inline-a-temporary-variable-with-c"></a>Встроенная временной переменной с помощью C# #
**Что:** позволяет удалить использование временной переменной и замените его фактический код.

**Когда:** использование этой временной переменной делает затрудняет их понимание кода.  

**Почему:** удаление временной переменной может сделать код более удобочитаемым в определенных ситуациях

**Как:**

1. Выделите или наведите курсор текста внутри этой временной переменной для встраивания:

   ![Выделенный код](media/inline_highlight.png)

1. Затем выполните одно из следующих действий.
   * **Клавиатура**
     * Нажмите клавишу **Ctrl +.** для запуска **Быстрые действия и рефакторинг** и выбрать пункт **встроенной временной переменной** из контекстного меню окна предварительного просмотра.
   * **Мышь**
     * Щелкните его правой кнопкой мыши, выберите **Быстрые действия и рефакторинг** и выбрать пункт **встроенной временной переменной** из контекстного меню окна предварительного просмотра.

   Переменная будет удален и его использования сразу же заменяется значением переменной.

   ![Встроенная результат](media/inline_result.png)
