# <a name="extract-a-method-in-c"></a>Извлечение метода на языке C# #
**Что:** позволяет включать фрагмент кода в отдельный метод.

**Когда:** имеется фрагмент существующего кода в какой-либо метод, который должен вызываться с помощью другого метода.  

**Почему:** вам удалось скопировать и вставить этот код, но это приведет к дублированию.  Лучшим решением является оптимизация фрагменте в отдельный метод, который может быть вызван свободно другим методом.

**Как:**

1. Выделите код для извлечения:

   ![Выделенный код](media/extractmethod_highlight.png)

1. Затем выполните одно из следующих действий.
   * **Клавиатура**
     * Нажмите клавишу **Ctrl + R**, затем **Ctrl + M**.  (Обратите внимание, что сочетание клавиш может отличаться в зависимости от выбранного профиля.)
     * Нажмите клавишу **Ctrl +.** для запуска **Быстрые действия и рефакторинг** и выбрать пункт **извлечение метода** из контекстного меню окна предварительного просмотра.
   * **Мышь**
     * Выберите **Правка > рефакторинг > извлечь метод**.
     * Щелкните его правой кнопкой мыши и выберите **рефакторинг > извлечь > извлечение метода**.
     * Щелкните его правой кнопкой мыши, выберите **Быстрые действия и рефакторинг** и выбрать пункт **извлечение метода** из контекстного меню окна предварительного просмотра.

   Метод будет создаваться немедленно.  Здесь вы теперь переименовать метод просто введя новое имя.

   > [!TIP]
   > Можно также обновить комментарии и другие строки для использования этого нового имени в том числе в [предварительного просмотра изменений](../../ide/preview-changes.md) до сохранения, с помощью флажков на **Переименовать** поле, которое отображается в верхней правой части интегрированная СРЕДА.

   ![Переименуйте метод](media/extractmethod_rename.png)

1. Если вы довольны изменения, нажмите кнопку **применить** или клавишу **ввод** и изменения будут зафиксированы.
