# <a name="convert-get-method-to-property--convert-property-to-get-method"></a>Преобразовать в свойство, метод Get / преобразовать свойство в метод Get
## <a name="convert-get-method-to-property"></a>Преобразовать в свойство, метод Get
**Что:** позволяет преобразовывать метод Get в свойство (и при необходимости Set-метод) и наоборот.

**Когда:** имеется метод Get, который не содержит никакой логики.

**Как:**

1. Поместите курсор в имени метода Get.

1. Затем выполните одно из следующих действий.
   * **Клавиатура**
     * Нажмите клавишу **Ctrl +.** для запуска **Быстрые действия и рефакторинг** и выбрать пункт **замените метод свойство** из контекстного меню окна предварительного просмотра. Если у вас есть метод Set, также можно преобразовать Set-метод в данный момент, выбрав **метод замените Get и Set-метод со свойством**.
   * **Мышь**
     * Щелкните его правой кнопкой мыши, выберите **Быстрые действия и рефакторинг** и выбрать пункт **замените метод свойство** из контекстного меню окна предварительного просмотра. Если у вас есть метод Set, также можно преобразовать Set-метод в данный момент, выбрав **метод замените Get и Set-метод со свойством**.

1. Если вы довольны с изменением в окне просмотра кода, нажмите клавишу **ввод** или выберите команду меню, и изменения будут зафиксированы.

Пример

```csharp
private int MyValue;

// Before
public int GetMyValue()
{
    return MyValue;
}

// Replace 'GetMyValue' with property

// After
public int MyValue
{
    get { return MyValue; }
}
```

## <a name="convert-property-to-get-method"></a>Преобразовать свойство в метод Get
**Что:** позволяет преобразовать свойство в метод Get

**Когда:** имеет свойства, которое включает в себя более немедленно задание и получение значения 

**Как:**

1. Поместите курсор в имени метода Get.

1. Затем выполните одно из следующих действий.
   * **Клавиатура**
     * Нажмите клавишу **Ctrl +.** для запуска **Быстрые действия и рефакторинг** и выбрать пункт **Заменить свойство с методами** из контекстного меню окна предварительного просмотра.
   * **Мышь**
     * Щелкните его правой кнопкой мыши, выберите **Быстрые действия и рефакторинг** и выбрать пункт **Заменить свойство с методами** из контекстного меню окна предварительного просмотра.

1. Если вы довольны с изменением в окне просмотра кода, нажмите клавишу **ввод** или выберите команду меню, и изменения будут зафиксированы.
