# <a name="use-the-msbuild-api"></a>Использование API MSBuild

MSBuild предоставляет общедоступный интерфейс API, который позволяет программам выполнять сборку и проверять проекты.

Документацию по API MSBuild можно найти в статье [Пространства имен Microsoft.Build](https://msdn.microsoft.com/library/mt476050(v=vs.110).aspx).
