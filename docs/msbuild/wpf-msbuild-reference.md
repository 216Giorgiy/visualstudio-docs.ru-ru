# Справочные сведения о WPF для MSBuild
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Windows Presentation Foundation \(WPF\) расширяет обработчик построения Майкрософт \(MSBuild\) с помощью дополнительной поддержки построения, которая описывается в данном разделе.  
  
## В этом подразделе  
 [Файлы TARGETS](../msbuild/wpf-dot-targets-files.md)  
 Сведения о файлах .Targets WPF.  
  
 [Справочные сведения о задачах](../msbuild/wpf-msbuild-task-reference.md)  
 Список доступных задач построения WPF.  
  
 <xref:Microsoft.Build.Tasks>  
 Сборка задачи построения.  
  
 <xref:Microsoft.Build.Tasks.Deployment.Bootstrapper>  
 Сборка загрузчика для развертывания задачи построения.  
  
 <xref:Microsoft.Build.Tasks.Deployment.ManifestUtilities>  
 Сборка программы манифеста для развертывания задачи построения.  
  
 <xref:Microsoft.Build.Tasks.Hosting>  
 Сборка для размещения задачи построения.  
  
 <xref:Microsoft.Build.Tasks.Windows>  
 Сборка окон задачи построения.
