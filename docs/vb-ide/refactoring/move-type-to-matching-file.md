# <a name="move-type-to-a-matching-file-in-visual-basic"></a>Переместить тип в соответствующий файл в Visual Basic
**Что:** позволяет переместить выбранный тип в отдельный файл с тем же именем.

**Когда:** имеется несколько классов, структуры, интерфейсы, т. д. в том же файле, который нужно разделить.  

**Почему:** размещение нескольких типов в одном файле может усложнить поиск этих типов.  Переместив типов файлов с тем же именем, код становится более читаемым и простым для перехода.

**Как:**

1. Выделите или наведите курсор текста внутри имя типа для перемещения:

   ![Выделенный код](media/movetype_highlight.png)

1. Затем выполните одно из следующих действий.
   * **Клавиатура**
     * Нажмите клавишу **Ctrl +.** для запуска **Быстрые действия и рефакторинг** и выбрать пункт **переместить тип в *TypeName*.vb** из контекстного меню окна предварительного просмотра, где *TypeName* — имя выбранного типа.
   * **Мышь**
     * Щелкните его правой кнопкой мыши, выберите **Быстрые действия и рефакторинг** и выбрать пункт **переместить тип в *TypeName*.vb** из контекстного меню окна предварительного просмотра, где  *TypeName* — имя выбранного типа.

   Тип будет мгновенно перемещен в новый файл с таким именем внутри решения.

   ![Встроенная результат](media/movetype_result.png)
