# <a name="implement-an-interface-in-visual-basic"></a>Реализация интерфейса в Visual Basic
**Что:** позволяет сразу создавать код, необходимый для реализации интерфейса. 

**Когда:** требуется наследовать от интерфейса.  

**Почему:** можно вручную реализовать все интерфейса по одному, однако эта функция автоматически создаст все сигнатуры метода. 

**Как:**

1. Поместите курсор в строке которого красной волнистой линией, указывающее, ссылка на интерфейс, но не реализован все необходимые элементы.

   ![Выделенный код](media/interface_highlight.png)

1. Затем выполните одно из следующих действий.
   * **Клавиатура**
     * Нажмите клавишу **Ctrl +.** для запуска **Быстрые действия и рефакторинг** и выбрать пункт **(явно) реализует интерфейс** из контекстного меню окна предварительного просмотра.
   * **Мышь**
     * Щелкните правой кнопкой мыши и выберите **Быстрые действия и рефакторинг** и выбрать пункт **(явно) реализует интерфейс** из контекстного меню окна предварительного просмотра.
     * Наведите указатель мыши на красной волнистой линией и нажмите кнопку ![Лампочки](media/bulb.png) значок, который отображается.
     * Нажмите кнопку ![Лампочки](media/bulb.png) значок, который отображается в левом поле, если текст курсор уже находится в строке с красной волнистой линией.

   ![Реализуйте класс preview](media/interface_preview.png)

   >[!TIP]
   >Используйте [ **предварительного просмотра изменений** ](../../ide/preview-changes.md) ссылку в нижней части окна предварительного просмотра для просмотра всех изменений, сделанных перед началом выделения.
   >
   >Кроме того, можно использовать **документа**, **проекта**, и **решения** ссылки в нижней части окна предварительного просмотра для создания правильного метода подписи в нескольких классы, реализующие интерфейс.

1. Сигнатуры методов интерфейса будет автоматически создан, Готово к реализации.

   ![Реализуйте класс результата](media/interface_result.png)
