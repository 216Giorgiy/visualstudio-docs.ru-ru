# <a name="generate-a-constructor-in-visual-basic"></a>Создается конструктор в Visual Basic
**Что:** позволяет немедленно сформировать код для нового конструктора для класса. 

**Когда:** вводит новый конструктор и хотите правильно объявите ее, автоматически.  

**Почему:** можно объявить конструктор перед их использованием, однако этот компонент, с соответствующими параметрами, автоматически создаст. 

**Как:**

1. Поместите курсор в строке которого красной волнистой линией, указывающее, используется конструктор, который еще не существует.

   ![Выделенный код](media/constructor_highlight.png)

1. Затем выполните одно из следующих действий.
   * **Клавиатура**
     * Нажмите клавишу **Ctrl +.** для запуска **Быстрые действия и рефакторинг** и выбрать пункт  **создать конструктор в "*TypeName*" ** из контекстного меню окна предварительного просмотра.
   * **Мышь**
     * Щелкните правой кнопкой мыши и выберите **Быстрые действия и рефакторинг** и выбрать пункт  **создать конструктор в "*TypeName*" ** из контекстного меню окна предварительного просмотра.
     * Наведите указатель мыши на красной волнистой линией и нажмите кнопку ![Лампочки](media/bulb.png) значок, который отображается.
     * Нажмите кнопку ![Лампочки](media/bulb.png) значок, который отображается в левом поле, если текст курсор уже находится в строке с красной волнистой линией.

   ![Создание предварительного просмотра конструктора](media/constructor_preview.png)

   >[!TIP]
   >Используйте [ **предварительного просмотра изменений** ](../../ide/preview-changes.md) ссылку в нижней части окна предварительного просмотра для просмотра всех изменений, сделанных перед началом выделения.

1. Конструктор будут автоматически созданы со всеми параметрами выводиться из использования.

   ![Создать результат конструктора](media/constructor_result.png)
