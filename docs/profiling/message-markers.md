# Маркеры сообщений
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Метка сообщения представляет выходные данные журнала.  Сообщение — строка, сгенерированная конкретным потоком в определенное время.  Можно экспортировать сообщения в текстовый файл для использования с другими средствами.  Можно навести указатель на сообщении в визуализаторе параллелизма для просмотра строки сообщения.  Можно просматривать все метки сообщений в [Отчет по маркерам](../profiling/markers-report.md).  На следующем рисунке показана метка сообщения.  
  
## Метки объединенных сообщений  
 Иногда несколько сообщений в визуализаторе параллелизма находятся настолько близко друг к другу, что они не могут быть отображены индивидуально.  Когда это происходит, показывается *метка объединенных сообщений*, которая представляет базовые сообщения.  Когда вы наводите указатель на один из этих значков, отображается подсказка о количестве представленных базовых сообщений.  Для просмотра сообщений увеличьте масштаб. Если при постоянно увеличении все равно получается маркер статистической обработки, можно просматривать основные сообщения в [Отчет по маркерам](../profiling/markers-report.md).
