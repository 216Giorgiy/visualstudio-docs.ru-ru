# Знак вставки на временной шкале
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Когда выбирается точка на графике выполнения сегмента потока, над ним появляется курсор времени выполнения.  Стек вызова, который отображается на текущей вкладке стека, \- это ближайший по времени стек для точки, где выбран сегмент.  Курсор используется для сопоставления стека вызова, который отображается на вкладке **Текущий** с моментом времени, когда он был получен.  Курсор указывает точное местоположение стека вызовов, который является ближайшем к месту, выбранному пользователем.
