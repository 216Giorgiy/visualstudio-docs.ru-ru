# Наличие несколько категорий
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Выбран раздел, который содержит более одного вида операций за заданный диапазон времени.  Обратите внимание, что отображаются два цвета, вертикально друг над другом.  Эти цвета представляют только две наиболее частые операции в заданном диапазоне времени.  Частота этих двух операций отображается относительно друг друга в виде третей или половин вертикального столбца.  Чтобы просмотреть дополнительные подробности, попробуйте увеличить масштаб этой секции представления потока.
