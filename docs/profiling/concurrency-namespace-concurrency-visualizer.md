# Пространство имен Concurrency (визуализатор параллелизма)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Пространство имен `Concurrency` содержит классы и функции, предоставляющие доступ к среде выполнения с параллелизмом, которая является платформой параллельного программирования для C\+\+.  Для получения дополнительной информации см. [Среда выполнения с параллелизмом](/visual-cpp/parallel/concrt/concurrency-runtime).  
  
## Синтаксис  
  
```  
namespace Concurrency;  
```  
  
## Члены  
  
### Пространства имен  
  
|Name|Описание|
