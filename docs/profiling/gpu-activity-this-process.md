# Активность GPU (этот процесс)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Сегменты **Активности GPU \(этот процесс\)** в представлении потоков в визуализаторе параллелизма представляют случаи, когда GPU обработал запросы от имени текущего процесса.  Эти запросы отправлены к GPU, как пакеты прямого доступа к памяти \(DMA\).  Длина сегмента представляет время, которое GPU обрабатывал пакет прямого доступа к памяти от имени текущего процесса.  
  
 При выборе сегмента действия GPU, на **Текущей** вкладке отображаются сведения о пакете прямого доступ к памяти, который был обработан.  Эти сведения включают время, которое пакет ожидал в аппаратной очереди, связанной с обработчиком DirectX, процесс, отправивший пакет, и время необходимое для обработки пакета.  Процесс, отличного от текущего процесса, может физически отправить пакет прямого доступ к памяти на GPU.  Визуализатор параллелизма может обнаружить, когда другой процесс отправил задачу GPU от имени текущего процесса.
