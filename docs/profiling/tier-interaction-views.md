# Представления взаимодействия между уровнями
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Профилирование взаимодействия между уровнями позволяет получить дополнительные сведения о времени выполнения функций многоуровневых приложений, взаимодействующих с базой данных посредством [!INCLUDE[vstecado](../data-tools/includes/vstecado_md.md)].  Сбор данных ведется только для синхронных вызовов функций.  Данные о взаимодействии уровней можно включить во все методы профилирования.
