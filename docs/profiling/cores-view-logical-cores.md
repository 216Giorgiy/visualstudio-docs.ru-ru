# Логические ядра представления &quot;Ядра&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Каждое имя канала представляет собой одно из логических ядер, которые существуют в профилируемой системе.  При наименовании нумерация начинается с 0 и включает общее количество логических ядер.  Например, четырехъядерная система будет включать логические ядра с номерами от 0 до 3.
