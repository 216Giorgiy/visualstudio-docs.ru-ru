# Представление &quot;Сводка&quot; — данные инструментирования профилировщика
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

В представлении "Сводка" отображается информация о создающих наибольшую нагрузку функциях или объектах в ходе сеанса профилирования.  Дополнительные сведения, включая описание ссылок на уведомления и списков отчетов, см. в разделе [Представление "Сводка"](../profiling/summary-view.md).  
  
## Временная шкала  
 Временна шкала представления "Сводка" отображает использование ЦП профилированным приложением за время профилирования.  Временная шкала позволяет фильтровать представление в соответствии с заданным интервалом времени.  Для получения дополнительной информации см. [Практическое руководство. Фильтрация представлений отчетов на сводной временной шкале](../Topic/How%20to:%20Filter%20Report%20Views%20from%20the%20Summary%20Timeline.md).  
  
## Критический путь  
 **Критический путь** отображает путь выполнения, потребовавший наибольшего времени.  Чтобы отобразить представление сведений о функции, щелкните функцию.  Для просмотра других представлений щелкните правой кнопкой мыши функцию и выберите необходимое представление из списка.  
  
 **Критический путь** включает следующие данные для каждой функции:  
  
|Столбец|Описание|  
|-------------|--------------|  
|**Name**|Имя функции.|  
|**Затраченное инклюзивное время в процентах**|Процент всего времени в данных профилирования, затраченного функцией на выполнение кода в теле самой функции и в вызванных ею функциях.|  
|**Затраченное эксклюзивное время в процентах**|Процент всего времени в данных профилирования, затраченного функцией на выполнение кода в теле самой функции.  Не включается время выполнения функций, вызванных выбранной функцией.|  
  
## Функции с максимальным объемом индивидуальной работы  
 Список функций, требующих наибольшего времени на выполнение кода в теле функции, но не в вызываемых функциях.  
  
 Раздел **Функции с максимальной индивидуальной работой** включает следующие данные о каждой функции.  
  
|Столбец|Описание|  
|-------------|--------------|  
|**Name**|Имя функции.|  
|**Эксклюзивное время %**|Процент всего времени в данных профилирования, затраченного функцией на выполнение кода в теле самой функции.  Не включается время выполнения функций, вызванных выбранной функцией.|
