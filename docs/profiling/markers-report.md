# Отчет по маркерам
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

В отчете о маркерах перечислены маркеры показанного периода времени.  Панорамирование или масштабирование, или скрытие строк может привести к появлению или исчезновению маркеров.  Отчет содержит следующие сведения о каждом маркере.  
  
-   Время, когда он начался, относительно начала трассировки.  
  
-   Его длительность.  Длительность равна нулю для флагов и сообщений, поскольку они представляют момент времени.  
  
-   Идентификатор потока, вызвавшего его.  
  
-   Поставщик отслеживания событий для Windows \(ETW\), который создал его.  
  
-   Серия маркеров, из которой он был записан.  
  
-   Категория событий, к которой он принадлежит.  
  
-   Его уровень важности.  
  
-   Его тип \(диапазон, флаг или сообщение\).  
  
-   Подробное описание того, что он представляет  
  
 Нажмите кнопку **Экспорт** для сохранения отчета о маркерах как CSV\-файла.  Можно использовать данные в CSV\-файле с другими приложениями и инструментами.  
  
> [!NOTE]
>  Отчет о маркерах может отображать 1000 маркеров.  Чтобы просмотреть все маркеры, экспортируйте полный отчет в CSV\-файл.
