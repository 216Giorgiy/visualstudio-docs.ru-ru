# Представление меток
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Представление меток указывает выборки и события трассировки событий Windows, вставленные в приложение.  
  
 По умолчанию в отчет предварительно вставлены метки, обозначающие начало и концу программы.  
  
 В этом представлении также отображаются данные счетчиков Windows, собранные на основе автоматически созданных меток.  Для получения дополнительной информации см. [Практическое руководство. Сбор данных счетчиков производительности Windows](../profiling/how-to-collect-windows-counter-data.md).  
  
 Чтобы создать фильтр для двух меток, выберите метки, щелкните их правой кнопкой мыши и выберите команду **Добавить фильтр по меткам** или **Добавить фильтр по метке времени**.  
  
 В следующей таблице представлены определения столбцов, доступных в представлении меток.  
  
 **ИД метки**  
 Уникальный идентификатор метки профилирования.  
  
 **Имя метки**  
 Имя события.  
  
 **Отметка времени**  
 Время от начала профилирования до момент, когда событие записано.  
  
 Данные счетчика производительности Windows  
 При сборе данных счетчика производительности Windows значения отображаются в столбце с именем счетчика.
