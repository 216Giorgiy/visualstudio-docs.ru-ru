# Временная шкала представления &quot;Ядра&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Каждая строка на временной шкале представляет ядро логического процессора профилируемой системы.  Для каждой строки по горизонтальной оси указывается поток, который выполняется в логическом ядре в любой заданный момент времени.  Если подвести курсор к нужному цвет на временной шкале, отобразится подсказка, позволяющая идентифицировать поток.  Чтобы упростить идентификацию потока, в расположенной в нижней части окна легенде указывается значение каждого цвета.  С помощью средства масштабирования можно увеличивать и уменьшать объекты путем перетаскивания или прокручивания колесика мыши с нажатой клавишей CTRL.  При переходе между представлением "Ядра" и представлением "Потоки" масштабирование сохраняется.
