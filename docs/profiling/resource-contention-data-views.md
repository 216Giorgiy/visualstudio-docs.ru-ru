# Представления данных о конфликтах ресурсов
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

В этом разделе содержатся справочные сведения о представлениях и отчетах по файлам данных профилировщика, содержащих данные профилирования конфликтов потоков.  
  
## Содержание  
 [Summary View \- Profiler Contention Data](../profiling/resource-contention-data-views.md)  
 Отображает графическую временную шкалу данных профилирования, а также список функций и ресурсов, задействованных в крупнейших событиях блокировки.  
  
 [Представление "Дерево вызовов"](../profiling/call-tree-view-contention-data.md)  
 Отображает иерархическое дерево, представляющее пути выполнения и данные о конфликтах ресурсов для функций в сеансе профилирования.  
  
 [Представление "Модули"](../profiling/modules-view-contention-data.md)  
 Содержит упорядоченные по модулям данные о конфликтах потоков и ресурсов, а также список функций, строк исходного кода и инструкций, которые выполнялись во время событий блокировки.  
  
 [Представление "Вызывающий\/вызываемый"](../profiling/caller-callee-view-contention-data.md)  
 Содержит список данных о конфликтах потоков и ресурсов для выбранной функции, а также функций, вызвавших выбранную функцию и вызванные ею.  
  
 [Представление "Сведения о ресурсах"](../profiling/resource-details-view-contention-data.md)  
 Отображает графическую временную шкалу событий блокировки для каждого конфликтующего ресурса и выводит стек вызова для событий блокировки.  
  
 [Представление "Сведения о потоке"](../profiling/thread-details-view-contention-data.md)  
 Отображает графическую временную шкалу событий блокировки для каждого потока и список стеков вызова для событий блокировки.  
  
 [Представление "Функции"](../profiling/functions-view-contention-data.md)  
 Содержит список данных о конфликтах потоков и ресурсов по функциям.  
  
 [Представление "Конфликты обращения к ресурсам"](../profiling/resource-contentions-view-contention-data.md)  
 Содержит список данных о конфликтах ресурсов по заблокированным ресурсам.  
  
 [Представление строк](../profiling/lines-view-contention-data.md)  
 Содержит список данных о конфликтах ресурсов по строкам исходного кода.  
  
 [Представление указателей инструкций](../profiling/instruction-pointers-ips-view-contention-data.md)  
 Содержит список конфликтов ресурсов по инструкциям.  
  
 [Представление "Процесс"](../profiling/process-view-contention-data.md)  
 Содержит список конфликтов ресурсов по процессам и потокам.  
  
## Справочные сведения  
 [Представление сведений о функции](../profiling/function-details-view.md)  
 Отображает графическую схему связей между выбранной функцией и функциями, которые вызывали выбранную функцию и вызывались ею.
