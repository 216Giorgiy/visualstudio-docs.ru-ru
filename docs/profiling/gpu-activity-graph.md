# Граф активности GPU
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Диаграмма активности GPU в визуализаторе параллелизма показывает уровень активности DirectX в системе, измеряемой количеством ядер DirectX, которые используются в течение долгого времени.  Диаграмма не отображает отдельные обработчики, которые были использованы.  Считается, что обработчик используется, если он обрабатывает любые работы GPU.  
  
## Цвета диаграммы активности GPU  
 Зеленый цвет обозначает потребление обработчиков DirectX процессом в данный момент.  
  
 Светло\-серый цвет обозначает потребление обработчиков DirectX другими процессами в системе.  Чтобы уменьшить потребление обработчиков DirectX другими процессами, уменьшите количество других процессов в системе.  
  
 Белый цвет обозначает доступность неиспользуемые обработчиков DirectX в системе.  Эти обработчики доступны для данного процесса, если есть возможности для их использования.  Некоторые обработчики могут использоваться только для определенных типов задач.
