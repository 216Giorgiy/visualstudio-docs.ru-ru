# Практическое руководство. Сбор данных счетчиков производительности Windows
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Счетчики Windows — это системные счетчики производительности, данные которых собираются через установленные интервалы времени в процессе профилирования.  В представлении "Метки" отчета средств профилирования для каждого интервала сбора данных имеется строка **Автоматическая метка**.  Эта строка содержит столбцы, в которых отображаются значения счетчиков производительности, полученные в соответствующем интервале.  Чтобы ограничить период анализа интервалом времени между двумя определенными метками, выберите метки, щелкните их правой кнопкой мыши и выберите в контекстном меню команду **Фильтр по** \-\>  **Marks**.  
  
 **Требования**  
  
-   [!INCLUDE[vsUltLong](../code-quality/includes/vsultlong_md.md)], [!INCLUDE[vsPreLong](../code-quality/includes/vsprelong_md.md)], [!INCLUDE[vsPro](../code-quality/includes/vspro_md.md)]  
  
> [!NOTE]
>  Функции усиленной безопасности в Windows 8 и Windows Server 2012 требуют значительных изменений в способе сбора данных профилировщиком Visual Studio на этих платформах.  Для Приложений Магазина Windows также требуются новые методы сбора.  См. раздел [Профилирование приложений для Windows 8 и Windows Server 2012](../profiling/performance-tools-on-windows-8-and-windows-server-2012-applications.md).  
  
### Сбор данных счетчиков производительности Windows  
  
1.  В обозревателе производительности щелкните правой кнопкой мыши сеанс, для которого выполняется настройка счетчиков Windows, и выберите **Свойства**.  
  
2.  В окне **Страницы свойств** щелкните **Счетчики Windows**.  
  
3.  Установите флажок **Сбор счетчиков Windows**.  
  
4.  В текстовом поле **Интервал сбора \(мс\)** введите временной интервал.  
  
5.  Выберите категорию в раскрывающемся списке **Категория счетчика**.  
  
6.  Выберите экземпляр из раскрывающегося списка **Экземпляр**.  
  
7.  Выберите счетчики, которые необходимо использовать при профилировании приложения.  
  
8.  Нажмите кнопку **Применить**.
