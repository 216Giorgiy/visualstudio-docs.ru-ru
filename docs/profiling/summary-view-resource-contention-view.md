# Представление &quot;Сводка&quot; — представление &quot;Конфликты обращения к ресурсам&quot; профилировщика
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

В представлении "Сводка" отображаются сведения о событиях в приложении, в которых поток или процесс был приостановлен во время ожидания доступа к ресурсам.  
  
 Дополнительные сведения, включая описание ссылок на уведомления и списков отчетов, см. в разделе [Представление "Сводка"](../profiling/summary-view.md).  
  
## Временная шкала  
 Временная шкала представления "Сводка" отображает количество конфликтных ситуаций профилируемого приложения в сеансе профилирования.  Временная шкала позволяет фильтровать представление в соответствии с заданным интервалом времени.  Для получения дополнительной информации см. [Практическое руководство. Фильтрация представлений отчетов на сводной временной шкале](../Topic/How%20to:%20Filter%20Report%20Views%20from%20the%20Summary%20Timeline.md).  
  
## Ресурсы, за которые происходит больше всего состязаний  
 В представлении **Ресурсы, за которые происходит больше всего состязаний** перечисляются ресурсы в приложении, которые стали причиной большинства конфликтных ситуаций.  Для отображения представления "Состязания" необходимо щелкнуть имя ресурса.  В представлении "Состязания" представляется подробная временная шкала конфликтов ресурсов по потокам.  
  
 Представление **Ресурсы, за которые происходит больше всего состязаний** содержит следующие данные для каждого ресурса.  
  
|Столбец|Описание|  
|-------------|--------------|  
|**Name**|Имя ресурса.|  
|**% конфликтов**|Процент общего количества конфликтов из данных профилирования, относящийся к конфликтам для этого ресурса.|  
  
## Потоки, за которые происходит больше всего состязаний  
 В представлении **Потоки, за которые происходит больше всего состязаний** перечисляются потоки в приложении, которые характеризуются наибольшим числом конфликтных ситуаций.  Можно щелкнуть имя потока, чтобы отобразить представление "Состязания", которое содержит подробную временную шкалу конфликтов ресурсов по потокам.  
  
 Представление **Потоки, за которые происходит больше всего состязаний** содержит следующие данные для каждого потока.  
  
|Столбец|Описание|  
|-------------|--------------|  
|**Идентификатор**|Идентификатор потока.|  
|**Name**|Имя процесса — владельца потока.|  
|**% конфликтов**|Процент общего количества конфликтов из данных профилирования, относящийся к конфликтам для этого ресурса.|
