# Только мой код (представление &quot;Потоки&quot;)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

В случае выбора этого параметра вы будете фильтровать стек вызова, чтобы показывать только код и один уровень вызываемых функций.  
  
 Включая этот параметр, можно значительно уменьшить сложность стека вызова и, возможно, облегчить диагностирование определенной ошибки.  
  
 В некоторых случаях выбор этого параметра может фильтровать блокирующий вызов.  Если вам необходимы полные данные стека вызова, чтобы выполнить определение, отмените выбор этого параметра, чтобы использовать полный стек вызова.
