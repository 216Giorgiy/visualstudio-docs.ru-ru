# Редактирование данных в приложении
После заполнения модели данных данными вы обычно добавляете, изменяете или удаляете некоторую их часть перед отправкой обратно в источник данных либо в другой процесс или приложение.  
  
 В следующих разделах приведены сведения о добавлении, изменении и удалении данных, а также о работе с данными на разных этапах приложения.

 Содержит информацию о создании, обновлении и удалении данных в Entity Framework и службах данных WCF.
