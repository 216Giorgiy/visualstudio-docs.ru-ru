# Практическое руководство. Создание таблиц данных
Данные могут храниться в памяти в объектах <xref:System.Data.DataTable>. \(Наборы данных состоят из объектов <xref:System.Data.DataTable>.\) Обычно новая таблица данных с адаптером таблицы создается при помощи [мастер настройки адаптера таблицы](../Topic/TableAdapter%20Configuration%20Wizard.md) или путем перетаскивания объектов базы данных из **Обозревателя серверов** в **Конструктор наборов данных**.  
  
 Таблицы данных также создаются как побочный продукт при создании нового адаптера таблицы в [мастер настройки источника данных](../data-tools/media/data-source-configuration-wizard.png), но могут быть созданы и независимо от него.  Автономную таблицу данных можно создать путем перетаскивания объекта <xref:System.Data.DataTable> из вкладки **Набор данных** пункта **Панель элементов** в **Конструктор наборов данных**.  
  
> [!NOTE]
>  Сведения о том, как создавать таблицы данных программным способом, см. в разделе [Создание DataTable](../Topic/Creating%20a%20DataTable.md).  
  
 [!INCLUDE[note_settings_general](../data-tools/includes/note_settings_general_md.md)]  
  
## Создание таблицы данных при помощи адаптера таблицы  
 Таблицы данных с адаптером таблиц включают методы заполнения таблицы данными и записи изменений обратно в базу данных.  Адаптеры таблиц создаются путем выполнения **Мастера настройки адаптеров таблиц** или путем перетаскивания объектов базы данных из **Обозревателя серверов** в **Конструктор наборов данных**.  
  
#### Создание новой таблицы данных с адаптером таблицы  
  
1.  Откройте набор данных в **Конструкторе наборов данных**.  Дополнительные сведения см. в разделе [Практическое руководство. Открытие набора данных в конструкторе наборов данных](../Topic/How%20to:%20Open%20a%20Dataset%20in%20the%20Dataset%20Designer.md).  
  
2.  Перетащите **Адаптер таблицы** из вкладки **Набор данных Панели элементов** на **Конструктор наборов данных**.  
  
     Откроется **Мастер настройки адаптеров таблиц**.  
  
3.  Выполните указания мастера.  Дополнительные сведения см. в разделе [мастер настройки адаптера таблицы](../Topic/TableAdapter%20Configuration%20Wizard.md).  
  
#### Создание новой таблицы данных с адаптером таблицы из обозревателя серверов  
  
1.  Откройте набор данных в **Конструкторе источников данных**.  
  
2.  Перетащите объект базы данных \(например, таблицу\) из **Обозревателя серверов** в **Конструктор наборов данных**.  
  
## Создание автономной таблицы данных  
 Для заполнения данными автономные таблицы должны иметь реализованную логику `Fill`.  Сведения о заполнении автономных таблиц данных содержатся в разделе [Заполнение DataSet из адаптера данных DataAdapter](../Topic/Populating%20a%20DataSet%20from%20a%20DataAdapter.md).  
  
#### Создание новой автономной таблицы данных  
  
1.  Откройте набор данных в **Конструкторе наборов данных**.  
  
2.  Перетащите <xref:System.Data.DataTable> из вкладки **Набор данных Панели элементов** в **Конструктор наборов данных**.  
  
3.  Добавьте столбцы для определения таблицы данных.  Для получения дополнительной информации см. [Практическое руководство. Добавление столбцов в объект DataTable](../Topic/How%20to:%20Add%20Columns%20to%20a%20DataTable.md).
