# <a name="warning-changes-have-been-made-to-the-configure-behavior-dialog-box-that-have-not-been-applied"></a>Предупреждение: Были внесены изменения в диалоговое окно Конфигурирование поведения, которые еще не применены

Предупреждение. В диалоговом окне «Настройка режима» внесены изменения, которые еще не применены. Применить изменения?

**Конфигурирование поведения** диалоговое окно позволяет настроить поведение инструкции Insert, Update и Delete для всех имеющихся классов. Это сообщение появляется при выборе новой **класса** и **поведение** сочетание и предыдущее изменение имеет еще не применена.

## <a name="to-apply-the-change-and-continue"></a>Чтобы применить изменение и продолжить

- Нажмите кнопку **Да** для применения изменений к выбранному **класса** и **поведение**.

## <a name="to-cancel-the-previous-change-and-continue"></a>Чтобы отменить предыдущее изменение и продолжить

- Нажмите кнопку **нет** отменить изменение выбранного **класса** и **поведение**.
