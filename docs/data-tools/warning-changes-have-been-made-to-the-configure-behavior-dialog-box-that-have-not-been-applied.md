# Предупреждение. В диалоговом окне &#171;Настройка режима&#187; внесены изменения, которые еще не применены
Предупреждение.В диалоговом окне «Настройка режима» внесены изменения, которые еще не применены.Применить изменения?  
  
 Диалоговое окно **Конфигурирование поведения** позволяет конфигурировать поведение операций Вставить, Обновить или Удалить для всех имеющихся классов.Это сообщение появляется при выборе нового сочетания **Класса** и **Поведения**, когда предыдущее изменение еще не было применено.  
  
### Чтобы применить изменение и продолжить  
  
-   Нажмите кнопку **Да**, чтобы применить изменение к выбранному **Классу** и **Поведению**.  
  
### Чтобы отменить предыдущее изменение и продолжить  
  
-   Нажмите кнопку **Нет**, чтобы отменить изменение выбранного **Класса** и **Поведения**.
