# Практическое руководство. Открытие окна &quot;Источники данных&quot;
В окне **Источники данных** отображаются источники данных в проекте.  Можно использовать окно **Источники данных** для создания интерфейса пользователя \(состоящего из элементов управления с привязкой данных\) путем перетаскивания элементов из окна на форму в проекте.  См. раздел [Привязка элементов управления Windows Forms к данным в Visual Studio](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md).  
  
 Можно заполнить окно **Источники данных**, запустив **Мастер конфигурации источников данных**.  Дополнительные сведения см. в разделе [мастер настройки источника данных](../data-tools/media/data-source-configuration-wizard.png).  
  
### Открытие окна "Источники данных"  
  
-   В строке меню выберите **Вид**, **Другие окна**, **Источники данных** \(или нажмите клавиши Shift\+Alt\+D\).  
  
    > [!IMPORTANT]
    >  Можно добавить источник данных к проекту, только если он поддерживает создание источников данных и работу с ними.  Например, невозможно открыть окно **Источники данных** в проекте приложения Windows для Магазина.
