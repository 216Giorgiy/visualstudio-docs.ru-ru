# <a name="saving-data"></a>Сохранение данных
Сохранение данных — процесс переноса измененных данных в модели данных приложения обратно в исходное хранилище данных, обычно реляционной базе данных таких как SQL Server.

 Обновление источника данных посредством модели данных обычно состоит из двух этапов. Первым шагом является обновление модели данных с новыми данными — новых записей, измененных или удаленных записей. Второй шаг — чтобы сохранить изменения в модели данных в базе данных.
