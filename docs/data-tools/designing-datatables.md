# Разработка объектов DataTable
[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] предоставляет средства разработки для создания и редактирования таблиц данных \(**DataTable**\).  Общие сведения о том, что такое **DataTable**, см. в разделе [DataTables](../Topic/DataTables.md).  
  
 В следующих разделах объясняется, как создать таблицу данных, добавить ее в типизированные наборы данных с помощью **Конструктора наборов данных**, создать и изменить столбцы данных \(**DataColumn**\), определяющие их.  
  
> [!NOTE]
>  Отображаемые диалоговые окна и команды меню могут отличаться от описанных в справке в зависимости от текущих настроек или выпуска.  Чтобы изменить параметры, в меню **Сервис** выберите команду **Импорт и экспорт параметров**.  Дополнительные сведения см. в разделе [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ru-ru/22c4debb-4e31-47a8-8f19-16f328d7dcd3).

## Ссылка  
 <xref:System.Data.DataSet>  
 Представляет расположенный в памяти кэш данных.  
  
 <xref:System.Data.DataTable>  
 Представляет одну таблицу с данными в памяти.  
  
 <xref:System.Data.DataColumn>  
 Представляет схему столбца в <xref:System.Data.DataTable>.  
  
 <xref:System.Data.DataRelation>  
 Представляет отношение "родительский\/дочерний объект" между двумя объектами <xref:System.Data.DataTable>.
