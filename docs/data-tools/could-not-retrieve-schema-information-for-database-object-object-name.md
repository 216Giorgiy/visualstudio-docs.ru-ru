# Не удалось получить схему объекта базы данных &lt;имя объекта&gt;
Данное сообщение обычно появляется, когда объект в окне **Обозреватель серверов\/Обозреватель баз данных** копируется в буфер обмена, удаляется из базы данных, а затем вставляется в конструктор.Поскольку объект базы данных больше не существует, появляется это сообщение.  
  
### Исправление этой ошибки  
  
-   Нажмите кнопку **OK**, чтобы закрыть диалоговое окно.
