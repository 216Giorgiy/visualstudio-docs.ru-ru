# Справочные материалы по неуправляемым API (разработка решений Office в Visual Studio)
  Начиная с системы Microsoft Office 2007 приложения Office используют интерфейс [Интерфейс IManagedAddin](../vsto/imanagedaddin-interface.md) для вызова компонента загрузчика надстроек VSTO, который входит в состав среды выполнения [!INCLUDE[vsto_runtime](../vsto/includes/vsto-runtime-md.md)]. Этот компонент используется для загрузки управляемых надстроек VSTO. Вы можете создать собственный компонент загрузчика надстроек VSTO, реализовав этот интерфейс.  
  
## В этом подразделе  
 [Интерфейс IManagedAddin](../vsto/imanagedaddin-interface.md)  
 COM\-интерфейс, который можно реализовать для загрузки и выгрузки управляемых надстроек VSTO в приложениях Office.
