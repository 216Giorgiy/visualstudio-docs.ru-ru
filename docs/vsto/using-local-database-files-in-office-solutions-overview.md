# Общие сведения об использовании файлов локальной базы данных в решениях для Office
  Можно включить файл базы данных, например файл SQL Server Express \(mdf\) или Microsoft Office access \(mdb\) в решении office.  Это позволит конечному пользователю поддерживать работу локальной базы без подключения к централизованной базе данных, как, например, в решении локальных данных инвентаризации, которые осуществляются только на одном компьютере.  
  
 [!INCLUDE[appliesto_all](../vsto/includes/appliesto-all-md.md)]  
  
## Импортирование файла базы данных в проект  
 Для того чтобы импортировать файлы в свой проект, используйте функцию **Мастер настройки источника данных** для создания источника данных в файле базы данных.  Мастер добавит файл базы данных и типизированный набор данных в проект.  
  
## Развертывание файла базы данных  
 При создании соединения с файлом локальной базы данных **Мастер настройки источника данных** использует относительный путь.  Это позволяет копировать решение с одного компьютера на другой, если при этом поддерживается относительное расположение файлов по отношению друг к другу.  
  
 При развертывании решения на сервере и распространении документа среди конечных пользователей необходимо также вручную распространить файл базы данных и установить его в тот же каталог, по отношению к документу.  Это означает, что конечный пользователь не может перемещать документ на новое место расположения в своем компьютере, не переместив сам файл базы данных.  
  
## Файлы локальной базы данных и кэширование набора данных  
 В решениях на уровне документов Microsoft Office Excel и Microsoft Office Word набор данных добавляется в кэш\-память документа с помощью создания элементов набора данных и атрибутов <xref:Microsoft.VisualStudio.Tools.Applications.Runtime.CachedAttribute>.  После добавления файла базы данных в проект с помощью **Мастера настройки источника данных** типизированный набор данных добавляется в проект автоматически.  Набор данных редко требует применения приложения<xref:Microsoft.VisualStudio.Tools.Applications.Runtime.CachedAttribute>, поскольку данные являются локальными для компьютера пользователя.  Дополнительные сведения см. в разделе [Кэширование данных](../vsto/caching-data.md).
