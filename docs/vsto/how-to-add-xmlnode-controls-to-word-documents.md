# Практическое руководство. Добавление элементов управления XMLNode в документы Word
  **Важным** набор сведений в этом разделе, касающиеся Microsoft Word исключительно для использования преимущества и цене и организаций, найдены вне Соединенные Штаты и его территорий или используют или разработки программ, запущенных на продуктов Microsoft Word, которые были лицензированы Майкрософт до января 2010, когда выполненного реализация конкретной функциональности Майкрософт, связанное с пользовательским XML в Microsoft Word.  Данная информация, относящаяся к Microsoft Word, не предназначена для чтения и использования лицами или организациями, расположенными в США или их территориях, которые используют продукты Microsoft Word, лицензированные корпорацией Майкрософт после 10 января 2010 г., или разрабатывают программы для этих продуктов; поведение данных продуктов отличается от поведения продуктов, лицензированных до указанной даты или приобретенных и лицензированных за пределами США.  
  
 [!INCLUDE[appliesto_wdalldoc](../vsto/includes/appliesto-wdalldoc-md.md)]  
  
 При сопоставлении неповторяющегося элемента схемы XML с документом Microsoft Office Word Visual Studio автоматически добавляет в документ элемент управления <xref:Microsoft.Office.Tools.Word.XMLNode>.  Дополнительные сведения о сопоставлении повторяющихся элементов схем XML см. в разделе [Практическое руководство. Добавление элементов управления XMLNodes в документы Word](../vsto/how-to-add-xmlnodes-controls-to-word-documents.md).  
  
> [!NOTE]  
>  Элемент управления <xref:Microsoft.Office.Tools.Word.XMLNode> недоступен в **Панели элементов** или в окне **Источники данных** и не может быть создан программными средствами.  
  
 [!INCLUDE[note_settings_general](../sharepoint/includes/note-settings-general-md.md)]  
  
### Добавление в документ элемента управления XMLNode  
  
1.  В документе, размещенном в конструкторе Visual Studio, выберите на ленте вкладку **Разработчик**.  
  
    > [!NOTE]  
    >  Если вкладка **Разработчик** не отображается в ленте, то ее следует сначала отобразить.  Дополнительные сведения см. в разделе [Практическое руководство. Отображение вкладки разработчика на ленте](../vsto/how-to-show-the-developer-tab-on-the-ribbon.md).  
  
2.  В группе **XML** нажмите кнопку **Схема**.  
  
     Появится диалоговое окно **Шаблоны и надстройки**.  
  
3.  Перейдите на вкладку **Схема XML**.  
  
4.  Откройте вкладку **Добавить схему**.  
  
     Появится диалоговое окно **Добавить схему**.  
  
5.  В диалоговом окне **Добавление схемы** выберите XML\-схему, которая содержит неповторяющиеся элементы схемы, и нажмите кнопку **Открыть**.  
  
     Открывается диалоговое окно **Параметры схемы**.  
  
6.  Укажите псевдоним или нажмите кнопку **OK**, чтобы добавить схему без назначения псевдонима.  
  
     Схема добавляется в диалоговое окно **Добавление схемы**.  
  
7.  В диалоговом окне **Добавление схемы** нажмите кнопку **ОК**.  
  
8.  Открывается область задач **Структура ХML**.  
  
9. Щелкните неповторяющийся элемент схемы в области задач **Структура ХML**, чтобы добавить его в документ.  
  
     Элемент управления <xref:Microsoft.Office.Tools.Word.XMLNode> создается и добавляется в проект.
