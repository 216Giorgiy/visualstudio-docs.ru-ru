# <a name="one-or-more-properties-in-the-ofs-file-are-not-valid-for-the-message-class-selected"></a>Одно или несколько свойств OFS-файла являются недопустимыми для выбранного класса сообщений
  Эта ошибка возникает при импорте области формы, разработанной в Outlook, но одно или несколько полей в области формы несовместимы с классами сообщений, выбранными на последней странице **новая область формы** мастера.  

Например, вы могли выбрать тип **Задача (IPM.Task)** на последней странице мастера **Создать область формы** . Если область формы, имеет **рабочий адрес** поле, вы получите эту ошибку, так как у задачи нет рабочего адреса. Таким образом **рабочий адрес** несовместимо с `IPM.Task` класс message.  
  
 Можно выбрать **задачи (Начальник. Задача)** на последней странице **новая область формы** мастера. Если область формы, имеет **рабочий адрес** поле, вы получите эту ошибку, так как у задачи нет рабочего адреса. Таким образом **рабочий адрес** несовместимо с `IPM.Task` класс message.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   На последней странице мастера **Создать область формы** выберите класс сообщений, который совместим с полями в области формы.  
  
-   В конструкторе форм Outlook удалите поля, несовместимые с классами сообщений. Удаление полей, которые вы планируете выбрать на последней странице **новая область формы** мастера.
