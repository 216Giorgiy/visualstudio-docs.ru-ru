# Общие сведения об управлении правами на доступ к данным и расширениях управляемого кода
  Microsoft Office Word и Microsoft Office Excel предоставляют функцию управления правами на доступ к данным, позволяющую предотвратить возможность просмотра и изменения важной информации несанкционированными пользователями.  Подробнее про работу функции управления правами на доступ к данным см. в специальном приложении Office "Справка".  
  
 [!INCLUDE[appliesto_alldoc](../vsto/includes/appliesto-alldoc-md.md)]  
  
## Выполнение кода программной части документов с ограниченными разрешениями  
 Если решение содержит документ или книгу, по умолчанию использующую IRM, Word и Excel не допускают выполнение кода.  Если вы являетесь автором документа или разрешениями "Полный доступ" на него, то вы можете изменить значения по умолчанию, чтобы приведет к выполнению решения.  Дополнительные сведения см. в разделе [Практическое руководство. Выполнение с выделенным кодом документов с ограниченными разрешениями](../vsto/how-to-permit-code-to-run-behind-documents-with-restricted-permissions.md).  
  
 IRM не позволяет <xref:Microsoft.VisualStudio.Tools.Applications.Runtime.ServerDocument> извлекать или управлять данными, которые кэшируются в документе.  
  
## Ограничения доступа пользователей к документам, использующим расширения управляемого кода  
 Любой пользователь, обладающий разрешениями "Полный доступ" на документ или книгу в решении, может использовать управление правами на доступ к данным \(IRM\) для ограничения разрешений.  Например, если работник бухгалтерии использует решение для автоматического заполнения книги данными из базы, он может предоставить доступ к изменению документов только своему отделу и разрешить доступ в режиме "Только чтение" остальным пользователям.  Если пользователь добавляет ограничение разрешений по умолчанию, лист с выделенным кодом не будет выполняться и соответственно заполняться данными.  
  
 Чтобы устранить эту проблему, пользователь, обладающий правами полного доступа к документу или книге, должен изменить настройки разрешения для активации программного доступа к модели объекта.  Дополнительные сведения см. в разделе [Практическое руководство. Выполнение с выделенным кодом документов с ограниченными разрешениями](../vsto/how-to-permit-code-to-run-behind-documents-with-restricted-permissions.md).
