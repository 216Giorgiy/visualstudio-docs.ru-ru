# Чтобы получить возможность создания или открытия проекта Visual Studio Tools для системы Microsoft Office, необходимо явно открыть доступ к системе проектов Microsoft Office Visual Basic for Applications
  Проектам по разработке программного обеспечения Office требуется доступ к системе проектов Visual Basic для приложений в Microsoft Office Word и Microsoft Office Excel, даже если эти проекты не используют Visual Basic для приложений.  Поддержка элементов управления времени разработки в проектах Visual Basic и C\# зависит от системы проектов Visual Basic для приложений.  
  
 Некоторые макровирусы Microsoft Office пытаются автоматизировать систему проектов Visual Basic для приложений как способ своего распространения.  Разрешая доступ к системе проектов Visual Basic для приложений, вы отключаете средство защиты, которое позволяет предотвратить распространение макровирусов.  Тем не менее, остаются стандартные средства обеспечения безопасности макросов: уровень безопасности макросов и список надежных издателей, поддерживаемый для приложений Office, помогут выявить выполнение макросов на компьютере.  
  
> [!NOTE]  
>  Это касается только компьютера разработки.  Компьютеры конечных пользователей не требуют включения этой возможности для выполнения решений Office.  
  
 Важно отметить, что отключение доступа к системе проектов Visual Basic для приложений само по себе не защитит вас от вирусов, оно просто помогает предотвратить распространение некоторых вирусов в другие документы в случае заражения компьютера макровирусом.  Эта возможность отключена по умолчанию, что обеспечивает дополнительный уровень защиты компьютера. Однако ее включение не делает ваш компьютер более подверженным атакам, если вы выполняете рекомендации по обеспечению безопасности.  
  
 Лучший способ защиты от макровирусов Office — выполнение приложений Office с высоким или очень высоким уровнем безопасности. В этом случае используются макросы только из проверенных и известных источников, применяются последние исправления безопасности и обновления средств поиска вирусов.  
  
 Дополнительные сведения о защите компьютера от вирусов и другого вредоносного кода см. на веб\-сайте [http:\/\/www.microsoft.com\/protect](http://www.microsoft.com/protect).  
  
 Параметр **Доверять доступ к Visual Basic Project** можно включить и отключить вручную.  
  
 При появлении ошибок VBA или COM можно восстановить установку Office.  
  
### Включение и отключение доступа к проектам Visual Basic  
  
1.  В меню **Сервис** в Word или Excel наведите указатель на пункт **Макрос** и щелкните **Безопасность**.  
  
2.  В диалоговом окне **Безопасность** перейдите на вкладку **Надежные издатели**.  
  
3.  Установите или снимите флажок **Доверять доступ к Visual Basic Project**.  
  
4.  Нажмите кнопку **ОК**.  
  
### Настройка уровня безопасности макросов в Office  
  
1.  В меню **Сервис** в Word или Excel наведите указатель на пункт **Макрос** и щелкните **Безопасность**.  
  
2.  На вкладке **Уровень безопасности** выберите нужную настройку.  
  
     Вкладка **Уровень безопасности** содержит сведения о каждом уровне.  Дополнительные сведения см. в разделе "Уровни безопасности макросов" справки Microsoft Office.  
  
### Установка VBA в выпуске 2007 системы Microsoft Office  
  
1.  На панели управления выберите пункт **Добавление и удаление программ** или **Программы и компоненты**.  
  
2.  В списке **Установленные программы** выберите Office.  
  
3.  Нажмите кнопку **Изменить**.  
  
4.  Выберите **Добавить или удалить компоненты** и нажмите кнопку **Продолжить**.  
  
5.  Установите флажок **Выбрать расширенную настройку приложений** и нажмите кнопку **Далее**.  
  
6.  Разверните узел **Общие средства Office** в списке **Выбор параметров обновления для приложений и средств**.  
  
7.  Откройте раскрывающееся меню в поле **Visual Basic для приложений** и выберите **Запускать с моего компьютера**.  
  
8.  Нажмите кнопку **Продолжить**.  
  
9. Нажмите кнопку **Закрыть**.  
  
### Восстановление установки Office  
  
1.  На панели управления выберите пункт **Добавление и удаление программ** или **Программы и компоненты**.  
  
2.  В списке **Установленные программы** выберите нужную версию Office.  
  
3.  Нажмите кнопку **Изменить**.  
  
4.  Выберите **Переустановка или восстановление** и нажмите кнопку **Далее**.  
  
5.  Установите флажок **Исправить ошибки в установке Office** и нажмите кнопку **Установить**.
