# <a name="understand-coauthoring-and-add-ins"></a>Сведения о совместной работе над книгой и Add-ins

Соавторство — это функция Excel для Windows desktop, который позволяет редактировать книги, размещенной в облаке (то есть, OneDrive, OneDrive для бизнеса или SharePoint Online) одновременно с другими пользователями. С появлением этой функции может потребоваться внести изменения для решения Excel, чтобы гарантировать, что оно работает без проблем даже в том случае, когда пользователи в совместном редактировании. См. в разделе [о совместной работе над книгой в Excel](/office/vba/excel/concepts/about-coauthoring-in-excel).

Примечание. Соавторство для Windows Desktop Excel была введена в 2017 г. и сейчас доступна для подписчиков Office 365. Пользователи, которые приобрели бессрочную лицензию для Office 2016 или более ранней версии в настоящее время нет доступа к компоненту совместного редактирования. (Excel Online, Excel для Android, Excel для iOS и Excel Mobile в Windows Store также поддерживает совместного редактирования).
