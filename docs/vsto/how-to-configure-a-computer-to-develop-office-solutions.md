# Практическое руководство. Настройка компьютера для разработки решений Office
  Для настройки компьютера разработчика таким образом, чтобы было можно использовать Microsoft Office Developer Tools в Visual Studio, следуйте инструкциям в этой статье.  Для выполнения следующих действий требуются права администратора на компьютере разработчика.  
  
### Настройка компьютера разработчика  
  
1.  Установите версию Visual Studio, которая содержит Office Developer Tools.  Office Developer Tools устанавливаются по умолчанию.  Если во время установки Visual Studio вы указываете, какие функции нужно установить, обязательно выберите пункт **Microsoft Office Developer Tools**. Дополнительные сведения о версиях Visual Studio, которые содержат Office Developer Tools, см. в статье [Настройка компьютера для разработки решений Office](../vsto/configuring-a-computer-to-develop-office-solutions.md).  
  
2.  Установите версию Office, которую поддерживают Office Developer Tools в Visual Studio.  Дополнительные сведения см. в статье [Настройка компьютера для разработки решений Office](../vsto/configuring-a-computer-to-develop-office-solutions.md).  
  
     Убедитесь, что вы также устанавливаете основные сборки взаимодействия для нужной версии Office.  Основные сборки взаимодействия устанавливаются вместе с Office по умолчанию.  Если вы изменяете настройки установки Office, убедитесь, что компонент **Поддержка программирования .NET** выбран для тех приложений, с которыми вы будете работать.  
  
3.  Если у вас есть англоязычная версия Visual Studio, но вы используете неанглоязычные параметры для Windows, можно установить языковой пакет [!INCLUDE[vsto_runtime](../vsto/includes/vsto-runtime-md.md)], что позволит видеть сообщения [!INCLUDE[vsto_runtime](../vsto/includes/vsto-runtime-md.md)] на том же языке, что и используется в Windows.  Неанглоязычные версии Visual Studio автоматически устанавливают языковой пакет.  Языковой пакет можно скачать из [Центра загрузки Майкрософт](http://go.microsoft.com/fwlink/?LinkId=140386).
