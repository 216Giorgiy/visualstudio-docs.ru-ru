# <a name="web-projects"></a>Веб-проекты
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Веб-проекта системы — это система, создающий веб-проектов. Веб-проектов, в свою очередь, создайте веб-приложений. Вы можете программировать с помощью традиционных языках программирования, таких как веб-страницы [!INCLUDE[vbprvb](../../includes/vbprvb-md.md)] или [!INCLUDE[csprcs](../../includes/csprcs-md.md)], для сбора и обработки информации от пользователя, сохранить его в базу данных и так далее. Веб-страницы с соответствующий код, называются смарт-веб-страниц.
