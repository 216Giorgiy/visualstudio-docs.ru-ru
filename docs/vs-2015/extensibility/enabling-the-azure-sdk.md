# <a name="enabling-the-azure-sdk"></a>Включение в пакет Azure SDK
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Azure SDK для .NET позволяет создавать, настраивать, построения, отладки, запуска и упаковки масштабируемых веб-приложений и служб в Azure.  
  
 Можно скачать пакет Azure SDK из **новый проект** диалоговое окно.  
  
 [!INCLUDE[note_settings_general](../includes/note-settings-general-md.md)]  
  
### <a name="to-enable-the-azure-sdk"></a>Для включения пакета SDK для Azure  
  
1. На **файл** меню, создайте новый проект. (Щелкните **New** и нажмите кнопку **проекта**.)  
  
2. Разверните узел для либо **Visual Basic** или **C#** и нажмите кнопку **Cloud**.  
  
3. В средней области выберите **получить пакет Azure SDK для .NET**, а затем нажмите кнопку **ОК**.  
  
     Следуйте инструкциям установщика.  
  
    > [!NOTE]
    >  Чтобы установить инструменты Azure, необходимо иметь Internet Information Services 7.0 и установлены компоненты разработки приложений ASP.NET. Если эти компоненты не установлены, программа установки средства Azure предложит установить их.
