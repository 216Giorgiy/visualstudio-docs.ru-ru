# <a name="debugger-components"></a>Компоненты отладчика
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

[!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] Отладчик реализуется в виде пакета VSPackage и управление сеансом отладки всего. Сеанс отладки включает в себя следующие элементы:  
  
-   **Пакет отладки:** [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] отладчик предоставляет тот же интерфейс пользователя, независимо от того, что выполняется отладка.  
  
-   **Диспетчер отладки сеансов (SDM):** предоставляет согласованный программный интерфейс для [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] отладчика для управления широкий набор ядер отладки. Она реализуется [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)].  
  
-   **Диспетчер отладки процессов (PDM):** управляет для всех выполняющихся экземплярах [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)], список всех программ, которые могут быть или выполняется отладка. Она реализуется [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)].  
  
-   **Отладка ядра (DE):** отвечает за мониторинг отлаживаемой программы взаимодействия состоянием выполняющейся программе SDM и PDM и взаимодействия со средством оценки выражений и поставщик символов для анализа в режиме реального времени из состояние памяти и переменные программы. Она реализуется [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] (для языков, поддерживаемых) и сторонних поставщиков, которые требуется поддерживать собственные время выполнения.  
  
-   **Средство оценки выражений (EE):** обеспечивает поддержку динамической оценки переменных и выражений, предоставленные пользователем, при остановке программы в определенной точке. Она реализуется [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] (для языков, поддерживаемых) и сторонних поставщиков, которые требуется поддерживать собственные языки.  
  
-   **Поставщик символов (SP):** также вызывается обработчик символов, сопоставляет символы отладки программы к запущенному экземпляру программы, можно предоставить полезные сведения (например, исходный код уровня отладки и оценки выражений). Она реализуется [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] (для среда CLR [со средой CLR] символов и базы данных программы [PDB] символов формата файла) и сторонних поставщиков, имеющих собственные собственный метод хранения отладочную информацию.
