# <a name="idebugengine3setallexceptions"></a>IDebugEngine3::SetAllExceptions
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [IDebugEngine3::SetAllExceptions](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugengine3-setallexceptions).  
  
Этот метод задает состояние всех необработанных исключений.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT SetAllExceptions(  
   EXCEPTION_STATE dwState  
);  
```  
  
```csharp  
int SetAllExceptions(  
   enum_EXCEPTION_STATE dwState  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `dwState`  
 [in] Один из [EXCEPTION_STATE](../../../extensibility/debugger/reference/exception-state.md) значения.  
  
## <a name="return-value"></a>Возвращаемое значение  
 В случае успешного выполнения возвращает `S_OK`; в противном случае возвращает код ошибки.
