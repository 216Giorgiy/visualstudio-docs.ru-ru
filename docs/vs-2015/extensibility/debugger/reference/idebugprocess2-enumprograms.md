# <a name="idebugprocess2enumprograms"></a>IDebugProcess2::EnumPrograms
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [IDebugProcess2::EnumPrograms](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugprocess2-enumprograms).  
  
Возвращает список всех программ, входящих в процессе.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp#  
HRESULT EnumPrograms(   
   IEnumDebugPrograms2** ppEnum  
);  
```  
  
```csharp  
int EnumPrograms(   
   out IEnumDebugPrograms2 ppEnum  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `ppEnum`  
 [out] Возвращает [IEnumDebugPrograms2](../../../extensibility/debugger/reference/ienumdebugprograms2.md) , содержащий список всех программ в процессе.  
  
## <a name="return-value"></a>Возвращаемое значение  
 В случае успешного выполнения возвращает `S_OK`; в противном случае возвращает код ошибки.
