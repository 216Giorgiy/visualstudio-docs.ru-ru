# <a name="idebugcoreserver2enumports"></a>IDebugCoreServer2::EnumPorts
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [IDebugCoreServer2::EnumPorts](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugcoreserver2-enumports).  
  
Возвращает список всех доступных портов.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp#  
HRESULT EnumPorts(   
   IEnumDebugPorts2** ppEnum  
);  
```  
  
```csharp  
int EnumPorts(   
   out IEnumDebugPorts2 ppEnum  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `ppEnum`  
 [out] Возвращает [IEnumDebugPorts2](../../../extensibility/debugger/reference/ienumdebugports2.md) , содержащий список всех портов из всех поставщиков порт.  
  
## <a name="return-value"></a>Возвращаемое значение  
 В случае успешного выполнения возвращает `S_OK`; в противном случае возвращает код ошибки.
