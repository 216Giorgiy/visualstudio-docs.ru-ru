# <a name="idebugdocumenttextevents2ondestroy"></a>IDebugDocumentTextEvents2::onDestroy
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [IDebugDocumentTextEvents2::onDestroy](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugdocumenttextevents2-ondestroy).  
  
Указывает, что весь документ был удален.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp#  
HRESULT onDestroy(   
   void   
);  
```  
  
```csharp  
int onDestroy();  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 В случае успешного выполнения возвращает `S_OK`; в противном случае возвращает код ошибки.
