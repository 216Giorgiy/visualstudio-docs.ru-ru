# <a name="namematch"></a>NAME_MATCH
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [NAME_MATCH](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/name-match).  
  
Выбирает параметр case для сопоставления имен.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp#  
typedef enum {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
} NAME_MATCH;  
```  
  
```csharp  
public enum NameMatchOptions {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
}  
```  
  
## <a name="members"></a>Участники  
 nmNone  
 Параметры не указаны.  
  
 nmCaseSensitive  
 Указывает, что имена для сравнения с учетом регистра.  
  
 nmCaseInsensitive  
 Указывает, что имена для сопоставления не учитывается регистр.
