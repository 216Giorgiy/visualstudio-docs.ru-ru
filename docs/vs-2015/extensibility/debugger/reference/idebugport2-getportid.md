# <a name="idebugport2getportid"></a>IDebugPort2::GetPortId
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [IDebugPort2::GetPortId](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugport2-getportid).  
  
Получает идентификатор порта.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp#  
HRESULT GetPortId(   
   GUID* pguidPort  
);  
```  
  
```csharp  
int GetPortId(   
   out Guid pguidPort  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `pguidPort`  
 [out] Возвращает идентификатор GUID, определяющий порт.  
  
## <a name="return-value"></a>Возвращаемое значение  
 В случае успешного выполнения возвращает `S_OK`; в противном случае возвращает код ошибки.
