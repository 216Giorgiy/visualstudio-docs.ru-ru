# <a name="troubleshooting-clickonce-deployments"></a>Устранение неполадок развертывания ClickOnce

Этот раздел поможет вам диагностировать и устранить наиболее распространенные проблемы с [!INCLUDE[ndptecclick](../includes/ndptecclick-md.md)] развертываний.  
  
 В большинстве случаев [!INCLUDE[ndptecclick](../includes/ndptecclick-md.md)] загрузить на компьютер пользователя и выполнения без проблем приложения. Существуют случаи, тем не менее, где веб-сервер или приложение направляет конфигурации может привести к непредвиденным проблемам.
