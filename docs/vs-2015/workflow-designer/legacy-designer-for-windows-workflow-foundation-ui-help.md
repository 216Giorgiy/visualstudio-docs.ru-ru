Используйте конструктор прежних версий, если приложение ориентируется на [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] или [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)].  
  
 Вы можете открыть эти разделы [!INCLUDE[vs_current_long](../includes/vs-current-long-md.md)] пользовательского интерфейса, нажав клавишу **F1** при прежних версий [!INCLUDE[wfd2](../includes/wfd2-md.md)] диалоговое окно открыто.
