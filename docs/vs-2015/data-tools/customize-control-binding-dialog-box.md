# <a name="customize-control-binding-dialog-box"></a>Диалоговое окно "Настройка привязки элементов управления"
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Используйте **Настройка привязки элементов управления** диалоговое окно, чтобы указать, какие элементы управления доступны для элементов в [окна "Источники данных"](http://msdn.microsoft.com/library/0d20f699-cc95-45b3-8ecb-c7edf1f67992).  
  
 Каждый элемент в **источников данных** окна имеется список элементов управления, которые могут быть привязаны к элементу. Элементы управления, доступные для каждого элемента определяется типом данных элемента. Каждый тип данных имеет список допустимых элементов управления, определенных в этом диалоговом окне, включая элемент управления по умолчанию.  
  
 Прежде чем перетащить элемент в рабочую область конструирования для создания элемента управления с привязкой данных, можно выбрать, какой элемент управления для создания. Если перетащить элемент из **источников данных** окно в область конструктора, не устанавливая флажок элемента управления по умолчанию для типа данных выбранного элемента, добавленного в область конструктора.  
  
 Дополнительные сведения о том, как использовать это диалоговое окно для настройки элементов управления для элементов в списке **источников данных** окно, см. в разделе [добавлять пользовательские элементы управления окна "Источники данных"](../data-tools/add-custom-controls-to-the-data-sources-window.md).  
  
## <a name="uielement-list"></a>Список элементов пользовательского интерфейса  
 **Тип данных**  
 Отображает список типов, которые связаны с элементами управления:  
  
- Таблицы, сущности и объекты отображаются в виде **[список]** типов.  
  
- Столбцы или общие свойства сущностей и объектов отображаются в виде фактических данных тип столбца или свойства в базовом хранилище данных.  
  
- Отображаются в виде объектов с пользовательскими типами **[Other]**. Например, если приложение содержит пользовательский элемент управления, отображающий данные из более чем одного свойства объекта, выберите **[Other]** тип данных для элемента управления.  
  
  **Связанные элементы управления**  
  Отображает список элементов управления, которые можно сопоставить с типом данных. Если вы хотите связать определенный элемент управления с типом данных, выбранным в **тип данных** выберите соответствующий флажок. Снимите флажок, чтобы удалить связь. Флажок установлен, элементы управления отображаются в контекстном меню, открывающемся по **источников данных** окна для элемента, связанного типа данных.  
  
  Элементы управления можно добавить в список путем добавления элементов управления с одним из нескольких атрибутов привязки данных в **элементов**. Дополнительные сведения см. в разделе [добавлять пользовательские элементы управления окна "Источники данных"](../data-tools/add-custom-controls-to-the-data-sources-window.md).  
  
  **Значение по умолчанию**  
  Устанавливает тип выбранного элемента управления по умолчанию для элементов выбранного типа данных. По умолчанию элемент управления отображается как первый выбранный пункт в контекстном меню, открывающемся по **источников данных** окна для элемента. Можно назначить только один тип элементов управления по умолчанию для типа данных.  
  
  **Очистить значение по умолчанию**  
  Удаляет назначение пространства имен элемента управления по умолчанию для выбранного типа данных. Если по умолчанию для выбранного типа данных, отсутствует **[нет]** отображается как первый выбранный пункт в контекстном меню, открывающемся по **источников данных** окна для элемента, соответствующего типа.
