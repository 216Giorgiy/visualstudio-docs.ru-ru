# <a name="saving-entity-data"></a>Сохранение данных сущностей
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

|Раздел|Описание|  
|-----------|-----------------|  
|[Практическое руководство. Добавление, изменение и удаление сущностей](http://msdn.microsoft.com/library/a00f8933-b232-4445-95ba-adc634f055d8)|Показано, как сохранить изменения данных сущности обратно через службу данных.|  
|[Практическое руководство. Выполнение асинхронных запросов к службе данных](http://msdn.microsoft.com/library/902a2dc1-d0e9-4b00-90a8-becc4cb1f6a7)|Показано, как асинхронное сохранение изменений через службу данных.|
