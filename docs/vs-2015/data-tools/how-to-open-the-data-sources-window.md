# <a name="how-to-open-the-data-sources-window"></a>Практическое руководство. Открытие окна "Источники данных"
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**Источников данных** окна отображаются источники данных в проекте. Использовании **источников данных** окно для создания пользовательского интерфейса элементов управления с привязкой к данным путем перетаскивания элементов из окна на форму в проекте. См. в разделе [управляет привязки Windows Forms к данным в Visual Studio](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md).  
  
 Можно заполнить **источников данных** окна, выполнив **конфигурации источника данных** мастера. Дополнительные сведения см. в разделе [мастер настройки источника данных](http://msdn.microsoft.com/library/c4df7de5-5da0-4064-940c-761dd6d9e28f).  
  
### <a name="to-open-the-data-sources-window"></a>Открытие окна "Источники данных"  
  
-   В строке меню выберите **представление**, **Other Windows**, **источников данных** (или нажмите сочетание клавиш Shift + Alt + D).  
  
    > [!IMPORTANT]
    >  Можно добавить источник данных к проекту, только если он поддерживает создание источников данных и работу с ними. Например, не удается открыть **источников данных** окна в проекте приложения Windows Store.
