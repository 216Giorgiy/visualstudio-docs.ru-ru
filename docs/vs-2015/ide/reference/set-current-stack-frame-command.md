# <a name="set-current-stack-frame-command"></a>Команда Set Current Stack Frame
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [задать текущую команду кадра стека](https://docs.microsoft.com/visualstudio/ide/reference/set-current-stack-frame-command).  
  
  
Позволяет задать определенный кадр стека.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
Debug.SetCurrentStackFrame index  
```  
  
## <a name="arguments"></a>Аргументы  
 `index`  
 Обязательно. Выбирает кадр стека по индексу.  
  
## <a name="example"></a>Пример  
  
```  
>Debug.SetCurrentStackFrame 1  
```
