# <a name="import-and-export-settings-environment-options-dialog-box"></a>Страница "Параметры импорта и экспорта", папка "Среда", диалоговое окно "Параметры"
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [Импорт и экспорт параметров, среда, диалоговое окно параметров](https://docs.microsoft.com/visualstudio/ide/reference/import-and-export-settings-environment-options-dialog-box).  
  
  
Эта страница служит для задания места сохранения файла параметров пользователя и указания необходимости использования файла параметров команды. Дополнительные сведения о параметрах см. в разделе [Настройка параметров разработки в Visual Studio](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3).
