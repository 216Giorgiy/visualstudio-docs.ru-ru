# <a name="retain-topics-for-later-use"></a>Сохранение разделов для дальнейшего использования
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [Сохранение разделов для последующего использования](https://docs.microsoft.com/visualstudio/ide/retain-topics-for-later-use).  
  
Полезные статьи можно сохранить, скопировав и вставив их в текстовый файл, распечатав или добавив в избранное. Например, можно сохранить материалы справки на будущее, открыв контекстное меню для соответствующей темы и выбрав команду **Копировать**, **Печать**, **Предварительный просмотр** или **Добавить в избранное**.  
  
## <a name="using-the-favorites-tab"></a>Использование вкладки "Избранное"  
 На вкладке **Избранное** можно открыть контекстное меню для любой записи и затем переименовать или удалить эту запись либо упорядочить записи по папкам.
