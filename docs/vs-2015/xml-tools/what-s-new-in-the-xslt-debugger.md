# <a name="what39s-new-in-the-xslt-debugger"></a>Что&#39;возможности отладчика XSLT
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Среда Visual Studio 2010 содержит следующие новые возможности.  
  
## <a name="xslt-profiler"></a>Профилировщик XSLT  
 Профилировщик XSLT - это средство, которое позволяет разработчикам измерять, оценивать и устранять в XSLT-коде проблемы, связанные с производительностью, посредством создания подробных отчетов о производительности XSLT. Дополнительные сведения см. в разделе [XSLT Profiler](../xml-tools/xslt-profiler.md).
