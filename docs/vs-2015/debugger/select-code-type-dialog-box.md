# <a name="select-code-type-dialog-box"></a>Диалоговое окно "Выбор типа кода"
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [диалоговое окно Выбор типа кода](https://docs.microsoft.com/visualstudio/debugger/select-code-type-dialog-box).  
  
Чтобы открыть это диалоговое окно, откройте **присоединение к процессу** диалоговое окно, а затем нажмите кнопку **выберите** кнопки.  
  
 **Автоматически определять тип отлаживаемого кода**  
 В зависимости от типа выполняемого кода будет выбран соответствующий отладчик.  
  
 **Отладить код следующих типов:**  
 В предоставленном списке выберите тип кода, который планируется отлаживать.
