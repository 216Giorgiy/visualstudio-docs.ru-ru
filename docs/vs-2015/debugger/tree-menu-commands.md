# <a name="tree-menu-commands"></a>Команды меню "Дерево"
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [команды меню "дерево"](https://docs.microsoft.com/visualstudio/debugger/tree-menu-commands).  
  
**Дерева** меню предоставляет команды, чтобы развернуть или свернуть представления.  
  
 **Развернуть один уровень**  
 Разворачивает выбранный элемент в системном представлении, отображая один уровень ниже его.  
  
 **Разверните ветвь**  
 Разворачивает все уровни ниже текущего выбранного элемента.  
  
 **Развернуть все**  
 Разворачивается и отображает все уровни в системном представлении.  
  
 **Свернуть**  
 Свертывает все уровни ниже текущего выбранного элемента.
