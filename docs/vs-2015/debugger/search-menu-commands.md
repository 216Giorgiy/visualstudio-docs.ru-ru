# <a name="search-menu-commands"></a>Команды меню "Поиск"
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**Поиска** меню доступны команды для поиска конкретного окна.  
  
 **Найти окно/процесс/поток/сообщение**  
 Открывает **окно**/**процесс**/**потоков**/**сообщения** диалоговое окно где можно использовать **средство поиска** или определение критериев поиска.  
  
 **Найти далее**  
 Поиск в текущем представлении для следующего соответствующего окна, процесса, потока или сообщения. Данная команда меню (и связанные кнопки) доступна только при наличии допустимых результатов поиска, не является уникальным.  
  
 **Найти ранее**  
 Поиск в текущем представлении предыдущего соответствующего окна, процесса, потока или сообщения. Данная команда меню (и связанные кнопки) доступна только при наличии допустимых результатов поиска, не является уникальным.
