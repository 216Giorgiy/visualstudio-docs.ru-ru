# <a name="idiaenumsourcefilesreset"></a>IDiaEnumSourceFiles::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [IDiaEnumSourceFiles::Reset](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenumsourcefiles-reset).  
  
Сбрасывает последовательность перечислений в начало.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp#  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Возвращает `S_OK`.
