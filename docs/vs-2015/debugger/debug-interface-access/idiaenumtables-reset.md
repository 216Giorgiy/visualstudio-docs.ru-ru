# <a name="idiaenumtablesreset"></a>IDiaEnumTables::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [IDiaEnumTables::Reset](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenumtables-reset).  
  
Сбрасывает последовательность перечислений в начало.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp#  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Возвращает `S_OK`.
