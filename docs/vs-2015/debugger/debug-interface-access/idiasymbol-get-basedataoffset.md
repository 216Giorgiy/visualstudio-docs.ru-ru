# <a name="idiasymbolgetbasedataoffset"></a>IDiaSymbol::get_baseDataOffset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [IDiaSymbol::get_baseDataOffset](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-basedataoffset).  
  
Получает смещение базовых данных.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT get_baseDataOffset(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>Параметры  
 `pRetVal`  
 [out] Указатель на `DWORD` , содержащий смещение базовых данных.  
  
## <a name="return-value"></a>Возвращаемое значение  
 В случае успешного выполнения возвращает `S_OK`; в противном случае возвращает `S_FALSE` или код ошибки.
