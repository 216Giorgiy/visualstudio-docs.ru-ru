# <a name="idiastackframegetmaxstack"></a>IDiaStackFrame::get_maxStack
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [IDiaStackFrame::get_maxStack](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiastackframe-get-maxstack).  
  
Извлекает максимальное число байтов, в стек в кадре.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp#  
HRESULT get_maxStack (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `pRetVal`  
 [out] Возвращает максимальное число байтов, в стек.  
  
## <a name="return-value"></a>Возвращаемое значение  
 В случае успешного выполнения возвращает `S_OK`. Возвращает `S_FALSE` Если свойство не поддерживается. В противном случае возвращается код ошибки.
