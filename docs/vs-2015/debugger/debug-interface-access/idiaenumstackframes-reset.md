# <a name="idiaenumstackframesreset"></a>IDiaEnumStackFrames::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [IDiaEnumStackFrames::Reset](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenumstackframes-reset).  
  
Сбрасывает последовательность перечисления в начало.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp#  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Возвращает `S_OK`.
