# <a name="idialinenumbergetaddressoffset"></a>IDiaLineNumber::get_addressOffset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [IDiaLineNumber::get_addressOffset](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idialinenumber-get-addressoffset).  
  
Извлекает часть смещения адрес памяти, где начинается блок.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp#  
HRESULT get_addressOffset (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `pRetVal`  
 [out] Возвращает часть смещения адрес памяти, где начинается блок.  
  
## <a name="return-value"></a>Возвращаемое значение  
 В случае успешного выполнения возвращает `S_OK`. Возвращает `S_FALSE` Если это свойство не поддерживается. В противном случае возвращается код ошибки.  
  
## <a name="example"></a>Пример  
  
```cpp#  
CComPtr< IDiaLineNumber > pLine;  
DWORD offset;  
pLine->get_addressOffset( &offset);  
```
