# <a name="window-menu-commands"></a>Команды меню "Окно"
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [команды меню "окно"](https://docs.microsoft.com/visualstudio/debugger/window-menu-commands).  
  
**Окно** меню доступны команды для управления окнами Spy ++.  
  
 **Cascade**  
 Упорядочение окон каскадом.  
  
 **Слева направо**  
 Упорядочение окон горизонтально.  
  
 **Сверху вниз**  
 Упорядочение окон вертикально.  
  
 **Упорядочить значки**  
 Упорядочение значков представлений, если окна свернуты.  
  
 **Закрыть / Закрыть все**  
 Закрывает один или все окна в Spy ++.  
  
 **Обновление**  
 Обновление выбранного представления.  
  
 **Окно**  
 Выбор отдельного окна.
