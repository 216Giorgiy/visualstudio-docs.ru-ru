# <a name="how-to-display-process-properties"></a>Практическое руководство. Отображение свойств процесса
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [как: отображение свойств процесса](https://docs.microsoft.com/visualstudio/debugger/how-to-display-process-properties).  
  
Чтобы получить дополнительные сведения о процессах, перечисленных в [представление процессов](../debugger/processes-view.md), использовать [диалоговое окно "Свойства процесса"](../debugger/process-properties-dialog-box.md).  
  
### <a name="to-open-a-process-properties-dialog-box"></a>Чтобы открыть диалоговое окно "Свойства процесса"  
  
1.  Выберите дерево, отображаемое в процесс **представление процессов**.  
  
2.  Из **представление** меню, выберите **свойства**.  
  
 Диалоговые окна свойств не модальным, поэтому вы можете щелкнуть другой элемент в окне представления и диалоговом окне будут отображены сведения о выбранного элемента.
