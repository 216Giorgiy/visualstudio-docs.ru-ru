# <a name="debugging-preparation-windows-services"></a>Подготовка к отладке: службы Windows
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [Подготовка к отладке: службы Windows](https://docs.microsoft.com/visualstudio/debugger/debugging-preparation-windows-services).  
  
Служба Windows – это программа, которая выполнятся в фоновом режиме в Microsoft Windows. Примерами таких служб является служба Telnet и служба времени Windows, изменяющая часы, отображаемые на рабочем столе. Служба Windows не может быть запущена из [!INCLUDE[vsprvs](../includes/vsprvs-md.md)]; ее необходимо запускать из диспетчера управления службами. Дополнительные сведения см. в разделе [создание служб Windows](http://msdn.microsoft.com/library/0f5e2cbb-d95d-477c-b2b5-4b990e6b86ff), [отладка служб Windows](http://msdn.microsoft.com/library/63ab0800-0f05-4f1e-88e6-94c73fd920a2), и [приложений служб Windows](http://msdn.microsoft.com/library/ba72d648-9553-4849-b829-069ad5ea014b).
