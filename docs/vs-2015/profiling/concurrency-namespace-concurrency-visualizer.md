# <a name="concurrency-namespace-concurrency-visualizer"></a>Пространство имен Concurrency (визуализатор параллелизма)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Пространство имен `Concurrency` предоставляет классы и функции, предоставляющие доступ к среде выполнения с параллелизмом, которая является платформой параллельного программирования для C++. Дополнительные сведения см.в статье [Concurrency Runtime](http://msdn.microsoft.com/library/874bc58f-8dce-483e-a3a1-4dcc9e52ed2c) (Среда выполнения с параллелизмом).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
namespace Concurrency;  
```  
  
## <a name="members"></a>Участники  
  
### <a name="namespaces"></a>Пространства имен  
  
|Имя|Описание:|  
|----------|-----------------|  
|[Пространство имен diagnostic](../profiling/diagnostic-namespace.md)|Пространство имен `diagnostics` предоставляет функциональные возможности для выпуска маркеров визуализатора параллелизма.|
