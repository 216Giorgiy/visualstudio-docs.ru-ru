# <a name="vspackage-essentials"></a>Основные сведения о пакетах VSPackage
Пакеты VSPackage — это программные модули, которые образуют и расширяют среду Visual Studio с помощью элементов пользовательского интерфейса, служб, проектов, редакторов и конструкторов. Пакеты VSPackage являются центральным элементом архитектуры [!INCLUDE[vsprvs](../includes/vsprvs-md.md)]. [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] сам записывается в основном как коллекцию пакетов VSPackage.  
  
 Чтобы создать [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] приложение расширяемости, необходимо сначала создать VSPackage. Затем можно добавить команды меню, окна инструментов, окна документов или другой код для расширения [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] интегрированной среды разработки.  
  
 Для использования пакетов VSPackage необходимо установить пакет SDK для Visual Studio. Для выполнения этого пошагового руководства необходимо установить пакет SDK для Visual Studio. Дополнительные сведения см. в разделе [пакет SDK для Visual Studio](../extensibility/visual-studio-sdk.md).
