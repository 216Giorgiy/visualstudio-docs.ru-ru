# <a name="managed-vspackages"></a>Управляемые пакеты VSPackage
Следующие разделы описывают создание пакета VSPackage. VSPackage — это модуль программного обеспечения, который расширяет [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] интегрированной среды разработки (IDE), предоставляя элементы пользовательского интерфейса (UI), службы, проекты, редакторы и конструкторы. Дополнительные сведения см. в разделе [пакетов VSPackage](../extensibility/internals/vspackages.md).
