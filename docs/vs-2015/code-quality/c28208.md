# <a name="c28208"></a>C28208
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28208](https://docs.microsoft.com/visualstudio/code-quality/c28208).  
  
Предупреждение C28208: функция \<функции > была ранее определена с другим списком параметров в \<файл > (\<строки >). Некоторые средства анализа дадут неверные результаты  
  
 Это предупреждение возникает в том случае, когда известных определение функции не соответствует другим вхождения.
