# <a name="c28228"></a>C28228
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28228](https://docs.microsoft.com/visualstudio/code-quality/c28228).  
  
Предупреждение C28228: параметр аннотации: невозможно использовать тип в аннотации  
  
 Это предупреждение указывает, что параметр имеет тип, который не поддерживается. Заметки можно использовать только ограниченный набор типов в качестве параметров. Это не должно быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не ожидается возникают в примерах.
