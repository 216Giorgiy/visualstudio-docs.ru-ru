# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28216](https://docs.microsoft.com/visualstudio/code-quality/c28216).  
  
Предупреждение C28216: _Check_return\_ Аннотация применяется только к постусловиям для конкретного параметра функции.  
  
 `_Check_return_` Заметки применен в контексте, отличное от `__post`; он может понадобиться `__post` (или `__drv_out`) модификатор, или он может размещаться неправильно.
