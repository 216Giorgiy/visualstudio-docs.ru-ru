# <a name="c1250"></a>C1250
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C1250: не удалось загрузить подключаемый модуль.  
  
 Средство анализа кода сообщает это предупреждение при возникновении внутренней ошибки в подключаемый модуль, а не в коде выполняется анализ.
