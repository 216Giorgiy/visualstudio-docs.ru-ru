# <a name="c28280"></a>C28280
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28280](https://docs.microsoft.com/visualstudio/code-quality/c28280).  
  
Предупреждение C28280: для символа «end» обнаружен без соответствующего «begin»  
  
 Заметки языка поддерживает begin и end (`{` и `}` на языке C) конструкции и связывание стало несбалансированной. Такой ситуации можно избежать, если используются макросы.
