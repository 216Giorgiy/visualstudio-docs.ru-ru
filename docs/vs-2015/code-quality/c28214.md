# <a name="c28214"></a>C28214
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28214: имена параметров атрибута должно быть p1... p9  
  
 Это предупреждение означает, что при создании объявление заметки, имена параметров ограничены p1... p9. Это предупреждение не должна возникать при нормальном использовании.
