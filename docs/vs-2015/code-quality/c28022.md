# <a name="c28022"></a>C28022
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28022](https://docs.microsoft.com/visualstudio/code-quality/c28022).  
  
Предупреждение C28022: классами функции \<список_классов1 > данной функции не совпадают с классами функции \<список_классов2 > в typedef, используемом для ее определения.  
  
 Это предупреждение возникает, когда возникает ошибка в аннотации. Определение типа и самой функции имеют `_Function_class_` заметок, но они не совпадают. Если указаны оба они должны соответствовать.
