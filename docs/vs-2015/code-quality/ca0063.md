# <a name="ca0063"></a>CA0063

Не удалось загрузить файл набора правил или один из его зависимых файлов наборов правил.

Не удалось загрузить файл набора правил или один из его зависимых файлов наборов правил.

Не удалось найти указанный набор правил, или один из файлов набора правил, включенных в ваш набор правил не найден. Убедитесь, что все наборы правил, которые включены в ваш набор правил существует на диске и что соответствующие ruleset каталоги были указаны в проекте через **CodeAnalysisRuleSetDirectories** свойства MSBuild.

Отладка ошибок, проверьте файл набора правил в текстовом редакторе. Чтобы найти путь к файлу набора правил, щелкните правой кнопкой мыши проект в обозревателе решений, щелкните **свойства**, а затем нажмите кнопку **анализа кода**. Убедитесь, что выбран файл набора правил в **выполнить этот набор правил**. Путь в набор правил отображается в поле описания.

Изучите **путь** все значения атрибута **Include** элементов. Включить пути можно использовать относительные пути к родительским/текущий файл набора правил, переменные среды и абсолютные пути. Пример:

```
<Include Path="%PUBLIC%\Documents\RuleSets\alldesign.ruleset" Action="Default" />
<Include Path="..\alldesign.ruleset" Action="Default" />
<Include Path="C:\Rulesets\alldesign.ruleset" Action="Default" />
```

Проверьте каждый из этих путей включения и проверьте, они все являются допустимыми.

В некоторых случаях набор правил может находиться зависит от свойства MSBuild. Свойства MSBuild нельзя ссылаться из набора правил. Чтобы обойти эту проблему, укажите дополнительные пути поиска в **CodeAnalysisRuleSetDirectories** свойства MSBuild. В этом случае указать только имя набора правил в **путь** атрибут **Include** элемент.
