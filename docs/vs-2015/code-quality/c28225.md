# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28225: не удалось найти правильное число обязательных параметров в аннотации  
  
 Это предупреждение означает, что именованной аннотации используется с неверное число параметров. Это не должно быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не ожидается возникают в примерах.
