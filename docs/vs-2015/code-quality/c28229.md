# <a name="c28229"></a>C28229
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28229: Аннотация не поддерживает параметры  
  
 Это предупреждение означает, что заметка использовалась с параметром, когда заметка объявлена без параметров. Это не должно быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не ожидается возникают в примерах.
