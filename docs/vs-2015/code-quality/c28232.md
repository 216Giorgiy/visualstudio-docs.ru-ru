# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28232](https://docs.microsoft.com/visualstudio/code-quality/c28232).  
  
Предупреждение C28232: _Pre\_, компонент _Post\_, или _Deref\_ к одной аннотации не применено  
  
 Это предупреждение указывает, что `_Pre_`, `_Post_`, или `_Deref_` появляется оператор в выражении аннотации без последующих функциональной заметки; модификатор было проигнорировано, но это указывает на заметку с неправильным кодом.
