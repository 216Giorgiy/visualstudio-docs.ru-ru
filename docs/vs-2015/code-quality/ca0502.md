# <a name="ca0502"></a>CA0502

Язык не поддерживается.

Эта ошибка возникает, когда пользователь пытается выполнить анализ проекта ASP.NET, содержащий один или несколько страниц или классы, написанные на языке, отличном от C# или Visual Basic. Чтобы устранить эту ошибку, удалите файл из проекта или отключите анализ кода.
