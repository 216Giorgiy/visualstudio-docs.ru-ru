# <a name="ca0501"></a>CA0501

Не удалось прочитать отчет о результатах.

Это происходит, когда не удается записать выходной отчет по одной из следующих причин:

-   Диск заполнен

-   Нет разрешения на запись в каталог

-   Каталог не существует
