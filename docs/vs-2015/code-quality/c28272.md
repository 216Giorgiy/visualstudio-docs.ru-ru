# <a name="c28272"></a>C28272
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28272](https://docs.microsoft.com/visualstudio/code-quality/c28272).  
  
Предупреждение C28272: аннотация для функции, параметр при изучении не согласуется с объявлением функции  
  
 Это предупреждение указывает на ошибку в аннотациях, а не в код, который анализируется. Заметки, появляющиеся в определении функции не совпадают с аналогичными появления в объявлении. Две заметки должны разрешаться в соответствии с.
