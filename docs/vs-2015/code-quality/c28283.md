# <a name="c28283"></a>C28283
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28283](https://docs.microsoft.com/visualstudio/code-quality/c28283).  
  
Предупреждение C28283: для символа: Указанная спецификация размера пока не поддерживается  
  
 Предупреждение указывает, что заметка спецификация размера «метки» получено значение, отличное от нуля. По сути вызывающий объект пытался сказать, что строка должна завершаться с символа, отличного от двоичный нуль.
