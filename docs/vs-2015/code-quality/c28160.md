# <a name="c28160"></a>C28160
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28160](https://docs.microsoft.com/visualstudio/code-quality/c28160).  
  
Предупреждение C28160: ошибка аннотации  
  
 Это предупреждение отображается, когда `__drv_error` обнаружил заметки. Эта заметка используется для пометки методов программирования, которые должны быть исправлены и может использоваться с `__drv_when` заметки для указания конкретных комбинаций параметров.
