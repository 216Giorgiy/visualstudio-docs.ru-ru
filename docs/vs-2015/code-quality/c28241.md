# <a name="c28241"></a>C28241
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28241: Аннотация для функции в параметре не распознана  
  
 Использовалось имя Нераспознанная заметка. Это не должно быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не ожидается возникают в примерах.
