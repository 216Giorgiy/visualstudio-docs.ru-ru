# <a name="ca0057"></a>CA0057

Нарушений не может быть сопоставлен текущий набор целевых объектов и правил.

Эта ошибка возникает, если FxCopCmd.exe вызывается вручную с файлом проекта или импортированным XML-отчет, имеющий нарушений, которые не может быть сопоставлен текущий набор целевых объектов и правил.

Происходит только в том случае, если проводится анализ с помощью средства командной строки версии.
