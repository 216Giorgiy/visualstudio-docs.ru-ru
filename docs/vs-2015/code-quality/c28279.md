# <a name="c28279"></a>C28279
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28279](https://docs.microsoft.com/visualstudio/code-quality/c28279).  
  
Предупреждение C28279: для символа «begin» обнаружен без сопоставления «end»  
  
 Заметки языка поддерживает begin и end (`{` и `}` на языке C) конструкции и связывание стало несбалансированной. Такой ситуации можно избежать, если используются макросы.
