# <a name="c28223"></a>C28223
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28223](https://docs.microsoft.com/visualstudio/code-quality/c28223).  
  
Предупреждение C28223: Не удалось найти ожидаемый токен или идентификатор для аннотации, параметр  
  
 Это предупреждение означает, что должен быть идентификатор параметра в заметку и обнаружен другой символ. Обычно это означает макроса неправильным кодом заметки. Использование ключевых слов C или C++ в этой позиции приведет к этой ошибке.
