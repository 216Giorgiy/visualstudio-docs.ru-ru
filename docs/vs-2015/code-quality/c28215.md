# <a name="c28215"></a>C28215
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28215: typefix нельзя применять к параметру, который уже имеет typefix  
  
 Применение `__typefix` заметки к параметру, который уже содержит аннотацию, что является ошибкой. `__typefix` Заметки используются только в некоторых особых случаях и это предупреждение не должна отображаться при нормальном использовании.
