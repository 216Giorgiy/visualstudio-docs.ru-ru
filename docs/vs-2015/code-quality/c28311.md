# <a name="c28311"></a>C28311
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28311](https://docs.microsoft.com/visualstudio/code-quality/c28311).  
  
Предупреждение 28311: Аннотация < имя_аннотации > в \<функция > \<параметра > является устаревшей версией SAL.  
  
 Заметка имеет старую версию и должна быть обновлена в эквивалентную [SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md). Это предупреждение не выдается, если ранее было выдано предупреждение о несогласованном примечании; оно выводится только один раз для каждого объявления. Проверьте остальную часть объявления на наличие других устаревших SAL.
