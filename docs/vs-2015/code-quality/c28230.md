# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28230](https://docs.microsoft.com/visualstudio/code-quality/c28230).  
  
Предупреждение C28230: тип параметра не содержит членов.  
  
 Это предупреждение означает, что аргумент в заметку пытается получить доступ к `struct`, `class`, или `union` и именованный элемент не существует.
