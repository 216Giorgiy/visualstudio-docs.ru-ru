# <a name="c28231"></a>C28231
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28231: Аннотация допустима только для массива  
  
 Это предупреждение означает, что аргумент заметки должны быть массивом и обнаружен другого типа.
