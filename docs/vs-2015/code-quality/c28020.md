# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28020](https://docs.microsoft.com/visualstudio/code-quality/c28020).  
  
Предупреждение C28020: выражение \<expr > не имеет значение true, при этом вызове  
  
 Это предупреждение отображается, когда _Satisfies\_ выражения в списке не имеет значение true. Часто это означает неверный параметр.  
  
 В этом случае в объявлении функции заметок указывают условие невозможно.
