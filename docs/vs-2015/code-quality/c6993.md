# <a name="c6993"></a>C6993
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение 6993: при анализе кода структуры OpenMP игнорируются; анализируется код одного потока  
  
 Это предупреждение означает, что анализатор кода обнаружил открытые директивы pragma MP, которые не удалось проанализировать.
