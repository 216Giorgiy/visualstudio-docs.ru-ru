# <a name="c28235"></a>C28235
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28235](https://docs.microsoft.com/visualstudio/code-quality/c28235).  
  
Предупреждение C28235: функция не может использоваться изолированно как аннотация  
  
 Это предупреждение означает, что была предпринята попытка использовать функцию, которая не объявлена как заметки в контексте заметки. Сюда входят, используя простые операции (PrimOp) в контексте автономного. Это не должно быть возможно в том случае, если стандартные макросы, используемые для заметки. Это предупреждение не ожидается возникают в примерах.
