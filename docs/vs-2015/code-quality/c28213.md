# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28213](https://docs.microsoft.com/visualstudio/code-quality/c28213).  
  
Предупреждение C28213: `_Use_decl_annotations_` Аннотация должен использоваться для ссылки, без изменения, перед объявлением. \<Почему >  
  
 Это предупреждение возникает при ссылке на предыдущее объявление без требуемого `_Use_decl_annotations_` заметки.
