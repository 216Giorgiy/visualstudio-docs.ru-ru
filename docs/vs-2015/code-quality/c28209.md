# <a name="c28209"></a>C28209
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28209](https://docs.microsoft.com/visualstudio/code-quality/c28209).  
  
Предупреждение C28209: объявление символа содержит конфликтующее объявление  
  
 Это предупреждение означает объявление неправильно сформированного заметки. Это предупреждение не должна возникать при нормальном использовании.
