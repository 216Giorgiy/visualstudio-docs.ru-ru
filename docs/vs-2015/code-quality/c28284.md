# <a name="c28284"></a>C28284
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28284](https://docs.microsoft.com/visualstudio/code-quality/c28284).  
  
Предупреждение C28284: для символа: предикаты в настоящее время не поддерживаются для символов, не являющихся функциями  
  
 Это предупреждение указывает, что условной аннотации (предикат, `__drv_when`) был найден на узле функции.
