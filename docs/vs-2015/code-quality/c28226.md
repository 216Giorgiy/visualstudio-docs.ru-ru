# <a name="c28226"></a>C28226
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28226: Аннотация не может также быть PrimOp (в текущем объявлении)  
  
 Это предупреждение означает, что объявлен PrimOp именованной аннотации и также была ранее объявлена как обычный заметки. Это не должно быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не должен рассматриваться при нормальном использовании.
