# <a name="c28221"></a>C28221
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28221](https://docs.microsoft.com/visualstudio/code-quality/c28221).  
  
Предупреждение C28221: для параметра аннотации ожидалось строковое выражение  
  
 Это предупреждение означает, что параметр заметки ожидается в виде строки и обнаружен другого типа. Обычно это указывает макроса неправильным кодом заметки и не должна отображаться при нормальном использовании.
