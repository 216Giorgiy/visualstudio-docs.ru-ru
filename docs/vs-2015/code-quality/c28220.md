# <a name="c28220"></a>C28220
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28220](https://docs.microsoft.com/visualstudio/code-quality/c28220).  
  
Предупреждение C28220: ожидалось целочисленное выражение для аннотации параметра в аннотации  
  
 Это предупреждение означает, что параметр заметки должен быть целочисленным выражением и обнаружен другого типа. Обычно это означает макроса неправильным кодом заметки.
