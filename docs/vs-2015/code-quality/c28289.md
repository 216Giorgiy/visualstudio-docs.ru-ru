# <a name="c28289"></a>C28289
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Последнюю версию этого раздела можно найти в [C28289](https://docs.microsoft.com/visualstudio/code-quality/c28289).  
  
Предупреждение C28289: для функции: ReadableTo или WritableTo не содержало спецификацию ограничения как параметр  
  
 Средство анализа кода сообщает это предупреждение, когда заметка/параметр функции является miscoded как указано.
