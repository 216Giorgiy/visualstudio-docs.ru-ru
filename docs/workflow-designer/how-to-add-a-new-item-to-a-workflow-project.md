# <a name="how-to-add-a-new-item-to-a-workflow-project"></a>Практическое руководство. Добавление нового элемента в проект рабочего процесса
После создания проекта рабочего процесса в него можно добавлять действия рабочих процессов, конструкторы и другие привычные элементы [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].  
  
 В следующей таблице перечислены элементы [!INCLUDE[wf](../workflow-designer/includes/wf_md.md)], которые можно добавлять в проект рабочего процесса.  
  
|Имя|Описание|  
|----------|-----------------|  
|Действие|Действие, которое составляется из других действий. При выборе этого элемента добавляет один и тот же файл XAML в проект как получается при выборе **библиотеки действий** шаблона для нового проекта. [!INCLUDE[crabout](../test/includes/crabout_md.md)]на эту процедуру в разделе [как: создать библиотеку действий](../workflow-designer/how-to-create-an-activity-library.md).|  
|конструктор действий|Конструктор для настройки поведения действия во время разработки. При выборе этого элемента добавляются те же файлы в проект, которые получатся при выборе **библиотека конструктора действий** шаблона для нового проекта. [!INCLUDE[crabout](../test/includes/crabout_md.md)]на эту процедуру в разделе [как: Создание библиотеки конструктора рабочих процессов](../workflow-designer/how-to-create-an-activity-designer-library.md).|  
|CodeActivity|Действие с логикой выполнения, записанное в код. Файл с исходным кодом с переопределением метода <xref:System.Activities.CodeActivity.Execute%2A> уже сформирован.|  
|Служба рабочего процесса WCF|Служба [!INCLUDE[indigo2](../workflow-designer/includes/indigo2_md.md)], построенная с использованием действий рабочего процесса. При выборе этого элемента добавляются те же файлы в проект, которые получатся при выборе **приложение службы рабочего процесса WCF** шаблона для нового проекта. [!INCLUDE[crabout](../test/includes/crabout_md.md)]на эту процедуру в разделе [как: Создание приложения службы рабочего процесса WCF](../workflow-designer/how-to-create-a-wcf-workflow-service-application.md).|  
  
### <a name="to-add-a-new-item-to-a-workflow-project"></a>Добавление нового элемента в проект рабочего процесса  
  
1.  На **проекта** меню, нажмите кнопку **добавить новый элемент...** .  
  
     **Добавить новый элемент** откроется диалоговое окно.  
  
2.  В **установленные шаблоны** выберите **рабочего процесса** группы.  
  
3.  Выберите один из четырех элементов. Доступные варианты перечислены в таблице выше.  
  
4.  Введите соответствующее имя для элемента в **имя** внизу диалогового окна.  
  
5.  Нажмите кнопку **добавить** следует добавить элемент в текущий проект рабочего процесса.
