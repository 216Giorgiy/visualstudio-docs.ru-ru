# Использование тем в рабочих процессах (для прежних версий)
В данном разделе описывается использование тем в [!INCLUDE[wf](../workflow-designer/includes/wf_md.md)] при работе со средством [!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)] более ранней версии, ориентированного на работу с [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] или [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)].  
  
 Используя темы [!INCLUDE[wf2](../workflow-designer/includes/wf2_md.md)] можно настроить внешний вид рабочих процессов с помощью конструктора более ранней версии.Тема задает основной цвет и цвет фона, стили, значки и другие визуальные элементы рабочего процесса.Темы можно сохранить для повторного использования другими рабочими процессами.  
  
 Создание и изменение тем выполняется с помощью диалогового окна [Диалоговое окно «Конфигурация темы» \(для прежних версий\)](../workflow-designer/theme-configuration-dialog-box-legacy.md).Для открытия диалогового окна выберите команду **Создать новую тему** в меню **Рабочий процесс**, либо щелкните правой кнопкой по рабочей области конструктора и выберите команду **Создать новую тему**.  
  
 Используя диалоговое окно [Диалоговое окно «Темы, конструктор рабочих процессов, параметры» \(для прежних версий\)](../workflow-designer/themes-workflow-designer-options-dialog-box-legacy.md) можно выбрать тему по умолчанию и сохраненные пользовательские темы.Для открытия этого диалогового окна, выберите команду **Выбрать пользовательскую тему** в меню **Рабочий процесс** \(или щелкните правой кнопкой мыши по рабочей области конструктора рабочих процессов и выберите команду **Выбрать пользовательскую тему**\), далее выберите команду **Конструктор рабочих процессов**.Выберите необходимую тему и нажмите кнопку **ОК**.
