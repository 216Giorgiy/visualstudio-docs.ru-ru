# <a name="control-flow-activity-designers"></a>Конструкторы действия управления потоком

Конструктор рабочих процессов Windows включает определенное количество системных действий, которые можно использовать при создании рабочих процессов. В этом разделе содержатся действия, предусмотренные системой, и служащие для управления потоком в пределах рабочего процесса. В следующих подразделах приводится описание этих действий, а также описание их использования.
