# <a name="theme-configuration-dialog-box-legacy"></a>Диалоговое окно «Конфигурация темы» (для прежних версий)

В этом разделе описывается использование **конфигурация темы** диалоговое окно в конструкторе рабочих процессов прежних версий Windows. Используйте конструктор рабочих процессов прежних версий, для целевой платформы .NET Framework 3.5 или WinFX.

Тема задает цвета фона и переднего плана, стили, значки и другие визуальные элементы рабочего процесса. Темы можно сохранить для повторного использования другими рабочими процессами.

Можно создавать и изменять темы с помощью **конфигурация темы** диалоговое окно. Чтобы открыть диалоговое окно, выберите **создать новую тему** на **рабочего процесса** меню или щелкните правой кнопкой мыши рабочий процесс рабочую область конструирования и выберите **создать новую тему**.

В следующей таблице описаны элементы пользовательского интерфейса (UI) **конфигурация темы** диалоговое окно.

|Элемент пользовательского интерфейса|Описание|
|----------------|-----------------|
|**Имя темы:**|Имя, идентифицирующее темы в [темы, конструктор рабочих процессов, параметры (для прежних версий)](../workflow-designer/themes-workflow-designer-options-dialog-box-legacy.md). Для новой темы создается изменяемое имя.|
|**Местоположение темы:**|Имя файла и путь к файлу темы. Изменяемое имя для новой темы создается на основе имени созданной темы. При изменении имени созданной темы возможно потребуется изменить имя файла, чтобы оно соответствовало имени темы.|
|**...**|Щелкните, чтобы выбрать расположение для сохранения файла темы рабочего процесса, который использует расширение файла .wtm. Выбранный путь будет показан в **расположение темы** текстовое поле.|
|**Выберите конструктор и задайте свойства:**|На левой панели представление в виде дерева содержит перечень действий, для которых может быть настроена тема. Выберите в представлении в виде дерева действие, свойства темы для действия будут показаны на панели свойств, которая расположена справа от панели с представлением в виде дерева. Чтобы изменить значение свойства, щелкните его.|
|**Предпросмотр**|Щелкните, чтобы отобразить окно для предварительного просмотра эффектов изменения свойства.|
