# <a name="flowchart-activity-designers"></a>Конструктор действия блок-схемы

Конструктор рабочих процессов Windows включает определенное количество системных действий, которые можно использовать при создании рабочих процессов. Данный раздел содержит действия, предоставленные системой и используемые для управления блок-схемами в пределах рабочего процесса. В следующих подразделах приводится описание этих действий, а также описание их использования.
