# <a name="using-the-activity-designers"></a>Использование конструкторов действий

В конструкторе рабочих процессов включает в себя ряд предоставляемых системой действий, которые можно использовать при построении действий и рабочих процессов. В этом разделе описываются категории системных действий и их конструкторы, которые доступны как системные функции. Темы индивидуальных действий и их соответствующих конструкторов внутри каждой из этих категорий описывают, что эти действия могут предпринять и предоставить руководство, как их следует использовать.
