# <a name="developing-workflow-applications-targeting-the-net-30-or-net-35-framework"></a>Разработка приложений рабочих процессов, предназначенных для платформы .NET 3.0 или .NET 3.5
[!INCLUDE[vs2010](../misc/includes/vs2010_md.md)] предоставляет конструктор рабочих процессов Windows для создания [!INCLUDE[wf](../workflow-designer/includes/wf_md.md)] приложений, предназначенных для [!INCLUDE[netfx40_long](../workflow-designer/includes/netfx40_long_md.md)]. Конструктор использует привычный пользовательский интерфейс [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] для графического конструирования и отладки приложений рабочих процессов.

 [!INCLUDE[vs2010](../misc/includes/vs2010_md.md)] также предоставляет доступ к конструктору рабочих процессов прежних версий, которые можно использовать при создании приложений, ориентированных на [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] или [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)]. Этот раздел содержит руководство по использованию конструктора прежних версий.
