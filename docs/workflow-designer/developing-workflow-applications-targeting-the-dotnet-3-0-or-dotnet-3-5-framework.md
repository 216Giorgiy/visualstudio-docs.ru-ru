[!INCLUDE[vs2010](../misc/includes/vs2010_md.md)] предоставляет средство [!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)] для создания приложений [!INCLUDE[wf](../workflow-designer/includes/wf_md.md)], ориентированных на [!INCLUDE[netfx40_long](../workflow-designer/includes/netfx40_long_md.md)].
