# <a name="developing-workflow-applications-targeting-the-net-30-or-net-35-framework"></a>Разработка приложений рабочих процессов, предназначенных для платформы .NET 3.0 или .NET 3.5

Visual Studio 2010 предоставляет конструктор рабочих процессов Windows для создания приложений Windows Workflow Foundation (WF), предназначенных для .NET Framework 4. Конструктор использует знакомый пользовательский интерфейс Visual Studio для графического создания и отладки приложений рабочих процессов.

Visual Studio 2010 также предоставляет доступ к конструктор прежних версий рабочего процесса, который может использоваться для целевой платформы .NET Framework 3.5 или WinFX. Этот раздел содержит руководство по использованию конструктора прежних версий.
