# <a name="transaction-activity-designers"></a>Конструкторы действий транзакций

Конструкторы операций транзакций используются для создания и настройки действий транзакций, например, компенсации и подтверждения. Содержится в этом разделе и перечисленных в следующей таблице представлены рекомендации по использованию действий конструктора рабочих процессов и конструкторов шаблонов.
