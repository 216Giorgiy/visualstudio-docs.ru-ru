# <a name="welcome-to-r-tools-for-visual-studio"></a>Инструменты R для Visual Studio. Приветствие

Благодарим вас за установку инструментов R для Visual Studio (RTVS)! Ознакомьтесь с возможностями инструментов и поделитесь с нами своим мнением, воспользовавшись ссылками в разделе [ресурсов](#resources).

![Снимок экрана RTVS](media/installer-screenshot.png)

## <a name="resources"></a>Ресурсы

- [Документация](index.md)
- [Порядок установки инструментов R для Visual Studio](installing-r-tools-for-visual-studio.md)
- [Отправка и просмотр ошибок и запросов функций на GitHib](https://github.com/Microsoft/RTVS/issues)
- [Изучение исходного кода и участие в его разработке](https://github.com/Microsoft/RTVS/blob/master/LICENSE)
- Чтобы отправить отзыв, не используя GitHub (что предпочтительно), воспользуйтесь меню **Инструменты R > Отзыв** в Visual Studio.

## <a name="microsoft-r-distributions"></a>Распространяемые материалы Microsoft R

RTVS поддерживает CRAN R и распространяемые материалы Microsoft R, включая версию R, которая распространяется вместе с SQL Server 2016. Воспользуйтесь ссылками ниже, чтобы ознакомиться с дополнительной информацией о различных распространяемых материалах Microsoft R:

- [Microsoft R Open](https://mran.microsoft.com/download/)
- [Microsoft R Client](/machine-learning-server/r-client/what-is-microsoft-r-client)
- [Microsoft Machine Learning Server (прежнее название — R Server)](/machine-learning-server/)
