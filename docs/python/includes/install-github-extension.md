1. Чтобы подключиться к репозиториям GitHub, запустите установщик Visual Studio, выберите вариант **Изменить** и перейдите на вкладку **Отдельные компоненты**. Прокрутите содержимое вниз до раздела **Средства для работы с кодом**, выберите компонент **Расширение GitHub для Visual Studio**, а затем выберите **Изменить**.

    ![Выбор расширения GitHub в установщике Visual Studio](../media/installation-github-extension.png)
