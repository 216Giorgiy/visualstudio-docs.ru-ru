1. Чтобы работать с репозиториями GitHub из Visual Studio, нужно установить расширение Github для Visual Studio. Для этого запустите установщик Visual Studio, выберите **Изменить** и откройте вкладку **Отдельные компоненты**. Прокрутите содержимое вниз до раздела **Средства для работы с кодом**, выберите компонент **Расширение GitHub для Visual Studio**, а затем выберите **Изменить**.

    ![Выбор расширения GitHub в установщике Visual Studio](../media/installation-github-extension.png)
