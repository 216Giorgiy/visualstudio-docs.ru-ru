# <a name="publishing-to-azure-app-service"></a>Публикация в службу приложений Azure

Вы можете быстро создать в Visual Studio веб-сайт на языке Python и опубликовать его в службе приложений Azure, следуя инструкциям ниже.

- [Создание подписки Azure](#create-an-azure-subscription)
- [Создание и тестирование исходного проекта](#create-and-test-the-initial-project)
- [Публикация в службу приложений Azure](#publish-to-azure-app-service)

Краткое пошаговое руководство по этому процессу представлено в видео [Visual Studio Python Tutorial: Building a Website](https://www.youtube.com/watch?v=FJx5mutt1uk&list=PLReL099Y5nRdLgGAdrb_YeTdEnd23s6Ff&index=6) (Руководство по Python для Visual Studio. Создание веб-сайта, длительность 3 мин 10 с), размещенном на сайте youtube.com. 

> [!VIDEO https://www.youtube.com/embed/FJx5mutt1uk] 

## <a name="create-an-azure-subscription"></a>Создание подписки Azure

Для публикации в Azure нужна подписка Azure. Также вы можете использовать временный сайт.

Знакомство с подписками мы рекомендуем начать со страницы [Создайте бесплатную учетную запись Azure уже сегодня](https://azure.microsoft.com/en-us/free/), где вы сможете получить большой кредит на использование служб Azure. Также вы можете [присоединиться к Visual Studio Dev Essentials](https://azure.microsoft.com/en-us/pricing/member-offers/vs-dev-essentials/) и в течение целого года ежемесячно получить кредит на сумму 25 долларов США.

Чтобы создать временный сайт в службе приложений Azure без подписки Azure, выполните следующие действия.

1. Откройте в браузере адрес [try.azurewebsites.net](https://try.azurewebsites.net).
1. Выберите тип приложения **Веб-приложение**, затем нажмите **Далее**.
1. Выберите шаблон **Пустой сайт** и нажмите **Создать >**.
1. Выполните вход через любую социальную сеть, и через некоторое время ваш сайт будет готов к работе по указанному URL-адресу.
1. Выберите **Скачать профиль публикации** и сохраните файл `.publishsettings`, который вам понадобится позднее.

## <a name="create-and-test-the-initial-project"></a>Создание и тестирование исходного проекта

1. В Visual Studio выберите пункт меню **Файл > Новый > Проект**, выполните поиск по запросу Bottle, выберите **Веб-проект Bottle** и нажмите кнопку **ОК**.    

1. Когда вам будет предложено установить внешние пакеты, выберите вариант **Install into a virtual environment** (Установить в виртуальной среде). Обратите внимание на элемент управления **Show required packages** (Показывать требуемые пакеты) в нижней части диалогового окна, который сообщит вам, какие пакеты будут установлены.

  ![Установка необходимых пакетов](~/python/media/tutorials-common-external-packages.png)

1. Выберите предпочитаемый базовый интерпретатор для виртуальной среды (например, **Python 2.7** или **Python 3.4**) и нажмите кнопку **Создать**.

  ![Добавление виртуальной среды при создании проекта](~/python/media/tutorials-common-add-virtual-environment.png)

1. После создания проекта его можно тестировать локально, выбрав пункт меню **Отладка > Начать отладку** или нажав клавишу F5. По умолчанию приложение использует размещенный в памяти репозиторий, который не требует никакой настройки. При остановке веб-сервера все данные будут утеряны.

1. Откройте приложение и проверьте, как оно работает.

1. Когда вы закончите проверку, остановите отладчик (**Отладка > Остановить отладку** или клавиши Shift+F5).

## <a name="publish-to-azure-app-service"></a>Публикация в службу приложений Azure

1. В **Обозревателе решений** щелкните проект правой кнопкой мыши и выберите действие **Публикация**. 

1. В диалоговом окне **Публикация** выберите вариант **Служба приложений Microsoft Azure**.

  ![Публикация в Azure, шаг 1](~/python/media/tutorials-common-publish-1.png)

1. Выберите цель публикации.

    - Если у вас есть подписка Azure, выберите в качестве цели публикации **Служба приложений Microsoft Azure**, и в следующем диалоговом окне выберите существующую службу приложений или нажмите **Создать**, чтобы создать новую службу.
    - Если вы используете временный сайт, созданный на странице try.azurewebsites.net, выберите в качестве цели публикации **Импорт**, затем перейдите к файлу `.publishsettings`, который вы скачали с сайта ранее, и выберите **ОК**.

1. Сведения о службе приложения отображаются в диалоговом окне **Публиковать**, на вкладке **Подключение**, как показано ниже.

  ![Публикация в Azure, шаг 2](~/python/media/tutorials-common-publish-2.png)

1. Выберите действие **Далее >** и просмотрите дополнительные параметры. Если вам нужна [Удаленная отладка кода Python в Azure](debugging-azure-remote.md), для параметра **Конфигурация** выберите значение **Отладка**.
1. Нажмите **Публиковать**. Когда вы закончите публикацию приложения в Azure, созданный сайт откроется в браузере по умолчанию.
