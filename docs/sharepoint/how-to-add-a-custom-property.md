# Практическое руководство. Добавление пользовательского свойства
  С помощью **Редактора свойств** к модели можно добавлять пользовательские свойства.  Для извлечения такой информации, как строки подключения и другой информации, во время выполнения, можно обеспечить доступ к этим свойствам с помощью кода.  
  
### Добавление пользовательского свойства  
  
1.  В **Обозревателе BDC** выберите узел, представляющий элемент модели, к которому следует применить пользовательское свойство.  
  
2.  В меню **Вид** выберите **Окно свойств**.  
  
3.  В окне **Свойства** выберите свойство **Пользовательские свойства** и нажмите кнопку многоточия \(![Эллипс конструктора ASP.NET для мобильных устройств](~/sharepoint/media/mwellipsis.gif "Эллипс конструктора ASP.NET для мобильных устройств")\).  
  
     Появится диалоговое окно **Редактор свойств**.  
  
4.  В текстовом поле в столбце **Имя** укажите имя свойства.  
  
5.  В поле **Тип** пользовательского свойства выберите соответствующий тип данных.  
  
6.  В поле пользовательского свойства **Значение** укажите значение, а затем нажмите кнопку **ОК**.
