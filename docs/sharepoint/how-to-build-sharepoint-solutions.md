# Практическое руководство. Построение решений SharePoint
  Построение решения SharePoint выполняется аналогично построению других приложений Visual Studio.  Дополнительные сведения о построении приложений см. в разделе [Построение приложений в Visual Studio](../ide/compiling-and-building-in-visual-studio.md).  Дополнительные сведения по вопросам, относящимся к решениям SharePoint, см. в разделе [Практическое руководство. Развертывание и публикация решения SharePoint на локальном сайте SharePoint](../sharepoint/how-to-deploy-and-publish-a-sharepoint-solution-to-a-local-sharepoint-site.md).  
  
### Построение решений SharePoint  
  
1.  Выберите конфигурацию, для которой выполняется построение: "Отладка" или "Выпуск".  Для получения дополнительной информации см. [Параметры проекта отладчика](../debugger/debugger-project-settings.md).  
  
2.  Выберите соответствующий параметр построения или повторного построения проекта или решения.  Для получения дополнительной информации см. [Построение приложений в Visual Studio](../ide/compiling-and-building-in-visual-studio.md).  
  
3.  В строке меню последовательно выберите **Сборка** и **Собрать решение**.
