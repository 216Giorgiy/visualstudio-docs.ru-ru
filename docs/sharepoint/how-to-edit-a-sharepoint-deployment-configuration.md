# Практическое руководство. Изменение конфигурации развертывания SharePoint
  Можно создать новую конфигурацию развертывания или изменить существующую.  Например, можно выполнить один этап процесса развертывания или изменить порядок выполнения его этапов.  Создание или изменение конфигурации развертывания может потребоваться потому, что встроенные и добавленные программным способом конфигурации изменять нельзя.  
  
## Создание конфигурации развертывания SharePoint  
  
#### Создание конфигурации развертывания SharePoint  
  
1.  В **Обозревателе Решений**, выберите проект SharePoint и затем в строке меню выберите **Project**, *ProjectName* **Properties**.  
  
2.  На вкладке **SharePoint**, нажмите кнопку **Создать**.  
  
     Откроется диалоговое окно **Добавление новой конфигурации развертывания**.  
  
3.  В поле **Имя** введите имя конфигурации развертывания.  
  
4.  В области **Действия при развертывании** выберите действия, которые требуется добавить в конфигурацию развертывания, нажмите кнопки \(**\>**\) и **OK**.  
  
    > [!NOTE]  
    >  Если заданы команды, выполняемые до или после развертывания, эти действия выполняются только в том случае, если они добавлены в пользовательскую конфигурацию развертывания.  
  
## Изменение активной конфигурации развертывания  
  
#### Изменение активной конфигурации развертывания  
  
1.  В **Обозревателе Решений**, выберите проект SharePoint и затем в строке меню выберите **Project**, *ProjectName* **Properties**.  
  
2.  Откройте вкладку **SharePoint**.  
  
3.  В списке **Активная конфигурация развертывания** выберите имя конфигурации развертывания, которую необходимо использовать.
