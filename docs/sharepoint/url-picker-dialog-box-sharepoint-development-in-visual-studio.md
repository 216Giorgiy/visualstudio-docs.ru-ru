# Диалоговое окно &quot;Выбор URL-адреса&quot; (Разработка приложений SharePoint в Visual Studio)
  В диалоговом окне выбора URL\-адреса можно выбрать такие файлы, как файл главной страницы или файлы изображений, расположенные в вашем проекте или на локальном сервере, где выполняется SharePoint.  
  
 Это диалоговое окно появится, когда требуется выбрать файл для задания свойства.  Для доступа к этому диалоговому окну следует нажать кнопку с многоточием \(![Эллипс конструктора ASP.NET для мобильных устройств](~/docs/sharepoint/media/mwellipsis.gif "Эллипс конструктора ASP.NET для мобильных устройств")\) рядом с различными свойствами в окне **Свойства**.  Кнопка с многоточием также отображается в виде подсказки IntelliSense при присваивании значений некоторым атрибутам в представлении конструктора **Исходный код**.  
  
## Список элементов пользовательского интерфейса  
 **Папки проекта**  
 Отображает список папок, определенных в проекте или на локальном сервере, на котором выполняется SharePoint.  Нажмите кнопку расширения, чтобы отобразить вложенные папки.  
  
 Разверните узел **Проект** для выбора файлов в вашем проекте.  Чтобы они отобразились в диалоговом окне для выбора, файлы проекта должны отвечать следующим критериям:  
  
-   файл должен содержаться в сопоставленной папке;  
  
-   файл должен быть добавлен в пакет решения;  
  
-   файл не должен быть расположен в другом проекте.  
  
 Если необходимо сослаться на файлы, которые не отвечают этим критериям, путь к ним придется ввести вручную.  
  
 Разверните узел **Сервер**, чтобы выбрать файлы, расположенные на локальном сервере, на котором выполняется SharePoint.  Чтобы эти файлы отобразились в диалоговом окне выбора URL\-адреса, они должны отвечать следующим критериям:  
  
-   файл должен быть расположен в одной из сопоставленных папок: **Images**, **Layouts** или **ControlTemplates**;  
  
-   файл не может быть расположен в базе данных контента SharePoint.  
  
 Если необходимо сослаться на файлы, которые не отвечают этим критериям, путь к ним придется ввести вручную.  
  
 **Содержимое папки**  
 Отображает список файлов в выбранной папке.  Выберите файл и нажмите кнопку **ОК**, чтобы закрыть диалоговое окно и отправить сведения о выборе процессу, который вызвал его.  
  
 **Тип файлов**  
 Позволяет выбрать из списка файлов, которые подходят для выполняемой задачи.
