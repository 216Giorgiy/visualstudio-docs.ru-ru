# <a name="how-to-install-a-custom-diagnostic-data-adapter"></a>Практическое руководство. Установка настраиваемого адаптера диагностических данных

Если вы создали или получили настраиваемый адаптер диагностических данных, его сборку можно установить, скопировав файл сборки в подходящий каталог на локальном компьютере.

 Если нужно применять настраиваемый адаптер диагностических данных для одной из ролей в среде, необходимо установить его на всех компьютерах, выполняющих агенты тестирования, которые могут использоваться для этой роли.

 Выполните следующие действия, чтобы установить настраиваемый адаптер диагностических данных в соответствующие расположения. Потребуются административные разрешения на всех компьютерах, на которых устанавливается адаптер диагностических данных.

## <a name="installing-a-custom-diagnostic-data-adapter"></a>Установка настраиваемого адаптера диагностических данных

### <a name="to-install-a-custom-diagnostic-data-adapter"></a>Чтобы установить настраиваемый адаптер диагностических данных, выполните следующие действия.

1.  Для использования адаптера диагностических данных во время выполнения тестов на клиентском компьютере или на компьютере агента скопируйте все файлы из каталога построения в следующий каталог на целевом компьютере (в зависимости от каталога установки):

     *%ProgramFiles(x86)%\Microsoft Visual Studio\2017\Enterprise\Common7\IDE\PrivateAssemblies\DataCollectors*

     Требуется скопировать следующие файлы.

    -   Сборка адаптера диагностических данных (DLL) (обязательно).

    -   Файл данных для отладки (PDB) адаптера (необязательно).

    -   Файл конфигурации адаптера (`<diagnostic data adapter name>.dll.config`), если имеются параметры конфигурации по умолчанию (необязательно).

    -   Сборка редактора конфигурации, если он был создан для изменения параметров конфигурации адаптера (необязательно). Эта сборка предназначена только для клиентских компьютеров. На компьютерах агентов редактор не используется.

    > [!NOTE]
    > Хотя адаптер диагностических данных и редактор конфигурации могут быть созданы в том же самом проекте и встроены в ту же самую сборку, при желании для них можно создать отдельные проекты и отдельные сборки.

     Дополнительные сведения о том, как настраивать параметры тестирования для использования среды при выполнении тестов, см. в разделе [Сбор данных диагностики в ручных тестах (VSTS)](/vsts/manual-test/mtm/collect-more-diagnostic-data-in-manual-tests).

2.  Чтобы выбрать адаптер диагностических данных для теста, необходимо сначала выбрать существующие параметры тестирования или создать новые параметры с помощью Microsoft Test Manager или Visual Studio, а затем выбрать адаптер диагностических данных на вкладке **Данные и диагностика** выбранных параметров тестирования.

3.  Если создан и установлен редактор конфигурации адаптера диагностических данных, для настройки адаптера диагностических данных в соответствии с выбранными параметрами тестирования нажмите кнопку **Настройка** рядом с адаптером и выполните нужные изменения. Нажмите кнопку **Сохранить**. Дополнительные сведения о создании редактора конфигурации для сборщика диагностических данных см. в разделе [Практическое руководство. Создание специализированного редактора данных для адаптера диагностических данных](../test/how-to-create-a-custom-editor-for-data-for-your-diagnostic-data-adapter.md).

4.  При выполнении тестов в Microsoft Test Manager эти параметры тестирования можно присвоить плану тестирования перед выполнением тестов или использовать команду **Запуск с параметрами** для присвоения или переопределения параметров тестирования. Дополнительные сведения о параметрах тестирования см. в статье [Сбор диагностических данных с помощью параметров тестирования](../test/collect-diagnostic-information-using-test-settings.md).

     Если тесты выполняются из Visual Studio, необходимо сделать эти параметры тестирования активными. Дополнительные сведения о параметрах тестирования см. в статье [Сбор диагностических данных с помощью параметров тестирования](../test/collect-diagnostic-information-using-test-settings.md).

5.  Выполните тесты с данными параметрами тестирования, выбрав адаптер диагностических данных.
