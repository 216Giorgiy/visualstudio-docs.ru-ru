# <a name="how-to-add-comments-while-analyzing-a-completed-load-test"></a>Практическое руководство. Добавление комментариев во время анализа выполненного нагрузочного теста

После завершения нагрузочного теста или загрузки его результатов можно добавить в нагрузочный тест описание и комментарий по анализу с помощью анализатора тестовой нагрузки. Комментарий по анализу и описание окончательно сохраняются вместе с результатами нагрузочного теста. Дополнительные сведения см. в разделе [Практическое руководство. Обращение к результатам нагрузочного теста для их анализа](../test/how-to-access-load-test-results-for-analysis.md).

Введенное описание также отображается в связанном с нагрузочном тестом столбце **Описание** диалогового окна **Открытие и обработка результатов нагрузочных тестов** в **редакторе тестовой нагрузки**. Дополнительные сведения см. в разделе [Практическое руководство. Обращение к результатам нагрузочного теста для их анализа](../test/how-to-access-load-test-results-for-analysis.md). Кроме того, комментарии отображаются при создании отчета о результатах нагрузочного теста в формате Excel. Дополнительные сведения см. в статье [Создание отчетов о результатах нагрузочных тестов для сравнения тестов или анализ трендов](../test/compare-load-test-results.md).

![Диалоговое окно "Анализ" для добавления примечаний нагрузочного теста](../test/media/ltest_ananotes.png)

## <a name="to-add-a-comment-to-a-load-test"></a>Добавление комментария к нагрузочному тесту

1.  После завершения нагрузочного теста или загрузки его результатов выберите команду **Добавить заметки по анализу** на панели инструментов анализатора тестовой нагрузки.

     Откроется диалоговое окно **Анализ**.

2.  Введите описание в текстовом поле **Описание**.

3.  Введите комментарий по анализу в текстовом поле **Анализ**.

    Дополнительные сведения см. в разделе [Практическое руководство. Обращение к результатам нагрузочного теста для их анализа](../test/how-to-access-load-test-results-for-analysis.md).
