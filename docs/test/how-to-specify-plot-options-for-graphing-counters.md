# <a name="how-to-specify-plot-options-for-graphing-counters"></a>Практическое руководство. Определение параметров построения для отображения счетчиков

Диалоговое окно **Параметры построения** позволяет изменять цвет и тип линий для показаний счетчика на диаграмме. Кроме того, можно заблокировать диапазон на уровне конкретного значения или сделать его автоматически изменяемым в зависимости от данных выборки.

![Диалоговое окно "Параметры построения"](../test/media/ltest_plotoptions.png "LTest_PlotOptions")

## <a name="to-specify-plotting-options-for-graphs"></a>Задание параметров построения диаграмм

1.  В анализаторе тестовой нагрузки нажмите кнопку **Диаграммы** на панели инструментов нагрузочных тестов.

     Результаты нагрузочных тестов будут отображены в представлении диаграмм.

2.  В условных обозначениях или на диаграмме щелкните правой кнопкой мыши строку или текущую выводимую линию счетчика производительности, для которого требуется изменить параметры построения, и выберите команду **Параметры построения**.

     Откроется диалоговое окно **Параметры построения**.

3.  С помощью раскрывающегося списка **Цвет** выберите цвет для отображения на диаграмме данных счетчика производительности.

4.  С помощью раскрывающегося списка **Стиль** выберите стиль для отображения на диаграмме данных счетчика производительности.

5.  Выполните одно из следующих действий.

     Выберите **Автоматически корректировать диапазон** (по умолчанию)

     \- или -

     Снимите флажок **Автоматически корректировать диапазон** и с помощью раскрывающегося списка **Диапазон** укажите диапазон для построения диаграммы счетчика производительности.

6.  Нажмите кнопку **ОК**.

     Показания счетчика производительности, для которого были изменены параметры, будут показаны на диаграмме с учетом этих изменений.
