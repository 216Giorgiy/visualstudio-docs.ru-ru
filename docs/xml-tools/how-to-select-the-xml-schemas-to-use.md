# <a name="how-to-select-the-xml-schemas-to-use"></a>Как: выберите XML-схем для использования

XML Editor предоставляет кэш схем, расположенный в *%InstallDir%\Xml\Schemas* каталога. Кэш схем включает известные XML-схемы, используемые для поддержки технологии IntelliSense и проверки правильности XML-документа.

**Схемы** свойство документа используется для выбора одного или нескольких XML-схемы определения языка XSD схем для использования. Оно позволяет выбрать схемы из кэша схем или указать схему, которая не находится в кэше.

Схемы, можно указать сохраняются в скрытом файле пользовательских параметров решения (. *SUO*), вместе с другими XML свойства документа. В результате исключается необходимость снова вводить эти значения при следующем открытии решения.

> [!NOTE]
> Редактор может выполнять проверку правильности с помощью встроенной схемы или схемы, на которую ссылается атрибут `xsd:schemaLocation`. Дополнительные сведения см. в разделе [Проверка XML-документа](../xml-tools/xml-document-validation.md).

## <a name="to-select-an-xml-schema-from-the-schema-cache"></a>Чтобы выбрать XML-схемы из кэша схем

1.  Откройте файл в приложении XML Editor.

2.  В окне свойств документа нажмите кнопку в **схемы** поля.

     **XML-схем** диалоговое окно. В поле диалогового окна перечислены все схемы с. *xsd* расширения в кэше схем (включая схемы, на которые ссылается *catalog.xml* файла), а также любую схему, которая в текущее решение открыто в Visual Studio, указывается в `xsd:schemaLocation` атрибут или ссылающихся на них **схемы** свойство.

3.  Выберите схемы, которые будут использоваться для проверки правильности, одним из приведенных ниже способов.

    -   Выберите схему из списка в **XML-схем** диалоговое окно, нажмите кнопку **используйте** столбца, а затем выберите **использовать эту схему**.

     - или -

    -   Выделите несколько схем в **XML-схем** диалоговое окно, щелкните правой кнопкой мыши и выберите **использовать эту схему**.

4.  Нажмите кнопку **ОК**.

     Список выбранных схем копируется обратно в **схемы** свойство документа.

## <a name="to-add-an-xml-schema-to-the-schema-cache"></a>Добавление схемы XML в кэш схем

1.  В окне свойств документа нажмите кнопку в **схемы** поля.

2.  Нажмите кнопку **Добавить**.

     При этом откроется **Открытие XSD-схемы** диалогового окна.

3.  Выполните поиск и выберите схемы, которые нужно добавить в кэш схем.

4.  Нажмите кнопку **откройте**.

     Схемы, которые добавлены в схему кэширования и является **используйте** значение столбца равно **использовать эту схему**.

## <a name="to-delete-an-xml-schema-from-the-schema-cache"></a>Чтобы удалить XML-схемы из кэша схем

1.  В окне свойств документа нажмите кнопку в **схемы** поля.

2.  Выберите схему для удаления и нажмите кнопку **удалить**.

     Схема удаляется из кэша схем, находящегося в оперативной памяти, но не из файловой системы.

    > [!NOTE]
    > При наличии ссылки на схему через `schemaLocation` атрибута или соответствующего `targetNamespace` затем **удалить** не будет работать в этом случае из-за автоматическое связывание. В этом случае рекомендуется пометить схему как **не использовать выбранные схемы** в **использовать** столбца.
