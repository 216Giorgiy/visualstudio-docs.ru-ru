# Отладка XSLT
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Для отладки XSLT вы можете использовать отладчик среды Visual Studio.Отладчик поддерживает задание точек останова, просмотр состояний выполнения XSLT и т. д.Отладка XSLT доступна в Visual Studio Team System и в выпуске Professional Edition.
