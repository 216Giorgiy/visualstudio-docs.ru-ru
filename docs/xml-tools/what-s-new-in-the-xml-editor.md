# Новые возможности редактора XML
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Среда Visual Studio 2010 содержит следующие новые функции.  
  
## XSLT IntelliSense  
 Функция XSLT IntelliSense позволяет автоматически заполнять имена наборов атрибутов; режимы шаблонов и имена; а также имена параметров для указанного режима или именованного шаблона.Дополнительные сведения см. в разделе [Функции IntelliSense редактора XML](../xml-tools/xml-editor-intellisense-features.md).  
  
## Профилировщик XSLT  
 Профилировщик XSLT создает подробные отчеты о производительности XSLT, помогающие измерять, оценивать и выявлять проблемы в XSLT коде, связанные с производительностью.Профилировщик XSLT также содержит полезные указания по оптимизации таблиц стилей XSL и XSLT.Дополнительные сведения см. в разделе [Пошаговое руководство. Профилировщик XSLT](../xml-tools/walkthrough-xslt-profiler.md).  
  
## Иерархия XSLT  
 Средство управления иерархией XSLT позволяет добавлять точки останова во включенные таблицы стилей и\/или встроенные правила шаблона.Дополнительные сведения см. в разделе [Пошаговое руководство. Использование XSLT иерархии](../xml-tools/walkthrough-using-xslt-hierarchy.md).
