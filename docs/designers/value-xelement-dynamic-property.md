# Value (динамическое свойство XElement)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Возвращает или задает содержимое данного элемента.  
  
## Синтаксис  
  
```  
elem.Value  
```  
  
## Значение свойства или возвращаемое значение  
 Строка <xref:System.String>, представляющая объединенное содержимое элемента.  
  
## Заметки  
 Это свойство эквивалентно свойству <xref:System.Xml.Linq.XElement.Value%2A> класса <xref:System.Xml.Linq.XElement?displayProperty=fullName>, однако это динамическое свойство также поддерживает уведомление об изменениях.
