# <a name="uninstall-visual-studio"></a>Удаление Visual Studio
На этой странице приведены пошаговые инструкции по удалению Visual Studio, интегрированного набора средств для эффективной работы разработчиков.  

1.  Найдите установщик Visual Studio на своем компьютере.  

     Например, на компьютере с юбилейным обновлением Windows 10 нажмите кнопку **Пуск** и прокрутите список до буквы **У**. Вы увидите пункт **Установщик Microsoft Visual Studio**.  

     ![00-ModifyingDev15Prev5-FindTheVisualStudioInstaller](../install/media/00-modifyingdev15prev5-findthevisualstudioinstaller.png)

2.  Щелкните установщик, чтобы запустить его, щелкните значок ![сведений](media/vs2017uninstall-UninstallIcon.png) и выберите пункт **Удалить**.  

     ![Изменение версии-кандидата Visual Studio 2017: запуск или изменение](media/vs2017uninstall-ChooseUninstallFromInstaller.png "Восстановление или удаление версии-кандидата Visual Studio 2017")  

3.  Нажмите кнопку **ОК**, чтобы подтвердить действие.

     ![Изменение версии-кандидата Visual Studio 2017: запуск или изменение](media/vs2017uninstall-UninstallConfirm.png "Подтверждение удаления версии-кандидата Visual Studio 2017")  

Если в будущем вы захотите снова установить версию-кандидат Visual Studio 2017, просто запустите установщик Visual Studio еще раз и щелкните **Установить** на экране выбора.

> [!IMPORTANT]
> Хотя версию-кандидат Visual Studio 2017 в общем можно использовать в рабочей среде, рабочие нагрузки и компоненты, которые в пользовательском интерфейсе установки доступны в режиме предварительной версии, в рабочей среде не поддерживаются.
