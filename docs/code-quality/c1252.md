# <a name="c1252"></a>C1252
Предупреждение C1252: циклическая или Отсутствующая зависимость между подключаемыми модулями: требуется GUID  
  
 Средство анализа кода сообщает это предупреждение, когда возникает ошибка внутреннего с зависимости подключаемого модуля, а не в коде выполняется анализ.
