# C28272
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28272: Аннотация для функции: параметр не согласуется с объявлением функции при проверке  
  
 Это предупреждение указывает на ошибку в аннотациях, а не в анализируемом коде.  Аннотации, отображающиеся в определении функции, несовместимы с отображаемыми в объявлении.  Сопоставление двух аннотаций должно быть разрешено.
