# <a name="ca0069"></a>CA0069
UsingAlternatePlatform. *FrameworkVersion1* не найден. С помощью *FrameworkVersion2* вместо этого. Для получения оптимальных результатов анализа убедитесь, что установлена правильная платформа .NET Framework.
