# <a name="c6288"></a>C6288
Предупреждение C6288: неверный оператор: результат взаимного включения через & & всегда равно нулю. Предполагалось использовать &#124; &#124; вместо?

 Это предупреждение означает, что в выражении проверки переменная проверяется для двух различных констант, и результат зависит от оба условия имели значение true. Код в этих случаях указывает неправильно записано намерения программиста. Очень важно проверить код и исправить ошибку; в противном случае код не будет работать как ожидалось.

 Эта ошибка обычно вызвана с помощью `&&`; вместо `||`, но может также быть вызвана использованием `==` где `!=` был предназначен.

## <a name="example"></a>Пример
 Это предупреждение возникает в следующем коде:

```
void f(int x)
{
  if ((x == 1) && (x == 2)) // warning
  {
    // code ...
  }
}
```

 Чтобы устранить это предупреждение, используйте следующий код:

```
void f(int x)
{
     if ((x == 1) || (x == 2))
     {
          // logic
     }

    /* or */
    if ((x != 1) && (x != 2))
    {
        // code ...
    }
}
```

 Средство анализа не предупреждать, если у выражения есть побочные эффекты.
