# C28209
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28209: Объявление символа содержит конфликтующее объявление  
  
 Это предупреждение означает, что объявление аннотации было построено неправильно.  Это предупреждение не возникает при обычном использовании.
