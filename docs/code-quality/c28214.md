# <a name="c28214"></a>C28214
Предупреждение C28214: имена параметров атрибута должно быть p1... p9  
  
 Это предупреждение означает, что при построении объявление заметки имена параметров ограничены p1... p9. Это предупреждение не должна возникать при нормальном использовании.
