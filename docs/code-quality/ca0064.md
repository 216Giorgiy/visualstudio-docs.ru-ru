# <a name="ca0064"></a>CA0064

Анализ не был выполнен, так как указанный набор правил не содержал правила FxCop.

Это предупреждение может возникать в одном из следующих ситуаций:

- Если возникнет это предупреждение в сочетании с предупреждениями CA0063 произошла ошибка при загрузке файла набора правил. Дополнительные сведения см. в разделе [CA0063](ca0063.md) документации.

- При возникновении ошибки в сочетании с предупреждениями CA0062 наиболее вероятно, анализ кода не удалось найти сборки или сборки, которые содержат правила, заданные с помощью набора правил. Дополнительные сведения см. в разделе [CA0062](ca0062.md) документации.

- В противном случае это предупреждение обычно возникает при вашего набора правил является пустой строкой или все правила, которые включены в набор правил дочерних были отключены. С помощью Visual Studio [редактор набора правил](../code-quality/working-in-the-code-analysis-rule-set-editor.md) включить некоторые правила в ваш набор правил.
