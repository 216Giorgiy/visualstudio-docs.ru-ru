# C28244
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28244: Аннотация для функции содержит аннотацию параметра или внешнюю аннотацию, не поддающуюся синтаксическому анализу  
  
 В случае использования стандартных макросов для аннотаций это не представляется возможным в настоящее время, таким образом, при типичном применении можно не ожидать появления данного предупреждения.
