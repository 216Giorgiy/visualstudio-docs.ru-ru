# <a name="c28244"></a>C28244
Предупреждение C28244: Аннотация для функции содержит аннотацию параметра или внешнюю аннотацию

 Это в настоящее время не должно быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не ожидается возникают в примерах.
