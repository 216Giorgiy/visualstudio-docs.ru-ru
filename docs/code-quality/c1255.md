# <a name="c1255"></a>C1255
Предупреждение C1255: неправильная длина имеет данных PCH для подключаемого модуля.  
  
 Средство анализа кода сообщает это предупреждение при внутренней ошибки в средстве, а не в коде выполняется анализ.
