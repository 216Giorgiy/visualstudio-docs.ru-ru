# <a name="c1254"></a>C1254
Предупреждение C1254: несоответствие версий подключаемого модуля: версия не совпадает с версией драйвера PREfast  
  
 Средство анализа кода отображает это предупреждение, когда возникает ошибка внутренней версией подключаемого модуля, а не в коде выполняется анализ.
