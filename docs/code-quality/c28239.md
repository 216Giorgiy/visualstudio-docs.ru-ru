# <a name="c28239"></a>C28239
Предупреждение C28239: аннотация в параметре имеет значение, stringValue или longValue и выражение paramn = xxx. Используйте только paramn=xxx  
  
 Внутренняя ошибка в файле PREfast модели. Это предупреждение не будет выполняться в типичных применений.
