# C28239
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

предупреждение C28239: аннотация параметра одновременно содержит значение value, stringValue или longValue и выражение paramn\=xxx.  Используйте только paramn\=xxx  
  
 Внутренняя ошибка возникла в файле модели PREfast.  Это предупреждение не должно появляться при обычном использовании.
