# <a name="c28227"></a>C28227
Предупреждение C28227: Аннотация не может также быть PrimOp (см. предыдущее объявление).  
  
 Это предупреждение указывает именованной аннотации объявлен как обычный примечание, которое также было объявлено как PrimOp. Это не должна быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не должны отображаться в типичное использование.
