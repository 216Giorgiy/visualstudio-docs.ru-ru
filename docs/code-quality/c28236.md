# <a name="c28236"></a>C28236
Предупреждение C28236: Аннотация не может использоваться в выражении  
  
 Это предупреждение означает, что была предпринята попытка использования функции, объявленной как заметка в контексте выражения. Это не должна быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не должны отображаться в типичное использование.
