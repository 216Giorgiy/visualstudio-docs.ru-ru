# C28236
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28236: Аннотацию невозможно использовать в выражениях  
  
 Данное предупреждение показывает, что была предпринята попытка использовать функцию, объявленную в качестве аннотации, в контексте выражения.  В случае использования стандартных макросов для аннотаций это не представляется возможным, таким образом, при типичном применении можно не ожидать появления данного предупреждения.
