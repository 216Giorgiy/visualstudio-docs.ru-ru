# <a name="c28236"></a>C28236
Предупреждение C28236: Аннотацию нельзя использовать в выражениях

 Это предупреждение означает, что была предпринята попытка использовать функции, объявленной как заметки в контексте выражения. Это не должно быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не ожидается возникают в примерах.
