# C28225
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28225: Не удалось найти правильное число необходимых параметров в аннотации  
  
 Это предупреждение означает, что именованная аннотация используется с неверным количеством параметров.  В случае использования стандартных макросов для аннотаций это не представляется возможным, таким образом, при типичном применении можно не ожидать появления данного предупреждения.
