# <a name="c28225"></a>C28225
Предупреждение C28225: не удалось найти правильное число обязательных параметров в аннотации

 Это предупреждение означает, что именованной аннотации используется с неверным числом параметров. Это не должна быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не должны отображаться в типичное использование.
