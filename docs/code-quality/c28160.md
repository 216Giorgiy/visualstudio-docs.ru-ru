# C28160
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28160: Ошибка аннотации  
  
 Это предупреждение показывается, если аннотация `__drv_error` была обнаружена.  Эта аннотация используется, чтобы пометить приемы программирования, которые должны быть устранены, и может использоваться с аннотацией `__drv_when`, чтобы указать конкретные сочетания параметров.
