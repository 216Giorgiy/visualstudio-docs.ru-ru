# <a name="c28160"></a>C28160
Предупреждение C28160: ошибка аннотации

 Это предупреждение отображается, когда `__drv_error` произошла заметки. Эта заметка используется для пометки приемы программирования, которые должны быть исправлены и может использоваться с `__drv_when` заметки, чтобы указать определенные сочетания параметров.
