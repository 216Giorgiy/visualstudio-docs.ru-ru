# C28226
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

предупреждение C28226: Примечание не может также быть PrimOp \(в текущем объявлении\).  
  
 Это предупреждение возникает, если именованное примечание, объявленное ранее как обычное, объявляется как PrimOp.  В случае использования стандартных макросов для аннотаций это не представляется возможным, таким образом, при типичном применении можно не ожидать появления данного предупреждения.
