# <a name="c28226"></a>C28226
Предупреждение C28226: Аннотация не может также быть PrimOp (в текущем объявлении)

 Это предупреждение указывает, что объявлен PrimOp именованной аннотации и также было объявлено как обычный заметки. Это не должна быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не должны отображаться в обычных условиях использования.
