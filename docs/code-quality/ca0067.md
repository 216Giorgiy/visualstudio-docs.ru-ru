# <a name="ca0067"></a>CA0067
Каталог не найден.  
  
 Значение **/directory** параметр в командной строке FxCopCmd не найден.  
  
 Это предупреждение может возникать, если вы используете **CodeAnalysisDependentAssemblyPaths** свойство [!INCLUDE[vstecmsbuild](../extensibility/internals/includes/vstecmsbuild_md.md)] для указания пути поиска дополнительных ссылок на сборки и один из этих путях не существует.  
  
-   Если появляется предупреждение CA0067 с [ошибки CA0058](ca0058.md) ошибок или [CA0060](ca0060.md) CA0067 разрешает предупреждения, обычно устранении других ошибок.  
  
-   Если появляется предупреждение CA0067 без ошибок или предупреждений, обычно можно пропустить это предупреждение, так как каталог не требуется.
