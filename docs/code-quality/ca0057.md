# <a name="ca0057"></a>CA0057
Не удается сопоставить нарушения с текущим набором целевых объектов и правил  
  
 Эта ошибка возникает, если FxCopCmd.exe вызывается вручную с файлом проекта или импортированным XML-отчетом с нарушениями, которые не удается сопоставить с текущим набором целевых объектов и правил.  
  
 Происходит только в том случае, если проводится анализ с помощью средства командной строки версии.
