# C28194
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28194: Функция объявлена как присваивающая псевдоним значению в переменной, однако она не выполнила эту операцию до своего выхода  
  
 Это предупреждение означает, что прототип для проанализированной функции имеет аннотацию `__drv_isAliased`, свидетельствующую о том, что она будет присваивать *псевдоним* указанному аргументу \(то есть, присвоит значение таким образом, что оно останется после возврата из функции\).  Однако функция не присвоила псевдоним аргументу по пути, обозначенному аннотацией.  Большинство функций, присваивающих псевдоним переменной, сохраняют ее значение в глобальной структуре данных.
