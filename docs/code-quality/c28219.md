# <a name="c28219"></a>C28219
Предупреждение C28219: ожидался член перечисления заметки для параметров в аннотации

 Должно быть членом именованного параметра для заметки `enum` обнаружен тип и другой символ; используйте член, `enum` типа. Обычно это означает макрос неправильным кодом заметки.
