# C28219
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28219: Для параметра аннотации ожидался член перечисления  
  
 Предполагается, что параметр заметки является членом именованного типа `enum`, и был обнаружен какой\-то другой символ; используйте член данного типа `enum`.  Обычно это означает неправильный закодированный макрос аннотации.
