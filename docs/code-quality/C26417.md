# <a name="c26417-nolvaluerefsharedptr"></a>C26417 NO_LVALUE_REF_SHARED_PTR
«Общим указателем параметр передается по ссылке и не сбрасывается или переназначить. Используйте T * или T & вместо.»

**Правила C++ Core**:   
R.35: Занять shared_ptr<widget>& параметр для выражения, функция может повторно подключите общий указатель

Передача по ссылке общие указатели можно использовать в случаях, когда вызываемый код обновляет целевой объект интеллектуального указателя и вызывающий объект ожидает, что для просмотра таких обновлений. С помощью ссылки только для сокращения расходов на передачу общего указателя не гарантируется. Если вызываемый код только обращается к целевой объект и никогда не управляет время существования, безопаснее для передачи необработанный указатель или ссылка, а не для предоставления подробных сведений об управлении ресурса.

## <a name="remarks"></a>Примечания
-  Эта проверка распознает std::shared_pointer и определяемые пользователем типы, которые, скорее всего, ведут себя как общие указатели. Для определяемых пользователем общие указатели ожидаются следующие признаки:
-  перегруженные разыменования или доступа к члену операторов (общедоступный и удаления);
-  конструктор копии или оператором назначения копии (общедоступный и удаления);
-  открытого деструктора, которой не удален и не используемые по умолчанию. Пустые деструкторы, по-прежнему рассматриваются как определяемые пользователем.
-  Для выполнения сброса или переназначение интерпретируется в более общем виде:
-  любой вызов метода неконстантное на общий указатель потенциально могут сбросить указатель;
-  Каждый вызов функции, которая принимает ссылку на общий указатель неконстантное может сбросить или переназначить этот указатель.

## <a name="example"></a>Пример 
сложность неиспользуемые интерфейса

```cpp
bool unregister(std::shared_ptr<event> &e) // C26417, also C26415 SMART_PTR_NOT_NEEDED
{
    return e && events_.erase(e->id());
}

void renew(std::shared_ptr<event> &e)
{
    if (unregister(e))
        e = std::make_shared<event>(e->id());
    // ...
}
```

## <a name="example"></a>Пример 
ненужные интерфейс усложнения — упрощенное письмо

```cpp
bool unregister(const event *e)
{
    return e && events_.erase(e->id());
}

void renew(std::shared_ptr<event> &e)
{
    if (unregister(e.get()))
        e = std::make_shared<event>(e->id());
    // ...
}
```
