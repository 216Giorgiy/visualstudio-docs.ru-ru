# <a name="c28241"></a>C28241
Предупреждение C28241: аннотация для функции в параметре не распознана

 Было использовано имя нераспознанный заметки. Это не должна быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не должны отображаться в типичное использование.
