# <a name="ca0504"></a>CA0504
Каталог правил пропущен, так как он не существует  
  
 Предупреждение CA504 создается в том случае, если задан каталог, в **CodeAnalysisRuleDirectories** не удалось найти свойство в файле конфигурации проекта или сборки. Чтобы устранить эту проблему, введите допустимый путь.
