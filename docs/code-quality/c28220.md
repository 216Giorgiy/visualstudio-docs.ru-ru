# <a name="c28220"></a>C28220
Предупреждение C28220: целое число со знаком для аннотации ожидалось выражение параметра аннотации

 Это предупреждение указывает на параметр в заметку должно быть целочисленным выражением, которое обнаружена другого типа. Обычно это означает макрос неправильным кодом заметки.
