# <a name="c28286"></a>C28286
Предупреждение C28286: Для функции синтаксическая ошибка рядом с окончанием

 Средство анализа кода сообщает это предупреждение, когда возможная ошибка встречается в файле модели. Несколько ошибок исходный файл также может вызывать такие ошибки.
