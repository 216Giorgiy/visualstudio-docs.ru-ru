# Использование аннотаций SAL для сокращения количества дефектов в коде C/C++
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

SAL — это язык аннотации кода, разработанный Microsoft.  С помощью аннотаций можно сделать назначение кода явным.  Аннотации также включают инструменты автоматизированного статического анализа кода для более тщательного анализа кода, с меньшим числом ложнопозитивных и ложнонегативных срабатываний.  
  
 Статьи в этом разделе документации рассматривают аспекты SAL, предоставляют руководство по синтаксису SAL и примеры его использования.  
  
-   [Основные сведения о языке SAL](../code-quality/understanding-sal.md)  
  
     Предоставляет сведения и примеры, показывающие аннотации SAL.  
  
-   [Создание примечаний к параметрам и возвращаемым значениям функций](../code-quality/annotating-function-parameters-and-return-values.md)  
  
     Перечисляет аннотации SAL для функций и параметров функций.  
  
-   [Аннотация поведения функций](../code-quality/annotating-function-behavior.md)  
  
     Перечисляет аннотации SAL для функций и параметров функций.  
  
-   [Аннотация структур и классов](../code-quality/annotating-structs-and-classes.md)  
  
     Перечисляет аннотации SAL для структур и классов.  
  
-   [Аннотация поведения блокировки](../code-quality/annotating-locking-behavior.md)  
  
     Объясняет, как использовать аннотации SAL с механизмами блокировки.  
  
-   [Указание времени и места применения примечания](../code-quality/specifying-when-and-where-an-annotation-applies.md)  
  
     Перечисляет аннотации SAL, определяющие состояние или область \(размещение\) других аннотаций SAL.  
  
-   [Встроенные функции](../code-quality/intrinsic-functions.md)  
  
     Перечислены встроенные аннотаций SAL.  
  
-   [Рекомендации и примеры](../code-quality/best-practices-and-examples-sal.md)  
  
     Содержит примеры, которые показывают, как использовать аннотаций SAL.  Также поясняются распространенные ошибки.
