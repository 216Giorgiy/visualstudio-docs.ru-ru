# C6705
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C6705: Ошибка аннотации: ожидалось \<ожидаемое\_число\> аргументов для аннотации \<параметр\>, найдено \<фактическое\_число\>.  
  
 Это предупреждение появляется при ошибке в примечаниях.
