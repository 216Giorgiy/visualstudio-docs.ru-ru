# <a name="ca0055"></a>CA0055
Не удалось загрузить файл:\<путь >

 Средство анализа кода не удалось загрузить файл, указанный для анализа. Это происходит, если файл не найден, либо был запрещен доступ к файлу. Как правило, это происходит, если пользователь выбрал, переопределите свойство входной сборки в [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)], или вызвал fxcopcmd.exe вручную.
