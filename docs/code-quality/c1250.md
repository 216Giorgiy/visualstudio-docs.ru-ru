# C1250
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

предупреждение C1250: Не удается загрузить подключаемый модуль.  
  
 Средство анализа кода сообщает об этом предупреждении, когда произошла внутренняя ошибка в подключаемом модуле, а не в проанализированном коде.
