# <a name="c28020"></a>C28020
Предупреждение C28020: выражение \<expr > не имеет значение true, при этом вызове  
  
 Это предупреждение отображается, когда _Satisfies\_ выражения в списке не имеет значение true. Часто это указывает недопустимый параметр.  
  
 В этом случае в объявлении функции аннотации указывают условие невозможно.
