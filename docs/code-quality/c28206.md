# C28206
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28206: \<выражение\> : левый операнд указывает на структуру, используйте `->`  
  
 Это предупреждение возникает в том случае, если ожидался указатель на структуру в разыменованной нотации `->`.
