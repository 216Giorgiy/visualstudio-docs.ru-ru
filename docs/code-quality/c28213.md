# C28213
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28213: Аннотация `_Use_decl_annotations_` должна использоваться для ссылки, без изменения, перед объявлением. \<причина\>  
  
 Это предупреждение возникает, когда на предыдущее объявление ссылаются без необходимой аннотации `_Use_decl_annotations_`.
