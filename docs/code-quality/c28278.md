# <a name="c28278"></a>C28278
Предупреждение C28278: Имя функции отображается без прототипа в области.  
  
 Это предупреждение обычно указывает, что `__deref` , которую необходимо применить `__return` возвращается заметки к значению.  
  
 Средство анализа кода отображает это предупреждение, когда была вызвана функция без объявления и анализа, которое может быть выполнено ограничено, поскольку объявление содержит важные сведения.  
  
 В языке C разрешает (но не рекомендует) использование функции, для которого был объявлен без прототипа. Определение функции или объявлении функции (prototype) должен стоять до первого использования функции. Это предупреждение означает, что была вызвана функция без объявления и анализа, которое может быть выполнено ограничено, поскольку объявление содержит важные сведения. Если в объявлении функции содержат заметки, в объявлении функции еще полезнее, средство анализа кода.
