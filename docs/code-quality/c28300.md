# C28300
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28300: \<имя\_параметра\>: Несовместимые типы операндов выражения для оператора \<имя\_оператора\>  
  
 Это предупреждение возникает, если типы операндов в параметрах несовместимы с оператором.
