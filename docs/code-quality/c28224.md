# <a name="c28224"></a>C28224
Предупреждение C28224: аннотации требуются параметры

 Это предупреждение означает, что именованной аннотации используется без параметров, и по крайней мере один параметр является обязательным. Это не должна быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не должны отображаться в обычных условиях использования.
