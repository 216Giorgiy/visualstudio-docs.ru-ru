# C28224
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28224: Аннотации требуются параметры  
  
 Это предупреждение означает, что именованная аннотация используется без параметров, когда по крайней мере один параметр является обязательным.  В случае использования стандартных макросов для аннотаций это не представляется возможным, таким образом, при типичном применении можно не ожидать появления данного предупреждения.
