# <a name="c28289"></a>C28289
Предупреждение C28289: для функции: ReadableTo или WritableTo не содержало спецификацию ограничения как параметр  
  
 Средство анализа кода отображает это предупреждение, когда параметр функции или заметка miscoded, как указано.
