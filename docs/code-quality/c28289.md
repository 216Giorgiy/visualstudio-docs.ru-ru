# C28289
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

предупреждение C28289: Для функции: ReadableTo или WritableTo не содержало спецификацию ограничения в качестве параметра  
  
 Средство анализа кода выдает это предупреждение, если примечание для функции\/параметра закодировано не так, как описано.
