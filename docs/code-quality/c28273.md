# <a name="c28273"></a>C28273
Предупреждение C28273: для функции, причины не согласуются с объявлением функции

 Это предупреждение указывает на ошибку в аннотации, а не в коде, который анализируется. Заметки, появляющихся в определении функции не совпадают с аналогичными появления в объявлении. Две заметки должны разрешаться в соответствии.
