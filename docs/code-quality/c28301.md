# <a name="c28301"></a>C28301
Предупреждение C28301: никаких заметок для первого объявления \<функция >.\< note1 > в разделе \<имя файла > (\<строки >). \<note2 >  
  
 Это предупреждение отображается, если заметки не найдены в первом объявлении определенной функции.
