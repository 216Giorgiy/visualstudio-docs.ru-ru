# C28301
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28301: Для первого объявления функции отсутствуют аннотации \<функция\>.\<примечание1\> См. \<имя\_файла\>\(\<строка\>\). \<примечание2\>  
  
 Это предупреждение появляется, если заметки не были найдены в первом объявлении данной функции.
