# <a name="c28217"></a>C28217
Предупреждение C28217: для функции число параметров у аннотации не совпадает с параметром в файле

 Заметки в текущей строке или в строке в сообщении, не согласованы. Это не должна быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не должны отображаться в обычных условиях использования.
