# C28217
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение С28217: Для функции число параметров в аннотации не совпадает с числом параметров, обнаруженным в файле.  
  
 Аннотации в текущей строке и в строке сообщения противоречат друг другу.  В случае использования стандартных макросов для аннотаций это не представляется возможным, таким образом, при типичном применении можно не ожидать появления данного предупреждения.
