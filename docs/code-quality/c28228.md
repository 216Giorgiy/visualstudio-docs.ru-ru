# C28228
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28228: Параметр аннотации: Невозможно использовать тип в аннотации  
  
 Это предупреждение возникает, если тип параметра не поддерживается.  В примечаниях может использоваться только ограниченный набор типов в качестве параметров.  В случае использования стандартных макросов для аннотаций это не представляется возможным, таким образом, при типичном применении можно не ожидать появления данного предупреждения.
