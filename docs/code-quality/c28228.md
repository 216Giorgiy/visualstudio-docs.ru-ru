# <a name="c28228"></a>C28228
Предупреждение C28228: параметр аннотации: невозможно использовать тип в аннотации  
  
 Это предупреждение означает, что параметр имеет тип, не поддерживается. Заметки можно использовать только ограниченный набор типов в качестве параметров. Это не должна быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не должны отображаться в типичное использование.
