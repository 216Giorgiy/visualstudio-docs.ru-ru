# <a name="c1257"></a>C1257
Предупреждение 1257: Сбой при инициализации подключаемого модуля.  
  
 Средство анализа кода отображает это предупреждение, когда имеется Внутренняя ошибка в подключаемый модуль, а не в коде выполняется анализ.
