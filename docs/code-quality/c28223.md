# C28223
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

предупреждение C28223: Не найден ожидаемые токен\/идентификатор для аннотации, параметра  
  
 Данное предупреждение указывает, что ожидается параметр для аннотации в виде идентификатора, но некоторые другие символы были обнаружены.  Обычно это означает неправильный закодированный макрос аннотации.  Использование ключевых слов С или С\+\+ в данном месте может вызывать данную ошибку.
