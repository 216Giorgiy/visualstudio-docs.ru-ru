# <a name="c28223"></a>C28223
C28223 предупреждение: Не удалось найти ожидаемый токен/идентификатор для аннотации, параметр

 Это предупреждение указывает на параметр в заметку должно быть идентификатором, что обнаружен другой символ. Обычно это означает макрос неправильным кодом заметки. Использование ключевых слов C или C++ в этой позиции вызовет эту ошибку.
