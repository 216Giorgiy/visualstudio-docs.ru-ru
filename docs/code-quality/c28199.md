# C28199
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28199: Использование памяти, которая, возможно, не инициализирована  
  
 Это сообщение говорит о том, что переменной присвоен адрес, однако назначение для нее не обнаружено.  
  
 Указанная переменная используется без явной инициализации, но в некоторый момент был принят ее адрес, что указывает на то, что она может быть инициализирована незаметно для средств анализа кода.  
  
 Это предупреждение может быть ошибочным, если переменная инициализируется за пределами функции.  
  
 Средство анализа кода выдает данное предупреждение на выходе из функции, если параметр имеет `_Out_` или `_Inout_` аннотацию, и переменная не инициализируется.
