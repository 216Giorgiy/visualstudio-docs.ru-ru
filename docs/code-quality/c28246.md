# C28246
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28246: Аннотация для функции "\<имя\>" \- parameter "\<параметр\>"не соответствует типу параметра  
  
 Оператор \_\_deref при создании заметки был применен к типу, отличному от указателя.
