# C28221
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28221: Для параметра в аннотации ожидалось строковое выражение  
  
 Данное предупреждение означает, что ожидается параметр аннотации типа строка, а обнаружен был какой\-либо другой тип.  Обычно это указывает на неправильно закодированный макрос аннотации. Данное предупреждение не должно появляться при обычном использовании.
