# <a name="c28221"></a>C28221
Предупреждение C28221: для параметра аннотации ожидалось строковое выражение

 Это предупреждение указывает, предполагается, что параметр в заметку является строка, что обнаружен другого типа. Это обычно указывает макрос заметки с неправильным кодом и не должно быть доступно при нормальном использовании.
