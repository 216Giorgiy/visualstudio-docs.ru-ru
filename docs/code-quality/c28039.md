# <a name="c28039"></a>C28039
Предупреждение C28039: тип фактического параметра \<операнд > должен точно соответствовать типу \<typename >

 Это предупреждение обычно возникает при формальных перечисления не передан член перечисления, но может также использоваться для других типов.

 Поскольку C разрешает перечисления являются взаимозаменяемыми, а взаимозаменяемо константы, можно легко передать значение неверного перечисления функции без ошибок.

 Для перечислимых типов, если тип параметра перечисления помечается с помощью `_Enum_is_bitflag_`, арифметические операции разрешено для параметра. В противном случае параметр должен быть ровно правильного типа. Если константа является строго обязательным, предупреждение C28137 могут также применяться.

 Это правило используется для других типов параметров; см. в документации функции для почему типы должны в точности совпадать.
