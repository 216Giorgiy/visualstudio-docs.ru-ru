# <a name="c26464-useconstpointerforelements"></a>C26464 USE_CONST_POINTER_FOR_ELEMENTS
  Значения, на которые указывают элементы массива «% массива %» назначаются только один раз; пометьте элементы как указатель на `const`. См. в разделе [C++ Core рекомендации con.4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
