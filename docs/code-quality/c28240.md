# C28240
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

предупреждение C28240: Аннотация в параметре содержит параметр2, но не содержит параметр1.  
  
 Внутренняя ошибка возникла в файле модели PREfast.  Это предупреждение не должно появляться при обычном использовании.
