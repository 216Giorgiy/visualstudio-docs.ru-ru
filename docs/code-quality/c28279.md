# <a name="c28279"></a>C28279
Предупреждение C28279: для символа «begin» обнаружен без сопоставления «end»

 Язык заметок поддерживает begin и end (`{` и `}` в C) конструкции и сопряжение оказался несбалансированной. Такой ситуации можно избежать, если используются макросы.
