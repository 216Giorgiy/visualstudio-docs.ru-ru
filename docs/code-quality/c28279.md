# <a name="c28279"></a>C28279
Предупреждение C28279: Для символа обнаружен begin без соответствующего end

 Заметки языка поддерживает begin и end (`{` и `}` на языке C) конструкции и связывание стало несбалансированной. Такой ситуации можно избежать, если используются макросы.
