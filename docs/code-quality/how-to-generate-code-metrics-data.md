# Практическое руководство. Создание данных для метрик кода
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Результаты метрик кода можно генерировать для всего решения или только для выбранного проекта.  
  
 **Требования**  
  
 Visual Studio 2012 Ultimate, Visual Studio 2012 Premium  
  
### Генерирование результатов метрик кода для всего решения  
  
-   Выберите пункт **Рассчитать метрики кода для решения** в меню **Анализ**.  
  
     \- либо \-  
  
-   В **обозревателе решений** щелкните правой кнопкой мыши соответствующее решение и выберите пункт **Рассчитать метрики кода для решения**.  
  
     \- либо \-  
  
-   В окне **Результаты метрик кода** нажмите кнопку **Рассчитать метрики кода для решения**.  
  
     Результаты генерируются, и на экран выводится окно **Результаты метрик кода**.  
  
### Генерирование результатов метрик кода для одного или нескольких выбранных проектов  
  
1.  Выберите проект или проекты в **обозревателе решений**.  
  
2.  Щелкните **Анализ** и выберите пункт **Создать метрики кода для выбранных проектов**.  
  
     \- либо \-  
  
     Щелкните правой кнопкой мыши выбранные проекты и выберите пункт **Рассчитать метрики кода**.  
  
 Результаты генерируются, и на экран выводится окно **Результаты метрик кода**.  
  
### Просмотр подробных результатов  
  
-   Разверните дерево в столбце **Иерархия**.
