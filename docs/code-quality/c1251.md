# C1251
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

предупреждение C1251: Не удалось загрузить модели  
  
 Средство анализа кода сообщает это предупреждение, когда произошла внутренняя ошибка в файле модели, а не в проанализированными коде.
