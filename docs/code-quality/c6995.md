# C6995
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

предупреждение C6995: Не удалось сохранить XML\-файл журнала.  
  
 Данное предупреждение указывает, что средству анализа кода не удалось создать журнал о дефектах, выявленных в результате анализа кода.  
  
 Эта ошибка может указывать на ошибку диска или указывать, что у вас нет разрешения на создание файла в указанном каталоге.
