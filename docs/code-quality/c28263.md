# C28263
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28263: Во встроенной функции обнаружена синтаксическая ошибка условной аннотации  
  
 Средство анализа кода выводит данное предупреждение, если возвращаемое значение для заданной функции имеет условное значение.  Это предупреждение указывает на ошибку в аннотациях, а не в анализируемом коде.  Если объявление функции находится в заголовочном файле, то аннотация должна быть исправлена таким образом, чтобы она соответствовала заголовочному файлу.  
  
 Список результатов для определенной функции и указанного параметра имеет несколько безусловные значения.  
  
 Как правило, это указывает на то, что несколько безусловных аннотаций `_Null_` и `__drv_valueIs` использовались для определения значения результата.
