# <a name="c28021"></a>C28021
Предупреждение C28021: параметр \<param >, аннотированный с использованием \<anno > должен быть указателем  
  
 Это предупреждение отображается, когда, аннотированный объект не является типом указателя. Эта заметка не может использоваться с `void` или целочисленных типов.
