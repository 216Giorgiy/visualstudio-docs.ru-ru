# C28021
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28021: Параметр \<параметр\>, , аннотированный с использованием \<аннотация\>, должен быть указателем  
  
 Это предупреждение появляется, когда аннотируемый объект не является типом указателя.  Эту аннотацию нельзя использовать с `void` или целочисленными типами.
