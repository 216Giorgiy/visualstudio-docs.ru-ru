# <a name="c28351"></a>C28351
Предупреждение C28351: Аннотация \<заметки > описывает где динамическое значение (переменная) не может использоваться в условии.  
  
 Это предупреждение отображается, когда заметки применяется где проверяемое типа C/C++.  
  
 Выражение в `_When_` должны оцениваться как константы. `_When_` Учитывается.
