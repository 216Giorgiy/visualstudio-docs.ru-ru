# C28290
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28290: Количество внешних элементов в аннотации функции превышает фактическое число параметров  
  
 Средство анализа кода выдает данное предупреждение, если количество внешних параметров аннотации функции превышает количество фактических параметров.
