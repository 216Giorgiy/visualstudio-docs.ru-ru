# <a name="c28290"></a>C28290
Предупреждение C28290: аннотация для функции содержит внешних элементов превышает фактическое число параметров  
  
 Средство анализа кода отображает это предупреждение, когда заметка для функции содержит внешних элементов превышает фактическое число параметров.
