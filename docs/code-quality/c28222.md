# <a name="c28222"></a>C28222
Предупреждение 28222: _да\_, _нет\_, или _Maybe\_ для аннотации  
  
 Это предупреждение означает, что параметр заметки должно быть одним из символов `_Yes_`, `_No_`, или `_Maybe_`, и обнаружен другой символ. Обычно это означает макрос неправильным кодом заметки.
