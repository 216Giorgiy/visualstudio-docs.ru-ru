# C28222
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

предупреждение 28222: \_Yes\_, \_No\_ или \_Maybe\_ ожидаются для аннотации  
  
 Данное предупреждение указывает, что ожидаемый для аннотации параметр должен быть одним из символов `_Yes_`, `_No_` или `_Maybe_`, однако был обнаружен другой символ.  Обычно это означает неправильный закодированный макрос аннотации.
