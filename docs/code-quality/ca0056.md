# <a name="ca0056"></a>CA0056
Файл проекта имеет используется неверная версия файла средство анализа.

 Это предупреждение возникает, если FxCopCmd.exe вызывается вручную с файлом проекта, в котором установлена версия средства анализа неправильный код.
