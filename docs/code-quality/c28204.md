# <a name="c28204"></a>C28204
Предупреждение C28204: \<функция >: только один из Эта перегрузка и по одному \<имя файла > (\<строки >) помечены для \<paramname >: оба или ни должен быть помечен.  
  
 Это предупреждение отображается, если имеется ошибка в аннотации.
