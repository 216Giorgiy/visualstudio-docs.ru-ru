# C28204
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28204: \<функция\> : только данная перегрузка или перегрузка из \<имя\_файла\>\(\<строка\>\) аннотирована для \<имя\_параметра\>: необходимо аннотировать обе перегрузки или ни одну из них.  
  
 Это предупреждение появляется при ошибке в примечаниях.
