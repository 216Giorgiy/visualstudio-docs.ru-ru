# C28285
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28285: Для функции: синтаксическая ошибка в параметре  
  
 Средство анализа кода выдает это предупреждение при обнаружении вероятной ошибки в файле модели.  Несколько ошибок файла источника также могут привести к таким ошибкам.
