# <a name="ca0059"></a>CA0059
Ошибка параметра командной строки.  
  
 Эта ошибка возникает, если обязательный параметр командной строки отсутствует, например,  
  
 / out или не задан параметр/Console. Эта ошибка также возникает при наличии недопустимого параметра, например неверного значения времени ожидания.
