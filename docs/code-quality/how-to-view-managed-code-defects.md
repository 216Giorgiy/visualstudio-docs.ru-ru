# Практическое руководство. Просмотр дефектов управляемого кода
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

При анализе кода ошибки, предупреждения и сообщения отображаются в окне анализа кода.  Окно анализа кода появляется автоматически при запуске анализа кода в проекте или решении.  
  
 В окне анализа кода:  
  
-   **Выберите имя предупреждения.**  
  
     Дополнительные сведения о предупреждении отображаются в окне анализа кода.  Строка исходного файла, которая высвечивает предупреждение, отображается и выделяется в редакторе исходного кода.  
  
-   **Выберите идентификатор предупреждения.** Идентификаторы предупреждений предваряются **CA** и включают четырехзначные номера.  Например, CA1234.  
  
     Отображается раздел справки библиотеки MSDN для предупреждения.  Раздел справки содержит детальную информацию о причине возникновения предупреждения и включает пример, который показывает, как решить возникшую проблему.
