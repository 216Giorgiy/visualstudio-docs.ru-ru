# <a name="how-to-view-managed-code-defects"></a>Практическое руководство. Просмотр дефектов управляемого кода
При анализе кода ошибки, предупреждения и сообщения отображаются в окне анализа кода. Анализ кода окно открывается автоматически при выполнении анализа кода для проекта или решения.  
  
 В окне анализа кода можно:  
  
-   **Выберите заголовок предупреждения.**  
  
     Дополнительные сведения о предупреждении отображается в окне анализа кода. Строки в исходном файле, инициировавший это предупреждение отображается и выделяются в редакторе исходного кода.  
  
-   **Выберите идентификатор предупреждения.** Предупреждение идентификаторы имеют префикс в виде **ЦС** и включать номер из четырех цифр. Например, CA1234.  
  
     Откроется раздел справки библиотеки MSDN для предупреждения. Раздел справки содержит подробные сведения о причинах предупреждения и включает примеры, которые демонстрируют, как возникает предупреждение ошибка и разрешается.
