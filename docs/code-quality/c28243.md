# <a name="c28243"></a>C28243
Предупреждение C28243: аннотация для функции в параметре требуется больше разыменований, чем допускает фактический Аннотируемый тип

 Число операторов `__deref` в примечании больше количества уровней указателя, определенных типом параметра. Устраните причину данного предупреждения путем изменения числа в примечании или уровней указателя параметра, на который была ссылка.
