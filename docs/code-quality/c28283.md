# <a name="c28283"></a>C28283
Предупреждение C28283: для символа: Указанная спецификация размера пока не поддерживается  
  
 Предупреждение указывает, что заметка спецификация размера «sentinel» получено значение, отличное от нуля. По существу вызывающий объект попытался сказать, что строка завершается символ, отличный от двоичного нуля.
