# <a name="c28275"></a>C28275
Предупреждение C28275: Параметр \_макрос\_значение\_ имеет значение null

 Это предупреждение означает, что имеется Внутренняя ошибка в файле модели, а не в коде выполняется анализ. *MacroValue* функция была вызвана без параметров.
