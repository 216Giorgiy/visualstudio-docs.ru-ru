# C28275
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

предупреждение C28275: Значение параметра \_Macro\_value\_ равно NULL.  
  
 Данное предупреждение указывает, что произошла внутренняя ошибка в файле модели, а не в анализируемом коде.  Функция *macroValue* была вызвана без параметров.
