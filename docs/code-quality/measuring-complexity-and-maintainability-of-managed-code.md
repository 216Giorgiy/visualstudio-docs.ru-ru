# Оценка сложности и удобства сопровождения управляемого кода
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Повышенная сложность современных программных приложений также усложняет задачу обеспечения надежности и сопровождаемости кода.  В последние годы было разработано множество методов оценки программного обеспечения, известных как метрики кода, которые призваны помочь разработчикам увидеть участки кода, требующие переработки или усиленного тестирования.  
  
 Разработчики могут использовать [!INCLUDE[vsprvsts](../code-quality/includes/vsprvsts_md.md)] для получения данных метрик кода, оценивающих сложность и удобство сопровождения созданного ими управляемого кода.  Данные метрик кода можно получить для всего решения или для отдельного проекта.  
  
 В данном подразделе описывается порядок получения и использования метрик кода в [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].
