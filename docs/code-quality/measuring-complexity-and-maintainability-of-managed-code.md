# <a name="measuring-complexity-and-maintainability-of-managed-code"></a>Оценка сложности и удобства сопровождения управляемого кода
Повышенная сложность современных приложений также затрудняют надежным и простым в обслуживании, что делает код. В последние годы, многие меры программного обеспечения, известных как метрики кода были разработаны, призваны помочь разработчикам увидеть участки кода, требующие переработки или увеличение тестирования.  
  
 Разработчики могут использовать [!INCLUDE[vsprvsts](../code-quality/includes/vsprvsts_md.md)] для получения данных метрик кода, оценивающих сложности и удобства сопровождения управляемого кода. Данные метрик кода может быть создан для всего решения или для отдельного проекта.  
  
 В этом разделе описываются способы создания и использования метрик кода в [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].
