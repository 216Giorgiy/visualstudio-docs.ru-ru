# <a name="c26471-noreinterpretcastfromvoidptr"></a>C26471 NO_REINTERPRET_CAST_FROM_VOID_PTR

Не используйте `reinterpret_cast`. Можно использовать приведения из void * `static_cast`. См. в разделе [Type.1 правила C++ Core](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-reinterpretcast).
