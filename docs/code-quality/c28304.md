# <a name="c28304"></a>C28304
Предупреждение C28304: C++ для ссылочного параметра < имя_параметра > "," неправильно размещенные `_Notref_` найден оператор применяется к \<token >.  
  
 `_Notref_` Оператор должен использоваться только в особых обстоятельствах, включающие параметры ссылка C++ и только в макросах, предоставляемых системой. Он должен сразу следовать `_Deref_` оператор или функциональной заметки.
