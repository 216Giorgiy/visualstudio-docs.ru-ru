# <a name="c28216"></a>C28216
Предупреждение C28216: _Check_return\_ Аннотация применяется только к постусловиям для конкретного параметра функции.

 `_Check_return_` Заметки применен в контексте, отличный от `__post`; он может понадобиться `__post` (или `__drv_out`) модификатор, или она может размещаться неправильно.
