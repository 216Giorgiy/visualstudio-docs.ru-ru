# C28216
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

предупреждение C28216: Аннотация \_Check\_return\_ применяется только к постусловиям для параметра конкретной функции.  
  
 Аннотация `_Check_return_` применялась в контексте отличном от `__post`; Возможно необходим модификатор `__post` \(или `__drv_out`\) или данное размещение является неверным.
