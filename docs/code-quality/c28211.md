# <a name="c28211"></a>C28211
Предупреждение C28211: Для SAL_context ожидалось имя статического контекста

 Это предупреждение указывает, что операнд `_Static_context_` заметки должно быть имя контекста, определенное средством. Это предупреждение не должна возникать при нормальном использовании.
