# C28211
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28211: Для SAL\_context ожидалось имя статического контекста  
  
 Это предупреждение указывает, что операнд в аннотации `_Static_context_` должен быть именем, определенным контекстом инструмента.  Это предупреждение не возникает при обычном использовании.
