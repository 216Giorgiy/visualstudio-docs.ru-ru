# <a name="c28211"></a>C28211
Предупреждение C28211: для SAL_context Ожидалось имя статического контекста  
  
 Это предупреждение означает, что операнд `_Static_context_` заметки должно быть имя, определенное средством контекста. Это предупреждение не должна возникать при нормальном использовании.
