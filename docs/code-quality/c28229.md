# C28229
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28229: Аннотация не поддерживает параметры  
  
 Это предупреждение возникает, если аннотация, объявленная без параметров, используется с параметрами.  В случае использования стандартных макросов для аннотаций это не представляется возможным, таким образом, при типичном применении можно не ожидать появления данного предупреждения.
