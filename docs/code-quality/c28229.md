# <a name="c28229"></a>C28229
Предупреждение C28229: Аннотация не поддерживает параметры  
  
 Это предупреждение означает, что заметки использовался с параметром, когда заметка объявлена без параметров. Это не должна быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не должны отображаться в типичное использование.
