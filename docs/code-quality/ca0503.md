# <a name="ca0503"></a>CA0503
Свойство является устаревшим. Заменяющее свойство  
  
 CA0503 предупреждение создается в том случае, когда значение в файле конфигурации сборки или проекта содержит свойства анализа кода не рекомендуется. Чтобы устранить это предупреждение, используйте указанное свойство, которое заменяет его.
