# C28208
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Предупреждение C28208: Функция \<функция\> была ранее определена с другим списком параметров в \<файл\>\(\<строка\>\).  Некоторые средства анализа дадут неверные результаты.  
  
 Это предупреждение возникает, если известное объявление функции не соответствует другому вхождению.
