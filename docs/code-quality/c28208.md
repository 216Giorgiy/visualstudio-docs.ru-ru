# <a name="c28208"></a>C28208
Предупреждение C28208: функция \<функция > был ранее задан с другим списком параметров в \<файл > (\<строки >). Некоторые средства анализа вернут неверные результаты

 Это предупреждение отображается, когда известная определение функции не соответствует другим вхождения.
