# <a name="ca0505"></a>CA0505
Нерекомендуемое свойство будет игнорироваться, так как определено более приоритетное свойство.  
  
 Предупреждение СА505 создается, когда указанные нерекомендуемые и приоритетные свойства анализа кода содержат значения в проект или файл конфигурации сборки. Чтобы устранить эту проблему, удалите свойство не рекомендуется.
