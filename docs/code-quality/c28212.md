# <a name="c28212"></a>C28212
Предупреждение C28212: для аннотации ожидалось выражение указателя  
  
 Это предупреждение указывает параметр нумерованной заметки (не функцию, аннотированный) должно быть указателя или тип массива, что обнаружен другого типа. Заметку необходимо исправить.
