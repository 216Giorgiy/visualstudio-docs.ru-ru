# Невозможно прочитать поток, переданный в TextFieldParser.
[Объект TextFieldParser](/dotnet/visual-basic/language-reference/objects/textfieldparser-object) не может прочитать поток, который был в него передан. Это может быть результатом попытки чтения файла, который не является текстовым.  
  
### Исправление ошибки  
  
-   Убедитесь, что файл является текстовым файлом.
