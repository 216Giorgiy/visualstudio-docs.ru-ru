# &lt;name&gt; is not valid editor for this file type.
Эта ошибка обычно возникает, если в окне **Команда** или поле **Найти\/Команда** введено неверное значение аргумента \/e:*имя\_редактора*.  
  
### Чтобы исправить эту ошибку  
  
1.  Проверьте по документации правильность синтаксиса \/e:*имя\_редактора* и введите команду еще раз.
