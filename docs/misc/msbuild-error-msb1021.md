# Ошибка MSBuild MSB1021
**Не удается создать экземпляр средства ведения журнала.**  
  
 Экземпляр средства ведения журнала не может быть создан из его сборки.  
  
### Чтобы исправить эту ошибку  
  
-   Дополнительные сведения о причинах и способах устранения ошибки MSBuild см. в разделе [Дополнительные ресурсы](../msbuild/additional-msbuild-resources.md).
