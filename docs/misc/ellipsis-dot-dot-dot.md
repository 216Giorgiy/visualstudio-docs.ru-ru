# Многоточие (...)
Многоточие \(`...`\) как оператор в C и C\+\+ имеет несколько предназначений, которые зависят от контекста.  Изначально в языке C он использовался как абстрактный декларатор для списков с переменным количеством аргументов. Кроме того, он используется при программировании макросов препроцессора в форме макросов с переменным количеством аргументов.  В C\+\+ оператор `...` используется в блоках обработки исключений `catch`, а в C\+\+11 — для шаблонов с переменным числом аргументов.
