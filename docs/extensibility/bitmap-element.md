# Битовая карта, элемент
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Определяет растровое изображение. Точечный рисунок загружается из ресурса или из файла.  
  
## Синтаксис  
  
```  
<Bitmap guid="guidImages" href="images\MyImage.bmp" usedList="bmp1, bmp2, bmp3" />  
```  
  
## Атрибуты и элементы  
 В следующих разделах описаны атрибуты, дочерние и родительские элементы.  
  
### Атрибуты  
  
|Атрибут|Описание|  
|-------------|--------------|  
|Идентификатор GUID|Обязательный. Идентификатор GUID идентификатор GUID и идентификатор команды.<br /><br /> Атрибут guid для точечного рисунка не связан с любой VSPackage или другие группы команд.  Он должны быть уникальными для определения точечного рисунка и не должен использоваться для других целей.|  
|resID|Идентификатор идентификатор GUID и идентификатор команды. Требуется resID или атрибут href.<br /><br /> Атрибут resID имеет целочисленный идентификатор ресурса, определяющий панель изображений, который должен быть загружен во время слияния таблицы команд.  При загрузке таблицы команд из ресурса тот же модуль загружается растровых изображений, заданных с помощью идентификатора ресурса.|  
|usedList|Требуется, если присутствует атрибут resID. Выбор доступных образов в полосе точечного рисунка.|  
|href|Путь к точечному рисунку. Требуется resID или атрибут href.<br /><br /> Включить пути к файлу указанное изображение, внедренная в результирующий двоичный файл.  Во время слияния таблицы команды образ копируется и не поиск дополнительных ресурсов или нагрузки не требуется.  Если отсутствует атрибут usedList, доступны все образы в полосе. **Note:**  Изображения может быть предоставлен в одном из следующих форматов, включающие .bmp, .gif и .png.  Более ранние версии компилятора не поддерживали 32\-разрядных точечных рисунков, которые имели данные альфа\-канала для частичную прозрачность. Решение для этих версий — использовать формат PNG.|  
|Условие|Необязательный. См. раздел [Условные атрибуты](../extensibility/vsct-xml-schema-conditional-attributes.md).|  
  
### Дочерние элементы  
 Отсутствует.  
  
### Родительские элементы  
  
|Элемент|Описание|  
|-------------|--------------|  
|[Растровые изображения элемента](../extensibility/bitmaps-element.md)|Группирует элементы точечного рисунка.|  
  
## Пример  
  
```  
<Bitmap guid="guidWidgetIcons" href="WidgetToolbarIcons_32.bmp" /> <Bitmap guid="guidWidgetIcons2" resID="IDBMP_WIDGETICONS" usedList="1, 2, 3, 4"/>  
```
