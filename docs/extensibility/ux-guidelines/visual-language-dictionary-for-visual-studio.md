# <a name="visual-language-dictionary-for-visual-studio"></a>Словарь визуального языка для Visual Studio
Visual Studio использует общий язык visual продемонстрировать основные понятия, которые являются составными частями его иллюстрациях и других визуальных представлений. При разработке новых значков, этот словарь можно используйте для определения понятия, которые вы хотите передать свой собственный образ.  
  
 **Общие рекомендации**  
  
-   Эти концепции visual должен использоваться согласованно с их определение словаря. В противном случае — риска, толку пользователей, которые предполагают Общие элементы для имеют согласованные значения.  
  
-   Выберите только наиболее соответствующие или важные основные понятия для использования в вашей значков, чтобы изображение четким и легко отличить.  
  
-   Используйте только один или два понятия каждого изображения.  
  
-   Дополнительные стратегии проектирования значков см [значок конструктора](../../extensibility/ux-guidelines/images-and-icons-for-visual-studio.md#BKMK_IconDesign).
