# <a name="extensibility-in-visual-studio"></a>Расширяемость в Visual Studio
Visual Studio предоставляет множество возможностей расширяемости. Можно создать собственные пакеты SDK, Windows, Windows Phone и Azure SDK (который устанавливаются как часть Visual Studio) и расширения Visual Studio.  
  
## <a name="extend-visual-studio"></a>Расширение Visual Studio  
 Пакет SDK для Visual Studio можно использовать для расширения почти любой части Visual Studio: команд, меню, windows, редакторы и проекты. Для получения дополнительных см [пакет SDK для Visual Studio](../extensibility/visual-studio-sdk.md).  
  
## <a name="create-your-own-sdks"></a>Создать собственные пакеты SDK  
 Узнайте, как создать, упаковать и развернуть собственные пакеты SDK платформы и расширение: [создание средств разработки программного обеспечения](../extensibility/creating-a-software-development-kit.md).  
  
## <a name="roslyn-extensibility"></a>Расширяемость Roslyn  
 Узнайте, как использовать Roslyn для расширения: [расширяемость платформы компилятора .NET («Roslyn»)](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md).
