# Диспетчер процессов отладки
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Процесс отладки \(диспетчер PDM\) \- это компонент Visual Studio, которое управляет программы и процессы, что делает их доступными для сеанса отладки диспетчера и обработчиков отладки.  
  
 PDM управляет всеми процессами, которые могут быть отлажены.  Быть отлаживанным программа должна быть зарегистрирована с PDM.  Это делается при запуске программы или портом или обработчиком отладки.
