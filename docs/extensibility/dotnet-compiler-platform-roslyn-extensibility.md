# Расширяемость платформы (&#171;Roslyn&#187;) компилятора .NET
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Основная миссия платформы компилятора .NET \(«Roslyn»\) откройте компиляторы C\# и Visual Basic и позволяя средствам и разработчикам для совместного использования в компиляторах, подробные сведения о программах. Средства анализа кода повысить качество кода и кода поможет генераторы построение приложения. Как средства получения быстрее, они должны доступ к более и более сложного кода знания, обладают только компиляторы. А не непрозрачные преобразователи \(исходный код и код объекта ожидания\), компиляторы Roslyn предоставляют интерфейсы API, которые можно использовать для задач, связанных с кодом, средств и приложений.  
  
 Лучше всего то, что компиляторы Roslyn, их интерфейсы API, примеры и пошаговые руководства и реальные средства, которые базируются на эти API\-интерфейсы являются полностью открытым исходным кодом на [github.com\/dotnet\/roslyn](https://github.com/dotnet/Roslyn). Перейдите на сайт ОС для получения дополнительных сведений и начало работы с Roslyn. Вы найдете ссылки для получения последних C\# и VB функции, которые можно использовать в качестве конечного пользователя, а также ссылки на начало работы как построитель средства, используя интерфейсы API Roslyn.
