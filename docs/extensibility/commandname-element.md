# Элемент CommandName
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

`CommandName` Элемент указывает текст, который отображается в категории «клавиатура» **Параметры** диалоговом и в **команды** список **Настройка** диалоговое окно.  
  
## Синтаксис  
  
```  
<CommandName>MyCommand</CommandName>  
```  
  
## Атрибуты и элементы  
 В следующих разделах описаны атрибуты, дочерние и родительские элементы.  
  
### Атрибуты  
 Отсутствует.  
  
### Дочерние элементы  
 Отсутствует.  
  
### Родительские элементы  
  
|Элемент|Описание|  
|-------------|--------------|  
|[Элемент строки](../extensibility/strings-element.md)|Группирует текстовые элементы, такие как `ButtonText` и `CommandName`.|
