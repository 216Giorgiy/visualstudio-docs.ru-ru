# Распространение приложений изолированной оболочки
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Чтобы создать приложение изолированной оболочки необходимо установить Visual Studio и Visual Studio SDK. Для распространения приложения на компьютерах других пользователей или клиентов, необходимо включить специальные распространяемого пакета для изолированной оболочки.  
  
## Необходимые условия для распространения приложений изолированной оболочки  
  
|Имя|Описание|  
|---------|--------------|  
|SDK для Visual Studio|Пакет SDK, необходимо иметь для разработки и тестирования расширений [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]. Также можно использовать пакет SDK для создания собственного экземпляра оболочки Visual Studio изолированной.<br /><br /> Visual Studio является необходимым условием для пакета SDK.|  
|Распространяемый пакет оболочки изолированного Microsoft Visual Studio|Распространяемый пакет, включить в программу установки, при сборке средства и среду Visual Studio изолированной оболочки. Распространяемый пакет оболочки изолированной включает .NET Framework 4.5.|  
  
## Создание программы установки для приложения  
 Программа специальной установки необходимо создать для приложения встроенной или изолированной оболочки. Для получения дополнительной информации см. [При установке приложения изолированной оболочки](../extensibility/installing-an-isolated-shell-application.md).  
  
## Поддержка обновлений приложения  
 Программе установки необходимо разрешить вероятность того, что приложения будут обновляться, центра обновления Майкрософт или обновления вашей компании. Дополнительные сведения об обновлениях см. в разделе [Обслуживание рекомендации для изолированной оболочки приложений](../extensibility/servicing-guidelines-for-isolated-shell-applications.md).
