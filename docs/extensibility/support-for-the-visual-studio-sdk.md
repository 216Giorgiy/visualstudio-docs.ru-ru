# <a name="support-for-the-visual-studio-sdk"></a>Поддержка Visual Studio SDK
Несколько вариантов поддержки доступны для пользователей с вопросами о способах расширения Visual Studio интегрированной среды разработки (IDE).  
  
## <a name="free-support"></a>Бесплатная поддержка  
  
### <a name="msdn-development-center"></a>Центр разработчиков MSDN  
 [Центр разработчиков расширенных возможностей Visual Studio](http://go.microsoft.com/fwlink/?LinkID=84381) на сайте MSDN содержит ресурсы, которые позволяют отвечать на вопросы о том, как расширить возможности Visual Studio. Например существует несколько ссылок на блоги разработчиков VSIP.  
  
### <a name="microsoft-newsgroups"></a>Группы новостей Microsoft  
 Члены группой разработки Visual Studio и другими разработчиками, часто расширения Visual Studio участвовать в группе новостей microsoft.public.vstudio.extensibility. Этой группе новостей является управляемые группы новостей. При наличии универсального подписчиков MSDN, убедитесь, что регистрации псевдонимом высылать спам, позволяют корпорации Майкрософт для ответа.  
  
### <a name="msdn-forums"></a>Форумы MSDN  
 [Форумы MSDN](http://go.microsoft.com/fwlink/?LinkID=76632) предоставляют другой способ общение с коллегами и экспертами. Графический пользовательский интерфейс (UI) форумы призвана сделать их более удобными в использовании, чем групп новостей. [Форум расширяемости Visual Studio](http://go.microsoft.com/fwlink/?LinkID=121964) является неплохим началом для поиска ответов на вопросы расширяемости. Всегда искать, прежде чем задать вопрос, поскольку другой пользователь может уже задавали тот же вопрос.  
  
### <a name="visual-studio-gallery"></a>Галерея Visual Studio  
 [Коллекции Visual Studio](http://visualstudiogallery.msdn.microsoft.com/) — место для расширения и примеры кода. Можно найти средства, повышающие производительность и пользователь опыт работы с Visual Studio, примеры приложений, которые демонстрируют использование VSPackages и Managed Extensibility Framework (MEF) и другие виды образцов.  
  
### <a name="codeplex"></a>CodePlex  
 [CodePlex](http://go.microsoft.com/fwlink/?LinkId=76627) является открытым исходным кодом проекта размещения веб-узел Microsoft. CodePlex можно использовать для создания и совместного использования новых проектов, участвовать в существующих проектах или работы с приложениями на узле и отправки отзывов. Все средства PowerToy (и сопутствующая документация), которые были включены в более ранних версиях [!INCLUDE[vsipsdk](../extensibility/includes/vsipsdk_md.md)] теперь доступны на сайте CodePlex.  
  
## <a name="included-support"></a>Включить поддержку  
  
### <a name="visual-studio-product-support"></a>Поддержка продуктов Visual Studio  
 Некоторые поддержка включается при покупке [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] продуктов. Дополнительные сведения см. в разделе [поддержка Visual Studio](http://msdn.microsoft.com/vstudio/cc136615.aspx). Корпорация Майкрософт поддерживает выделенный поддержки для ответов на вопросы о [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] расширяемости.  
  
### <a name="msdn-subscription-support"></a>Поддержка подписки MSDN  
 Пакеты подписки MSDN могут также включать некоторую поддержку. Дополнительные сведения см. в разделе [поддержки для подписчиков MSDN](https://msdn.microsoft.com/subscriptions/aa718661.aspx).  
  
## <a name="paid-support"></a>Платная поддержка  
  
### <a name="developer-tools-support"></a>Поддержка средств разработчика  
 Можно также приобрести поддержки. Доступны различные возможности поддержки категории средств разработчика на [центр справки и поддержки центры](http://go.microsoft.com/fwlink/?LinkID=82383).  
  
 Клиенты, желающие использовать [!INCLUDE[vsipsdk](../extensibility/includes/vsipsdk_md.md)] для разработки сложных пакетов можно приобрести контракт с Microsoft Premier Support. Дополнительные сведения см. в разделе [Microsoft Premier Support](http://go.microsoft.com/fwlink/?LinkID=76660). Для небольших компаний [преимущества поддержки партнеров](http://www.microsoft.com/services/microsoftservices/srv_mspa.mspx) может быть Заманчивой альтернативой традиционных Premier Support предложения.  
  
> [!NOTE]
>  Нельзя использовать стандартной поддержки, консультации и получить рекомендации.  
  
 Вопросы о предложения Premier Support вместе с программе VSIP могут отправляться на [ vsipsvs@microsoft.com ](mailto:vsipsvs@microsoft.com). Вопросы о новой программе Partner Advantage могут отправляться на [ advantg@microsoft.com ](mailto:advantg@microsoft.com).  
  
 Сведения о поддержки отдельных запросов можно найти на [отдельных запросов поддержки](http://go.microsoft.com/fwlink/?LinkID=82385).
