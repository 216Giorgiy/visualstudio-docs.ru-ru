# <a name="creating-an-extension-with-a-tool-window"></a>Создание расширения с помощью окна инструментов
В этой процедуре вы узнаете, как использовать шаблон проекта VSIX и **окно инструментов настраиваемый** шаблона элемента для создания расширения с окном инструментов.  
  
## <a name="prerequisites"></a>Предварительные требования  
 Начиная с Visual Studio 2015, не установить пакет SDK для Visual Studio из центра загрузки Майкрософт. Он включен в качестве дополнительного компонента в программе установки Visual Studio. VS SDK также можно установить позже. Дополнительные сведения см. в разделе [Установка пакета SDK для Visual Studio](../extensibility/installing-the-visual-studio-sdk.md).  
  
### <a name="creating-a-tool-window"></a>Создание окна инструментов  
  
1.  Создайте проект VSIX с именем **FirstWindow**. Шаблон проекта VSIX в можно найти **новый проект** диалогового окна в разделе **Visual C# / Extensibility**.  
  
2.  При открытии проекта, добавление шаблона элемента окно инструментов с именем **MyWindow**. В **обозревателе решений**, щелкните правой кнопкой мыши узел проекта и выберите **добавить / новый элемент**. В **Добавление нового элемента** диалоговое окно, последовательно выберите пункты **Visual C# / расширяемости** и выберите **собственное окно средства**. В **имя** в нижней части окна, измените имя файла инструмента окна **MyWindow.cs**.  
  
3.  Выполните сборку решения и запустите отладку.  
  
     Откроется экспериментальный экземпляр Visual Studio. Дополнительные сведения о экспериментальный экземпляр, в разделе [экспериментальный экземпляр](../extensibility/the-experimental-instance.md).  
  
4.  В экспериментальном экземпляре откройте **представления и другие окна**.  
  
     Вы увидите пункт меню для **MyWindow**. Щелкните его.  
  
     Должно появиться окно инструментов с заголовком **MyWindow** и кнопка фраза про **Click Me!.**
