# Автоматическое форматирование в языковую службу для прежних версий
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

С автоматическим форматированием, служба языка автоматически вставляет фрагмент кода, когда пользователь начинает введите известную конструкции кода.  
  
## Реакция на событие Автоматического форматирования  
 Например, при вводе `if`служба языка автоматически вставляет соответствующий фигурные скобки или при нажатии клавиши ВВОД, то служба языка точку вставки на новой линии к соответствующему уровню отступа в зависимости от, проверяет ли предыдущий открыть линии вверх в другую область.  
  
 Фильтр команды, используемый для остальной части языковой службы могут также использоваться для автоматического форматирования.  Можно также выбрать соответствующий фигурные скобки, вызвав <xref:Microsoft.VisualStudio.TextManager.Interop.IVsTextView.HighlightMatchingBrace%2A>.
