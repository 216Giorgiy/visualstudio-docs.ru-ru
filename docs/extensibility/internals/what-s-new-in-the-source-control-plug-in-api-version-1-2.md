# Новые возможности в подключаемый модуль API версия системы управления версиями 1.2
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Этот раздел дополнительно выбирает функции управления исходным кодом в версии 1.2 API системы управления версиями вставляемой.  Далее перечислены новые функции в версии 1.2.  
  
## Изменения  
 [Устранение ~ SAK файлов](../../extensibility/internals/elimination-of-tilde-sak-files.md)  
 Описание api\-интерфейса 1,2 системы управления версиями подключаемый модуль взаимодействует с временными файлами.  
  
 [Применение параметров в нескольких соединениях проекта](../../extensibility/internals/application-of-settings-across-multiple-project-connections.md)  
 Показывает, как подключаемый модуль системы управления версиями можно использовать пакетную операцию для выполнения одной и той же операция системы управления версиями с помощью нескольких проектов или нескольких контекстах соединения.  
  
 [Создание папки родительского контейнера для решения](../../extensibility/internals/creating-parent-container-folders-for-solutions.md)  
 Объясняет, как версия 1.2 API системы управления версиями подключаемых модулей позволяет пользователю задать единое назначение системы управления версиями корня для всех веб\-проекта в решении.  
  
 [Необязательные сравнения папки локального проекта в системе управления версиями](../../extensibility/internals/optional-comparison-of-local-project-folder-to-source-control-store.md)  
 Описание api\-интерфейса 1,2 подключаемый модуль системы управления версиями сравниваются версии файлов.  
  
 [Удаление сведения системы управления версиями из. Proj и. SLN\-файлы](../../extensibility/internals/removal-of-source-control-information-from-dot-proj-and-dot-sln-files.md)  
 Объясняет, как версия 1.2 API системы управления версиями подключаемых модулей хранит сведения о проектах и решениях.
