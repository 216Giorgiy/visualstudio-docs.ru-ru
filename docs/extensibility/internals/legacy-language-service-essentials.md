# <a name="legacy-language-service-essentials"></a>Essentials устаревших языковой службы
Укажите языковую службу для интеграции языка программирования в Visual Studio. В этом разделе объясняется, функции, доступные в устаревших языковые службы.  
  
 Устаревший языковые службы реализуются как частью VSPackage, но это использование расширений MEF новый способ реализации возможностей службы языка. Чтобы больше узнать о новый способ реализации службы языка, в разделе [редактора и расширения языка службы](../../extensibility/editor-and-language-service-extensions.md).  
  
> [!NOTE]
>  Мы рекомендуем начать использовать новый редактор API как можно быстрее. Это улучшает производительность службы языка и позволяют воспользоваться преимуществами новых функций редактора.  
  
 Устаревший языковые службы предоставляют следующие возможности:  
  
|Функция|Описание|  
|-------------|-----------------|  
|Цветовая подсветка синтаксиса|В результате представление редактора отобразить различные цвета и стили шрифтов для различных элементов языка. Этих различий может упростить чтение и изменение файлов.<br /><br /> Общие сведения см. в разделе [цветовой подсветки синтаксиса в языковую службу для прежних версий](../../extensibility/internals/syntax-coloring-in-a-legacy-language-service.md).<br /><br /> Сведения об этой функции в платформа управляемых пакетов (MPF) в разделе [синтаксиса, выделение цветом в языковую службу для прежних версий](../../extensibility/internals/syntax-colorizing-in-a-legacy-language-service.md).|  
|Завершение операторов|Выполняет инструкцию или ключевое слово, которое пользователь начинает ввод. Завершение операторов помогает пользователям более легко вводить сложных инструкций меньший объем ввода и меньше вероятность ошибок.<br /><br /> Общие сведения см. в разделе [завершение инструкции в языковую службу для прежних версий](../../extensibility/internals/statement-completion-in-a-legacy-language-service.md).<br /><br /> Сведения об этой функции в MPF разделе [завершение слов в языковую службу для прежних версий](../../extensibility/internals/word-completion-in-a-legacy-language-service.md).|  
|Проверка парности фигурных скобок|Основные особенности пару символов, таких как фигурные скобки. Когда пользователь вводит закрывающего символа, таких как «}», парные фигурные скобки выделяет соответствующий открывающий символ, такой как «{». При наличии нескольких уровней окружающие символы, эта функция позволяет пользователям получить подтверждение, что окружающие символы указаны правильно.<br /><br /> Сведения об этой функции в MPF разделе [парные фигурные скобки в языковую службу для прежних версий](../../extensibility/internals/brace-matching-in-a-legacy-language-service.md).|  
|Параметр сведения подсказки|Отображает список возможных подписей для перегруженный метод, который в настоящее время, вводимого пользователем.<br /><br /> Общие сведения см. в разделе [сведения о параметрах в языковую службу для прежних версий](../../extensibility/internals/parameter-info-in-a-legacy-language-service1.md).<br /><br /> Сведения об этой функции в MPF разделе [сведения о параметрах в языковую службу для прежних версий](../../extensibility/internals/parameter-info-in-a-legacy-language-service2.md).|  
|Маркеры ошибок|Отображает красной волнистой линией, также известный как волнистая, под текстом, синтаксически неверен. Ошибки обычно используются маркеры в информировании пользователей о ключевых слов с ошибками, незакрытые скобки, недопустимые символы и аналогичные сообщения об ошибке.<br /><br /> В классах MPF маркеры ошибки обрабатываются автоматически в <xref:Microsoft.VisualStudio.Package.AuthoringSink.AddError%2A>метод <xref:Microsoft.VisualStudio.Package.AuthoringSink>класса.</xref:Microsoft.VisualStudio.Package.AuthoringSink> </xref:Microsoft.VisualStudio.Package.AuthoringSink.AddError%2A>|  
  
 Многие из этих функций требуется служба языка для анализа исходного кода. Часто можно повторно использовать маркирование и анализа кода компилятором или интерпретатором.  
  
 Следующие характеристики относятся к поддержка языков программирования, но не являются частью языка службы:  
  
|Функция|Описание|  
|-------------|-----------------|  
|Вычислители выражений|Поддерживает [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] отладчик путем проверки точек останова и указания списка выражений для отображения в **видимые** окно отладки.<br /><br /> Дополнительные сведения см. в разделе [языковой службы поддержки для отладки](../../extensibility/internals/language-service-support-for-debugging.md).|  
|Средства обзора символов|Поддерживает **обозреватель объектов**, **представление классов**, **Обозреватель вызовов**, и **результаты поиска символа**.|
