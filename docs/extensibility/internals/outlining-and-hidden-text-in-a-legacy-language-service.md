# <a name="outlining-and-hidden-text-in-a-legacy-language-service"></a>Структуры и скрытый текст в языковую службу для прежних версий
Структурирование позволяет свернуть последовательность текстовых строк в одну строку. Например в C++ все методы можно свернуть, чтобы одна строка, показывающая сигнатуру метода. Скрытый текст — это последовательность текстовых строк, которые могут быть отображены или скрыты.
