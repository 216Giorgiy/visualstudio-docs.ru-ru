# Способствовали модели автоматизации
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Visual Studio предоставляет набор интерфейсов автоматизации для настройки среды. Модель автоматизации — объектная модель, которая позволяет пользователям создавать надстройки Visual Studio и расширения.  
  
 Кроме того это уместно, как разработчик VSPackage внести свой вклад в модели автоматизации. Таким образом, включение конечных пользователей для создания надстроек и предоставляют пользователю согласованный интерфейс модели при использовании VSPackage в VSPackage [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)].  
  
 Чтобы согласовать работы конечного пользователя, необходимо выполнить набор рекомендаций по разработке VSPackage, чтобы модель автоматизации для VSPackage соответствует идеи в [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)].
