# <a name="web-projects"></a>Веб-проекты
Веб-проект системы — это система, создающий веб-проектов. Веб-проектов, в свою очередь, создавать веб-приложения. Вы можете программировать с использованием традиционных языков программирования, таких как веб-страницу [!INCLUDE[vbprvb](../../code-quality/includes/vbprvb_md.md)] или [!INCLUDE[csprcs](../../data-tools/includes/csprcs_md.md)], для сбора и обработки сведений от пользователя, сохранить его в базе данных и т.д. Веб-страницы с помощью кода называются смарт-веб-страниц.
