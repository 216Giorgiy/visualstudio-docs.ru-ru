# <a name="wizard-vsz-file"></a>Мастер (. Файл VSZ)
Интегрированную среду разработки (IDE) использует VSZ-файлы для запуска мастера. Эти VSZ-файлы содержат информацию, которую использует интегрированной среды разработки, чтобы определить, какой мастер следует вызвать и какие сведения для передачи в мастер.  
  
 VSZ-файл — это версия файла .ini форматированный текст, который не содержит разделов. В Интегрированной среде разработки хранится в начале файла. Это обеспечивает связь между мастер, который вызывает интегрированной среды разработки и параметры, которые находятся в VSZ-файл будет передан в Интегрированной среде разработки. Остальная часть файла предоставляет параметры, заданные в мастере и собирать в интегрированной среде разработки или переданы конкретного мастера.  
  
 Следующий пример показывает содержимое VSZ-файл.  
  
```  
VSWizard 8.0  
Wizard=VsWizard.CBlankSiteWizard -or- {123-1234556-123334}  
Param="WIZARDNAME = Wizard One"  
Param="WIZARDUI = FALSE"  
```  
  
 Ниже перечислены компоненты в VSZ-файле.  
  
|Отделение|Описание|  
|----------|-----------------|  
|VSWizard|Первый параметр в файле — номер версии формата файла шаблона. Этот номер версии должен быть 6.0, 7.0, 7.1 или 8.0. Другие номера не может быть запущена и приводят к ошибке Недопустимый формат.|  
|Мастер|Это поле содержит OLE идентификатор ProgID мастера, или строковое представление GUID для CLSID мастера, который создается в интегрированной среде разработки.|  
|Param|Эти элементы являются необязательными. Можно добавить столько необходимые.|  
  
 Параметры позволяют передавать дополнительные пользовательские параметры мастера VSZ-файл. Каждое значение передается как строковый элемент в массиве значений типа Variant в мастер. Дополнительные сведения см. в разделе [пользовательские параметры](../../extensibility/internals/custom-parameters.md). Дополнительные сведения об использовании VSZ-файл в разработке пользовательских мастеров в разделе [. VSZ-файл (элемент управления проекта)](/visual-cpp/ide/dot-vsz-file-project-control)  
  
 Чтобы добавить код языка по умолчанию VSZ-файла, укажите `FALLBACK_LCID`= xxxx, где xxxx — идентификатор языка, например 1033 для английского языка. Когда `FALLBACK_LCID` определен параметр, мастер использует идентификатор предоставленного резервного языкового стандарта, если текущий идентификатор не найден.
