# <a name="default-command-group-and-toolbar-placement"></a>Команда по умолчанию, группы и расположение панели инструментов
Для продукта единообразие и стабильности, пользовательский Интерфейс отображает определенных групп, команда по умолчанию и [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] предоставляет определения для команд и группы команд. Пакеты VSPackage также можно использовать стандартные команды и группы команд.  
  
 Группы по умолчанию команда делятся на три категории: IDE команды, команды продукта и команд редактора.  
  
## <a name="default-ide-commands"></a>По умолчанию команд интегрированной среды разработки  
 Панели инструментов по умолчанию интегрированная среда разработки включает команды, используемые все продукты, содержащиеся в [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]. Сюда относятся команды, относящиеся к операции универсального проекта, такие как **Сохранить** команды и **добавить элемент** команды. Пакеты VSPackage не следует добавить или вычесть из этой панели инструментов, за одним исключением: Если продукт или VSPackage добавляет новое окно инструментов, то окно должны быть добавлены в список окон инструментов, доступных на **представление** меню. Новые продукты или пакеты VSPackage может добавлять свои собственные панели инструментов.  
  
## <a name="default-product-commands"></a>Команды продукта по умолчанию  
 Каждый продукт может предоставить среду IDE собственную панель инструментов по умолчанию, содержащий важные и часто используемые команды. Рекомендуется, однако для использования существующего меню и панелей инструментов, когда это возможно и дополнить других панелей инструментов конкретных задач при необходимости.  
  
 Поле «приоритет» для панели инструментов определяет его размещение строк. Ноль приоритет помещает панели инструментов в третьей строке (строка 3) под строкой меню (строка 1) и **Стандартная** панель инструментов (строка 2). Таким образом, другие панели инструментов отображаются в строке (приоритет + 3). Последующие панели инструментов, помещаются в ту же строку, если есть место; в противном случае они автоматически сдвигаются на следующую строку.  
  
## <a name="default-editor-commands"></a>По умолчанию для команд редактора  
 Пакет VSPackage, который предоставляет пользовательский редактор должен предоставить панель инструментов по умолчанию, которая содержит наиболее важные и часто используемые команды в этом редакторе. Панель инструментов редактора откроется при редактора активна и должен быть скрыт, если редактор не активен. Эта видимость осуществляется в `VisibilityConstraints Element` файла vsct.  
  
 Редактор панелей инструментов должен располагаться под панелей инструментов интегрированной среды разработки и продукта.
