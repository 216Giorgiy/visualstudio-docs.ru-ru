# <a name="default-command-group-and-toolbar-placement"></a>Команда, группы и размещения панели инструментов по умолчанию
Единообразие продукта и стабильность, пользовательский Интерфейс отображает определенные группы команд по умолчанию, и [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] содержит определения для команд и группы команд. Пакеты VSPackage можно также использовать стандартные команды и группы команд.  
  
 Группы по умолчанию команда делятся на три категории: IDE команды, команды продукта и команд редактора.  
  
## <a name="default-ide-commands"></a>По умолчанию команды интегрированной среды разработки  
 Панель инструментов интегрированной среды разработки по умолчанию содержит команды, общими для всех продуктов, содержащихся в [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]. Сюда относятся команды, относящиеся к универсальная операций, таких как **Сохранить** команды и **добавить элемент** команды. Пакеты VSPackage не следует добавить или вычесть из этой панели инструментов, за одним исключением: Если продукт или VSPackage добавляет новое окно инструмента, а затем окна должны добавляться в список доступных инструментов windows на **представление** меню. Новые продукты или VSPackages можно добавить собственные панели инструментов.  
  
## <a name="default-product-commands"></a>По умолчанию команды продукта  
 Каждый продукт может предоставить интегрированная среда разработки с собственную панель инструментов по умолчанию, содержит важные и часто используемые команды. Рекомендуется, однако для использования существующего меню и панелей инструментов, возможности и дополнить других панелей инструментов конкретных задач при необходимости.  
  
 Поле приоритета для панели инструментов определяет его размещение строк. Ноль приоритет помещает панели инструментов в третьей строке (строка 3) под строкой меню (строка 1) и **Стандартная** инструментов (строка 2). Таким образом, другие панели инструментов отображаются в строке (приоритет + 3). Последующие панелей инструментов помещаются в той же строке, если места; в противном случае они автоматически перемещаются на следующую строку.  
  
## <a name="default-editor-commands"></a>По умолчанию редактор команд  
 VSPackage, который предоставляет пользовательский редактор должен предоставить панель инструментов по умолчанию, которая содержит наиболее важные и часто используемые команды в этом редакторе. Панели инструментов редактора появится, когда редактор активен и должен быть скрыт, если редактор не активен. Эта видимость контролируется в `VisibilityConstraints Element` файла .vsct.  
  
 Редактор панелей инструментов должны находиться ниже панели инструментов интегрированной среды разработки и продукта.
