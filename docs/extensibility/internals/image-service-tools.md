# Средства обновления образа
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

VS SDK включает в себя несколько средств, помогающих построители расширения, которые используют службу образов в Visual Studio.  
  
-   [Манифеста из ресурсов](../../extensibility/internals/manifest-from-resources.md) Инструмент принимает список ресурсов образа \(файлы с расширением PNG или .xaml\) и создает файл манифеста изображения.  
  
-   [Манифеста для кода](../../extensibility/internals/manifest-to-code.md) Инструмент принимает файл манифеста изображения и создает файл программы\-оболочки для использования в файлах C\+\+, C\#, VB или .vsct.  
  
-   [Средство просмотра изображений библиотеки](../../extensibility/internals/image-library-viewer.md) Нагрузки, редактирования и поиска манифесты изображения.
