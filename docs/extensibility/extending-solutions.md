# Расширение решения
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Решение — это группа из одного или нескольких проектов, работающих совместно для создания приложения.  
  
## Содержание  
 В следующем примере демонстрируются некоторые способы, в которой можно расширять решения.  
  
1.  [Расширение фильтра обозревателя решений](../extensibility/extending-the-solution-explorer-filter.md) Описание фильтра, отображаются в обозревателе решений.  
  
2.  [Управление загрузкой проекта в решении](../extensibility/managing-project-loading-in-a-solution.md) содержит описание процесса управления загрузкой проекта для повышения производительности, когда пользователь загружает решение.
