# <a name="add-extensions-to-dsl-definitions"></a>Для добавления расширений определения DSL

Модуль определения DSL позволяет создать пакет расширения для доменного языка (DSL). Так же, как DSL расширения DSL, которое содержится в Visual Studio Integration Extension (VSIX), можно установить на компьютере пользователя. Дополнительные возможности можно динамически включать и отключать во время выполнения. Нет необходимости явно разрабатывать расширения доменного языка и расширения могут создать позже, или третьими сторонами, без изменения расширенной DSL.

DSL расширения могут включать следующие возможности:

-   Свойства элементов модели и представления

-   Декораторы фигур и соединителей

-   Классы, связи, фигуры и соединители

-   Ограничения проверки

-   Вкладки и элементы панели элементов

Пользователь расширенные DSL можно создать и сохранить модель, которая содержит экземпляры дополнительные возможности. Модель может быть прочитан другим пользователям, установившим соответствующее расширение. Пользователи, которых не установлены расширения не могут использовать дополнительные возможности, но их можно обновлять и сохранить модель без потери дополнительные возможности.
