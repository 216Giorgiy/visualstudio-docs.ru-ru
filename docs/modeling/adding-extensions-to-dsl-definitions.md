# <a name="adding-extensions-to-dsl-definitions"></a>Добавление расширений в определения доменных языков
Модуль определения DSL можно создать пакет расширения для доменного языка (DSL). Расширение доменного языка, который содержится в Visual Studio Integration Extension (VSIX), можно установить на компьютере пользователя, так же, как DSL. Дополнительные функции можно динамически включать и отключать во время выполнения. DSL нет необходимости явно разрабатываться для расширения и расширения могут создать позже или сторонними производителями без изменения расширенной DSL.  
  
 Дополнительные возможности могут включать следующее:  
  
-   Свойства для элементов модели и презентации  
  
-   Декораторов фигур и соединителей  
  
-   Классы, связи, фигур и соединителей  
  
-   Ограничения проверки  
  
-   Элементы панели элементов и вкладок  
  
 Пользователь расширенной DSL можно создать и сохранить модель, которая содержит экземпляры дополнительные возможности, и они могут быть прочитаны других пользователей, установивших соответствующего расширения. Пользователи, у которых не установлено расширение не могут использовать дополнительные функции, но их можно обновить и сохранить модель без потери дополнительные функции.
