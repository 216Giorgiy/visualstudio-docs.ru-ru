## <a name="deployment-editions"></a>Развертывание выпусков  
 [!INCLUDE[dsl](../modeling/includes/dsl_md.md)]поддерживает следующие конфигурации для доменных языков, создаваемые развертывания:  
  
-   Visual Studio Enterprise  
  
-   Visual Studio Professional  
  
-   Распространяемый пакет распространяемого пакета Visual Studio Shell (интегрированный режим)  
  
-   Оболочка Visual Studio Shell (изолированный режим), распространяемый пакет.  
  
> [!NOTE]
>  Чтобы работать на продукте оболочки доменный язык DSL, необходимо задать **поддерживается выпуск VS** в манифеста расширения. Дополнительные сведения см. в разделе [развертывание решений доменный язык](../modeling/deploying-domain-specific-language-solutions.md).
