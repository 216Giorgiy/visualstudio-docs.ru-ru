## <a name="deployment-editions"></a>Развертывание выпусков  
 [!INCLUDE[dsl](../modeling/includes/dsl_md.md)]Развертывание доменного языка, создаваемые поддерживает следующие конфигурации:  
  
-   Visual Studio Enterprise  
  
-   Visual Studio Professional  
  
-   Visual Studio Shell (интегрированный режим) распространяемый пакет.  
  
-   Оболочка Visual Studio Shell (изолированный режим), распространяемый пакет.  
  
> [!NOTE]
>  Чтобы сделать DSL, работать на продукте оболочки, необходимо задать **поддерживаемая версия VS** в манифесте расширения. Дополнительные сведения см. в разделе [развертывание решений доменного языка](../modeling/deploying-domain-specific-language-solutions.md).
