Начиная с Visual Studio 2017, конструкторов UML были удалены из Visual Studio. Дополнительные сведения см. в этом [блога](https://blogs.msdn.microsoft.com/visualstudioalm/2016/10/14/uml-designers-have-been-removed-layer-designer-now-supports-live-architectural-analysis/).
