# <a name="create-uml-modeling-projects-and-diagrams"></a>Создание проектов и схем моделирования UML

Начиная с Visual Studio 2017 г., UML-конструкторов были удалены из Visual Studio. Дополнительные сведения см. в этой [блога](https://blogs.msdn.microsoft.com/visualstudioalm/2016/10/14/uml-designers-have-been-removed-layer-designer-now-supports-live-architectural-analysis/).

Если вам нужно использовать UML, можно использовать Visual Studio 2015 Enterprise edition, которая side-by-side совместим с более поздними версиями Visual Studio.

Сведения об использовании UML в Visual Studio 2015 см. в разделе:

* [Создание моделей для приложения](https://msdn.microsoft.com/library/dd409436%28v=vs.140%29.aspx)
* [Использование моделей в процессе разработки](https://msdn.microsoft.com/library/dd409423%28v=vs.140%29.aspx)
* [Проверка системы в ходе разработки](https://msdn.microsoft.com/library/dd409448%28v=vs.140%29.aspx)
