# Свойства дорожек
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Можно добавить прокрутки к схеме.  Вертикальные или отслеживание разбивают схему в горизонтальных области.  Можно указать других фигур, отображаемых в дорожек.  Дополнительные сведения см. в разделе [Определение доменного языка](../modeling/how-to-define-a-domain-specific-language.md).  Дополнительные сведения о методах использования этих свойств см. в разделе [Настройка и расширение доменного языка](../modeling/customizing-and-extending-a-domain-specific-language.md).  
  
 Дорожки имеют свойства, которые перечислены ниже в таблице.  
  
|Свойство.|Описание|Default|  
|---------------|--------------|-------------|  
|Цвет заливки тела сообщения|Цвет заливки для тела отслеживания.|Белый|  
|Цвет заливки заголовка|Цвет заливки для заголовка отслеживания.|DarkGray|  
|Цвет разделителя|Цвет линий разделителя.|LightGray|  
|Стиль линий разделителя|Стиль линий разделителя \(`Solid`"  `Dash`"  `Dot`"  `DashDot`"  `DashDotDot`или  `Custom`\).|`Dash`|  
|Ширина разделителя|Толщина линий разделителя в дюймах.|0.03125|  
|Цвет текста|Цвет, используемый для оформителей текст, связанных с данной дорожкой.|Черный|  
|Модификатор доступа|Уровень доступа класса \(`public` OR  `internal`\).|Открытый|  
|Пользовательские атрибуты|Используется для добавления атрибутов к классу кода, сформированного из этой отслеживания.|\<отсутствует\>|  
|Создает производный тип double|If `True`и будут создаваться базовый класс и разделяемый класс \(поддержки настройку до конца переопределяет\).  Дополнительные сведения см. в разделе [Переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md).|False|  
|Имеет пользовательский конструктор|If `True`, обеспечат пользовательский конструктор в исходном коде.  Дополнительные сведения см. в разделе [Переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md).|False|  
|Модификатор наследования|Описывает тип наследования класса исходного кода, созданных прокрутки \(`none`"  `abstract` OR  `sealed`\).|нет|  
|Базовая дорожка|Базовый класс этой отслеживания.|\(нет\)|  
|Имя|Имя данной отслеживания.|Текущее имя|  
|Пространство имен|Пространство имен, которое принимано с данной дорожкой.|Текущее пространство имен|  
|Тип подсказки|Подсказка указана \(как`fixed`"  `variable`или  `none`\).  If `fixed`затем значение  `Fixed Tooltip Text` свойство используется. If  `variable`после этого подсказка задана в пользовательском коде.|\<отсутствует\>|  
|Примечания|Неофициальные заметки, связанные с дорожкой.|\<отсутствует\>|  
|Выравнивание|Горизонтальное или вертикальное выравнивание.|Вертикальный|  
|Начальная высота|Начальная высота данной отслеживания, в дюймах.  Применимо только к горизонтальным дорожкам.|0|  
|Начальная ширина|Начальная ширина этой отслеживания, в дюймах.  Применимо только к вертикальной дорожкам.|0|  
|Предоставляет цвет текста|If `True`пользователь может задать цвет прокрутки в конструкторе.  Чтобы задать этот параметр, щелкните правой кнопкой мыши форму и выберите команду отслеживать **Добавьте предоставляется**.|False|  
|Описание|Используется в документ создаваемый конструктор.|\<отсутствует\>|  
|Отображаемое имя|Имя, которое будет отображаться в конструкторе для обращения к этому классу отслеживания.|\<отсутствует\>|  
|Фиксированный текст всплывающей подсказки|Текст, используемый для фиксированной подсказки.|\<отсутствует\>|  
|Ключевое слово справки|Ключевое слово, используемое для справки F1 индекса для этой отслеживания.|\<отсутствует\>|
