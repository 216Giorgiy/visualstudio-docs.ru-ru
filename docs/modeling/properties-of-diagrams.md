# <a name="properties-of-diagrams"></a>Свойства схем
Можно задать свойства, определяющие, каким образом диаграммы будет отображаться в созданном конструкторе. Например можно указать цвет по умолчанию для текста на схеме.

 Дополнительные сведения см. в разделе [способ определения доменного языка](../modeling/how-to-define-a-domain-specific-language.md). Дополнительные сведения об использовании этих свойств см. в разделе [Настройка и расширение доменного языка](../modeling/customizing-and-extending-a-domain-specific-language.md).

 Ниже перечислены свойства схемы.

|Свойство.|Описание|Значение по умолчанию|
|-|-|-|
|Цвет заливки|Цвет заливки для схемы.|Белый|
|Цвет текста|Цвет текста, который отображается на схеме.|Черный|
|Модификатор доступа|Модификатор доступа класса (общедоступного или внутреннего).|Public|
|Настраиваемые атрибуты|Используется, чтобы добавить атрибуты в классе сформированного кода.|\<None >|
|Создает двойной производным|Если `True`, будет создан базовый класс и разделяемый класс (для поддержки настройки посредством переопределений). Дополнительные сведения см. в разделе [переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md).|False|
|Имеет пользовательский конструктор|Если `True`, пользовательский конструктор будет предоставляться в исходном коде. Дополнительные сведения см. в разделе [переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md)...|False|
|Модификатор наследования|Описывает тип наследования класса источника кода, который создается из схемы (`none`, `abstract` или `sealed`).|Нет|
|Базовая схема|Базовый класс на этой схеме.|(нет)|
|name|Имя этой схеме.|Текущее имя|
|Пространство имен|Пространство имен, которое входит в состав этой схеме.|Текущее пространство имен|
|Класс, представленный|Корневой класс домена, на этой схеме представлены.|Текущий корневой класс, если применимо|
|Примечания|Неофициальные примечания, связанный с данным элементом.|\<None >|
|Цвет заливки предоставляет как свойство|Если `True`, пользователь может задать цвет заливки схемы созданного конструктора. Для этого щелкните правой кнопкой мыши форму диаграммы и нажмите кнопку **добавить Explosed**.|False|
|Предоставляет цвет текста в виде свойства|Если `True`, пользователь может задать цвет текста диаграммы в созданном конструкторе. Для этого щелкните правой кнопкой мыши форму диаграммы и нажмите кнопку **добавить Explosed**.|False|
|Описание|Описание, которое используется для документирования созданного конструктора.|\<None >|
|Отображаемое имя|Имя, которое будет отображаться в созданном конструкторе для этой схеме.|\<None >|
|Ключевое слово Help|Ключевое слово, используемое для индексации справки F1 для этой схеме.|\<None >|
