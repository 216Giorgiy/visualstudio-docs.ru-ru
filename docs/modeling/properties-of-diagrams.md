# <a name="properties-of-diagrams"></a>Свойства схем
Можно задать свойства, определяющие, как схемы отображается в конструкторе созданный. Например можно указать цвет по умолчанию для текста на диаграмме.

 Дополнительные сведения см. в разделе [способ определения доменного языка](../modeling/how-to-define-a-domain-specific-language.md). Дополнительные сведения об использовании этих свойств см. в разделе [настройку и расширение доменного языка](../modeling/customizing-and-extending-a-domain-specific-language.md).

 Ниже перечислены свойства схемы.

|Свойство.|Описание|Значение по умолчанию|
|--------------|-----------------|-------------|
|Цвет заливки|Цвет заливки для диаграммы.|Белый|
|Цвет текста|Цвет текста, который отображается на диаграмме.|Черный|
|Модификатор доступа|Модификатор доступа класса (public или internal).|Public|
|Настраиваемые атрибуты|Используется для добавления атрибутов к классу сформированного кода.|\<None >|
|Приводит к возникновению ошибки двойной производный|Если `True`, будут создаваться базового класса и разделяемого класса (для поддержки настройки с помощью переопределения). Дополнительные сведения см. в разделе [переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md).|False|
|Имеет пользовательский конструктор|Если `True`, пользовательский конструктор будет предоставляться в исходном коде. Дополнительные сведения см. в разделе [переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md)...|False|
|Модификатор наследования|Описывает тип наследования класса исходного кода, созданной из схемы (`none`, `abstract` или `sealed`).|Нет|
|Диаграмма, базовый|Базовый класс на этой схеме.|(нет)|
|name|Имя этой схемы.|Текущее имя|
|Пространство имен|Пространство имен, которое связано с этой схеме.|Текущее пространство имен|
|Класс, представленный|Корневой класс домена, представляющий эту диаграмму.|Текущий корневой класс, если применимо|
|Примечания|Неформальные заметки, связанные с данным элементом.|\<None >|
|Предоставляет цвет заливки как свойство|Если `True`, пользователь может задать цвет заливки диаграммы созданный конструктором. Для этого щелкните правой кнопкой мыши фигуру диаграммы и нажмите кнопку **добавить Explosed**.|False|
|Представляет цвет текста в виде свойства|Если `True`, пользователь может задать цвет текста диаграммы в конструкторе созданный. Для этого щелкните правой кнопкой мыши фигуру диаграммы и нажмите кнопку **добавить Explosed**.|False|
|Описание|Описание, которое используется для документирования созданный конструктором.|\<None >|
|Отображаемое имя|Имя, которое будет отображаться в конструкторе, созданный для этой схемы.|\<None >|
|Ключевое слово Help|Ключевое слово, используемое для индексации справки F1 для этой схемы.|\<None >|
