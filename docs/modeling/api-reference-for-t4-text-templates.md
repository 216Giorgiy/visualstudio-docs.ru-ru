# API Reference for T4 Text Templates
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

API текстовых шаблонов позволяет вызывать и настраивать преобразование [текстовые шаблоны](../modeling/code-generation-and-t4-text-templates.md).  
  
## Пространства имен  
  
|Пространство имен|Назначение|  
|-----------------------|----------------|  
|<xref:Microsoft.VisualStudio.TextTemplating>|Содержит классы, позволяющие использовать функциональные возможности преобразования текстовых шаблонов.  Модуль преобразования текстовых шаблонов интегрирован в Visual Studio и преобразует файлы текстовых шаблонов в сгенерированные выходные текстовые файлы.|  
|<xref:Microsoft.VisualStudio.TextTemplating.Modeling>|Предоставляет средства преобразования текста в UML\-моделях и доменных языках, например доступ к шине [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] ModelBus.|  
|<xref:Microsoft.VisualStudio.TextTemplating.VSHost>|Предоставляет доступ к службе текстовых шаблонов в [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].|
