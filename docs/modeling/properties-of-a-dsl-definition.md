# <a name="properties-of-a-dsl-definition"></a>Свойства определения доменного языка
Определение свойства DslDefinition *доменного языка* определение свойств, например нумерация версий. Свойства DslDefinition отображаются в **свойства** окно при щелкните пустую область схемы в *конструктора доменного языка*.  
  
 Дополнительные сведения см. в разделе [способ определения доменного языка](../modeling/how-to-define-a-domain-specific-language.md). Дополнительные сведения об использовании этих свойств см. в разделе [Настройка и расширение доменного языка](../modeling/customizing-and-extending-a-domain-specific-language.md).  
  
 DslDefinition имеет свойства в таблице ниже.  
  
|Свойство|Описание|По умолчанию|  
|--------------|-----------------|-------------|  
|Модификатор доступа|Определяет, является ли модификатор доступа для класса домена public или internal.|public|  
|Настраиваемые атрибуты|Пользовательские атрибуты для класса домена.<br /><br /> **Примечание** используйте кнопку обзора, чтобы добавить атрибут.|\<Нет настроек|  
|Название организации|Имя текущего имени компании в системном реестре.|Текущее имя компании|  
|Имя|Имя этого класса домена.|Текущее имя|  
|Пространство имен|Пространство имен, связанный с этим классом домена.|Текущее пространство имен|  
|Идентификатор Guid пакета|Идентификатор guid для [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] пакет, созданный для этого доменного языка.|\<Нет настроек|  
|Пространство имен пакетов|Пространство имен для [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] пакет, созданный для этого доменного языка.|\<Нет настроек|  
|Название продукта|Имя продукта, который будет зарегистрирован для [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] пакет, созданный для этого доменного языка.|\<Нет настроек|  
|Примечания|Заметки, связанные с этим классом домена.|\<Нет настроек|  
|Описание|Описание этого класса домена.|\<Нет настроек|  
|Отображаемое имя|Имя, которое будет отображаться в созданном конструкторе для этого класса домена.|\<Нет настроек|  
|Ключевое слово Help|Ключевое слово справки, связанное с этим классом домена.|\<Нет настроек|  
|Построить|Номер инкрементное построение данного определения доменного языка.|0|  
|Основной номер версии|Добавочное основного номера построения для данного определения доменного языка.|1|  
|Дополнительный номер версии|Добавочное дополнительного номера построения для данного определения доменного языка.|0|  
|Номер редакции|Добавочные версии номер сборки для данного определения доменного языка.|0|
