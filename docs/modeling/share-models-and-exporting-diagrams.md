# <a name="share-models-and-exporting-diagrams"></a>Общий доступ к моделям и экспорт схем
В Visual Studio есть несколько способов организовать совместное использование моделей и схем с коллегами.  
  
 Чтобы узнать, какие версии Visual Studio поддерживают эту функцию, см. раздел [поддержка версий для инструментов моделирования и архитектуры](../modeling/what-s-new-for-design-in-visual-studio.md#VersionSupport).  
  
## <a name="working-on-a-model-as-a-team"></a>Работа с моделью в команде  
 Пользователи Visual Studio можно создавать и редактировать модели, схемы зависимостей и направленные графы.  
  
 Разные пользователи могут работать одновременно на модели, но должны работать в разных пакетах.  
  
 Мы рекомендуем использовать [!INCLUDE[esprscc](../code-quality/includes/esprscc_md.md)].  
  
 Дополнительные сведения см. в разделе [управление моделей и схем в систему управления версиями](../modeling/manage-models-and-diagrams-under-version-control.md).  
  
 Рекомендации по организации моделей в среднем или большом проекте см. в разделе [структуры решения моделирования](../modeling/structure-your-modeling-solution.md).  
  
## <a name="reading-and-reviewing-models"></a>Чтение и рецензирование моделей  
 Участники команды, использующие некоторые версии Visual Studio (который нельзя создавать модели) могут читать модели, схемы моделирования и направленные графы, но не позволяет их редактировать.  Чтобы узнать, какие версии Visual Studio поддерживают эту функцию, см. раздел [поддержка версий для инструментов моделирования и архитектуры](../modeling/what-s-new-for-design-in-visual-studio.md#VersionSupport).  
  
 Диаграмма зависимостей может быть открыт только в пределах модели, в котором он был создан. Чтобы предоставить доступ к схеме, нужно предоставить доступ ко всем файлам проекта моделирования или к их копиям.  
  
 Дополнительные сведения см. в разделе [чтение моделей и схем в других выпусках Visual Studio](../modeling/read-models-and-diagrams-in-other-visual-studio-editions.md).  
  
## <a name="using-diagrams-in-documents-and-presentations"></a>Использование схем в документах и презентациях  
 Вы можете копировать части схем в документы или презентации. Также можно печатать схемы, например, в XPS-файлы.  
  
 Дополнительные сведения см. в разделе [экспортировать схемы в виде изображения](../modeling/export-diagrams-as-images.md).
