# <a name="creating-a-wpf-based-domain-specific-language"></a>Создание доменного языка на основе WPF
Можно создать доменный язык, который имеет конструктор WPF, а не графический конструктор.  
  
 Сведения и примеры для этой функции можно найти на визуализации Visual Studio средства моделирования веб-сайта и на [http://go.microsoft.com/fwlink/?LinkId=186128](http://go.microsoft.com/fwlink/?LinkId=186128)
