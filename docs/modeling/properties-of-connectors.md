# Свойства соединителей
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Соединители \- доменных ссылки в конструкторе.  
  
 Дополнительные сведения см. в разделе [Определение доменного языка](../modeling/how-to-define-a-domain-specific-language.md).  Дополнительные сведения о методах использования этих свойств см. в разделе [Настройка и расширение доменного языка](../modeling/customizing-and-extending-a-domain-specific-language.md).  
  
 Соединители имеют свойства, которые перечислены ниже в таблице.  
  
|Свойство.|Описание|Default|  
|---------------|--------------|-------------|  
|Цвет|Цвет данного соединителя.|Черный|  
|Стиль штриха|Стиль пунктирных линий для линии для данного соединителя \(твердого тела сообщения, тире, эллипсы, DashDot, DashDotDot или пользовательского\).|Заливка|  
|Стиль элемента источника|Стиль окончания источника для этого соединителя \(HollowArrow, EmptyArrow, FilledArrow, EmptyDiamond, FilledDiamond или никакого\).|None|  
|Стиль элемента целевого объекта|Стиль окончания целевого объекта для данного соединителя \(HollowArrow, EmptyArrow, FilledArrow, EmptyDiamond, FilledDiamond или никакого\).|None|  
|Цвет текста|Цвет, используемый для оформителей текст, связанные с этим соединителем.|Черный|  
|Thickness|Толщина линии для данного соединителя, измеренной в дюймах.|0.03125|  
|Модификатор доступа|Уровень доступа класса \(`public` OR  `internal`\).|Открытый|  
|Пользовательские атрибуты|Используется для добавления атрибутов к классу исходного кода, созданного из этого соединителя.|\<отсутствует\>|  
|Создает производный тип double|If `True`и будут создаваться базовый класс и разделяемый класс \(поддержки настройку до конца переопределяет\).  Дополнительные сведения см. в разделе [Переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md).|False|  
|Имеет пользовательский конструктор|If `True`, обеспечат пользовательский конструктор в исходном коде.  Дополнительные сведения см. в разделе [Переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md).|False|  
|Модификатор наследования|Описывает тип наследования класса исходного кода, сформированного из соединителей \(`none`"  `abstract` OR  `sealed`\).|нет|  
|Базовый соединитель|Базовый класс этого соединителя.|\(нет\)|  
|Имя|Имя данного соединителя.|Текущее имя|  
|Пространство имен|Пространство имен, которое принимано с этим соединителем.|Текущее пространство имен|  
|Тип подсказки|Подсказка задана в качестве \(зафиксировано, переменная или нет\).  Если зафиксировано, затем значение `Fixed Tooltip Text` свойство используется в качестве подсказки; если переменная, затем подсказка задана в пользовательском коде.|\<отсутствует\>|  
|Примечания|Неофициальные заметки, связанные с этим соединителем.|\<отсутствует\>|  
|Стиль маршрутизации|Стиль, используемый для маршрутизации соединителя.  A `Rectilinear` соединитель делает прав\-angled превращается в зависимости от необходимости. a  `Straight` соединитель не делает.|Прямолинейно|  
|Цвет, предоставляемый как свойство<br /><br /> Предоставленный стиль штриха, как свойство<br /><br /> Толщина предоставляемая как свойство<br /><br /> Предоставляет цвет текста|If `True`пользователь может задать заявленное свойство фигуры.  Чтобы задать этот параметр, щелкните правой кнопкой мыши определение фигур и нажмите кнопку **Добавьте предоставляется**.|False|  
|Описание|Используется в документ создаваемый конструктор.|\<отсутствует\>|  
|Отображаемое имя|Имя, которое будет отображаться в конструкторе для данного соединителя.|\<отсутствует\>|  
|Фиксированный текст всплывающей подсказки|Текст, используемый для фиксированной подсказки.|\<отсутствует\>|  
|Ключевое слово справки|Ключевое слово, используемое для справки F1 индекса для данного элемента.|\<отсутствует\>|
