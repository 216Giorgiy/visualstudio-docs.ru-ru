# <a name="properties-of-geometry-shapes"></a>Свойства геометрических фигур
Чтобы указать, как экземпляры классов доменов отображаются в доменный язык можно использовать геометрических фигур. Дополнительные сведения см. в разделе [способ определения доменного языка](../modeling/how-to-define-a-domain-specific-language.md). Дополнительные сведения об использовании этих свойств см. в разделе [Настройка и расширение доменного языка](../modeling/customizing-and-extending-a-domain-specific-language.md).  
  
 Геометрических фигур, имеют свойства, которые перечислены в следующей таблице.  
  
|Свойство|Описание|По умолчанию|  
|--------------|-----------------|-------------|  
|Цвет заливки|Цвет заливки для этой фигуры.|Белый|  
|Заливка градиентный режим|Режим заливки градиента этой фигуры (горизонтального и вертикального, диагональный вперед, назад по диагонали или нет).|Горизонтально|  
|Геометрия|Геометрия эта фигура (прямоугольник, прямоугольник со скругленными углами, эллипс или кружок).|Прямоугольник|  
|Содержит точки подключения по умолчанию|Если `True`, фигура будет использовать верхней, нижней, левой и правой соединения указывает в созданном конструкторе.|False|  
|Цвет контура|Цвет контура для этой фигуры.|Черный|  
|Структура пунктирных линий|Тип штриха структуры этой фигуры (сплошная, тире, точки, DashDot, DashDotDot или пользовательского).|Сплошная|  
|Толщина контура|Толщина контура этой фигуры.|0.03125|  
|Цвет текста|Цвет, используемый для декораторы текста, которые связаны с этой фигуры.|Черный|  
|Модификатор доступа|Модификатор доступа класса (public или internal).|Public|  
|Настраиваемые атрибуты|Используется для добавления атрибутов класса исходного кода, созданный для этой фигуры.|\<Нет настроек|  
|Создает двойной производный|Если `True`, будет создан базовый класс и разделяемого класса (для поддержки настройки с помощью переопределения). Дополнительные сведения см. в разделе [переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md).|False|  
|Имеет пользовательский конструктор|Если `True`, пользовательский конструктор будет предоставляться в исходном коде. Дополнительные сведения см. в разделе [переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md).|False|  
|Модификатор наследования|Описывает тип наследования класса исходного кода, созданной из фигуры (`none`, `abstract` или `sealed`).|Нет|  
|Базовый геометрическая фигура|Базовый класс для этой фигуры.|(нет)|  
|Имя|Имя этой фигуры.|Текущее имя|  
|Пространство имен|Пространство имен, которое связано с этой фигуры.|Текущее пространство имен|  
|ToolTip-тип|Определение подсказки (Фиксированная, переменной или нет). Если так, то значение `Fixed Tooltip Text` свойство используется в качестве подсказки; Если переменной, то подсказки определен в пользовательском коде.|Нет|  
|Примечания|Неофициальные примечания, связанные с этим элементом.|\<Нет настроек|  
|Начальная высота|Начальная высота эту фигуру в дюймах.|1|  
|Начальная ширина|Начальная ширина эту фигуру в дюймах.|1.5|  
|Цвет заливки предоставляется как свойство<br /><br /> Градиентный режим предоставляется заливки<br /><br /> Цвет контура в виде свойства<br /><br /> Свойства в виде пунктирных линий структуры<br /><br /> Толщина контура как свойство предоставляется<br /><br /> Предоставляет цвет текста|Если `True`, пользователь может задать свойство заявленного фигуры. Для этого щелкните правой кнопкой мыши определение фигуры и нажмите кнопку **добавить предоставленный**.|False|  
|Описание|Описание, которое используется для документирования созданный конструктором.|\<Нет настроек|  
|Отображаемое имя|Имя, которое будет отображаться в конструкторе, созданный для этой фигуры.|\<Нет настроек|  
|Фиксированным текстом всплывающей подсказки|Текст, который используется для основных всплывающей подсказки.|\<Нет настроек|  
|Ключевое слово Help|Ключевое слово, используемое для индексации справки F1 для этой фигуры.|\<Нет настроек|
