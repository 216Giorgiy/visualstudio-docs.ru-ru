# Общие сведения о платформах построения
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Можно сохранять различные версии решения и свойств проекта, которые применяются к платформам другого целевого объекта.  Например, можно создать конфигурацию отладки, на которую нацелен x86 платформы и конфигурации отладки, на которую нацелен платформа x64.  Можно быстро изменить активная среда так, чтобы можно было легко нескольких конфигураций построения.
