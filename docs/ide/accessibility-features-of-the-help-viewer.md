# Специальные возможности средства просмотра справки
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Корпорация Microsoft стремится к тому, чтобы сделать использование своих продуктов и служб простым и удобным для каждого.  В этот раздел входит информация о функциях, продуктах и службах, помогающих предоставить окно справки \(Майкрософт\) 2.2 для людей с разными физическими возможностями.  
  
## Доступ с клавиатуры  
 Вы можете получить доступ ко всем функциям окна справки, используя клавиатуру.  Дополнительные сведения см. в разделе [Сочетания клавиш \(средство просмотра справки\)](../ide/shortcut-keys-help-viewer.md).  
  
## Font size  
 Вы можете изменить размер шрифта текста, отображаемого в окне документа.  Дополнительные сведения см. в разделе [Настройка средства просмотра справки](../ide/customize-the-help-viewer.md).  
  
## Размер окна  
 Вы можете изменить ширину окна навигации или окна документа с помощью разделителя между двумя окнами.  Когда курсор изменится на двустороннюю стрелку, используйте основную кнопку мыши, чтобы перетащить разделитель вправо или влево.  
  
## Положение окна справки  
 Вы можете изменить положение окна справки, перетащив заголовок окна в другое место.  
  
## Параметры Internet Explorer  
 Вы можете сделать окно справки более доступным, изменив параметры Internet Explorer, связанные с отображением веб\-страниц.  Дополнительные сведения см. в разделе [Настройка средства просмотра справки](../ide/customize-the-help-viewer.md).
