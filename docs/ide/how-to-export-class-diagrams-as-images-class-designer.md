# How to: Export Class Diagrams As Images (Class Designer)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Чтобы экспортировать схему классов, созданную на основе кода в проекте, сохраните ее как изображение.  Если вместо этого необходимо экспортировать схемы класса UML, см. раздел [Экспорт схем в виде изображений](../modeling/export-diagrams-as-images.md).  
  
1.  Откройте файл со схемой классов \(CD\).  
  
2.  В меню **Схема классов** или в контекстном меню поверхности схемы выберите пункт **Экспортировать схему как рисунок**.  
  
3.  Выберите схему.  
  
4.  Выберите нужный формат.  
  
5.  Выберите **Экспорт**, чтобы завершить процедуру экспорта.  
  
     Чтобы автоматически обновить экспортированные изображения, на которые есть ссылки в других документах, еще раз экспортируйте схему в Visual Studio.
