# <a name="generate-a-method-in-visual-basic"></a>Создание метода в Visual Basic
**Что?** Вы можете сразу же добавить метод в класс. 

**Когда?** Вы представляете новый метод и хотите правильно его объявить автоматически.  

**Зачем?** Вы можете объявить метод и параметры перед их использованием, но эта функция позволяет создать объявление автоматически. 

**Как?**

1. Поместите курсор в строку с красной волнистой линией. Она указывает, что вы использовали метод, который еще не существует.

   ![Выделенный код](media/method-highlight-vb.png)

1. Затем выполните одно из следующих действий:
   * **Клавиатура**
     * Нажмите клавиши **Ctrl+.**, чтобы активировать меню **Быстрые действия и рефакторинг**. Затем во всплывающем окне предварительного просмотра выберите пункт **Создать метод**.
   * **Мышь**
     * Щелкните правой кнопкой мыши и выберите меню **Быстрые действия и рефакторинг**. Затем выберите во всплывающем окне предварительного просмотра пункт **Создать метод**.
     * Наведите указатель мыши на красную волнистую линию и щелкните ![значок лампочки,](media/bulb-vb.png) который отображается.
     * Нажмите кнопку ![значок лампочки,](media/bulb-vb.png) который отображается в левом поле, если текстовый курсор уже находится в строке выбора с красной волнистой линией.

   ![Предварительный просмотр создаваемого метода](media/method-preview-vb.png)

   >[!TIP]
   >Щелкните ссылку [**Просмотреть изменения**](../../ide/preview-changes.md) в нижней части окна предварительного просмотра, чтобы просмотреть все будущие изменения перед выбором.

1. Метод будет создан автоматически со всеми параметрами с учетом его использования.

   ![Результат создания метода](media/method-result-vb.png)
