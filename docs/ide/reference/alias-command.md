# Команда Alias
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Создается новый псевдоним для полной команды, полной команды с аргументами или для другого псевдонима.  
  
> [!TIP]
>  Если набрать `>alias` без каких\-либо аргументов отображается текущий список псевдонимов и их определений.  
  
## Синтаксис  
  
```  
Tools.Alias [/delete] [/reset] [aliasname] [aliasstring]  
```  
  
## Аргументы  
 `aliasname`  
 Необязательный.  Имя для нового псевдонима.  Если для `aliasname` не предоставляется значение, выводится список текущих псевдонимов и их определений.  
  
 `aliasstring`  
 Необязательный.  Полное имя команды или существующий псевдоним и все параметры, которые должен заменять создаваемый псевдоним.  Если для `aliasstring` не предоставлено значение, для указанного псевдонима отображается имя псевдонима и строка псевдонима.  
  
## Переключатели  
 \/delete или \/del или \/d  
 Необязательный.  Указанный псевдоним удаляется; одновременно он удаляется из функции автозавершения.  
  
 \/reset  
 Необязательный.  Для списка стандартных псевдонимов устанавливаются исходные параметры.  то есть все стандартные псевдонимы восстанавливаются, а все псевдонимы, определенные пользователем, удаляются.  
  
## Заметки  
 Поскольку псевдонимы представляют команды, они должны располагаться в начале командной строки.  
  
 При вводе данной команды переключатели следует включать непосредственно после команды, а не после псевдонимов, иначе переключатель будет включен как часть строки псевдонима.  
  
 Переключатель `/reset` используется для запроса подтверждения перед восстановлением псевдонимов.  Для переключателя `/reset` нет краткой формы.  
  
## Примеры  
 В этом примере создается новый псевдоним `upper` для полной команды Edit.MakeUpperCase.  
  
```  
>Tools.Alias upper Edit.MakeUpperCase  
```  
  
 В этом примере удаляется псевдоним `upper`.  
  
```  
>Tools.alias /delete upper  
```  
  
 В этом примере отображается список всех текущих псевдонимов и их определений.  
  
```  
>Tools.Alias  
```
