# <a name="introduce-a-local-variable-in-visual-basic"></a>Ввод локальной переменной на Visual Basic
**Что?** Этот компонент позволяет сразу же создать локальную переменную, чтобы заменить существующее выражение.

**Когда?** У вас есть код, который вы могли бы позже повторно использовать, если бы он был в виде локальной переменной.  

**Зачем?** Вы можете много раз копировать и вставлять код, чтобы использовать его в различных расположениях. Но лучше выполнить операцию один раз, сохранить результат в виде локальной переменной и использовать эту переменную, где необходимо. 

**Как?**

1. Выделите выражение, которое нужно назначить новой локальной переменной.

   ![Выделенный код](media/local-highlight-vb.png)

1. Затем выполните одно из следующих действий:
   * **Клавиатура**
     * Нажмите клавиши **CTRL + .**, чтобы активировать меню **Быстрые действия и рефакторинг**. Затем во всплывающем окне предварительного просмотра выберите пункт **Введите локальное значение для всех вхождений *выражения***, где *выражение* — это выделенный вами код.
   * **Мышь**
     * Щелкните правой кнопкой мыши и выберите меню **Быстрые действия и рефакторинг**. Затем во всплывающем окне предварительного просмотра выберите пункт **Введите локальное значение для всех вхождений *выражения***, где *выражение* — это выделенный вами код.
     * Нажмите кнопку ![Значок лампочки](media/bulb-vb.png) который отображается в поле слева, если текстовый курсор уже находится в строке выбора с красной волнистой линией.

   ![Ввод локальной переменной — предварительный просмотр](media/local-preview-vb.png)

   >[!TIP]
   >Щелкните ссылку [**Просмотреть изменения**](../../ide/preview-changes.md) в нижней части окна предварительного просмотра, чтобы просмотреть все изменения перед выбором.

1. Локальная переменная будет создана автоматически с типом с учетом ее использования.  Присвойте новой локальной переменной имя.

   ![Результат ввода локальной переменной](media/local-result-vb.png)

   >[!NOTE]
   >Вы можете использовать пункт меню **...все вхождения...**, чтобы заменить все найденные экземпляры выбранного выражения, а не только выделенное вами.
