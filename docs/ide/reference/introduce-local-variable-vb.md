# <a name="introduce-a-local-variable-in-visual-basic"></a>Ввод локальной переменной на Visual Basic
**Что?** Этот компонент позволяет сразу же создать локальную переменную, чтобы заменить существующее выражение.

**Когда?** У вас есть код, который позже мог бы легко использоваться еще раз, если бы он был представлен в виде локальной переменной.  

**Зачем?** Вы можете много раз копировать и вставлять код, чтобы использовать его в различных расположениях. Но было бы лучше выполнить операцию один раз, сохранить результат в виде локальной переменной и использовать эту переменную во всем коде. 

**Как?**

1. Выделите выражение, которое нужно назначить новой локальной переменной.

   ![Выделенный код](media/local-highlight-vb.png)

1. Затем выполните одно из следующих действий:
   * **Клавиатура**
     * Нажмите клавиши **CTRL+.**, чтобы активировать меню **Быстрые действия и рефакторинг**. Затем во всплывающем окне предварительного просмотра выберите пункт **Введите локальное значение для всех вхождений "*expression*"**, где *expression* — это выделенный вами код.
   * **Мышь**
     * Щелкните правой кнопкой мыши и выберите меню **Быстрые действия и рефакторинг**. Затем во всплывающем окне предварительного просмотра выберите пункт **Введите локальное значение для всех вхождений "*expression*"**, где *expression* — это выделенный вами код.
     * Нажмите кнопку ![значок лампочки,](media/bulb-vb.png) который отображается в поле слева, если текстовый курсор уже находится в строке выбора с красной волнистой линией.

   ![Ввод локальной переменной — предварительный просмотр](media/local-preview-vb.png)

   >[!TIP]
   >Щелкните ссылку [**Просмотреть изменения**](../../ide/preview-changes.md) в нижней части окна предварительного просмотра, чтобы просмотреть все будущие изменения, прежде чем вы выберете элементы.

1. Локальная переменная будет автоматически создана с типом, который выводится исходя из ее использования.  Введите имя и присвойте его новой локальной переменной.

   ![Результат ввода локальной переменной](media/local-result-vb.png)

   >[!NOTE]
   >Вы можете использовать пункт меню **...all occurences of...** (...все вхождения...), чтобы заменить все найденные экземпляры выбранного выражения, а не только выделенное вами.
