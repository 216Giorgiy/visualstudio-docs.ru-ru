# <a name="-devenvexe"></a>/? (devenv.exe)
Отображает окно сообщения, где перечислены все параметры `devenv` с кратким описанием каждого из них.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
devenv /?  
```
