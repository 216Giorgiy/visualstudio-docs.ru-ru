# <a name="generate-a-method-in-c"></a>Создание метода на C# #
**Что?** Позволяет немедленно добавить метод в класс. 

**Когда?** Вы представляете новый метод и хотите правильно его объявить автоматически.  

**Зачем?** Вы можете объявить метод и параметры перед их использованием, но эта функция позволяет создать объявление автоматически. 

**Как?**

1. Поместите курсор в строку с красной волнистой линией. Она указывает, что вы использовали метод, который еще не существует.

   ![Выделенный код](media/method-highlight-cs.png)

1. Затем выполните одно из следующих действий:
   * **Клавиатура**
     * Нажмите клавиши **CTRL + .**, чтобы активировать меню **Быстрые действия и рефакторинг**. Затем во всплывающем окне предварительного просмотра выберите пункт **Создать метод**.
   * **Мышь**
     * Щелкните правой кнопкой мыши и выберите меню **Быстрые действия и рефакторинг**. Затем выберите во всплывающем окне предварительного просмотра пункт **Создать метод**.
     * Наведите указатель мыши на красную волнистую линию и щелкните ![Значок лампочки](media/bulb-cs.png) .
     * Нажмите кнопку ![Значок лампочки](media/bulb-cs.png) который отображается в поле слева, если текстовый курсор уже находится в строке выбора с красной волнистой линией.

   ![Создание метода — предварительный просмотр](media/method-preview-cs.png)

   >[!TIP]
   >Щелкните ссылку [**Просмотреть изменения**](../../ide/preview-changes.md) в нижней части окна предварительного просмотра, чтобы просмотреть все будущие изменения, прежде чем вы выберете элементы.

1. Метод будет автоматически создан со всеми параметрами с учетом его использования.

   ![Результат создания метода](media/method-result-cs.png)
