# Практическое руководство. Управление переносом слов в редакторе
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Для управления переносом слов можно устанавливать и снимать флажок **Переносить по словам**.  Если этот флажок установлен, любая часть длинной строки, выступающая за пределы текущего окна редактора, будет автоматически отображаться на следующей строке.  Если этот флажок снят, например для активации нумерации строк, длинные строки можно просматривать путем прокрутки вправо.  
  
 Дополнительные сведения см. в разделе [How to: Set General Editor Options](http://msdn.microsoft.com/ru-ru/704e4a7b-2162-4bed-8a47-f4f6ffec98c2).  
  
> [!NOTE]
>  Отображаемые диалоговые окна и команды меню могут отличаться от описанных в **Справке** в зависимости от текущих настроек или выпуска.  Чтобы изменить параметры, в меню **Сервис** выберите команду **Импорт и экспорт параметров**.  Дополнительные сведения см. в разделе [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ru-ru/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
## Процедура  
  
#### Настройка параметров переноса слов  
  
1.  В меню **Сервис** выберите пункт **Параметры**.  
  
2.  В папке **Текстовый редактор** выберите **Общие** параметры в подпапке **Все языки**, чтобы установить общий параметр.  
  
     — либо —  
  
     Выберите **Общие** параметры в подпапке для используемого языка программирования.  
  
3.  В разделе **Параметры** установите или снимите флажок **Переносить по словам**.  
  
     Установка флажка **Переносить по словам** делает доступным параметр **Показывать графические метки в местах переноса слов**.  
  
4.  Если установлен флажок **Показывать графические метки в местах переноса слов**, в местах переноса длинных строк будут отображаться обратные стрелки.  Для выключения отображения этих стрелок снимите флажок.  
  
    > [!NOTE]
    >  Эти стрелки\-напоминания не добавляются в код; они предназначены только для просмотра.
