# <a name="generate-an-override-in-c"></a>Создание переопределения на C# #

**Что.** Немедленное создание кода для любого метода, который можно переопределить из базового класса.

**Когда.** Необходимо переопределить метод базового класса и автоматически создать сигнатуру.

**Зачем.** Сигнатуру метода можно написать самостоятельно, но эта функция автоматически создает сигнатуру.

**Как?**

1. Введите ключевое слово **переопределение**, за которым следует пробел, там, где нужно вставить сигнатуру переопределенного метода. Появится раскрывающийся список IntelliSense.

   ![Переопределение IntelliSense](media/override-intellisense-cs.png)

1. Выберите метод, который хотите переопределить из базового класса. Для этого щелкните его с помощью мыши или перейдите к нему с помощью клавиатуры и нажмите клавишу **ВВОД**.

   >[!TIP]
   >* Используйте значок свойства, ![Значок свойства](media/override-property-cs.png) чтобы отобразить или скрыть свойство в списке.
   >* Используйте значок метода, ![Значок метода](media/override-method-cs.png) чтобы отобразить или скрыть метод в списке.

1. Выбранный метод или свойство будет добавлено в класс как переопределение, готовое к реализации.

   ![Результат переопределения](media/override-result-cs.png)
