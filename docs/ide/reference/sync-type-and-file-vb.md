# <a name="sync-a-type-to-a-filename-or-a-filename-to-a-type-in-visual-basic"></a>Синхронизация типа с именем файла или имени файла с типом в Visual Basic

<!-- VERSIONLESS -->
**Эта возможность доступна в Visual Studio 2017 и более поздних версиях.  Кроме того, рефакторинг для проектов .NET Standard пока не поддерживается.**

**Что?** Вы можете переименовать тип в соответствии с именем файла или изменить имя файла в соответствии с типом, который он содержит.

**Когда?** Вы переименовали файл или тип и еще не обновили их в соответствии с их именами. 

**Зачем?** Если поместить тип в файл с другим именем (или наоборот), вам трудно будет найти нужный файл или тип.  Если переименовать тип или файл, будет проще читать код и удобнее переходить по нему.

**Как?**

1. Выделите имя типа, который требуется синхронизировать, или поместите в него текстовый курсор.

   ![Выделенный код](media/synctype-highlight-vb.png)

1. Затем выполните одно из следующих действий:
   * **Клавиатура**
     * Нажмите клавиши **CTRL + .**, чтобы активировать меню **Быстрые действия и рефакторинг**. Затем во всплывающем окне предварительного просмотра выберите пункт **Переименовать файл в *Имя типа*.vb**, где *Имя типа* — это имя выбранного вами типа.
     * Нажмите клавиши **Ctrl+.**, чтобы активировать меню **Быстрые действия и рефакторинг**. Затем во всплывающем окне предварительного просмотра выберите пункт **Переименовать тип в _имя_файла_**, где *имя_файла* — это имя текущего файла.
   * **Мышь**
     * Щелкните код правой кнопкой мыши и выберите меню **Быстрые действия и рефакторинг**. Затем во всплывающем окне предварительного просмотра пункт выберите пункт **Переименовать файл в *Имя типа*.vb**, где *Имя типа* — это имя выбранного вами типа.
     * Щелкните код правой кнопкой мыши и выберите меню **Быстрые действия и рефакторинг**. Затем во всплывающем окне предварительного просмотра выберите пункт **Переименовать тип в _Имя файла_**, где *имя файла* — это имя выбранного вами файла.

   Тип или файл будут сразу же переименованы.  В примере ниже файл **Employee.cs** был переименован в **Person.cs**, чтобы его имя совпадало с именем типа.

   ![Встроенный результат](media/synctype-result-vb.png)
