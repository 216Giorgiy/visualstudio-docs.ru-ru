# Страница &quot;Автовосстановление&quot;, папка &quot;Среда&quot;, диалоговое окно &quot;Параметры&quot;
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

На этой странице диалогового окна "Параметры" можно включить или отключить автоматическое резервное копирование файлов.  Также можно включить или отключить автоматическое восстановление измененных файлов после неожиданного завершения работы среды разработки.  Чтобы открыть этот окно, в меню **Сервис** выберите **Параметры**, затем откройте папку **Среда** и перейдите на страницу **Автоматическое восстановление**.  Если эта страница отсутствует в списке, установите флажок **Показать все параметры** в диалоговом окне **Параметры**.  
  
> [!NOTE]
>  Отображаемые диалоговые окна и команды меню могут отличаться от описанных в справке в зависимости от текущих настроек или выпуска.  Для изменения настроек выберите Параметры импорта и экспорта в меню Сервис.  Дополнительные сведения см. в разделе [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ru-ru/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
 **Сохранять данные автоматического восстановления каждые \<n\> минут**  
 Используйте этот параметр, чтобы указать, насколько часто файл должен автоматически сохраняться.  Копии ранее сохраненных файлов помещаются в папку \\...  \\My Documents\\Visual Studio \<*версия*\>\\Backup Files\\\<*имя\_проекта*\>.  Если файл является новым и не был сохранен вручную, то при его автоматическом сохранении имя файла формируется случайным образом.  
  
 **Хранить данные автоматического восстановления в течение \<n\> дней**  
 С помощью этого параметра можно указать, в течение какого срока должны храниться файлы, созданные для автоматического восстановления.
