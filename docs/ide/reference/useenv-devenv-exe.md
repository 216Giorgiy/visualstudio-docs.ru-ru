# <a name="useenv-devenvexe"></a>/UseEnv (devenv.exe)
Запускает [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] и загружает переменные среды в диалоговом окне **Каталоги VC++**.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
Devenv /useenv  
```  
  
## <a name="example"></a>Пример  
 Приведенный ниже пример запускает [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] и загружает переменные среды в диалоговом окне **Каталоги VC++**.  
  
```  
Devenv.exe /useenv  
```
