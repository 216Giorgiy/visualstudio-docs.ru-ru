# Параметры, текстовый редактор, все языки
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Это диалоговое окно позволяет изменять заданное по умолчанию поведение редактора кода.  Данные параметры также применяются к другим редакторам, использующим в своей основе редактор кода, например к средству просмотра исходного кода в конструкторе HTML.  Чтобы открыть это диалоговое окно, выберите пункт **Параметры** в меню **Сервис**.  В папке **Текстовый редактор** разверните подпапку **Все языки** и выберите **Общие**.  
  
> [!CAUTION]
>  На этой странице устанавливаются параметры по умолчанию для всех языков разработки.  Следует помнить, что сброс значения параметра в этом диалоговом окне сбросит общие параметры для всех языков вне зависимости от выбранного значения.  Для изменения параметров текстового редактора только для одного языка раскройте вложенную папку для этого языка и выберите страницы его параметров.  
  
 Затененная пометка отображается в том случае, если параметр был выбран на странице "Общие" для некоторого определенного языка программирования, но не для остальных языков.  
  
> [!NOTE]
>  Отображаемые диалоговые окна и команды меню могут отличаться от описанных в справке в зависимости от текущих настроек или выпуска.  Чтобы изменить параметры, в меню **Сервис** выберите команду **Импорт и экспорт параметров**.  Дополнительные сведения см. в разделе [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ru-ru/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
## Завершение операторов  
 Автоматически отображать список членов  
 Если этот флажок установлен, при вводе в редакторе отображаются всплывающие списки доступных членов, свойств, значений или методов.  Чтобы вставить в код какой\-либо элемент, выберите его в списке.  Выбор этого параметра разрешает установку параметра **Скрывать дополнительные члены**.  
  
 Скрывать дополнительные члены  
 Если этот флажок установлен, всплывающий список завершения операторов будет состоять только из наиболее часто используемых элементов.  Другие элементы будут отфильтрованы из списка.  
  
 Сведения о параметрах  
 Если этот флажок установлен, полный синтаксис для текущего объявления или процедуры будет отображаться непосредственно под позицией вставки в редакторе со всеми доступными параметрами.  Очередной параметр, который может быть присвоен, выделяется полужирным шрифтом.  
  
## Параметры  
 Включить виртуальное пространство  
 При выборе этого параметра и снятии флажка **Перенос по словам** можно щелкнуть в любом месте после конца строки в редакторе кода и ввести текст.  Эта функция может использоваться для размещения комментариев на одинаковом расстоянии от кода.  
  
 Переносить по словам  
 Если этот флажок установлен, любая часть строки, выступающая за пределы видимой области редактора по горизонтали, будет автоматически отображаться на следующей строке.  Выбор этого параметра включает параметр **Отображать символы переноса по словам**.  
  
> [!NOTE]
>  Если включен параметр **Переносить по словам**, параметр **Виртуальное пространство** отключается.  
  
 Показывать графические метки в местах переноса слов  
 Если этот флажок установлен, в местах переноса длинной строки на следующую строку будет отображаться символ в виде обратной стрелки.  
  
 Чтобы эти индикаторы не отображались, снимите флажок.  
  
> [!NOTE]
>  Эти стрелки\-напоминания не добавляются в код и не выводятся на печать.  Они используются исключительно для удобства.  
  
 Применять команды "Вырезать" и "Копировать" к пустым строкам, когда текст не выделен  
 Этот параметр задает поведение редактора в ситуации, когда пользователь помещает курсор в пустую строку, ничего не выбирает и затем пытается выполнить копирование или вырезание.  
  
-   Установка этого флажка позволяет вырезать и копировать пустые строки.  При последующей вставке будет вставлена новая пустая строка.  
  
-   Если параметр не установлен, пустые строки будут удалятся по команде “Вырезать”.  При этом данные в буфере обмена будут сохраняться.  Таким образом, если воспользоваться командой “Вставить”, вставляется содержимое, скопированное в буфер обмена последним.  Если ничего не было ранее скопировано, вставка не произойдет.  
  
 Этот параметр не влияет на команды "Копировать" и "Вырезать" в случае, если строка не пуста.  Если ничего не выделено, копируется или вырезается вся строка.  Если затем произвести вставку, будет вставлен текст всей строки и ее завершающий символ.  
  
> [!TIP]
>  Чтобы включить отображение индикаторов для пробелов, табуляций и концов строк и таким образом визуально отделить строки с отступом от полностью пустых строк, в меню **Правка** выберите пункты **Дополнительно** и **Показать пустое пространство**.  
  
## Отображение  
 Номерам строк.  
 Если этот флажок установлен, рядом с каждой строкой кода будет отображаться ее номер.  
  
> [!NOTE]
>  Номера строк не добавляются в код и не выводятся на печать.  Они используются исключительно для удобства.  
  
 Открывать URL\-адреса однократным щелчком  
 Если этот флажок установлен, при наведении курсора на URL\-адрес в редакторе указатель курсора принимает форму руки.  Можно щелкнуть URL\-адрес для отображения указанной страницы в браузере.  
  
 Панель переходов  
 Если этот флажок установлен, в верхней части редактора кода отображается **Область переходов**.  Раскрывающиеся списки **Объекты** и **Члены** этой области позволяют выбирать определенные объекты в коде, их члены и переходить к объявлениям выбранных членов.
