# How to: Copy Class Diagram Elements to a Microsoft Office Document (Class Designer)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Можно копировать фигуры из схемы классов .NET \(CD\-файл\) в другие документы.  При вставке будет получена либо копия фигуры, либо ее базовый код, в зависимости от типа документа, в который выполняется вставка.  Сведения о копировании фигуры из схем классов UML в проекте моделирования см. в разделе [Экспорт схем в виде изображений](../modeling/export-diagrams-as-images.md).  
  
### Копирование одного элемента  
  
-   Щелкните правой кнопкой мыши фигуру и выберите **Копировать изображение**.  
  
### Копирование нескольких элементов  
  
1.  Выберите на схеме фигуры, которые требуется скопировать.  
  
2.  Щелкните правой кнопкой мыши выбранное и воспользуйтесь пунктом меню **Копировать изображение**.  
  
### Копирование всех элементов в схеме классов  
  
1.  Щелкните правой кнопкой мыши рабочую область конструирования и выберите пункт меню **Выбрать все**.  \(Сочетание клавиш на клавиатуре: **CTRL \+ A**\)  
  
2.  В меню **Правка** выберите **Копировать изображение**.  
  
 Можно также выбрать **Копировать** вместо **Копировать изображение**.  Команда **Копировать** копирует изображение как обычное точечное изображение.  Команда **Копировать изображение** копирует изображение как векторное изображение, что является наиболее подходящим для большинства приложений Office.
