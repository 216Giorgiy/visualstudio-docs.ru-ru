# <a name="connected-environment"></a>Подключенная среда
Visual Studio представляет собой подключенную среду.  

- Вы можете войти в Visual Studio, чтобы получить доступ ко всем службам, связанным с вашей учетной записью пользователя, а также всем учетным записям, связанным с этой учетной записью. Параметры вашей учетной записи переносятся при каждом использовании Visual Studio.  
- Visual Studio сообщает вам о доступности какого-либо обновления для компонента вашей установки, например пакета SDK.  
- Вы можете скачать пакеты NuGet и расширения.  
- И многое другое!
