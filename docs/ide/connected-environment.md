# <a name="internet-resources-used-by-visual-studio"></a>Ресурсы в Интернете, используемые Visual Studio

Visual Studio представляет собой подключенную среду.

- Вы можете [войти](../ide/signing-in-to-visual-studio.md) в Visual Studio, чтобы получить доступ ко всем службам, связанным с вашей учетной записью пользователя, а также всем учетным записям, связанным с этой учетной записью. Параметры вашей учетной записи переносятся при каждом использовании Visual Studio.  
- Visual Studio [уведомляет](../ide/visual-studio-notifications.md) о доступности какого-либо обновления для компонента вашей установки, например пакета SDK.  
- Вы можете скачать пакеты [NuGet](/nuget/) и библиотеки.  
- Вы можете скачать [расширения](../ide/finding-and-using-visual-studio-extensions.md) для Visual Studio.  
- Вы можете подключиться к службам Azure.  
- Начальная страница обновляется и содержит последние новости для разработчиков с различных веб-сайтов  
- и т. д.  

В подразделах этого раздела описываются различные способы, которыми в Visual Studio используются интернет-ресурсы.
