# Практическое руководство. Управление режимами редактора
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Редактор кода Visual Studio можно отображать в различных режимах.  
  
> [!NOTE]
>  Отображаемые диалоговые окна и команды меню могут отличаться от описанных в справке в зависимости от текущих настроек или выпуска.  Чтобы изменить параметры, в меню **Сервис** выберите команду **Импорт и экспорт параметров**.  Дополнительные сведения см. в разделе [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ru-ru/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
## Включение режима "Во весь экран"  
 Включив режим **Во весь экран**, можно скрыть окна всех инструментов, оставив лишь окна документов.  
  
#### Включение режима "Во весь экран"  
  
-   Для включения и выключения режима **Во весь экран** используется сочетание клавиш ALT\+SHIFT\+ENTER.  
  
     \- или \-  
  
-   Выполните команду `View.Fullscreen` в окне **Команда**.  
  
## Включение режима виртуального пространства  
 В режиме **виртуального пространства** в конец каждой строки кода вставляются пробелы.  Этот параметр позволяет помещать комментарии на одинаковом расстоянии от кода.  
  
#### Включение режима виртуального пространства  
  
1.  В меню **Сервис** выберите команду **Параметры**.  
  
2.  Разверните папку **Текстовый редактор** и щелкните **Все языки**, чтобы установить общий параметр, либо выберите папку конкретного языка.  \(Например, чтобы включить номера строк только в Visual Basic, выберите команды "Basic", "Текстовый редактор".\)  
  
3.  Выберите **Общие**, а затем в разделе **Параметры** щелкните **Включить виртуальное пространство**.  
  
    > [!NOTE]
    >  **Виртуальное пространство** включается в режиме **Выбор столбцов**.  Когда режим **виртуального пространства** не включен, курсор переходит с конца одной строки непосредственно на первый символ последующей.
