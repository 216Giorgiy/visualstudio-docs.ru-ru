# <a name="globalize-applications"></a>Глобализация приложений

Глобализация — это процесс проектирования и разработки программных продуктов, которые работают в средах с различными языками и региональными параметрами. Следующая статья относится к страницам Windows Forms и Web Forms.

 В статье о [классах, соответствующих определенному языку и региональным параметрам, для глобальных форм Windows Forms и Web Forms](../ide/culture-specific-classes-for-global-windows-forms-and-web-forms.md) описаны классы, которые используются для форматирования дат, времени, чисел, валют и других сведений в соответствии с параметрами языка и региональных параметров.
