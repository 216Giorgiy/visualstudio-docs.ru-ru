# Практическое руководство. Настройка используемой по умолчанию веб-страницы для ClickOnce-приложения
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

При публикации ClickOnce\-приложения на веб\-узле вместе с приложением автоматически создается и публикуется веб\-страница.  Используемая по умолчанию страница содержит имя приложения и ссылки на установку приложения, на компоненты, необходимые для установки, или на справку доступа на MSDN.  
  
> [!NOTE]
>  Действительные ссылки, которые видны на странице, зависят от компьютера, на котором просматривается страница, и от того, какие необходимые компоненты установлены.  
  
 По умолчанию веб\-странице присваивается имя "Publish.htm"; имя можно изменить в **конструкторе проектов**.  Дополнительные сведения см. в разделе [Практическое руководство. Задание страницы публикации для приложения ClickOnce](../deployment/how-to-specify-a-publish-page-for-a-clickonce-application.md).  
  
 Веб\-страница Publish.htm публикуется, если только обнаруживается новая версия.  
  
> [!NOTE]
>  Изменения в настройках области **Публикация** не будут влиять на страницу Publish.htm, но с одним исключением: если после первоначальной публикации добавляются или удаляются необходимые компоненты, список необходимых компонентов будет неточным.  Необходимо изменить текст ссылки на необходимые компоненты для отражения изменений.  
  
### Настройка публикации веб\-страницы  
  
1.  Опубликуйте ClickOnce\-приложение в местоположении на веб\-узле.  Дополнительные сведения см. в разделе [Практическое руководство. Публикация приложения ClickOnce с помощью мастера публикации](../Topic/How%20to:%20Publish%20a%20ClickOnce%20Application%20using%20the%20Publish%20Wizard.md).  
  
2.  На веб\-сервере откройте файл Publish.htm в конструкторе Visual Web Designer или в другом HTML\-редакторе.  
  
3.  Настройте страницу, как необходимо, и сохраните ее.  
  
4.  Необязательный параметр.  Чтобы предотвратить перезапись программным обеспечением Visual Studio настроенной веб\-страницы публикации, в диалоговом окне "Параметры публикации" снимите флажок **Автоматически создавать веб\-страницу развертывания после каждой публикации**.
