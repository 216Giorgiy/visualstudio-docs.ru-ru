# Практическое руководство. Задание режима установки ClickOnce: автономного или через Интернет
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Режим установки `Install Mode` для приложения [!INCLUDE[ndptecclick](../deployment/includes/ndptecclick_md.md)] определяет, будет ли приложение доступно в автономном или интерактивном режиме.  При выборе **Приложение доступно только из сети** пользователь должен иметь доступ к расположению публикации [!INCLUDE[ndptecclick](../deployment/includes/ndptecclick_md.md)] \(веб\-страница или в общая папка\), чтобы запустить приложение.  При выборе **Приложение будет доступно в автономном режиме** приложение добавляет элементы в меню **Пуск** и в диалоговое окно **Установка и удаление программ**; пользователи могут запускать приложение, если они не подключены к сети Интернет.  
  
 Режим установки `Install Mode` может быть установлен на странице **Публикация** в **конструкторе проектов**.  
  
 **Примечание.** Режим установки `Install Mode` может быть также установлен с помощью мастера публикации.  Дополнительные сведения см. в разделе [Практическое руководство. Публикация приложения ClickOnce с помощью мастера публикации](../Topic/How%20to:%20Publish%20a%20ClickOnce%20Application%20using%20the%20Publish%20Wizard.md).  
  
### Задание доступности приложения ClickOnce только в интерактивном режиме  
  
1.  Выбрав проект в **обозревателе решений**, в меню **Проект** выберите пункт **Свойства**.  
  
2.  Выберите вкладку **Публикация**.  
  
3.  В области **Режим установки и параметры** нажмите кнопку параметра **Приложение доступно только из сети**.  
  
### Задание доступности приложения ClickOnce в интерактивном или автономном режиме  
  
1.  Выбрав проект в **обозревателе решений**, в меню **Проект** выберите пункт **Свойства**.  
  
2.  Выберите вкладку **Публикация**.  
  
3.  В области **Режим установки и параметры** нажмите кнопку параметра **Приложение будет доступно в автономном режиме**.  
  
     Если приложение установлено, оно добавляет элементы в меню **Пуск** и в меню **Установка и удаление программ** панели управления.
