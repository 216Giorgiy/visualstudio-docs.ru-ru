# Init
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Подготавливает компонент диагностики графики в приложении для активного захвата и записи данных графики в файл журнала графики.  
  
## Синтаксис  
  
```cpp  
void Init(  
  std::function<void (int len, wchar_t * pszBuffer)> vsgLogGetter  
);  
```  
  
#### Параметры  
 `vsgLogGetter`  
 Вызываемая сущность, такая как функция, указатель на функцию, лямбда или объект\-функция, которая принимает в качестве параметров длину буфера из `wchar_t` и указатель на этот буфер и возвращает `void`.  При вызове вызываемая сущность определяет имя файла, который будет использоваться для записи сведений о графики, и записывает его в указанный буфер перед возвратом.  
  
## Примечания  
 Функция `Init` вызывается автоматически при создании экземпляра класса `VsgDbg` с указанным параметром `bDefaultInit` его конструктора как `true`; в противном случае `Init` должна вызываться явным образом перед запуском активного захвата и записи данных графики.  
  
 Можно завершить и закрыть активный файл журнала графики, вызвав `UnInit`, а затем выполнять захват и запись других данных графики в новый файл журнала графики, вызвав `Init` снова.  Данное действие можно повторять сколько угодно раз, если необходимо использовать один и тот же экземпляр `VsgDbg` для создания нескольких независимых файлов журнала графики.
