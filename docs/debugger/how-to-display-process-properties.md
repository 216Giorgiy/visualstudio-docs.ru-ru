# How to: Display Process Properties
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Для получения дополнительных сведений о процессах, перечисленных в [представлении процессов](../debugger/processes-view.md), воспользуйтесь диалоговым окном [Свойства процесса](../debugger/process-properties-dialog-box.md).  
  
### Открытие диалогового окна "Свойства процесса"  
  
1.  Выберите процесс в дереве, отображенном в **представлении процессов**.  
  
2.  В меню **Вид** выберите пункт **Свойства**.  
  
 Диалоговые окна свойств не являются модальными, поэтому можно щелкнуть другой элемент в окне представления, и в диалоговом окне свойств будут отображены сведения о выбранном элементе.
