# <a name="how-to-display-process-properties"></a>Практическое руководство. Отображение свойств процесса
Для получения дополнительных сведений о процессах, перечисленных в [представление процессов](../debugger/processes-view.md), используйте [диалоговое окно "Свойства процесса"](../debugger/process-properties-dialog-box.md).  
  
### <a name="to-open-a-process-properties-dialog-box"></a>Чтобы открыть диалоговое окно "Свойства процесса"  
  
1.  Выберите дерево, отображаемое в процесс **представление процессов**.  
  
2.  Из **представление** меню, выберите **свойства**.  
  
 Диалоговые окна свойств не модальным, поэтому можно щелкнуть другой элемент в окно представления и диалоговом окне будут отображены сведения о выбранного элемента.
