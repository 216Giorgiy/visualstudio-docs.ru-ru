# Диалоговое окно &quot;Поиск окна&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Это диалоговое окно используется для поиска и выбора узла для конкретного окна в [представлении окон](../debugger/windows-view.md).  Чтобы открыть это диалоговое окно, переместите фокус на окно представления окон.  Затем в меню **Поиск** выберите команду **Найти окно**.  
  
 Для выбора окна можно использовать **инструмент поиска**, ввести дескриптор окна или определить окно по заголовку и классу.  
  
 Доступны следующие параметры:  
  
 **Инструмент поиска**  
 Для выбора окна используется перетаскивание.  
  
 **Скрыть Spy\+\+**  
 Удаляет главное окно Spy\+\+ и располагает диалоговое окно **Поиск окна** поверх всех приложений.  
  
 **Дескриптор**  
 Уникальный идентификатор окна, которое необходимо найти.  
  
 **Caption**  
 Текст заголовка окна, который необходимо найти.  
  
 **Класс**  
 Класс окна, которое необходимо найти.  
  
 **Направление поиска: вверх или вниз**  
 Начальное направление поиска.
