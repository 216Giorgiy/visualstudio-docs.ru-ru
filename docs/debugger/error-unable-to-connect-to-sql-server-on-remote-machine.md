# Ошибка: не удается подключиться к SQL Server на удаленном компьютере
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Не удалось подключиться к SQL Server на удаленном компьютере *name*.  Доступ запрещен.  Проверьте, что на удаленном компьютере установлен удаленный отладчик.  Если удаленный компьютер не входит в домен, или если Visual Studio выполняется с локальной учетной записью, удаленный компьютер должен иметь учетную запись с тем же именем и паролем, что и локальная учетная запись.  
  
### Исправление этой ошибки  
  
-   См. раздел [Удаленная отладка](../debugger/remote-debugging.md).
