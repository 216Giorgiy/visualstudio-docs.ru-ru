# Команды меню &quot;Сообщения&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

В меню **Сообщения** представлены команды управления представлениями сообщений.  
  
 **Начать\/Остановить запись**  
 Начинает или останавливает работу представления сообщений.  
  
 **Параметры журнала**  
 Открывает диалоговое окно **Параметры сообщения**, в котором можно указать условия поиска сообщений.  
  
 **Очистить журнал**  
 Очищает содержимое активного представления сообщений.
