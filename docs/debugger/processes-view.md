# Processes View
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

В представлении процессов отображается дерево всех активных процессов системы.  При этом указываются идентификаторы процессов и имена модулей.  Представление процессов используется для просмотра сведений по конкретному процессу системы, который обычно соответствует выполняемой программе.  Процессы идентифицируются по именам модулей или по связанным с ними "системным процессам".  
  
 Система Microsoft Windows поддерживает множественные процессы.  Каждый процесс может иметь как один, так и несколько потоков, а каждый поток может иметь одно или несколько окон верхнего уровня.  Каждому окну верхнего уровня может принадлежать серия окон.  Значок "\+" указывает на то, что уровень свернут.  Свернутое представление включает по одной строке для каждого процесса.  Чтобы развернуть уровень, щелкните значок "\+".  
  
 Представление процессов используется для просмотра сведений по конкретному процессу системы, который обычно соответствует выполняемой программе.  Процессы идентифицируются по именам модулей или по связанным с ними "системным процессам". Чтобы найти процесс, нужно свернуть дерево и выполнить поиск в списке.  
  
## Процедуры  
  
#### Открытие представления процессов  
  
1.  В меню **Spy** выберите пункт **Процессы**.
