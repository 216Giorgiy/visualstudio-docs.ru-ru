# <a name="process-properties-dialog-box"></a>Диалоговое окно "Свойства процесса"
Используйте это диалоговое окно предназначено для получения дополнительных сведений об определенном процессе. Чтобы открыть это диалоговое окно предназначено, переместите фокус элементу [представление процессов](../debugger/processes-view.md) окна. Выберите любой узел процесса в дереве, а затем выберите **свойства** из **представление** меню.  
  
 В Windows 98 и Windows 95 операционных системах только измененную версию **Общие** отображается вкладка.
