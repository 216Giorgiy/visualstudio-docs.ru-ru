# <a name="how-to-search-for-a-thread-in-threads-view"></a>Практическое руководство. Поиск потока в представлении потоков
Можно поиск определенного потока в представлении потоков с помощью его поток идентификатор или строку модуля как условия поиска. Можно также указать начальное направление поиска. Поля в диалоговом окне отобразятся атрибуты потока, выбранного в дереве потоков.  
  
### <a name="to-search-for-a-thread-in-threads-view"></a>Поиск потока в представлении потоков  
  
1.  Расположение окон, поэтому Spy ++ и активной [представление "Потоки"](../debugger/threads-view.md) окна являются видимыми.  
  
2.  Из **поиска** меню, выберите **найти поток**.  
  
     [Потоков диалоговое окно поиска](../debugger/thread-search-dialog-box.md) открывается.  
  
3.  Введите идентификатор потока или строку модуля в качестве условия поиска.  
  
4.  Очистите все поля, для которых не требуется задать значения.  
  
    > [!TIP]
    >  Чтобы найти все потоки, принадлежащие модулю, очистите **потоков** текстовое поле и введите модуль имя в **модуль** поле. Затем с помощью **Найти далее** для продолжения поиска потоков.  
  
5.  Выберите **копирование** или **работу** для исходное направление поиска.  
  
6.  Нажмите кнопку **ОК**.  
  
 Если искомый поток найден, он выделяется в окне представления потоков.
