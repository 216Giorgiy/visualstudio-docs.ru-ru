# How to: Search for a Thread in Threads View
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Поиск потока в представлении потоков можно выполнять, используя в качестве условия поиска его идентификатор или строку модуля.  Кроме того, можно указать начальное направление поиска.  В полях диалогового окна отобразятся атрибуты потока, выбранного в дереве потоков.  
  
### Поиск потока в представлении потоков  
  
1.  Расположите окна таким образом, чтобы программа Spy\+\+ и активное окно [представления потоков](../debugger/threads-view.md) были видны одновременно.  
  
2.  В меню **Поиск** выберите команду **Найти поток**.  
  
     Откроется диалоговое окно [Поиск потока](../debugger/thread-search-dialog-box.md).  
  
3.  Введите в качестве условия поиска идентификатор потока или строку модуля.  
  
4.  Очистите все поля, для которых не требуется задавать значения.  
  
    > [!TIP]
    >  Чтобы найти все потоки, принадлежащие модулю, очистите поле **Поток** и введите имя модуля в поле **Модуль**.  Нажмите кнопу **Найти далее** для продолжения поиска потоков.  
  
5.  Выберите пункт **Вверх** или **Вниз** в качестве начального направления поиска.  
  
6.  Нажмите кнопку **ОК**.  
  
 Если искомый поток найден, он будет выделен в окне представления потоков.
