# Подготовка к отладке: службы Windows
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Служба Windows – это программа, которая выполнятся в фоновом режиме в Microsoft Windows.  Примерами таких служб является служба Telnet и служба времени Windows, изменяющая часы, отображаемые на рабочем столе.  Служба Windows не может быть запущена из [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]; ее необходимо запускать из диспетчера управления службами.  Дополнительные сведения см. в разделах [Создание служб Windows](../Topic/How%20to:%20Create%20Windows%20Services.md), [Отладка служб Windows](../Topic/How%20to:%20Debug%20Windows%20Service%20Applications.md) и [Службы Windows](../Topic/Developing%20Windows%20Service%20Applications.md).
