# <a name="message-properties-dialog-box"></a>Диалоговое окно "Свойства сообщения"
Используйте это диалоговое окно предназначено для получения дополнительных сведений о конкретном сообщении. Чтобы открыть это диалоговое окно предназначено, переместите фокус элементу [представления сообщений](../debugger/messages-view.md) окна. Выберите любой узел сообщения в дереве, а затем выберите **свойства** из **представление** меню.  
  
 **Общие** отображается только вкладка. Доступны следующие параметры:  
  
 **Дескриптор окна**  
 Уникальный идентификатор этого окна. Номера дескрипторов окон используются неоднократно; они определяют окна только в течение времени существования этого окна. Выберите этот параметр для просмотра свойств этого окна.  
  
 **Уровень вложенности**  
 Глубина вложенности сообщения, где 0 означает отсутствие вложенности.  
  
 **Сообщение**  
 Номер, состояние и имя выбранного сообщения окна.  
  
 **lResult**  
 Значение *lResult* параметр, если таковые имеются.  
  
 **wParam**  
 Значение *wParam* параметр, если таковые имеются.  
  
 **lParam**  
 Значение *lParam* параметр, если таковые имеются. Это значение декодируется в том случае, если он является указателем на строку или структуру.
