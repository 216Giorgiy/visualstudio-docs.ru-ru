# Диалоговое окно &quot;Свойства сообщения&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Это диалоговое окно используется для получения дополнительных сведений о конкретном сообщении.  Чтобы открыть это диалоговое окно, переместите фокус на окно [представления сообщений](../debugger/messages-view.md).  Выберите любой узел сообщения в дереве, затем выберите в меню **Вид** пункт **Свойства**.  
  
 Вкладка **Общие** является единственной.  Доступны следующие параметры:  
  
 **Дескриптор окна**  
 Уникальный идентификатор этого окна.  Номера дескрипторов окон используются неоднократно; т. е. они определяют конкретное окно только в течение времени его существования \(и могут определять другое окно после завершения существования первого\).  Выберите этот параметр для просмотра свойств окна.  
  
 **Уровень вложенности**  
 Глубина вложенности сообщения, где 0 означает отсутствие вложенности.  
  
 **Сообщение**  
 Номер, состояние и имя выбранного сообщения окна.  
  
 **lResult**  
 Значение параметра *lResult*, если есть.  
  
 **wParam**  
 Значение параметра *wParam*, если есть.  
  
 **lParam**  
 Значение параметра *lParam*, если есть.  Значение декодируется, если это указатель на строку или структуру.
