# <a name="help-menu-commands"></a>Команды меню "Справка"
**Помочь** меню предоставляет команды, чтобы открыть файлы справки.

 **Индекс** Открывает экран содержимое справки по Spy ++.

 **Использование справки** открывает файл справки, в котором описано, как использовать справку.

 **О программе Spy ++** программного обеспечения версии и авторских правах для отображаются сведения Spy ++.
