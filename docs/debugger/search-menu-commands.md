# <a name="search-menu-commands"></a>Команды меню "Поиск"
**Поиска** предоставляет меню команд для поиска конкретного окна.  
  
 **Найти окно/процесс/поток/сообщение**  
 Открывает **окна**/**процесс**/**потоков**/**поиск сообщений** диалоговое окно «» где можно использовать **средство поиска** или укажите условия поиска.  
  
 **Найти далее**  
 Поиск в текущем представлении для следующего соответствующего окна, процесса, потока или сообщения. Этот пункт меню (и связанных инструментов) доступен только при наличии допустимых результатов поиска, не является уникальным.  
  
 **Найти ранее**  
 Поиск в текущем представлении предыдущего соответствующего окна, процесса, потока или сообщения. Этот пункт меню (и связанных инструментов) доступен только при наличии допустимых результатов поиска, не является уникальным.
