# <a name="page-file-tab-process-properties-dialog-box"></a>Вкладка "Файл подкачки" диалогового окна "Свойства процесса"
Используйте **файла подкачки** вкладку для просмотра сведений о файле подкачки процесса. Для отображения [диалоговое окно "Свойства процесса"](../debugger/process-properties-dialog-box.md), перемещение фокуса к [представление процессов](../debugger/processes-view.md) окна. Выберите любой узел процесса в дереве, а затем выберите **свойства** из **представление** меню.  
  
 Следующие параметры доступны на **файла подкачки** вкладки:  
  
|Ввод|Описание|  
|-----------|-----------------|  
|**Байт файла подкачки**|Текущее количество страниц, которые использует этот процесс в файл подкачки. В файле подкачки хранятся страниц данных, используемой процессом, но не содержащихся в других файлах. Файл подкачки используется всеми процессами, и нехватка места в файле подкачки может привести к ошибкам при выполнении других процессов.|  
|**Байт файла подкачки (пик)**|Максимальное число страниц, которые использовались в файле подкачки на этот процесс.|  
|**Ошибки страниц**|Количество ошибок страниц физической памяти потоками, выполняющимися в данном процессе. Сбой страницы происходит, когда поток ссылается на страницу виртуальной памяти, которая не находится в соответствующем рабочем наборе в основной памяти. Таким образом, страницы не быть получена с диска, если в списке ожидания и, следовательно, в основной памяти, или если он используется другим процессом, общие страницы.|
