# <a name="find-window-dialog-box"></a>Диалоговое окно "Поиск окна"
Это диалоговое окно используется для выбора окна и отображения его свойств или сообщений. Чтобы отобразить это диалоговое окно, выберите **найти окно** из **Spy** меню.  
  
 Доступны следующие параметры:  
  
 **Инструмент поиска**  
 Перетаскивание средство, используемое для выбора окна.  
  
 **Скрыть Spy ++**  
 Удаляет главное окно Spy ++ и оставляет **найти окно** диалоговое окно поверх других приложений.  
  
 **Дескриптор**  
 Уникальный идентификатор окна для поиска.  
  
 **Подпись**  
 Текст заголовка окна.  
  
 **Класс**  
 Класс окна.  
  
 **Стиль**  
 Стиль окна.  
  
 **Прямоугольник**  
 Ограничивающий прямоугольник для окна.  
  
 **Показать свойства или сообщения**  
 Определяет, отображаются ли нажатие ОК [диалоговое окно "Свойства окна"](../debugger/window-properties-dialog-box.md) или откройте [представления сообщений](../debugger/messages-view.md).
