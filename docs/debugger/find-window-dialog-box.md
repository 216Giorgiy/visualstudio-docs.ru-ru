# Диалоговое окно &quot;Поиск окна&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Это диалоговое окно используется для выбора окна и отображения его свойств или сообщений.  Чтобы открыть это диалоговое окно, выберите команду **Найти окно** в меню **Spy**.  
  
 Доступны следующие параметры:  
  
 **Инструмент поиска**  
 Для выбора окна используется перетаскивание.  
  
 **Скрыть Spy\+\+**  
 Удаляет главное окно Spy\+\+ и оставляет видимым диалоговое окно **Поиск окна** поверх всех приложений.  
  
 **Дескриптор**  
 Уникальный идентификатор окна, которое необходимо найти.  
  
 **Caption**  
 Текст заголовка окна.  
  
 **Класс**  
 Класс окна.  
  
 **Стиль**  
 Стиль окна.  
  
 **Прямоугольник**  
 Ограничивающий прямоугольник для окна.  
  
 **Отображать свойства или сообщения**  
 Определяет, что отображать при нажатии кнопки "ОК": [диалоговое окно свойств окна](../debugger/window-properties-dialog-box.md) или [представление сообщений](../debugger/messages-view.md).
