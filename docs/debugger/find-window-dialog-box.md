# <a name="find-window-dialog-box"></a>Диалоговое окно "Поиск окна"
Используйте это диалоговое окно для выбора окна и отображения его свойств или сообщений. Чтобы открыть это диалоговое окно предназначено выберите **Поиск окна** из **Spy** меню.  
  
 Доступны следующие параметры:  
  
 **Инструмент поиска**  
 Перетаскивание средство, используемое для выбора окна.  
  
 **Скрыть Spy ++**  
 Удаляет главное окно Spy ++ и оставляет **Поиск окна** диалоговое окно поверх других приложений.  
  
 **Дескриптор**  
 Уникальный идентификатор окна для поиска.  
  
 **Подпись**  
 Текст заголовка окна.  
  
 **Класс**  
 Класс окна.  
  
 **Стиль**  
 Стиль окна.  
  
 **Rect**  
 Ограничивающий прямоугольник для окна.  
  
 **Отображение свойств или сообщений**  
 Определяет, будут ли отображаться нажать кнопку ОК [диалоговое окно "Свойства"](../debugger/window-properties-dialog-box.md) или откройте [представления сообщений](../debugger/messages-view.md).
