# How to: Control Messages View
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Spy\+\+ предоставляет достаточно много возможностей управления содержимым окна [представления сообщений](../debugger/messages-view.md).  Можно [запускать и останавливать](../debugger/how-to-start-and-stop-the-message-log-display.md) отображение сообщений в любое время, а также определять:  
  
-   какие типы сообщений следует отображать;  
  
-   какие окна следует отслеживать;  
  
-   в каком формате отображать строки сообщений.  
  
 Эти параметры доступны в диалоговом окне [Параметры сообщения](../debugger/message-options-dialog-box.md) и применяются только к активному окну [представления сообщений](../debugger/messages-view.md).  Чтобы открыть это диалоговое окно, в меню **Spy** выберите команду **Записывать сообщения**.  
  
 Доступны три вкладки параметров.  Дополнительные сведения см. в разделах [Вкладка "Окна" диалогового окна "Параметры сообщения"](../debugger/windows-tab-message-options-dialog-box.md), [Вкладка "Сообщения" диалогового окна "Параметры сообщения"](../debugger/messages-tab-message-options-dialog-box.md) и [Вкладка "Вывод" диалогового окна "Параметры сообщения"](../debugger/output-tab-message-options-dialog-box.md).
