# <a name="run-uwp-apps-from-visual-studio"></a>Выполнение приложений UWP в Visual Studio
Приложения UWP выполняются на обширном диапазоне устройств, и Visual Studio предлагает вам несколько способов запуска и взаимодействия с этими приложениями. Процедура, вызываемая по нажатию клавиши F5, улучшена в целях упрощения отладки, профилирования и тестирования приложений во время разработки.
