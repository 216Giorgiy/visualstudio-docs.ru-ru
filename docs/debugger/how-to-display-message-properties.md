# <a name="how-to-display-message-properties"></a>Практическое руководство. Отображение свойств сообщения
Чтобы узнать больше о сообщениях, перечисленных в [представления сообщений](../debugger/messages-view.md), используйте [диалоговое окно Свойства сообщения](../debugger/message-properties-dialog-box.md).  
  
### <a name="to-open-a-message-properties-dialog-box"></a>Чтобы открыть диалоговое окно свойств сообщения  
  
1.  Выберите сообщение в дереве представления сообщений.  
  
2.  Из **представление** меню, выберите **свойства**.  
  
 Диалоговые окна свойств не модальным, поэтому можно щелкнуть другой элемент в окно представления и диалоговом окне будут отображены сведения о выбранного элемента.
