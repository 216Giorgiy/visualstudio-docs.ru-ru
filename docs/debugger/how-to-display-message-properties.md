# How to: Display Message Properties
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Для получения дополнительных сведений о сообщениях, перечисленных в [представлении сообщений](../debugger/messages-view.md), воспользуйтесь диалоговым окном [Свойства сообщения](../debugger/message-properties-dialog-box.md).  
  
### Открытие диалогового окна "Свойства сообщения"  
  
1.  Выберите сообщение в дереве представления сообщений.  
  
2.  В меню **Вид** выберите пункт **Свойства**.  
  
 Диалоговые окна свойств не являются модальными, поэтому можно щелкнуть другой элемент в окне представления, и в диалоговом окне свойств будут отображены сведения о выбранном элементе.
