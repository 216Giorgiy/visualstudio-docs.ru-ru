# How to: Change Fonts
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Можно изменить шрифт, начертание и размер шрифта, используемого в окнах Spy\+\+.  
  
### Изменение параметров шрифта  
  
1.  В меню **Вид** выберите пункт **Шрифт**.  
  
2.  Выберите шрифт, стиль и размер.  
  
3.  Нажмите кнопку **ОК**.  
  
 После нажатия кнопки **По умолчанию** выбранный шрифт будет использоваться во всех новых окнах Spy\+\+.
