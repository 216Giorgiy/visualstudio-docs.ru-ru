# <a name="output-window-debugging-options-dialog-box"></a>Окно "Вывод", узел "Отладка", диалоговое окно "Параметры"
Можно указать типы отладочной информации, появляющейся в **вывода** окна. Для отображения этих параметров откройте **средства** меню, нажмите кнопку **параметры**, разверните **Отладка** и щелкните **окно вывода**.

**Общие параметры вывода** этой категории содержатся элементы, управляющие появлением общих сообщений отладки в **вывода** окна. Здесь можно указать, какие типы сообщений будут отображаться.  

**Параметры трассировки WPF** этой категории содержатся элементы, позволяющие определить уровень сообщений трассировки WPF, появляющихся в **вывода** окна. Можно задать тип отображаемых сообщений и уровень: от **критическое** для **все**.

Дополнительные сведения см. в разделе [как: отображение сведений о трассировке WPF](../debugger/how-to-display-wpf-trace-information.md).

Если требуется восстановить параметры по умолчанию, можно сделать, используя **средства** > **Импорт и экспорт параметров** > **сбросить все параметры**. Если требуется только подмножество параметров сброса, сохранить настройки в **мастер импорта и экспорта параметров** перед внесением изменений, которые необходимо проверить, затем импортировать сохраненные параметры позже.
