# <a name="space-tab-process-properties-dialog-box"></a>Вкладка "Пространство" диалогового окна "Свойства процесса"
Используйте **пространства** вкладку для просмотра адресном пространстве процесса. Для отображения [диалоговое окно "Свойства процесса"](../debugger/process-properties-dialog-box.md), перемещение фокуса к [представление процессов](../debugger/processes-view.md) окна. Выберите любой узел процесса в дереве, а затем выберите **свойства** из **представление** меню.  
  
 Следующие параметры доступны на **пространства** вкладки:  
  
|Ввод|Описание:|  
|-----------|-----------------|  
|**Показать место, помеченное как**|Используйте этот список для выбора категории пространства (образ, сопоставлено, зарезервировано или неназначенные).|  
|**Исполняемый файл байт**|Для выбранной категории, сумма всех адресное пространство, которое использует этот процесс. Исполняемую память — это память, может выполняться программами, но может не быть чтение и запись.|  
|**EXEC только для чтения байтов**|Для выбранной категории сумму все адресное пространство с помощью свойства только для чтения, которые используют этот процесс. EXEC только для чтения памяти — это память, может быть выполнена, а также чтение.|  
|**EXEC чтения и записи байтов**|Для выбранной категории сумма адресное пространство со свойствами чтения и записи, которые используют этот процесс. Exec-read-write память — это память, исполняемый код программы, а также для чтения и изменения.|  
|**EXEC запись копии (байт)**|Для выбранной категории, сумма всех адресное пространство, которое можно исполняемый код программы, а также для чтения и записи. Этот тип защиты используется в том случае, когда памяти должен быть одновременно несколькими процессами. Если процесс управления доступом только для чтения объем памяти, затем они будут использовать общую память. Если процесс хочет записать данные, копия этой памяти будут выполняться для процесса.|  
|**Нет доступа (байт)**|Для выбранной категории, сумма всех адресное пространство, которое предотвращает процесс его использования. При попытке чтения или нарушение прав доступа создается в том случае, если запись.|  
|**Только для чтения байтов**|Для выбранной категории, сумма всех адресное пространство, которое может быть выполнена, а также чтение.|  
|**Байт чтения записи**|Для выбранной категории, сумма всех адресного пространства, используемого для чтения и записи.|  
|**Запись копии (байт)**|Для выбранной категории сумма адресное пространство, позволяющая памяти для управления доступом для чтения, но не для записи. Если процессы считывают данные, они могут использовать общую память. Тем не менее когда процесс, требуется доступ на чтение и запись к совместно используемой памяти, что память копия для записи.|
