# <a name="symbol-load-information-dialog-box"></a>Диалоговое окно "Сведения о загрузке символов"
**Сведения о загрузке символов** диалоговое окно открывается, если щелкнуть правой кнопкой мыши **модули** окна и выберите **сведения о загрузке символов**.  
  
## <a name="uielement-list"></a>Список элементов пользовательского интерфейса  
 **Информация**  
 Отображаются каталоги, в которых производится поиск файлов символов (PDB-файлы), основаны на Visual Studio установите параметры и изменения, внесенные в параметры символов в **параметры** диалоговое окно.  
  
 **Параметры символов**  
 Нажмите эту кнопку, чтобы изменить параметры символов в **параметры** диалоговое окно (**Отладка** категории **символы** страницы). На этой странице можно добавлять или удалять места расположения символов, определять место расположения локального кэша при использовании сервера символов и управлять поиском местоположений символов.
