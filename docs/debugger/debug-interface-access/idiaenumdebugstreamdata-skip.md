# <a name="idiaenumdebugstreamdataskip"></a>IDiaEnumDebugStreamData::Skip
Пропускает заданное число записей в последовательности, перечисленные.  
  
## <a name="syntax"></a>Синтаксис  
  
```C++  
HRESULT Skip (   
   ULONG celt  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 celt  
 [in] Количество пропускаемых перечисляемое последовательности записей.  
  
## <a name="return-value"></a>Возвращаемое значение  
 В случае успеха возвращает `S_OK`; в противном случае возвращает `S_FALSE` Если нет дополнительных записей для пропуска.
