# <a name="idiaenumtablesreset"></a>IDiaEnumTables::Reset
Сбрасывает последовательность перечисления в начало.  
  
## <a name="syntax"></a>Синтаксис  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Возвращает `S_OK`.
