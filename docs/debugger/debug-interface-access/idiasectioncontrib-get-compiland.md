# <a name="idiasectioncontribgetcompiland"></a>IDiaSectionContrib::get_compiland
Извлекает ссылку на символ компилируемого объекта, который участвует в этом разделе.  
  
## <a name="syntax"></a>Синтаксис  
  
```C++  
HRESULT get_compiland (   
   IDiaSymbol** pRetVal  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `pRetVal`  
 [out] Возвращает [IDiaSymbol](../../debugger/debug-interface-access/idiasymbol.md) объект, представляющий компилируемого объекта, который участвует в этом разделе.  
  
## <a name="return-value"></a>Возвращаемое значение  
 В случае успеха возвращает `S_OK`. Возвращает `S_FALSE` , если это свойство не поддерживается. В противном случае возвращается код ошибки.
