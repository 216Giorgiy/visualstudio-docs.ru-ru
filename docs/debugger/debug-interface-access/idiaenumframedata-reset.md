# <a name="idiaenumframedatareset"></a>IDiaEnumFrameData::Reset
Сбрасывает последовательность перечисления в начало.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Возвращает `S_OK`.
