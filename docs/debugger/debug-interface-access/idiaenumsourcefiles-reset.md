# <a name="idiaenumsourcefilesreset"></a>IDiaEnumSourceFiles::Reset
Сбрасывает последовательность перечислений в начало.

## <a name="syntax"></a>Синтаксис

```C++
HRESULT Reset ();
```

## <a name="return-value"></a>Возвращаемое значение
 Возвращает `S_OK`.
