# <a name="idiaenuminjectedsourcesreset"></a>IDiaEnumInjectedSources::Reset
Сбрасывает последовательность перечисления в начало.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Возвращает `S_OK`.
