# IDiaEnumInjectedSources::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Сбросить последовательность перечисления в начало.  
  
## Синтаксис  
  
```  
HRESULT Reset();  
```  
  
## Возвращаемое значение  
 Возвращает `S_OK`.
