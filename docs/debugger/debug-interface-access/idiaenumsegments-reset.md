# IDiaEnumSegments::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Сбросить последовательность перечисления в начало.  
  
## Синтаксис  
  
```cpp#  
HRESULT Reset ();  
```  
  
## Возвращаемое значение  
 Возвращает `S_OK`.
