# <a name="idiasymbolgetlocationtype"></a>IDiaSymbol::get_locationType
Получает тип расположения данных символа.  
  
## <a name="syntax"></a>Синтаксис  
  
```C++  
HRESULT get_locationType (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `pRetVal`  
 [out] Возвращает значение из [перечисление LocationType](../../debugger/debug-interface-access/locationtype.md) перечисления, которое указывает тип расположения данных символа, таких как `static` или `local`.  
  
## <a name="return-value"></a>Возвращаемое значение  
 В случае успеха возвращает `S_OK`; в противном случае возвращает `S_FALSE` или код ошибки.  
  
> [!NOTE]
>  Возвращаемое значение `S_FALSE` означает свойство недоступно для символа.
