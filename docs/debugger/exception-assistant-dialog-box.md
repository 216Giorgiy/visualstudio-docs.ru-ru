# Диалоговое окно &quot;Помощник по исключениям&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Диалоговое окно **Помощник по исключениям** появляется при возникновении исключения во время выполнения.  В окне **Помощник по исключениям** отображаются тип исключения, дополнительные сведения и ссылки на советы по устранению неполадок, предлагается способ получения дополнительной справки в Интернете, а также предоставляется возможность выполнить некоторые действия, например просмотреть подробные сведения об исключении.  
  
## Список задач  
 Чтобы просмотреть раздел, посвященный устранению неполадок, связанных с данным типом исключений, щелкните одно из сообщений с советом, отображенных в области **Советы по устранению неполадок**.  
  
 Чтобы выполнить действия, связанные с исключением, щелкните одно из действий, показанных в области действий.  
  
 Дополнительные сведения о том, как включить или отключить помощник по исключениям, см. в разделе [Страница "Общие", папка "Отладка", диалоговое окно "Параметры"](../debugger/general-debugging-options-dialog-box.md).  
  
## Список элементов пользовательского интерфейса  
 **Тип исключения**  
 Отображает тип вызванного исключения.  
  
 **Дополнительные сведения**  
 Отображает дополнительные сведения об исключении.  
  
 **Советы по устранению неполадок**  
 Отображает ссылки на советы по устранению неполадок, которые могут быть полезны для обнаружения источника исключения.  
  
 **Действия**  
 Перечисляет действия, которые можно выполнить, например просмотр дополнительной информации об объекте исключения.  
  
 **Получить справку в сети**  
 Позволяет выполнить поиск дополнительной справочной информации в Интернете.
