# Диалоговое окно &quot;Шрифт&quot; (Справка по Microsoft Spy++)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Это диалоговое окно служит для настройки шрифта, используемого в окне [Представления Spy\+\+](../debugger/spy-increment-views.md).  Чтобы открыть это диалоговое окно, в меню **Вид** выберите пункт **Шрифт**.  
  
 Доступны следующие параметры:  
  
 **Шрифт**  
 Шрифт для активного окна представлений Spy\+\+.  
  
 **Стиль шрифта**  
 Стиль шрифта для активного окна представлений Spy\+\+.  
  
 **Размер**  
 Размер шрифта для активного окна представлений Spy\+\+.  
  
 **По умолчанию**  
 Сохраняет текущие параметры шрифта как пользовательские настройки для представлений Spy\+\+.
