# <a name="how-to-display-window-properties"></a>Практическое руководство. Отображение свойств окна
Чтобы узнать больше об окнах, перечисленных в [представление окон](../debugger/windows-view.md), используйте [диалоговое окно "Свойства"](../debugger/window-properties-dialog-box.md).  
  
### <a name="to-open-the-window-properties-dialog-box"></a>Чтобы открыть диалоговое окно "Свойства окна"  
  
1.  Выберите элемент, а затем выберите **свойства** из **представление** меню.  
  
 Диалоговые окна свойств не модальным. Вариант от элемента к элементу в окне просмотра, диалоговое окно автоматически обновляются с учетом выбранного элемента данных.
