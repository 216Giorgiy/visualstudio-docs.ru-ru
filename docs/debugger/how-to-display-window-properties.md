# How to: Display Window Properties
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Для получения дополнительных сведений об окнах, перечисленных в [представлении окон](../debugger/windows-view.md), воспользуйтесь диалоговым окном [Свойства окна](../debugger/window-properties-dialog-box.md).  
  
### Открытие диалогового окна "Свойства окна"  
  
1.  Выберите элемент, а затем в меню **Вид** выберите пункт **Свойства**.  
  
 Диалоговые окна свойств не являются модальными.  При переходе от элемента к элементу в окне представлений диалоговое окно автоматически обновляется и в нем отображаются свойства выбранного элемента.
