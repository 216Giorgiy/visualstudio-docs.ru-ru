Для отладки приложений Windows 8.1, в разделе [отладка приложений магазина в Visual Studio](https://msdn.microsoft.com/en-us/library/hh441472.aspx). Для отладки приложений Windows Phone 8, в разделе [отладки приложений для Windows Phone 8](https://msdn.microsoft.com/library/windows/apps/ff402572(v=vs.105).aspx).
