# <a name="spy-menu-commands"></a>Команды меню Spy
**Spy** меню доступны команды для работы с Windows Spy ++ и представления.

 **Windows** открывает новое представление Windows.

 **Процессы** открывает новое представление процессов.

 **Потоки** откроется представление "Потоки".

 **Запись в журнал сообщений** открывает новое представление сообщений и [диалоговое окно "Параметры сообщения"](../debugger/message-options-dialog-box.md) для представления.

 **Окно поиска** отображает **FindWindow** диалоговое окно для выбора окна с помощью мыши.

 **Выход** завершает выполнение программы Spy ++.
