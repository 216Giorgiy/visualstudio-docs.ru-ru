# <a name="how-to-search-for-a-process-in-processes-view"></a>Практическое руководство. Поиск процесса в представлении процессов
Можно поиск определенного процесса в представлении процессов с помощью его процесс идентификатор или строку модуля как условия поиска. Можно также указать начальное направление поиска. Поля в диалоговом окне отобразятся атрибуты процессов, выбранных в дерево процесса.  
  
### <a name="to-search-for-a-process-in-processes-view"></a>Поиск процесса в представлении процессов  
  
1.  Расположение окон, поэтому Spy ++ и активной [представление процессов](../debugger/processes-view.md) окна являются видимыми.  
  
2.  Из **поиска** меню, выберите **найти процесс**  
  
     [Диалоговое окно "Поиск процесса"](../debugger/process-search-dialog-box.md) открывается.  
  
3.  Введите идентификатор процесса или строку модуля в качестве критерия поиска.  
  
4.  Очистите все поля, для которых не требуется задать значения.  
  
    > [!TIP]
    >  Чтобы найти все процессы, принадлежащие модулю, очистите **процесс** и введите имя модуля в **модуль** поле. Затем с помощью **Найти далее** для продолжения поиска.  
  
5.  Выберите **копирование** или **работу** для исходное направление поиска.  
  
6.  Нажмите кнопку **ОК**.  
  
 Если найден процесс сопоставления, выделяется в **представление "процесс"** окна.
