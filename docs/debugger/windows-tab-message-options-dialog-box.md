# <a name="windows-tab-message-options-dialog-box"></a>Вкладка "Окна" диалогового окна "Параметры сообщения"
Используйте **Windows** вкладку, чтобы задать область сообщений, перечисленных в [представления сообщений](../debugger/messages-view.md). Для отображения [диалоговое окно "Параметры сообщения"](../debugger/message-options-dialog-box.md), выберите **сообщений журнала** из **Spy** меню.  
  
 Следующие параметры доступны на **Windows** вкладки:  
  
 **Инструмент поиска**  
 Перетаскивание средство, используемое для выбора окна.  
  
 **Скрыть Spy ++**  
 Удаляет главное окно Spy ++ и оставляет **параметры сообщения** диалоговое окно поверх других приложений.  
  
 **Родительское окно**  
 Отображает сообщения для выбранного окна и его непосредственного родительского окна.  
  
 **Дочерние окна**  
 Отображает сообщения для выбранного окна и всех его дочерних окон, включая вложенные дочерние окна.  
  
 **Окна одного потока**  
 Отображает сообщения для выбранного окна и всех других окон, в том же потоке.  
  
 **Окна одного процесса**  
 Отображает сообщения для выбранного окна и всех других окон, в том же процессе.  
  
 **Все окна в системе**  
 Для отображения сообщений для всех окон.  
  
 **Сохранить параметры по умолчанию**  
 Сохраняет описанные выше параметры для новых окон потоков сообщений. Эти параметры также сохраняются при выходе из Spy ++.
