# <a name="graphics-frame-validation"></a>Проверка кадров графики
<!-- VERSIONLESS -->
Visual Studio 2017 г. и расширенную поддержку **проверки кадра** средства.  В окне фрейма проверки отображаются ошибки и предупреждения, связанные с помощью списка событий.  Чтобы открыть это окно, выберите **представление > Проверка кадра** меню.

![Проверка кадра](media/gfx_diag_frame_validation.png)

Нажмите кнопку **запустить проверку** кнопку в левом верхнем углу, чтобы начать анализ.  Он может занять несколько минут в зависимости от сложности кадра.  Данных, который отображается ниже представляет собой комбинацию из двух источников: сообщений, D3D себя выдает при [уровней SDK](https://msdn.microsoft.com/library/windows/desktop/ff476881(v=vs.85).aspx) включен и данные, собранные из этого инструмента собственные внутреннее состояние отслеживания. После завершения вы увидите несколько столбцов данных.

**Столбец**|**Описание**
---|---
Идентификатор события | Идентификатор, который соответствует записи в [список событий](graphics-event-list.md) окна.
Серьезность | Повреждение, ошибка, предупреждение, сведения или сообщения.
Категория | Приложения определено, прочие, инициализации, очистки, компиляции, состояния создания, параметр состояния, получение состояния, выполнения, обработки ресурсов, шейдера избыточностью и неиспользуемые.
Сообщение | Сообщение, связанное с событием.
событие | Событие, связанное с ошибкой или предупреждением.
