# <a name="view-menu-commands"></a>Команды меню "Вид"
**Представление** меню предоставляет команды для управления Spy ++ пользовательского интерфейса.

 **Шрифт** открывает **шрифта** диалоговое окно, в котором можно выбрать шрифт для использования в текущем представлении Spy ++.

 **Панель инструментов** скрытие или отображение панели инструментов.

 **Строка состояния** скрывает или отображает в строке состояния.

 **Свойства** открывает **свойства** диалоговое окно для активное представление.
