# <a name="message-search-dialog-box"></a>Диалоговое окно "Поиск сообщений"
Используйте это диалоговое окно предназначено для поиска и выбора узла для конкретного сообщения в [представления сообщений](../debugger/messages-view.md). Чтобы отобразить это диалоговое окно предназначено, переместите фокус на окно представления сообщений. Выберите **найти сообщение** из **поиска** меню.  
  
 Доступны следующие параметры:  
  
 **Инструмент поиска**  
 Перетаскивание средство, используемое для выбора окна.  
  
 **Скрыть Spy ++**  
 Удаляет главное окно Spy ++ и оставляет **поиск сообщений** диалоговое окно поверх других приложений.  
  
 **Дескриптор**  
 Уникальный идентификатор сообщения для поиска.  
  
 **Type**  
 Тип сообщения для поиска.  
  
 **Сообщение**  
 Идентификатор сообщения для поиска.  
  
 **Направление поиска вверх или вниз**  
 Исходное направление поиска.
