# Диалоговое окно &quot;Поиск процесса&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Это диалоговое окно используется для поиска и выбора узла для конкретного процесса в [представлении процессов](../debugger/processes-view.md).  Чтобы открыть это диалоговое окно, переместите фокус на окно представления **Процессы**.  В меню **Поиск** выберите команду **Найти процесс**.  
  
 Доступны следующие параметры:  
  
 **Процесс**  
 Идентификатор искомого процесса.  
  
 **Модуль**  
 Искомая строка модуля.  
  
 **Направление поиска: вверх или вниз**  
 Начальное направление поиска.
