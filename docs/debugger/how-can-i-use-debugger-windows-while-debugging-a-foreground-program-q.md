# Как использовать окна отладчика при отладке активной программы?
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

## Описание проблемы  
 Выполняются попытки отладить проблему отображения на экране.  Для этого приходится держать программу активной, а это означает, что отсутствует доступ к окнам отладчика.  Что можно сделать?  
  
## Решение  
 Если есть второй компьютер, используйте удаленную отладку.  С двумя компьютерами можно посмотреть отображение на экране на удаленном компьютере, работая в отладчике на основном хосте.  Дополнительные сведения по удаленной отладке см. в разделе [Настройка удаленной отладки](../Topic/Set%20Up%20the%20Remote%20Tools%20on%20the%20Device.md).
