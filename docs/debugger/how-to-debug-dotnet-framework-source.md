# <a name="how-to-debug-net-framework-source"></a>Практическое руководство. Отладка исходного кода .NET Framework
Чтобы отладить исходный код .NET Framework, необходимо иметь доступ к символам отладки для кода. Необходимо также Включение захода в исходный код .NET Framework.  
  
 .NET Framework можно включить пошаговое выполнение и загрузки отладочных символов в **параметры** диалоговое окно. При включении загрузки символов можно выбрать один из двух вариантов: загрузить символы немедленно или просто включить эту возможность для загрузки символов в дальнейшем. Если не загружать символы сразу же, они будут загружены в начале следующего сеанса отладки приложения. Также можно сделать вручную из **модули** окна или **стек вызовов** окна.  
  
### <a name="to-enable-net-framework-source-debugging"></a>Включение отладки исходного кода .NET Framework  
  
1.  На **средства** меню, нажмите кнопку **параметр**s.  
  
2.  В **параметры** диалоговое окно, нажмите кнопку **Отладка** категории.  
  
3.  В **Общие** установите **исходный код .NET Framework включить пошаговое выполнение.**  
  
    1.  Если был включен режим "Только мой код", появится диалоговое окно с предупреждением об отключении этого режима. Нажмите кнопку **ОК**.  
  
    2.  Если расположение кэша символов не было задано ранее, появится еще одно диалоговое окно — с сообщением о выборе расположения кэша символов по умолчанию. Нажмите кнопку **ОК**.  
  
4.  В разделе **Отладка** категории, нажмите кнопку **символы**.  
  
5.  Если вы хотите изменить расположение кэша символов, измените расположение в **кэшировать символы в каталоге** или нажмите кнопку **Обзор** следует выбрать местоположение.  
  
6.  Если требуется загрузить символы немедленно, нажмите кнопку **загрузить символы, используя вышеуказанные места**.  
  
     Эта кнопка недоступна в режиме конструктора, но доступна во время отладки.  
  
     Если не загружать символы прямо сейчас, они будут загружены автоматически при следующей отладке разрабатываемой программы.  
  
7.  Нажмите кнопку **ОК**, чтобы закрыть диалоговое окно **Параметры**.  
  
### <a name="to-load-framework-symbols-using-the-modules-window"></a>Загрузка символов Framework вручную с помощью окна "Модули"  
  
1.  В **модули** окна (во время отладки, выберите **отладки** > **Windows** > **модули**), Щелкните правой кнопкой мыши модуль, для которого не загружены символы. Можно определить символы загружены или не анализируя **состояние символов** столбца.  
  
2.  Пункты **параметры символов** и нажмите кнопку **серверы символов Майкрософт** на загрузку символов с Microsoft общедоступного сервера символов. Или щелкните правой кнопкой мыши модуль и выберите **загрузить символы** из каталога, где ранее были сохранены символы.  
  
### <a name="to-load-framework-symbols-using-the-call-stack-window"></a>Загрузка символов Framework вручную с помощью окна "Стек вызовов"  
  
1.  В **стек вызовов** окно, щелкните правой кнопкой мыши какой-либо кадр, для которого не загружены символы. Кадр затеняется.  
  
2.  Пункты **параметры символов** и нажмите кнопку **серверы символов Microsoft**, или щелкните правой кнопкой мыши модуль и выберите **путь к символам**.
