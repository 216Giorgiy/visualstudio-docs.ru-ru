# <a name="tree-menu-commands"></a>Команды меню "Дерево"
**Дерева** меню предоставляет команды, чтобы развернуть или свернуть представления.

 **Развернуть один уровень** Разворачивает выбранный элемент в системном представлении, отображая один уровень ниже его.

 **Разверните** Разворачивает все уровни ниже текущего выбранного элемента.

 **Развернуть все** разворачивается и отображает все уровни в системном представлении.

 **Свернуть** сворачивает все уровни ниже текущего выбранного элемента.
